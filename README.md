# <a name="microsoft-azure-documentation"></a>Documentação do Microsoft Azure

Bem-vindo ao código-fonte aberto [documentação](https://docs.microsoft.com/azure) dos [Microsoft Azure](https://azure.microsoft.com). Consulte este ficheiro Leia-me a compreender como pode ajudar a contribuir para a documentação do Microsoft Azure.

## <a name="getting-started"></a>Introdução

Contribuir para open source está fornecendo mais do que apenas as atualizações, ele também é dar-na saber quando existe um problema. Leia nossa [contribuir orientações](.github/CONTRIBUTING.md) para obter mais informações.

### <a name="prerequisites"></a>Pré-requisitos

Decidiu fornecer uma contribuição, que é ótima! Existem algumas ferramentas que são necessárias para contribuir para a documentação.

Contribuir para a documentação requer uma conta do GitHub, se não tiver uma conta, siga as instruções para [configuração da conta do GitHub](https://docs.microsoft.com/contribute/get-started-setup-github) do nosso guia do contribuinte.

#### <a name="download"></a>Transferência

Instale as ferramentas seguintes:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* [Pacote de criação do Docs](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) extensão para o Visual Studio Code

#### <a name="install"></a>Instalar

Siga as instruções fornecidas no [ferramentas de criação de conteúdo de instalação](https://docs.microsoft.com/contribute/get-started-setup-tools) do nosso guia do contribuinte.
