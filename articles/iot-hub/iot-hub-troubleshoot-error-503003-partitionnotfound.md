# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

Este artigo descreve as causas e soluções para **erros de PartitionNotFound 503003.**

## <a name="symptoms"></a>Sintomas

Os pedidos ao IoT Hub falham com o erro **503003 PartitionNotFound**.

## <a name="cause"></a>Causa

Este erro é interno no IoT Hub e é provavelmente transitório. Consulte a [causa do erro do servidor interno do IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Solução

Consulte [soluções para erros internos do servidor IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
