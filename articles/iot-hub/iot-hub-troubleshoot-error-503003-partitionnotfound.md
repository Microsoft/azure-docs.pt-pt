# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

Este artigo descreve as causas e soluções para **503003 Erros Do Término Do Partição.**

## <a name="symptoms"></a>Sintomas

Os pedidos ao IoT Hub falham com o erro **503003 PartitionNotFound**.

## <a name="cause"></a>Causa

Este erro é interno para o Hub IoT e é provavelmente transitório. Consulte a causa de [erro do servidor interno Do IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Solução

Consulte [soluções para erros do servidor interno IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
