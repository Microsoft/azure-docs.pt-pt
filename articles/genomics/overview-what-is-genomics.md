# <a name="what-is-microsoft-genomics"></a>O que é o Microsoft Genomics?
A Microsoft Genomics oferece uma implementação em nuvem do Burrows-Wheeler Aligner (BWA) e do Genome Analysis Toolkit (GATK) para análise secundária. O serviço é certificado pela ISO e está em conformidade com os regulamentos da HIPAA, e oferece previsibilidade de preço para as suas necessidades de sequenciação do genoma. Aprenda a usar o serviço Microsoft Genomics e integre-se com a nossa API lendo os nossos quickstarts, tutoriais e documentação.

## <a name="support-your-most-demanding-sequencing-needs"></a>Apoie as suas necessidades de sequenciação mais exigentes
Em vez de gerir os seus próprios datacenters, aproveite a escala e a experiência da Microsoft na execução de cargas de trabalho à escala de exabytes. A nossa implementação em nuvem do BWA-GATK é altamente concordante com o oleoduto de boas práticas do Instituto Broad.


## <a name="keep-your-business-running"></a>Mantenha o seu negócio a funcionar
A Microsoft Genomics oferece um contrato de nível de serviço de disponibilidade de 99,99% (SLA) para receber pedidos de fluxo de trabalho. Saiba mais sobre o Microsoft Genomics [SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Proteger os dados
O serviço Microsoft Genomics é ISO 27001, ISO 27018 e ISO 9001 certificado e em conformidade com os regulamentos da HIPAA. Visite o [Centro de Fidedignidade do Azure](https://www.microsoft.com/trustcenter/security) para obter informações sobre a segurança da plataforma do Azure.
