# <a name="version-release-history"></a>Histórico de versões
A equipe de Microsoft Genomics atualiza regularmente o Microsoft Genomics cliente Python para correções e novas funcionalidades. 

## <a name="latest-release"></a>Versão mais recente
O cliente Python atual é a versão 0.8.1. Ele foi lançado em dezembro de 3 2018 e dá suporte a fluxos de trabalho em execução com GATK 3,5 e GATK4. Ele também dá suporte à saída de gVCF.


## <a name="release-history"></a>Histórico de lançamentos 
Novas versões do cliente Microsoft Genomics Python são lançadas cerca de duas vezes por ano. À medida que novas versões do cliente Microsoft Genomics Python são lançadas, uma lista de correções e recursos é atualizada aqui. Quando novas versões são lançadas, as versões anteriores devem continuar a ter suporte por pelo menos 90 dias. Quando as versões anteriores não forem mais suportadas, elas serão indicadas nesta página. 

### <a name="version-081"></a>0\.8.1 da versão
A versão 0.8.1 inclui correções de bugs secundárias.  

### <a name="version-080"></a>0\.8.0 da versão
A versão 0.8.0 inclui suporte para GATK4 e saída de gVCFs.  

### <a name="version-074"></a>0\.7.4 da versão
A versão 0.7.4 inclui suporte para aceitar tokens SAS em vez de chaves de conta na entrada `config.txt`. Para obter mais informações, consulte [início rápido de tokens SAS de entrada](quickstart-input-sas.md). 

### <a name="version-073"></a>0\.7.3 da versão
A versão 0.7.3 inclui correções de bugs secundárias.

### <a name="version-072"></a>0\.7.2 da versão
A versão 0.7.2 é a versão inicial. Ele foi lançado em novembro de 1 2017.
