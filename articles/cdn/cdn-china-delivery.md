# <a name="china-content-delivery-with-azure-cdn"></a>Entrega de conteúdos na China com CDN Azure

A Azure Content Delivery Network (CDN) global pode servir conteúdo a utilizadores da China com localizações de ponto de presença (POP) perto da China ou qualquer POP que proporcione o melhor desempenho a pedidos originários da China. No entanto, se a China é um mercado significativo para os seus clientes e eles precisam de um desempenho rápido, considere usar a Azure CDN China em vez disso.

A Azure CDN China difere do Azure CDN global na medida em que fornece conteúdo de POPs dentro da China através da parceria com vários fornecedores locais. Devido à conformidade e regulação chinesas, deve registar uma subscrição separada para utilizar a Azure CDN China e os seus websites precisam de ter uma licença ICP. A experiência portal e API para permitir e gerir a entrega de conteúdos é idêntica entre a Azure CDN global e a Azure CDN China.

## <a name="comparison-of-azure-cdn-global-and-azure-cdn-china"></a>Comparação entre a Azure CDN global e a Azure CDN China

Azure CDN global e Azure CDN China têm as seguintes características:

- Azure CDN global:

     - Portal:https://portal.azure.com  

     - Realiza entrega de conteúdo fora da China

     - Quatro níveis de preços: padrão da Microsoft, padrão Verizon, Verizon premium e padrão Akamai

     - [Documentação](https://docs.microsoft.com/azure/cdn/)

- Azure CDN China:

     - Portal:https://portal.azure.cn

     - Realiza entrega de conteúdo dentro da China

     - Dois níveis de preços: Standard e premium

     - [Documentação](https://docs.azure.cn/en-us/cdn/)
