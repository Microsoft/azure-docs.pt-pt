# <a name="china-content-delivery-with-azure-cdn"></a>Entrega de conteúdo da China com Azure CDN

A Azure Content Delivery Network (CDN) global pode servir conteúdo para utilizadores da China com localizações de ponto de presença (POP) perto da China ou qualquer POP que forneça o melhor desempenho a pedidos originários da China. No entanto, se a China é um mercado significativo para os seus clientes e eles precisam de um desempenho rápido, considere usar a Azure CDN China.

A Azure CDN China difere da Azure CDN global na medida em que fornece conteúdo de POPs dentro da China, em parceria com vários fornecedores locais. Devido ao cumprimento e regulação chineses, você deve registar uma subscrição separada para usar Azure CDN China e seus sites precisam ter uma licença ICP. A experiência do portal e da API para permitir e gerir a entrega de conteúdos é idêntica entre a Azure CDN global e a Azure CDN China.

## <a name="comparison-of-azure-cdn-global-and-azure-cdn-china"></a>Comparação da Azure CDN global e Azure CDN China

AZure CDN global e Azure CDN China têm as seguintes funcionalidades:

- Azure CDN global:

     - Portal:https://portal.azure.com  

     - Executa a entrega de conteúdos fora da China

     - Quatro níveis de preços: padrão da Microsoft, padrão Verizon, prémio Verizon e padrão Akamai

     - [Documentação](https://docs.microsoft.com/azure/cdn/)

- Azure CDN China:

     - Portal:https://portal.azure.cn

     - Executa a entrega de conteúdos dentro da China

     - Dois níveis de preços: Standard e premium

     - [Documentação](https://docs.azure.cn/en-us/cdn/)
