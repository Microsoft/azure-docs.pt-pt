# <a name="azure-cdn-coverage-by-metro"></a>Cobertura da CDN do Azure por metro 
> [!div class="op_single_selector"]
> * [Localizações POP-por região](cdn-pop-locations.md)
> * [Localizações POP-por abreviatura](cdn-pop-abbreviations.md)
> 


Este artigo lista os metrôs atuais que contêm locais POP (ponto de presença), classificados por região, para os produtos da CDN (rede de distribuição de conteúdo) do Azure. Cada metro pode conter mais de um POP. Por exemplo, a CDN do Azure da Microsoft tem 129 POPs em 65 metros. 

> [!IMPORTANT]
> Os locais de cidade POP para a **CDN do Azure da Akamai** não são divulgados individualmente.  
> 
> Como cada produto da CDN do Azure tem uma maneira distinta de criar suas infraestruturas CDN, a Microsoft recomenda o uso de locais POP para decidir qual produto da CDN do Azure usar. Em vez disso, considere seus recursos e o desempenho do usuário final. Teste o desempenho com cada produto da CDN do Azure para escolher o produto certo para seus usuários.
