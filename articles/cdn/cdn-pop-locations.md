# <a name="azure-cdn-coverage-by-metro"></a>Cobertura Azure CDN por Metro 
> [!div class="op_single_selector"]
> * [Localizações POP-por região](cdn-pop-locations.md)
> * [Localizações POP-por abreviatura](cdn-pop-abbreviations.md)
> 


Este artigo lista os atuais Metros contendo localizações de ponto de presença (POP), ordenadas por região, para produtos da Rede de Entrega de Conteúdos Azure (CDN). Cada metro pode conter mais de um POP. Por exemplo, o Azure CDN da Microsoft tem 130 POPs em 80 Metros. 

> [!IMPORTANT]
> As localizações da cidade pop para **Azure CDN da Akamai** não são divulgadas individualmente.  
> 
> Uma vez que cada produto Azure CDN tem uma forma distinta de construir as suas infraestruturas DeCDN, a Microsoft recomenda não utilizar localizações POP para decidir qual o produto Azure CDN a utilizar. Em vez disso, considere as suas funcionalidades e desempenho do utilizador final. Teste o desempenho com cada produto Azure CDN para escolher o produto certo para os seus utilizadores.
