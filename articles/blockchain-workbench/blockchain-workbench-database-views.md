# <a name="database-views-in-azure-blockchain-workbench"></a>Vistas de base de dados no Azure Blockchain Workbench

Azure Blockchain Workbench fornece dados do ledgers distribuídos para um *desativar cadeia* base de dados de base de dados SQL. Isto torna possível utilizar o SQL Server e ferramentas existentes, tais como [SQL Server Management Studio](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017), para interagir com dados blockchain.

Azure Blockchain Workbench fornece um conjunto de vistas da base de dados que fornecem acesso aos dados que irão ser útil quando efetuar consultas. Estas vistas são fortemente denormalized torna mais fácil para rapidamente criar introdução relatórios, análise e caso contrário consumir blockchain dados com ferramentas existentes e sem ter de reparametrização dos empregados da base de dados.

Esta secção inclui uma descrição geral das vistas da base de dados e os dados que contêm.

> [!NOTE]
> Qualquer utilização direta da base de dados foram encontradas tabelas na base de dados fora destas vistas, ainda que possível, não é suportada.
>
