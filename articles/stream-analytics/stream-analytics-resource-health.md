# <a name="stream-analytics-resource-health-page-in-the-azure-portal"></a>Página de estado de funcionamento do Stream Analytics recursos no portal do Azure

O **estado de funcionamento do recurso** página no portal do Azure mostra o estado de funcionamento das suas tarefas do Azure Stream Analytics. Também mostra acionável orientações que podem ajudar a resolver problemas. 

O **estado de funcionamento do recurso** página destina-se para o ajudar a determinar mais rapidamente se a raiz de um problema está na aplicação, ou se este problema é causado por um evento no Azure. Pode ajudar a saber quando abrir um incidente de suporte para receber assistência.

## <a name="troubleshooting-workflow"></a>Fluxo de trabalho de resolução de problemas

Para ver o estado de funcionamento de recursos para as tarefas do Stream Analytics, selecione o **estado de funcionamento do recurso** página. 

![Fluxo - acesso de resolução de problemas do Stream Analytics](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

O fluxograma a seguir pode ajudar a tomar decisões sobre que procurar e abrir um incidente de suporte.

![Fluxo de resolução de problemas de análise de sequência - mapa](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
