# <a name="monitor-and-manage-stream-analytics-jobs-with-visual-studio"></a>Monitorize e gerencie trabalhos stream analytics com o Visual Studio

Este artigo demonstra como monitorizar o seu trabalho de Stream Analytics no Visual Studio. As ferramentas Azure Stream Analytics para o Estúdio Visual proporcionam uma experiência de monitorização semelhante ao portal Azure sem ter de sair do IDE. Pode começar a monitorizar um trabalho assim que **submeter para Azure** a partir do seu **Script.asaql,** ou pode monitorizar os trabalhos existentes independentemente da forma como foram criados. 

## <a name="job-summary"></a>Resumo do trabalho

O Resumo do **Trabalho** e as **Métricas** de Emprego dão uma foto rápida do seu trabalho. Num ápice, pode determinar o estado de um trabalho e a informação do evento.]

<img src="./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-job-summary-metrics.png" alt="Stream Analytics job summary and job metrics" width="300px"/> 


## <a name="job-metrics"></a>Métricas de trabalho

Pode colapsar o Resumo de **Trabalho** e clicar no separador **Métricas** de Trabalho para ver um gráfico com métricas importantes. Verifique e desverifique os tipos de métricas para adicionar e removê-los do gráfico.

![Métricas de Analytics stream em Estúdio Visual](./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-vs-metrics.png)


## <a name="error-monitoring"></a>Monitorização de erros

Também pode monitorizar erros clicando no separador **Errors.**

![Erros de Análise de Fluxo em Estúdio Visual](./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-vs-errors.png)
