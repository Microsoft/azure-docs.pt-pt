# <a name="monitor-and-manage-stream-analytics-jobs-with-visual-studio"></a>Monitore e gere os trabalhos de Stream Analytics com o Visual Studio

Este artigo demonstra como monitorizar o seu trabalho stream Analytics no Visual Studio. As ferramentas Azure Stream Analytics para o Visual Studio proporcionam uma experiência de monitorização semelhante ao portal Azure sem ter de sair do IDE. Pode começar a monitorizar um trabalho assim que **submeter ao Azure** a partir do seu **Script.asaql,** ou pode monitorizar os empregos existentes independentemente da forma como foram criados. 

## <a name="job-summary"></a>Resumo do trabalho

O **Resumo do Trabalho** e as **Métricas** de Emprego dão uma imagem rápida do seu trabalho. Num ápice, pode determinar o estado do trabalho e a informação do evento.]

<img src="./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-job-summary-metrics.png" alt="Stream Analytics job summary and job metrics" width="300px"/> 


## <a name="job-metrics"></a>Métricas de trabalho

Pode colapsar o Resumo do **Trabalho** e clicar no separador **Métricas** de Trabalho para visualizar um gráfico com métricas importantes. Verifique e desescoda os tipos de métricas para os adicionar e remova do gráfico.

![Métricas de Stream Analytics em Estúdio Visual](./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-vs-metrics.png)


## <a name="error-monitoring"></a>Monitorização de erros

Também pode monitorizar erros clicando no separador **Erros.**

![Erros de Stream Analytics no Estúdio Visual](./media/stream-analytics-monitor-jobs-use-vs/stream-analytics-vs-errors.png)
