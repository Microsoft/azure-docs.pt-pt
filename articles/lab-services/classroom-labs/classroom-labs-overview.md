# <a name="introduction-to-classroom-labs"></a>Introdução aos laboratórios de sala de aula
O Azure Lab Services permite-lhe configurar rapidamente um ambiente de laboratório de sala de aula na cloud. O educador cria um laboratório de sala de aula, aprovisiona máquinas virtuais do Linux ou Windows, instala os laboratórios de ferramentas e software necessários na sala de aula e disponibiliza-os aos utilizadores. Os alunos na sala de aula ligam-se a máquinas virtuais (VMs) no laboratório e utilizam-nas para os projetos, tarefas ou para fazer exercícios da sala de aula. 

Os laboratórios de sala de aula são tipos de laboratório gerenciados que são gerenciados pelo Azure. O próprio serviço lida com todo o gerenciamento de infraestrutura para um tipo de laboratório gerenciado, da rotação de máquinas virtuais (VMs) até o tratamento de erros e de dimensionamento da infraestrutura. Especifica o tipo de infraestrutura que precisa e instala quaisquer ferramentas ou software necessário para a sala de aula. 

## <a name="automatic-management-of-azure-infrastructure-and-scale"></a>Gerenciamento automático de infraestrutura e escala do Azure 
Azure Lab Services é um serviço gerenciado, o que significa que o provisionamento e o gerenciamento da infraestrutura subjacente de um laboratório são tratados automaticamente pelo serviço. Você pode se concentrar apenas na preparação da experiência de laboratório certa para seus usuários. Deixe o serviço lidar com o restante e distribua as máquinas virtuais do seu laboratório para seu público. Dimensione seu laboratório para centenas de máquinas virtuais com um único clique.

## <a name="simple-experience-for-your-lab-users"></a>Experiência simples para os usuários do laboratório 
Os usuários que são convidados para seu laboratório obtêm acesso imediato aos recursos que você lhes dá dentro de seus laboratórios. Eles só precisam entrar para ver a lista completa de máquinas virtuais que eles têm acesso em vários laboratórios. Eles podem clicar em um único botão para se conectar às máquinas virtuais e começar a trabalhar. Os usuários não precisam de assinaturas do Azure para usar o serviço. 

## <a name="cost-optimization-and-tracking"></a>Otimização e controlo de custos  
Mantenha seu orçamento em cheque controlando exatamente quantas horas seus usuários de laboratório podem usar as máquinas virtuais. Configure agendamentos no laboratório para permitir que os usuários usem as máquinas virtuais somente durante os intervalos de tempo designados ou configurem a ocorrência de desligamento automático e horários de início. Controlar o uso de usuários individuais e definir limites.
