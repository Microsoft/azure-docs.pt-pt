# <a name="introduction-to-classroom-labs"></a>Introdução aos laboratórios de sala de aula
O Azure Lab Services permite-lhe configurar rapidamente um ambiente de laboratório de sala de aula na cloud. O educador cria um laboratório de sala de aula, aprovisiona máquinas virtuais do Linux ou Windows, instala os laboratórios de ferramentas e software necessários na sala de aula e disponibiliza-os aos utilizadores. Os alunos na sala de aula ligam-se a máquinas virtuais (VMs) no laboratório e utilizam-nas para os projetos, tarefas ou para fazer exercícios da sala de aula. 

Os laboratórios de sala de aula são geridos por tipos de laboratório que são geridos pelo Azure. O próprio serviço trata de toda a gestão da infraestrutura para um tipo de laboratório gerido, desde girar máquinas virtuais (VMs) até erros de manuseamento, e dimensionar a infraestrutura. Especifica o tipo de infraestrutura que precisa e instala quaisquer ferramentas ou software necessário para a sala de aula. 

## <a name="automatic-management-of-azure-infrastructure-and-scale"></a>Gestão automática da infraestrutura e escala azure 
O Azure Lab Services é um serviço gerido, o que significa que o provisionamento e gestão da infraestrutura subjacente de um laboratório é tratado automaticamente pelo serviço. Pode concentrar-se em preparar a experiência de laboratório certa para os seus utilizadores. Deixe o serviço cuidar do resto e lançar as máquinas virtuais do seu laboratório para o seu público. Escala o teu laboratório para centenas de máquinas virtuais com um único clique.

## <a name="simple-experience-for-your-lab-users"></a>Experiência simples para os seus utilizadores de laboratório 
Os utilizadores que são convidados para o seu laboratório têm acesso imediato aos recursos que lhes dá dentro dos seus laboratórios. Só precisam de entrar para ver a lista completa das máquinas virtuais a que têm acesso em vários laboratórios. Podem clicar num único botão para se ligarem às máquinas virtuais e começarem a trabalhar. Os utilizadores não precisam de subscrições do Azure para utilizar o serviço. 

## <a name="cost-optimization-and-tracking"></a>Otimização e rastreio de custos  
Mantenha o seu orçamento sob controlo controlando exatamente quantas horas os seus utilizadores de laboratório podem usar as máquinas virtuais. Configurar horários no laboratório para permitir que os utilizadores utilizem as máquinas virtuais apenas durante as faixas horárias designadas ou configurar os horários de paragem automática recorrentes e os horários de início. Acompanhe a utilização individual dos utilizadores e estabeleça limites.

Consulte os tipos de classe Exemplo no artigo da [Azure Lab Services](class-types.md) para alguns tipos de exemplo sonantes de aulas para as quais você pode configurar laboratórios com Serviços de Laboratório Azure.
