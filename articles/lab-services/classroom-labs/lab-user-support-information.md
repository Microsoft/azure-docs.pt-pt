# <a name="view-support-information-lab-user-in-azure-lab-services"></a>Ver informações de suporte (utilizador de laboratório em Serviços de Laboratório Azure)
Este artigo explica como você (como utilizador de laboratório) pode ver as seguintes informações de suporte:

- URL
- Email
- Telefone
- Instruções adicionais

Pode usar esta informação para obter ajuda quando se depara com problemas técnicos enquanto usa um laboratório numa conta de laboratório.

 
## <a name="view-support-information"></a>Ver informações de suporte
1. Inscreva-se no [portal Azure Lab Services.](https://labs.azure.com)
2. Selecione o **laboratório ou a máquina virtual** para a qual precisa de ajuda e selecione? **?** no canto superior direito da página. 
3. Confirme que vê links para o site de suporte de **visualização,** **suporte de e-mail**e **número de telefone de suporte.**

    ![Ver informações de suporte](../media/lab-user-support-information/support-information.png)
4. Pode ver informações de contacto de suporte para outro laboratório, mudando para aquele laboratório na lista de entrega. 

    ![Mude para outro laboratório](../media/lab-user-support-information/switch-another-lab.png)
5. Agora, vê os dados de contacto de apoio para o outro laboratório. 

    ![Informação de apoio de outro laboratório](../media/lab-user-support-information/second-lab-support-information.png)
