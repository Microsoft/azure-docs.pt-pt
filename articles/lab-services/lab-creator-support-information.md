# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>Ver informações de suporte (criador de laboratório nos Serviços Azure Lab)
Este artigo explica como você (como um criador de laboratório) pode ver as seguintes informações de suporte:

- URL
- E-mail
- Telefone
- Instruções adicionais

Pode usar esta informação para obter ajuda quando se detendo com problemas técnicos enquanto cria um laboratório numa conta de laboratório.

 
## <a name="view-support-information"></a>Ver informações de apoio
1. Inscreva-se no [portal Azure Lab Services](https://labs.azure.com).
2. Selecione o ponto de interrogação **(?**) no canto superior direito da página. 
3. Confirme que vê links para o site de suporte à **vista,** **suporte de e-mail**e **número de telefone de suporte**.

    ![Ver informações de apoio](./media/lab-creator-support-information/support-information.png)
