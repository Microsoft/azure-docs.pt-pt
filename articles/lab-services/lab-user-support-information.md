# <a name="view-support-information-lab-user-in-azure-lab-services"></a>Ver informações de suporte (utilizador de laboratório nos Serviços Azure Lab)
Este artigo explica como você (como utilizador de laboratório) pode ver as seguintes informações de suporte:

- URL
- E-mail
- Telefone
- Instruções adicionais

Pode usar esta informação para obter ajuda quando se detendo com problemas técnicos enquanto usa um laboratório numa conta de laboratório.

 
## <a name="view-support-information"></a>Ver informações de apoio
1. Inscreva-se no [portal Azure Lab Services](https://labs.azure.com).
2. Selecione o **laboratório ou máquina virtual** para o qual necessita de ajuda e selecione? **?** no canto superior direito da página. 
3. Confirme que vê links para o site de suporte à **vista,** **suporte de e-mail**e **número de telefone de suporte**.

    ![Ver informações de apoio](./media/lab-user-support-information/support-information.png)
4. Pode ver as informações de contacto de apoio para outro laboratório, mudando para o laboratório na lista de espera. 

    ![Mude para outro laboratório](./media/lab-user-support-information/switch-another-lab.png)
5. Agora, vê a informação de contacto de apoio para o outro laboratório. 

    ![Informação de apoio de outros laboratórios](./media/lab-user-support-information/second-lab-support-information.png)
