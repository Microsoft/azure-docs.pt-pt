# <a name="azure-stack-edge-and-azure-data-box-gateway-1911-release-notes"></a>Notas de lançamento da Azure Stack Edge e Azure Data Box Gateway 1911

As seguintes notas de lançamento identificam as questões críticas em aberto e as questões resolvidas para o lançamento de 1911 para Azure Stack Edge e Data Box Gateway.

As notas de lançamento são continuamente atualizadas e, à medida que são descobertas questões críticas que exigem uma suver, são adicionadas. Antes de implementar o seu Portal de Porta-Porta sinuoso De Ponta/Data, reveja cuidadosamente as informações contidas nas notas de lançamento.

Esta versão corresponde às versões do software:

- **Ponta de pilha azure 1911 (1.6.1049.786)**
- **Data Box Gateway 1911 (1.6.1049.786)**

> [!NOTE]
> A atualização 1911 só pode ser aplicada a todos os dispositivos que estejam a executar versões de disponibilidade geral (GA) do software ou posteriormente.

## <a name="whats-new"></a>Novidades

Não existem novidades no lançamento de 1911, apenas correções de bugs.

## <a name="known-issues-in-this-release"></a>Questões conhecidas neste lançamento

Não são divulgados novos problemas para esta libertação. Todas as questões de lançamento foram transmitidas dos lançamentos anteriores. Para ver uma lista de questões conhecidas, vá a [questões conhecidas no lançamento da AG](data-box-gateway-release-notes.md#known-issues-in-ga-release).
