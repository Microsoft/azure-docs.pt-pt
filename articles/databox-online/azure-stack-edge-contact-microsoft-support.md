# <a name="open-a-support-ticket-for-azure-stack-edge-and-azure-data-box-gateway"></a>Abra um bilhete de suporte para Azure Stack Edge e Azure Data Box Gateway

Este artigo aplica-se ao Azure Stack Edge e ao Azure Data Box Gateway, ambos geridos pelo serviço Azure Stack Edge / Azure Data Box Gateway. Se encontrar algum problema com o seu serviço, pode criar um pedido de serviço de apoio técnico. Este artigo acompanha-o através de:

* Como criar um pedido de apoio.
* Como gerir um suporte solicitar ciclo de vida dentro do portal.

## <a name="create-a-support-request"></a>Criar um pedido de suporte

Execute os seguintes passos para criar um pedido de apoio:

1. Vá à sua encomenda de Gateway Azure Stack Edge ou Data Box. Navegue para Apoiar + secção de **resolução de problemas** e, em seguida, selecione **Novo pedido**de suporte .
   
2. Em Novo pedido de **apoio,** no separador Basics, tome as **seguintes etapas:**
    
    1. A partir da lista de dropdown do **tipo Emissão,** selecione **Técnico**.
    2. Escolha a sua **Subscrição**.
    3. Em **Serviço,** verifique **Os Meus Serviços.** A partir da lista de dropdown, selecione **Azure Stack Edge e Data Box Gateway**.
    4. Selecione o seu **Recurso**. Isto corresponde ao nome da sua encomenda.
    5. Forneça um breve **resumo** do problema que está a passar. 
    6. Selecione o seu **tipo problema**.
    7. Com base no tipo de problema selecionado, escolha um **subtipo problema**correspondente .
    8. Selecione **Seguinte: Soluções >>**.

        ![Noções básicas](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-support-request-1.png)

3. No separador **Detalhes,** tome os seguintes passos:
    
    1. Forneça a data e a hora de início para o problema.
    2. Forneça uma **descrição** para o seu problema.
    3. No upload do **Ficheiro,** selecione o ícone da pasta para navegar em quaisquer outros ficheiros que pretenda carregar.
    4. Verifique **partilhar informações de diagnóstico**.
    5. Com base na sua subscrição, um plano de **Suporte** é automaticamente povoado.
    6. A partir da lista de abandono, selecione a **Severity**.
    7. Especifique um método de **contacto preferido**.
    8. As horas de **Resposta** são automaticamente selecionadas com base no seu plano de subscrição.
    9. Forneça a linguagem que prefere para o Apoio.
    10. Nas **informações de Contacto**, forneça o seu nome, e-mail, telefone, contacto opcional, país/região. O Microsoft Support utiliza estas informações para lhe contactar para mais informações, diagnóstico e resolução. 
    11. Selecione **A seguir: Rever + Criar >>**.

        ![Problema](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-support-request-2.png)

4. No separador **Review + Criar,** reveja as informações relacionadas com o bilhete de suporte. Selecione **Criar**. 

    ![Problema](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-support-request-3.png)

    Depois de ter criado o bilhete de Suporte, um engenheiro de suporte entrará em contacto consigo o mais rapidamente possível para prosseguir com o seu pedido.

## <a name="get-hardware-support"></a>Obtenha suporte de hardware

Esta informação aplica-se apenas ao dispositivo Azure Stack. O processo de reportar problemas de hardware é o seguinte:

1. Abra um bilhete de suporte do portal Azure para uma questão de hardware. Sob **o tipo de problema,** selecione **Hardware De Stack Azure**. Escolha o **subtipo Problema** como falha de **hardware**. 

    ![Problema de hardware](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-hardware-issue-1.png)

    Depois de ter criado o bilhete de Suporte, um engenheiro de suporte entrará em contacto consigo o mais rapidamente possível para prosseguir com o seu pedido. 

2. Se o Microsoft Support determinar que se trata de um problema de hardware, então ocorre uma das seguintes ações: 

    - É enviada uma Unidade de Substituição de Campo (FRU) para a peça de hardware falhada. Atualmente, a unidade de alimentação é a única FRU suportada. 
    - Para qualquer outra falha de peça, a Microsoft faz uma substituição completa do sistema (FSR) ou uma troca de dispositivos.

3. Se for levantado um bilhete de apoio antes das 16:30 locais (de segunda a sexta-feira), um técnico no local é enviado no dia útil seguinte para a sua localização para realizar uma FRU ou uma substituição completa do dispositivo.

## <a name="manage-a-support-request"></a>Gerir um pedido de apoio

Depois de criar um pedido de suporte, pode gerir o ciclo de vida do mesmo no portal.

#### <a name="to-manage-your-support-requests"></a>Para gerir os seus pedidos de apoio

1. Para chegar à página de ajuda e suporte, navegue para **navegar > ajuda + suporte**.

    ![Gerir pedidos de apoio](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-manage-support-request-1.png)   

2. Uma listagem tabular de pedidos de **suporte recentes** é exibida em Ajuda **+ suporte**.

    <!--[Manage support requests](./media/azure-stack-edge-contact-microsoft-support/data-box-edge-support-request-1.png)--> 

3. Selecione e clique num pedido de suporte. Pode ver o estado e os detalhes para este pedido. Clique **+ Nova mensagem** se quiser acompanhar este pedido.
