# <a name="iot-solution-accelerators-documentation"></a>Documentação dos aceleradores de soluções de IoT

Comece a trabalhar rapidamente com os aceleradores de soluções de IoT para criar as suas soluções de IoT. Adicione novos dispositivos (e ligue outros já existentes) através de SDKs de dispositivos para várias plataformas, incluindo Linux, Windows e sistemas operativos em tempo real. Dimensione facilmente de apenas alguns sensores para milhões de dispositivos ligados em simultâneo e confie na disponibilidade global do Azure&mdash;, independentemente das dimensões do seu projeto.
