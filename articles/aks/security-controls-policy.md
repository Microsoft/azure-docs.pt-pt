# <a name="azure-policy-regulatory-compliance-controls-for-azure-kubernetes-service-aks"></a>Controlos de conformidade regulamentar da política Azure para o Serviço Azure Kubernetes (AKS)

[A Conformidade Regulamentar na Política Azure](../governance/policy/concepts/regulatory-compliance.md) fornece à Microsoft definições de iniciativa criadas e geridas, conhecidas como _incorporados,_ para os domínios de conformidade e **controlos** de segurança relacionados com **diferentes** padrões de conformidade. Esta página lista os **domínios** de conformidade e **os controlos de segurança** do Serviço Azure Kubernetes (AKS).
Pode atribuir os incorporados a um controlo de **segurança** individualmente para ajudar a tornar os seus recursos Azure conforme com o padrão específico.
