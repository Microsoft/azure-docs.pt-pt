# <a name="sfctl-mesh"></a>sfctl mesh
Exclua e gerencie aplicativos de malha Service Fabric.

## <a name="subgroups"></a>Subgrupos
|Subgrupo|Descrição|
| --- | --- |
| [app](service-fabric-sfctl-mesh-app.md) | Obter e excluir recursos do aplicativo. |
| [code-package-log](service-fabric-sfctl-mesh-code-package-log.md) | Obtenha os logs do contêiner do pacote de códigos especificado para a réplica de serviço fornecida. |
| [deployment](service-fabric-sfctl-mesh-deployment.md) | Crie Service Fabric recursos de malha. |
| [gateway](service-fabric-sfctl-mesh-gateway.md) | Obter e excluir recursos do gateway de malha. |
| [rede](service-fabric-sfctl-mesh-network.md) | Obter e excluir recursos de rede de malha. |
| [secret](service-fabric-sfctl-mesh-secret.md) | Obter e excluir recursos de segredo de malha. |
| [segredo](service-fabric-sfctl-mesh-secretvalue.md) | Obter e excluir recursos de segredo de malha. |
| [service](service-fabric-sfctl-mesh-service.md) | Obter detalhes do serviço e listar serviços de um recurso de aplicativo. |
| [service-replica](service-fabric-sfctl-mesh-service-replica.md) | Obter detalhes da réplica e listar réplicas de um determinado serviço em um recurso de aplicativo. |
| [volume](service-fabric-sfctl-mesh-volume.md) | Obter e excluir recursos de volume. |
