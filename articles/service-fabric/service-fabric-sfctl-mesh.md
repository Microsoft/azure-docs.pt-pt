# <a name="sfctl-mesh"></a>sfctl mesh
Eliminar e gerir aplicações de Mesh do Service Fabric.

## <a name="subgroups"></a>Subgrupos
|Subgrupo|Descrição|
| --- | --- |
| [app](service-fabric-sfctl-mesh-app.md) | Obter e eliminar recursos de aplicação. |
| [code-package-log](service-fabric-sfctl-mesh-code-package-log.md) | Obter os registos do contentor do pacote de código especificada para a réplica de determinado serviço. |
| [deployment](service-fabric-sfctl-mesh-deployment.md) | Crie recursos de malha do Service Fabric. |
| [gateway](service-fabric-sfctl-mesh-gateway.md) | Obter e eliminar recursos de gateway de malha. |
| [network](service-fabric-sfctl-mesh-network.md) | Obter e eliminar recursos de rede de malha. |
| [secret](service-fabric-sfctl-mesh-secret.md) | Obter e eliminar recursos secreta da malha. |
| [secretvalue](service-fabric-sfctl-mesh-secretvalue.md) | Obter e eliminar recursos de secretvalue da malha. |
| [service](service-fabric-sfctl-mesh-service.md) | Obtenha detalhes do serviço e os serviços de lista de um recurso de aplicação. |
| [service-replica](service-fabric-sfctl-mesh-service-replica.md) | Obtenha os detalhes de réplica e de réplicas de lista de um determinado serviço num recurso de aplicação. |
| [volume](service-fabric-sfctl-mesh-volume.md) | Obter e eliminar recursos de volume. |
