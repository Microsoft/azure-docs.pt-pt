# <a name="sfctl-mesh"></a>malha de sfctl
Eliminar e gerir aplicações de Mesh do Service Fabric.

## <a name="subgroups"></a>Subgrupos
|Subgrupo|Descrição|
| --- | --- |
| [Aplicação](service-fabric-sfctl-mesh-app.md) | Obter e eliminar recursos de aplicação. |
| [registo de pacote de código](service-fabric-sfctl-mesh-code-package-log.md) | Obter os registos do contentor do pacote de código especificada para a réplica de determinado serviço. |
| [deployment](service-fabric-sfctl-mesh-deployment.md) | Crie recursos de malha do Service Fabric. |
| [Gateway](service-fabric-sfctl-mesh-gateway.md) | Obter e eliminar recursos de gateway de malha. |
| [Rede](service-fabric-sfctl-mesh-network.md) | Obter e eliminar recursos de rede de malha. |
| [Segredo](service-fabric-sfctl-mesh-secret.md) | Obter e eliminar recursos secreta da malha. |
| [secretvalue](service-fabric-sfctl-mesh-secretvalue.md) | Obter e eliminar recursos de secretvalue da malha. |
| [service](service-fabric-sfctl-mesh-service.md) | Obtenha detalhes do serviço e os serviços de lista de um recurso de aplicação. |
| [réplica de serviço](service-fabric-sfctl-mesh-service-replica.md) | Obtenha os detalhes de réplica e de réplicas de lista de um determinado serviço num recurso de aplicação. |
| [volume](service-fabric-sfctl-mesh-volume.md) | Obter e eliminar recursos de volume. |
