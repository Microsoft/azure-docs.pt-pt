# <a name="get-started-with-built-in-iot-hub-integration"></a>Começar com integração do IoT Hub incorporado

Esta opção permite-lhe utilizar o serviço sem utilizar o Azure Security Center para agentes de segurança IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Permitir a integração do IoT Hub incorporado

Para permitir a monitorização da gestão da identidade do dispositivo, dispositivo para cloud e cloud para padrões de comunicação do dispositivo, faça o seguinte para iniciar o serviço:

1. Abra o seu **Hub IoT.**
1. Selecione o menu de visão geral da **Segurança.**
1. Clique **em Proteger a sua solução IoT** e complete o formulário de embarque.

Parabéns! Completou a ativação do Centro de Segurança Azure para o serviço IoT no seu IoT Hub.
