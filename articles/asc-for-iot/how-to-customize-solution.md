# <a name="customize-your-azure-security-center-for-iot-solution"></a>Personalize o seu Centro de Segurança Azure para solução IoT

Neste guia, aprenda a personalizar diferentes configurações no Azure Security Center para IoT.

> [!div class="checklist"]
> * Configure recomendações de solução
> * Alterar definições

## <a name="configure-solution-recommendations"></a>Configure recomendações de solução

Para configurar o seu Centro de Segurança Azure para recomendações de solução IoT, faça o seguinte:

1. Abra o seu **Hub IoT** no portal Azure.
1. Selecione e abra **definições** em **Segurança** no menu esquerdo.
1. Em **Definições**, selecione **Configurações Recomendadas**.
1. Desativar/ativar as recomendações de solução relevantes para a sua organização e fluxos de trabalho.

## <a name="change-settings"></a>Alterar definições

Gerencie o seu Centro de Segurança Azure para a definição de IoT:

Para configurar o seu Centro de Segurança Azure para as definições de IoT, faça o seguinte:

1. Abra o seu **Hub IoT** no portal Azure.
1. A partir do menu esquerdo em **Segurança,** selecione e abra **Definições**.
1. Em **Recolha de Dados***** selecione a definição de solução que pretende alterar.
1. Lembre-se de clicar sempre em **Guardar** na parte superior de qualquer ecrã de definição para guardar as alterações de definição.
