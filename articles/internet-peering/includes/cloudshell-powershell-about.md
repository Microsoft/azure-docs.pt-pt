Para executar os cmdlets, pode utilizar a Azure Cloud Shell, uma concha interativa gratuita. Tem as ferramentas comuns do Azure pré-instaladas e configuradas para utilização com a sua conta. Selecione **Copy** para copiar o código e cole-o em Cloud Shell. Em seguida, **selecione Enter** para executá-lo. Há algumas formas de lançar a Cloud Shell:


| Método de Lançamento | Captura de ecrã  |
|-----------------------------------------------|---|
| Abrir o Cloud Shell no seu browser. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Selecione o botão **Cloud Shell** na barra de ferramentas no canto superior direito do portal Azure. | [![Cloud Shell no portal](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Se não quiser utilizar o Azure Cloud Shell, pode instalar o PowerShell localmente. Se optar por instalar e utilizar o PowerShell localmente, certifique-se de instalar a versão mais recente dos cmdlets PowerShell do Gestor de Recursos Azure. Os cmdlets PowerShell são atualizados frequentemente. Normalmente, é necessário atualizar os cmdlets PowerShell para obter a funcionalidade mais recente. Se não o fizeres, podes encontrar problemas.

Para encontrar a versão do PowerShell que está a executar localmente, utilize o cmdlet **'Get-Module -ListAvailable Az'.** Para atualizar, consulte [instalar o módulo Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps). Para mais informações, consulte [como instalar e configurar a Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview).

Se utilizar o PowerShell no macOS, siga os passos na [Instalação do PowerShell no macOS.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
