# <a name="peering-service-partners"></a>Parceiros de serviço de observação

A Microsoft estabeleceu uma parceria com fornecedores de serviços de internet (ISPs), fornecedores de intercâmbio de internet (IXPs) e fornecedores de interligação de nuvem definida por software (SDCI) em todo o mundo para fornecer conectividade pública fiável e de alto desempenho com o encaminhamento ideal do cliente para a rede Microsoft.

Este artigo fornece informações sobre os fornecedores de conectividade que são parceiros com a Microsoft para oferecer a ligação Azure Peering Service aos clientes.
