# <a name="sdk-content"></a>Conteúdo do SDK
> [!IMPORTANT]
> O Azure Mobile Engagement ter extinguido em 3/31/2018. Esta página será eliminada pouco tempo depois.
> 

Este documento apresenta e descreve o conteúdo do arquivo SDK.

`mobile-engagement-VERSION.jar`

> Ficheiros de classe SDK do engagement.
> 
> 

`res` Pasta

> Recursos de SDK do alcance do engagement (drawable e ficheiros de esquema para incluir nas suas aplicações).
> 
> 

`src` Pasta

> Código de origem SDK do engagement.
> 
> Tenha em atenção que as origens do serviço não estão incluídas (`com.microsoft.azure.engagement.service.EngagementService` e os respetivos ficheiros de origem relacionados).
> 
> 

`documentation.html`

> Um atalho para o site de documentação correspondente para a versão transferida do SDK.
> 
> 

Outros ficheiros são uma cópia de licenças de software.
