# <a name="how-to-troubleshoot-common-azure-marketplace-publisher-issues"></a>Como resolver problemas comuns de publicador do Azure Marketplace
Este artigo explica como resolver ou obter suporte para alguns cenários comuns do Marketplace.

## <a name="credits-and-refunds"></a>Créditos e refunds
Em algumas circunstâncias, poderá ter de pedir um crédito ou um reembolso em nome de um cliente.  Support da Microsoft pode ajudar a garantir que os pedidos de crédito e reembolso de clientes são considerados e que uma resposta é que lhe é fornecida no due óbvio.

O seguinte mostra como pedir um crédito ou reembolso em nome de um cliente.

> [!NOTE]
> Se um cliente ou revendedor a partir do Azure contacta a Microsoft para pedir um reembolso ou crédito, iremos aplicar as mesmas políticas que utilizamos para o nosso aprovisionamento dos serviços do Azure. Se as nossas políticas indicarem que um reembolso ou crédito é devido, iremos irá emitir proativamente (e autorizar-nos para emitir em seu nome) um reembolso ou crédito para o cliente ou revendedor a partir do Azure, conforme aplicável. Se as nossas políticas não indicar que um reembolso ou crédito devido, poderá Convidamo-o cliente ou revendedor a partir do Azure para contactá-lo diretamente.
>
>

  ![1_RequestRefund][1]

1. Navigate to the support pages here: [https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0&ccsid=635562546535117884](https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0&ccsid=635562546535117884).
2. No **tipo de problema** caixa, selecione **gerir clientes**.
3. No **categoria** caixa, selecione **aprovar ou para requerer um reembolso de cliente**.
4. Clique em **pedido de início**.

   ![2_RequestRefund][2]

1. Forneça as informações de contacto.
2. No **o que é o problema?** box, introduza as seguintes informações:

   * O ID de subscrição do cliente afectado
   * O nome do cliente
   * Quanto o crédito deve ser por
   * O período do crédito
   * A data da fatura desse período
   * O motivo para o crédito

Em seguida, irá receber um e-mail de confirmação dentro de 24 horas que recebemos as suas informações. Depois de efetuada uma decisão em relação o pedido, será informado de que o resultado.

## <a name="customer-subscription-validation"></a>Validação de subscrição do cliente
Em algumas circunstâncias, poderá ter de validar os detalhes da subscrição para um dos seus clientes. Pode contactar a Microsoft Support para ajudar com esta. Support da Microsoft irá ajudá-lo para confirmar que um ID de subscrição e o endereço para faturação pertencem a um cliente conhecido.

  ![subscriptionvalidation][3]

* Um cliente poderá contactá-lo a pedir suporte.
* Não pode saber como validar que um cliente é um cliente para uma oferta genuíno.
* Pode iniciar um pedido de suporte para pedir a validação de subscrição do cliente.
* Terá de fornecer um ID de subscrição do cliente e o endereço para faturação.
* Microsoft irá contactá-lo para fornecer a confirmação de que o endereço para faturação e o ID de subscrição são reconhecidos-nos.

## <a name="how-do-i-update-my-publisher-profile"></a>Como atualizar o meu perfil de publicador
Se pretender atualizar o perfil do publicador, isto pode ser feito através do Microsoft Dev Center:

**1. Inicie sessão no Windows Dev Center**.

* Abra uma nova Chrome Incognito ou Internet Explorer InPrivate navegação sessão para se certificar de que tem não sessão iniciada para uma conta pessoal.
* Aceda a [http://dev.windows.com/registration?accountprogram=azure](http://dev.windows.com/registration?accountprogram=azure)

![InPrivate][4]

**2. Inicie sessão com a sua empresa conta escolar ou profissional** (ou seja, AzureStore@yourcompany.com).

![início de sessão][5]

**3. Concluir a verificação de "Ajuda-na proteger a sua conta"**.

> [!NOTE]
> Pode utilizar o seu telefone número ou endereço de e-mail para verificar.
>
>

![protectacct][6]

**4. Navegue até às informações de detalhes da conta e contacte**.

![acctdetails][7]

## <a name="how-do-i-add-or-delete-an-administrator-from-my-publishing-account"></a>Como adicionar ou eliminar um administrador a partir da minha conta de publicação?
![manageadmins][8]

1. Navegue para a publicação do Portal.
2. Selecione **publicadores**.
3. Selecione **administradores**.
4. Adicione o endereço de e-mail de coadministrador que pretende adicionar ou remover.
5. Selecione **Guardar**.
