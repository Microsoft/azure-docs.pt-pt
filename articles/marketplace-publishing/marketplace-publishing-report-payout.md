# <a name="understand-your-azure-marketplace-payout-reports"></a>Compreender os seus relatórios do Azure Marketplace dividendos
## <a name="access-and-view-your-payout-reports"></a>Aceder e ver os relatórios de dividendos
Enquanto que efetuar a transição para Dev Center alguns dos seus relatórios dividendos poderão estar disponíveis no Centro de desenvolvimento em https://dev.windows.com/en-us enquanto outros utilizadores ainda podem ser encontrados no Portal de publicação em https://publish.windowsazure.com.

Relatórios de dividendos estarão agora disponíveis em **Dev Center** para qualquer ofertas de Marketplace que estão associadas a payouts modernas; atualmente inclui:

* VMs
* Oferece B + C
* Dados e Dev serviços oferecidos em EA

Relatórios de dividendos continuarão a estar no **Portal publicação** para:

* Dados e Dev serviços oferecidos em Web direto (que continua a utilizar o sistema de dividendos legado).

Os relatórios estão disponíveis 45 dias após o fecho do trimestre e são calculados após qualquer refunds.

### <a name="access-payout-reports-in-dev-center"></a>Aceder aos relatórios de dividendos no Dev Center
1. Navegue para Dev Center em https://dev.windows.com/en-us.
2. Clique em **Dashboard**.

    ![LandingPageDashboardHighlight][1]
3. Clique em **dividendos resumo**.

    ![DashboardPayoutSummary][2]

## <a name="view-your-payout-reports-in-dev-center"></a>Ver os relatórios de dividendos no Dev Center
O relatório de dividendos para cada trimestre regista todas as transações que ocorreram dentro desse quarto.

* A quantidade reservada indica qualquer pagamentos que estão a acumular fora do ciclo de pagamento futuros (por exemplo, esta quantidade irá mudar para o mês seguinte de pagamento futuros).  Este valor será normalmente $0 (a menos que um cliente pays bem em Avançadas).
* Clique no pagamento de Upcoming ou um pagamento mais recente **ver detalhes** ligações para ver uma nota sobre esses payouts.
* Clique em **declarações de pagamento** para ver os detalhes em continua por aplicação/produto.
* Clique em de **vista** ligação para ver instruções individuais.

    ![PayoutSummaryUpcomingMostRecentLinksStatement][3]
* Utilize o **prossegue repartição** filtro na parte inferior da instrução individuais para ver a várias aplicações/produtos, caso existam.

    ![PayoutSummaryPaymentStatementsFilterControl][4]

## <a name="view-your-payout-reports-in-publishing-portal"></a>Ver os relatórios de dividendos no Portal de publicação
O relatório de dividendos para cada trimestre regista todas as transações que ocorreram dentro desse quarto.

1. Navegue para o portal de publicação em https://publish.windowsazure.com.
2. Do **publicadores** secção, clique em **dividendos relatórios**.
3. Clique na lista pendente para apresentar relatórios de dividendos trimestral disponíveis.

    ![accessingpayoutreport][5]

### <a name="read-your-payout-reports"></a>Leia os relatórios de dividendos
O relatório de dividendos para cada trimestre regista todas as transações que ocorreram dentro desse quarto.

* Se estiver à procura de entradas de ledger relacionados com um determinado trimestre, selecione o relatório de dividendos para esse trimestre da lista pendente. Por exemplo, se estiver interessado em entradas de ledger de Abril para Junho de 2015, selecione esse intervalo de datas da lista pendente.
* Se estiver à procura para obter detalhes sobre payouts relacionados com um determinado trimestre, selecione o relatório de dividendos para o trimestre subsequente. Por exemplo, se estiver interessado nos payouts para Abril para Junho de 2015, estes quantidades aparecerá no relatório de dividendos subsequentes para Julho para Setembro de 2015.
  ![readingpayoutreport][6]
* O painel de resumo financeiro Mostra saldos, créditos e debits por categoria.
* Entradas de Ledger mostram transações individuais.

## <a name="payout-questions"></a>Perguntas de dividendos
Se tiver uma pergunta relacionadas com a sua payouts, contacte a nossa equipa de suporte.

![payoutquestions][9]

1. Navegue para as páginas de suporte.
2. Selecione **Payouts**.
3. Selecione **dividendos relacionadas com pedidos de**.
4. Clique em **pedido de início**.
