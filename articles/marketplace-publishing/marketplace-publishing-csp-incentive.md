# <a name="view-and-change-your-cloud-solution-provider-reseller-incentive-in-the-azure-marketplace"></a>Ver e alterar o fornecedor de solução em nuvem "Revendedor incentivo" no Azure Marketplace
No dia 15 de Março de 2016 iremos irá efetuar a fornecedores de soluções de nuvem (CSPs) "Revendedor incentivo" disponível para publicadores do Marketplace.  A predefinição para este valor incentivo é **Sim**.  Como um publicador, pode optar ativamente do programa incentivo.  Este artigo mostra como alterar a definição no seu perfil do publicador, pelo que pode ativamente.

Para ver ou alterar o estado de "Revendedor incentivo", siga estes passos:

1. Inicie sessão no seu [conta de publicação do Microsoft Azure](https://publish.windowsazure.com/workspace)

   ![Ecrã de destino do perfil][1]
2. Clique em **editores**

   ![Ecrã de destino do perfil - editores de ligação][2]
3. Tenha em atenção as informações no seu perfil de CSPs.  Não existe um **mudar para diga Sim ou não** para "Incentivo revendedor" - o estado predefinido é "Sim".

   ![Perfil - editores][3]
4. Para alterar o "incentivo revendedor" Estado de aceder ao seu perfil do publicador, em seguida, selecione e guarde à sua escolha:

   * Selecione **Sim** escolher fornecer o desconto 20 por cento para os CSPs que são reselling as aplicações e serviços ter apresentados no Azure Marketplace.  Ou selecione **não** ativamente por este programa incentivo.  Em ambos os casos, o publicador recebe o mesmo dividendos.
   * Depois de efetuar uma seleção Certifique-se de que clica **guardar** no canto superior direito.

   ![Caixas de alteração de perfil - publicadores-][4]
