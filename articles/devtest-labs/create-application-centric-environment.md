# <a name="create-an-application-centric-environment"></a>Criar um ambiente centrado na aplicação

[Quali's CloudShell Colony](https://azuremarketplace.microsoft.com/marketplace/apps/quali_systems.cloudshell_colony?tab=Overview) é uma plataforma SaaS para fornecer Automatização de Infraestruturas em Escala para ambientes complexos centrados em aplicações em tecnologias de nuvem, incluindo Azure e Kubernetes. A CloudShell Colony complementa a Azure DevTest Labs para ajudar as equipas de desenvolvimento a implementar aplicações complexas em todo o fluxo de valor, até à produção.

Este artigo demonstra como criar um ambiente centrado em aplicações com a CloudShell Colony e a Microsoft Azure.

## <a name="set-up-the-environment-with-cloudshell-colony-and-microsoft-azure"></a>Configurar o ambiente com a CloudShell Colony e o Microsoft Azure

1. Inscreva-se para o seu teste gratuito de [Colónia.](https://azuremarketplace.microsoft.com/marketplace/apps/quali_systems.cloudshell_colony?tab=Overview)

    :::image type="content" source="./media/create-application-centric-environment/free-trial.png" alt-text="Inscrever-se numa avaliação gratuita":::    
1. Ligue a sua conta Azure[(ver passos aqui).](https://colonysupport.quali.com/hc/articles/360008222234)

    :::image type="content" source="./media/create-application-centric-environment/welcome.png" alt-text="Bem-vindo à Colónia":::     
1. Convide os utilizadores para o seu espaço.
1. Crie a sua primeira planta utilizando um ficheiro YAML[(veja os passos aqui).](https://colonysupport.quali.com/hc/articles/360001680807-Steps-to-Developing-a-Blueprint)
    1. Ligue o seu repo de plantas em GitHub ou BitBucket à Colónia.
    1. Utilize uma amostra de colónia como base e modifique conforme apropriado.

        :::image type="content" source="./media/create-application-centric-environment/performance-stress-tests.png" alt-text="Testes de esforço":::    
    1. Publique a sua planta para outros usarem.
1. Lance o seu Ambiente de Aplicação numa caixa de areia utilizando a Colónia.

    :::image type="content" source="./media/create-application-centric-environment/blueprints.png" alt-text="Lance o seu Ambiente de Aplicação numa caixa de areia usando a Colónia":::    

> [!NOTE]
> Também pode integrar a sua planta como parte de um fluxo de trabalho CI/CD em Azure DevOps[(ver passos aqui).](https://colonysupport.quali.com/hc/articles/360008464234)

:::image type="content" source="./media/create-application-centric-environment/devops-pipeline.png" alt-text="Ligue-se ao seu oleoduto Azure DevOps":::
