# <a name="service-bus-messaging-samples"></a>Amostras de mensagens de ônibus de serviço
As amostras de mensagens service Bus demonstram características-chave na [mensagem Service Bus](https://azure.microsoft.com/services/service-bus/). Atualmente, pode encontrar as amostras nos seguintes locais:

| Linguagem de programação | Localização de SDK ou amostras |
| -------------------- | ----------------------- |
| .NET, Java e Gestão | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Explorador do Service Bus

Além disso, o [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) é uma amostra hospedada no GitHub que lhe permite ligar-se a um espaço de nome de serviço de ônibus de serviço e gerir facilmente entidades de mensagens. A ferramenta fornece funcionalidades avançadas, como a funcionalidade de importação/exportação, e a capacidade de testar entidades de mensagens e serviços de retransmissão. Você pode encontrar a fonte completa do Service Bus Explorer e documentação no [GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
