# <a name="azure-database-for-postgresql-videos"></a>Base de dados do Azure para PostgreSQL vídeos

[Descrição geral: Base de dados do Azure para PostgreSQL e MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

Base de dados do Azure para PostgreSQL e base de dados do Azure para MySQL reúnem motores de base de dados de edição de Comunidade e as capacidades de um serviço totalmente gerido, para que se possa focar nas suas aplicações em vez de ter de gerir uma base de dados. Ligue-se para obter uma descrição geral das vantagens da utilização do serviço e ver alguns dos recursos em ação.

[Descrição aprofundada da capacidades de serviço gerido para o MySQL e PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

Base de dados do Azure para PostgreSQL e base de dados do Azure para MySQL reúnem motores de base de dados de edição de Comunidade e os recursos de um serviço totalmente gerido. Ligue-se para obter uma descrição profunda sobre como funcionam os estes serviços, como podemos garantir a elevada disponibilidade e rápida de dimensionamento (em segundos), para que pode atender às necessidades dos seus clientes. Também aprenderá sobre alguns dos investimentos subjacentes na segurança e a disponibilidade em todo o mundo.

[Desenvolver uma aplicação de análise inteligente com PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T149)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T149/player]

Base de dados do Azure para PostgreSQL reúne capacidades de um serviço totalmente gerido e de motor de base de dados de edição de Comunidade — para que se possa focar nas suas aplicações em vez de ter de gerir uma base de dados. Ligue-se para ver em ação, como é fácil criar novas experiências, como adicionar os serviços cognitivos para as suas aplicações por não diferenciar no Azure.

[Como começar com a nova base de dados do Azure para o serviço PostgreSQL](https://channel9.msdn.com/events/Build/2017/B8046)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8046/player]

Neste vídeo da conferência 2017 Microsoft //Build, aprenda com dois clientes de adoção antecipados como utilizaram base de dados do Azure para o serviço PostgreSQL para inovar mais rapidamente. Saiba como eles migrados para o serviço e discutir os passos seguintes no seu desenvolvimento de aplicativo. O vídeo explica alguns dos recursos principais de serviço e discute como, como um desenvolvedor pode migrar as suas aplicações existentes ou desenvolva novas aplicações que utilizam este serviço de PostgreSQL gerido no Azure.
