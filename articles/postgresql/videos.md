# <a name="azure-database-for-postgresql-videos"></a>Base de dados do Azure para PostgreSQL vídeos
[Como começar com a nova base de dados do Azure para o serviço de PostgreSQL](https://channel9.msdn.com/events/Build/2017/B8046)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8046/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Neste vídeo de conferências de //Build de Microsoft 2017, mais de dois clientes adoção precoce como tiver utilizadas base de dados do Azure para o serviço de PostgreSQL a inovar mais rapidamente. Saiba como serem migrados para o serviço e aborda os passos seguintes na respetiva desenvolvimento de aplicações. O vídeo explica algumas das funcionalidades do serviço de chave e descreve a forma como como um programador pode migrar as suas aplicações existentes ou desenvolver novas aplicações que utilizam este serviço PostgreSQL geridas no Azure.
