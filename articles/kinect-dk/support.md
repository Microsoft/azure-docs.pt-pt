# <a name="azure-kinect-support-options-and-resources"></a>Opções e recursos de suporte da Azure Kinect

Este artigo irá acompanhá-lo através de várias opções de suporte.

## <a name="community-support"></a>Suporte da comunidade

Existem várias formas de obter as suas perguntas respondidas através de fóruns públicos:

- [StackOverflow,](https://stackoverflow.com/search?q=azurekinect&s=3b855ed0-8564-4961-856f-9614aeab4c0d&s=fd9ea920-622c-4d8e-b908-ec996e1f1403)onde pode fazer perguntas ou pesquisar através da biblioteca de perguntas existente.
- [GitHub](https://github.com/Microsoft/Azure-Kinect-Sensor-SDK), onde pode fazer perguntas, abrir novos bugs ou contribuir para o desenvolvimento do sensor Azure Kinect SDK.
- [Forneça feedback,](https://feedback.azure.com/forums/920053-azure-kinect-dk)onde pode partilhar as suas ideias para o futuro do produto e votar numa ideia existente.

## <a name="assisted-support"></a>Apoio assistido

Há várias maneiras de obter um apoio assistido para Azure Kinect.

### <a name="microsoft-qa"></a>Microsoft Q&A

Para respostas rápidas e fiáveis sobre as suas questões técnicas de produtos da Microsoft Engineers, Azure Most Valuable Professionals (MVPs), ou da nossa comunidade especializada, envolva-se connosco no [Microsoft Q&A](/answers/products/azure), o destino preferido da Azure para apoio à comunidade.

- [Microsoft Q&A para Azure Kinect,](/answers/topics/azure-kinect-dk.html)onde pode fazer perguntas ou pesquisar através da biblioteca de perguntas existente.

### <a name="development-azure-kinect-on-azure"></a>Desenvolvimento Azure Kinect em Azure

Os subscritores do Azure podem criar e gerir pedidos de suporte no portal Azure. Suporte de desenvolvimento one-on-one para Body Tracking, Sensor SDK, Speech device SDK ou Azure Cognitive Services está disponível para assinantes do Azure com um [Plano de Suporte Azure](https://azure.microsoft.com/support/plans/) associado à sua subscrição.

  - Tem um [Plano de Apoio Azure](https://azure.microsoft.com/support/plans/) associado à sua subscrição Azure?  [Inscreva-se no portal Azure](https://ms.portal.azure.com/) para apresentar um incidente.
  - Precisa de uma assinatura Azure? [As opções de subscrição do Azure](https://azure.microsoft.com/pricing/purchase-options/) fornecerão mais informações sobre diferentes opções.
  - Precisa de um plano de apoio? [Selecione plano de suporte](https://azure.microsoft.com/support/plans/)

### <a name="azure-kinect-on-premises-or-other-cloud-services"></a>Azure Kinect no local ou outros serviços em nuvem

Para suporte técnico utilizando o Sensor SDK e o Body Tracking SDK no local, abra um bilhete para suporte profissional no portal de [suporte da Microsoft.](https://support.microsoft.com/supportforbusiness/productselection?sapId=c49ea5bb-2b09-8612-be35-d55159732667)

### <a name="azure-kinect-dk-device"></a>Dispositivo Azure Kinect DK

Antes de contactar o suporte de hardware, certifique-se de que configura e atualizou o Azure Kinect DK. Para testar se o dispositivo está a funcionar, utilize o [visualizador Azure Kinect](azure-kinect-viewer.md). Saiba mais na nossa página [de ajuda do Azure Kinect DK.](./index.yml)
Você também pode querer verificar os [nossos problemas conhecidos e resolução de problemas](troubleshooting.md).

[Obtenha ajuda](https://support.microsoft.com/supportforbusiness/productselection?sapId=f77b1b95-721e-43a0-2db8-b01e81a3f813) com uma funcionalidade de dispositivo ou sensor, atualizações de firmware ou opções de compra.

Para obter mais informações sobre ofertas de suporte, saiba mais sobre [o suporte da Microsoft para o negócio.](https://support.microsoft.com/help/4341255/support-for-business)
