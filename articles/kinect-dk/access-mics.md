# <a name="access-azure-kinect-dk-microphone-input-data"></a>Aceder aos dados de entrada do microfone DK do Azure Kinect

Os [quickstarts do Speech SDK](../cognitive-services/speech-service/index.yml) fornecem exemplos de como usar a matriz de microfone DK do Azure Kinect em várias linguagens de programação.
Por exemplo, consulte o discurso Reconhecer em C++ no Windows utilizando o quickstart **Do SDK do Discurso.** O código está disponível [no GitHub.](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp)

Aceda ao conjunto de microfones também através da API do Windows. Consulte os seguintes documentos para obter mais informações sobre a documentação do Windows:

* [Arquitetura de áudio do Windows](/windows-hardware/drivers/audio/windows-audio-architecture)
* [Windows.Media.Capture documentação](/uwp/api/Windows.Media.Capture)
* [Tutorial para captura de webcam](/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [Informação áudio USB](/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

Também pode rever a especificação de [hardware da matriz](hardware-specification.md#microphone-array)do microfone.
