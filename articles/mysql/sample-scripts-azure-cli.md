| |  |
|---|---|

|**Restaurar um servidor**||
| [Restaurar um servidor](./scripts/sample-point-in-time-restore.md?toc=%2fcli%2fazure%2ftoc.json) | Script da CLI do Azure que restaura um servidor único da Base de Dados do Azure para MySQL para um ponto anterior no tempo. |
|**Manipular com registos do servidor**||
| [Ativar e transferir registos do servidor](./scripts/sample-server-logs.md?toc=%2fcli%2fazure%2ftoc.json) | Script da CLI do Azure que permite e transfere registos do servidor de um servidor único da Base de Dados do Azure para MySQL. |
