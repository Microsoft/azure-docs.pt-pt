Esta página fornece conteúdo de vídeo para aprender sobre a Base de Dados Azure para o MySQL.

## <a name="overview-azure-database-for-postgresql-and-mysql"></a>Visão geral: Base de Dados Azure para PostgreSQL e MySQL

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player] 
[Aberto no Canal 9](https://channel9.msdn.com/Events/Connect/2017/T147)

A Azure Database for PostgreSQL and Azure Database for MySQL reúne motores de base de dados de edição comunitária e capacidades de um serviço totalmente gerido , para que possa concentrar-se nas suas apps em vez de ter de gerir uma base de dados. Sintonize para obter uma visão geral rápida das vantagens de usar o serviço, e ver algumas das capacidades em ação.

## <a name="create-a-postgresql-or-mysql-server"></a>Criar um servidor PostgreSQL ou MySQL
A base de dados Azure para PostgreSQL e Azure Database para o MySQL são serviços geridos que utiliza para executar, gerir e escalar motores de base de dados de edição comunitária altamente disponíveis na nuvem. Estes tutoriais de vídeo mostram-lhe como criar um servidor PostgreSQL ou MySQL em cerca de três minutos usando o portal Azure. Se não tiver uma subscrição do Azure, [crie uma conta Azure gratuita](https://azure.microsoft.com/free/) antes de começar.

* [Tutorial de vídeo pós-SQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-postgresql-server-in-the-azure-portal)

* [Tutorial de vídeo MySQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-mysql-server-by-using-the-azure-portal)
