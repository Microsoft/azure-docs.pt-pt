[Visão geral: Base de Dados Azure para PostgreSQL e MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

A Azure Database for PostgreSQL and Azure Database for MySQL reúne motores de base de dados de edição comunitária e capacidades de um serviço totalmente gerido , para que possa concentrar-se nas suas apps em vez de ter de gerir uma base de dados. Sintonize para obter uma visão geral rápida das vantagens de usar o serviço, e ver algumas das capacidades em ação. Veja como criar uma [Base de Dados Azure para PostgreSQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-postgresql-server-in-the-azure-portal) e [Azure Database para o MySQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-mysql-server-by-using-the-azure-portal).

[Mergulho profundo nas capacidades de serviço geridas para MySQL e PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

A Azure Database for PostgreSQL and Azure Database for MySQL reúne motores de base de dados de edição comunitária e as capacidades de um serviço totalmente gerido. Sintonize-se para obter um mergulho profundo no funcionamento destes serviços — como garantimos alta disponibilidade e escala rápida (em segundos), para que possa atender às necessidades dos seus clientes. [Como começar com a nova Base de Dados Azure para o serviço MySQL](https://channel9.msdn.com/events/Build/2017/B8045)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8045/player]

Neste vídeo da conferência de maio de 2017 da Microsoft //Build, conheça a oferta gerida pela Microsoft mySQL em Azure. O vídeo percorre a estratégia da Microsoft para apoiar sistemas de base de dados Open-Source em Azure. O vídeo discute o que significa para si como desenvolvedor desenvolver ou implementar aplicações que utilizem o MySQL em Azure. Este vídeo mostra uma visão geral da arquitetura do serviço, e demonstra que a Base de Dados Azure para o MySQL está integrada com outros Serviços Azure, como web Apps.
