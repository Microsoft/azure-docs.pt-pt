# <a name="what-is-azure-database-migration-service"></a>O que é o Azure Database Migration Service?

O Azure Database Migration Service é um serviço totalmente gerido para permitir migrações perfeitas de várias fontes de base de dados para plataformas de dados Azure com o mínimo de tempo de inatividade (migrações online).

## <a name="migrate-databases-to-azure-with-familiar-tools"></a>Migrar bases de dados para Azure com ferramentas familiares

O Azure Database Migration Service integra algumas das funcionalidades das nossas ferramentas e serviços existentes. Fornece aos clientes uma solução abrangente e altamente disponível. O serviço utiliza o Assistente de [Migração](https://aka.ms/dma) de Dados para gerar relatórios de avaliação que fornecem recomendações para guiá-lo através das alterações necessárias antes de realizar uma migração. Cabe-lhe a si fazer qualquer reparação necessária. Quando estiver pronto para iniciar o processo de migração, o Serviço de Migração da Base de Dados Azure executa todos os passos necessários. Pode iniciar os seus projetos de migração e deixá-los avançar autonomamente sem preocupações ao saber que o processo tira partido das melhores práticas determinadas pela Microsoft.

> [!NOTE]
> A utilização do Serviço de Migração de Bases de Dados Azure para realizar uma migração online requer a criação de uma instância baseada no nível de preços Premium.
