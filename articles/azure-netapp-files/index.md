# <a name="azure-netapp-files-documentation"></a>Documentação do Azure NetApp Files

Saiba como utilizar o Azure NetApp Files para criar e gerir contas NetApp, conjuntos de capacidade e volumes, configurar a política de exportação, montar um volume para uma máquina virtual e gerir instantâneos.

> [!IMPORTANT] 
> Todas as tarefas descritas nesta documentação exigem que faça parte do programa de Pré-visualização Pública e que esteja na lista de permissões para aceder ao Fornecedor de Recursos do Azure do Microsoft.NetApp. Para pedir a associação ao programa de Pré-visualização Pública, veja [Registar no Azure NetApp Files](azure-netapp-files-register.md).
