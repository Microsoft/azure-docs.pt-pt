# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Montar ou desmontar um volume para máquinas virtuais Windows ou Linux 

Pode montar ou desmontar um volume para máquinas virtuais Windows ou Linux conforme necessário.  As instruções de montagem para máquinas virtuais do Linux estão disponíveis nos ficheiros do Azure NetApp.  

1. Clique nas **Volumes** painel e, em seguida, selecione o volume para o qual pretende montar o NFS exporta. 
2. Clique em **instruções de montagem** do volume selecionado e, em seguida, siga as instruções.
