# <a name="introduction-to-azure-netapp-files"></a>Introdução ao Azure NetApp Files
O serviço Azure NetApp Files é um serviço de armazenamento de ficheiros com tráfego limitado, de elevado desempenho e de nível empresarial. O Azure NetApp Files suporta qualquer tipo de carga de trabalho e, por predefinição, possui uma elevada disponibilidade. Pode selecionar níveis de desempenho e de serviço garantidos e configurar a instantâneos através do serviço. 

A documentação do Azure NetApp Files fornece instruções sobre como criar e gerir volumes através do Azure NetApp Files. 

> [!IMPORTANT] 
> Todas as tarefas descritas nesta documentação exigem que faça parte do programa de Pré-visualização Pública e que esteja na lista de permissões para aceder ao Fornecedor de Recursos do Azure do Microsoft.NetApp. Para pedir a associar o programa de pré-visualização pública, consulte [registar para ficheiros do Azure NetApp](azure-netapp-files-register.md).
