# <a name="manage-volumes-by-using-azure-netapp-files"></a>Gerir volumes utilizando ficheiros Azure NetApp
Pode gerir um volume visualizando, atualizando ou eliminando o volume. 

Se pretender apagar um volume, deve primeiro eliminar todos os sub-objectos de um volume. 

1.  Clique no painel **Volumes**, a partir do painel Gerir Conjuntos de Capacidade. 
2.  Utilize a vista **Volumes de Navegação** para visualizar, atualizar ou eliminar volumes existentes.
