# <a name="azure-status-overview"></a>Visão geral do estado de Azure

[O estatuto de azul](https://status.azure.com/status/) proporciona-lhe uma visão global da saúde dos serviços e regiões da Azure. Com o estado de Azure, pode obter informações sobre a disponibilidade do serviço. O estatuto de Azure está disponível para todos para visualizar todos os serviços que reportam a sua saúde de serviço, bem como incidentes com impacto alargado. No entanto, se é um utilizador atual do Azure, encorajamo-lo a utilizar a experiência personalizada na [Azure Service Health](https://aka.ms/azureservicehealth). O Serviço Azure Health inclui todas as paragens, próximas atividades de manutenção planeadas e assessorias de serviços.

![Página de estado do Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Atualizações de estado do Azure

A página de estado do Azure é atualizada em tempo real à medida que a saúde dos serviços Azure muda. Se deixar a página do estado de Azure aberta, pode controlar a taxa a que a página se atualiza com novos dados. No topo, pode ver a última vez que a página foi atualizada.

![Atualização do estado de Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>História do estado de Azure

Embora a página do estado de Azure mostre sempre as últimas informações de saúde, pode ver eventos mais antigos utilizando a [página de histórico do estado de Azure](https://status.azure.com/status/history/). A página de história contém todos os RCAs para incidentes ocorridos no dia 20 de novembro de 2019 ou mais tarde e irá - a partir dessa data - fornecer um histórico de RCA de 5 anos. RCAs antes de 20 de novembro de 2019 não estão disponíveis.

## <a name="rss-feed"></a>Feed RSS

O estado de Azure também fornece [um feed RSS](https://status.azure.com/status/feed/) de alterações à saúde dos serviços Azure que você pode subscrever.
