# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Criar uma subscrição adicional no portal do Azure

Pode criar subscrições adicionais para conta no Azure. Talvez queira uma subscrição adicional para evitar atingir os limites da subscrição, para criar ambientes separados para faturação e segurança ou para isolar dados por motivos de conformidade.

Se quiser criar subscrições do Azure ao abrigo do Contrato Enterprise da sua organização, terá de ter a função de Proprietário da Conta na organização.

Se ainda não tiver uma subscrição do Azure, crie uma [conta gratuita](https://azure.microsoft.com/free).

## <a name="create-an-additional-azure-subscription"></a>Criar uma subscrição do Azure adicional

1. Inicie sessão no [portal do Azure](https://portal.azure.com).
1. Escreva **Subscrições** na caixa de pesquisa.

   ![Captura de ecrã que mostra a pesquisa de uma subscrição no portal](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. Selecione **Adicionar**.

   ![Captura de ecrã que mostra o botão Adicionar na vista Subscrições](./media/billing-create-subscription/subscription-add.png)

Também pode criar subscrições adicionais através de programação. Para obter mais informações, veja [Criar subscrições Enterprise do Azure através de programação](../azure-resource-manager/programmatically-create-subscription.md).
