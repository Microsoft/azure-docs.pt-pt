# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Criar uma subscrição adicional no portal do Azure

Pode criar subscrições adicionais para a sua conta no Azure. Pode desejar uma subscrição adicional para evitar atingir os limites de subscrição, para criar ambientes separados de faturação e de segurança ou para isolar os dados por motivos de conformidade.

Se quiser criar subscrições do Azure no contrato Enterprise da sua organização, tem de ter a função de proprietário da conta para a sua organização.

Se ainda não tiver uma subscrição do Azure, crie uma [conta gratuita](https://azure.microsoft.com/free).

## <a name="create-an-additional-azure-subscription"></a>Criar uma subscrição do Azure adicional

1. Inicie sessão no [portal do Azure](https://portal.azure.com).
1. Procure **subscrições**.

   ![Captura de ecrã que mostra a pesquisa no portal para a subscrição](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. Selecione **Adicionar**.

   ![Captura de ecrã que mostra o botão Adicionar na vista de subscrições](./media/billing-create-subscription/subscription-add.png)

Também pode criar subscrições adicionais por meio de programação. Para obter mais informações, consulte [programaticamente criar subscrições do Azure Enterprise](../azure-resource-manager/programmatically-create-subscription.md).
