# <a name="overview-of-the-swift-csp-cscf-v2020-blueprint-sample"></a>Descrição geral do exemplo de esquema SWIFT CSP-CSCF v2020

O exemplo de esquema SWIFT CSP-CSCF v2020 fornece proteções de governação com o [Azure Policy](../../../policy/overview.md) que ajudam a avaliar controlos de SWIFT CSP específicos. Este esquema ajuda os clientes a implementar um conjunto essencial de políticas para qualquer arquitetura implementada no Azure que tenha de implementar controlos de SWIFT CSP.

## <a name="control-mapping"></a>Mapeamento de controlo

A secção de mapeamento de controlo fornece detalhes sobre as políticas incluídas neste esquema e sobre como estas políticas ajudam a cumprir vários controlos no mais recente SWIFT CSP-CSCF. Quando atribuídos a uma arquitetura, os recursos são avaliados pelo Azure Policy para determinar a não conformidade face às políticas atribuídas. Para obter mais informações, veja [Azure Policy](../../../policy/overview.md).
