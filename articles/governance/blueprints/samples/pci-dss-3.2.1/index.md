# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Descrição geral do exemplo de esquema PCI-DSS v3.2.1

O exemplo de esquema PCI-DSS v3.2.1 é um conjunto de políticas que ajuda a garantir a conformidade com o PCI-DSS v3.2.1. Este esquema ajuda os clientes a gerir ambientes com base na cloud com cargas de trabalho PCI-DSS.
O esquema PCI-DSS implementa um conjunto essencial de políticas para qualquer arquitetura implementada no Azure que necessite desta acreditação.

## <a name="control-mapping"></a>Mapeamento de controlo

A secção de mapeamento de controlo fornece detalhes sobre as políticas incluídas nesta iniciativa e sobre como estas políticas ajudam a cumprir vários controlos definidos pelo PCI-DSS v3.2.1. Quando atribuídos a uma arquitetura, os recursos são avaliados pelo Azure Policy para determinar a não conformidade face às políticas atribuídas.

Depois de atribuir este esquema, veja o nível de conformidade dos seus ambientes do Azure no Dashboard de Conformidade do Azure Policy.
