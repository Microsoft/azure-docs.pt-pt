# <a name="overview-of-the-media-blueprint-sample"></a>Descrição geral do exemplo de esquema Media

O exemplo de esquema Media fornece um conjunto de princípios de proteção de governação através do [Azure Policy](../../../policy/overview.md) que ajudam a obter o atestado [Media](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Amostra de esquema

Este exemplo de esquema ajuda os clientes a implementar um conjunto essencial de políticas para qualquer arquitetura implementada no Azure que necessite de acreditação ou conformidade com a arquitetura Media. A secção de [mapeamento de controlo](./control-mapping.md) fornece detalhes sobre as políticas incluídas nesta iniciativa e sobre como estas políticas ajudam a cumprir vários controlos definidos pela arquitetura Media. Quando atribuídos a uma arquitetura, os recursos são avaliados pelo Azure Policy para determinar a conformidade face às políticas atribuídas.
