# <a name="migrate-to-azure-app-service"></a>Migrar para o Serviço de Aplicações Azure

Utilizando o Assistente de Migração do Serviço de [Aplicações,](https://azure.microsoft.com/services/app-service/migration-assistant/) pode migrar a sua aplicação no local para o Serviço de Aplicações Azure. O Assistente de Migração do Serviço de Aplicações foi concebido para simplificar a sua viagem à nuvem através de uma solução gratuita, simples e rápida para migrar aplicações do local para a nuvem.

Com o Assistente de Migração do Serviço de Aplicações Azure, pode rapidamente:

- Digitalize o URL da sua aplicação para avaliar se é um bom candidato à migração
- Descarregue o Assistente de Migração para iniciar a sua migração.
- Utilize a ferramenta para executar verificações de prontidão e avaliação geral das definições de configuração da sua aplicação
- Migrar a sua aplicação ou site para o Azure App Service através da ferramenta.
