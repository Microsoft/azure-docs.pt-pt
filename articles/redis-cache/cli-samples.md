A tabela seguinte inclui ligações para scripts bash criados com a CLI do Azure.

| | |
|---|---|
|**Criar cache**||
| [Criar uma cache](./scripts/create-cache.md) | Cria um grupo de recursos e um escalão básico a Cache de Redis do Azure. |

|**Eliminar cache**||
