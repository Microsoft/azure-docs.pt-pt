# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Começar com as capacidades cognitivas da U-SQL

## <a name="overview"></a>Descrição geral
As capacidades cognitivas para o U-SQL permitem que os desenvolvedores utilizem a inteligência nos seus grandes programas de dados. 

Estão disponíveis as seguintes amostras utilizando capacidades cognitivas:
* Imagem: [Detetar rostos](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Imagem: [Detetar emoção](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Imagem: [Detetar objetos (marcação)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Imagem: [OCR (reconhecimento de caracteres óticos)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Texto: [Análise de & de sentimento de extração de frases-chave](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
