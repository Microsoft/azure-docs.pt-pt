# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Começar a utilizar as capacidades cognitivos do U-SQL

## <a name="overview"></a>Descrição geral
As funcionalidades cognitivos para U-SQL permitem aos programadores utilize intelligence colocar os seus programas de macrodados. 

Estão disponíveis as seguintes capacidades de cognitivos:
* Imagens: Detetar faces [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Imagens: Detetar emoções [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Imagens: Detetar objetos (marcação) [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Imagens: OCR (reconhecimento de caráter optical) [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Texto: Chave frase extração & análise de dados de sentimento [exemplo](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

## <a name="registering-cognitive-extensions-in-u-sql"></a>Registar cognitivos extensões em U-SQL
