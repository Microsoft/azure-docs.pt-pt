# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Comece com as capacidades cognitivas do U-SQL

## <a name="overview"></a>Descrição geral
Capacidades cognitivas para U-SQL permitem aos programadores utilizar inteligência put em seus programas de macrodados. 

As seguintes capacidades cognitivas estão disponíveis:
* Geração de imagens: Detetar rostos [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Geração de imagens: Detetar emoções [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Geração de imagens: Detetar objetos (identificação) [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Geração de imagens: OCR (reconhecimento ótico de carateres) [exemplo](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Texto: Chave extração de expressões e análise de sentimentos [exemplo](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)

## <a name="registering-cognitive-extensions-in-u-sql"></a>Registar extensões cognitivas do U-SQL
Antes de começar, siga os passos neste artigo para registar extensões cognitivas do U-SQL: [registar extensões cognitivas do U-SQL](https://msdn.microsoft.com/azure/data-lake-analytics/u-sql/cognitive-capabilities-in-u-sql#registeringExtensions).
