# <a name="quickstart-create-an-azure-cache-for-redis-instance"></a>Quickstart: Criar um Cache Azure para a instância Redis

Azure Cache para Redis fornece [redis](https://redis.io/) de código aberto totalmente geridodentro de Azure. Pode começar com um Cache Azure para Redis de qualquer nível (Básico, Standard ou Premium) e tamanho, e dimensioná-lo para atender às necessidades de desempenho da sua aplicação. Este quickstart demonstra como usar o portal Azure para criar um novo Azure Cache para Redis.

## <a name="prerequisites"></a>Pré-requisitos

Vai precisar de uma assinatura Azure antes de começar. Se não tiver uma, crie uma [conta gratuita](https://azure.microsoft.com/free/) primeiro.
