# <a name="azure-policy-regulatory-compliance-controls-for-azure-networking-services"></a>Controlos de conformidade regulamentar da Política Azure para serviços de networking Azure

[A Conformidade Regulamentar na Política Azure](../governance/policy/concepts/regulatory-compliance.md) fornece à Microsoft definições de iniciativa criadas e geridas, conhecidas como _incorporados,_ para os domínios de conformidade e **controlos** de segurança relacionados com **diferentes** padrões de conformidade. Esta página lista os **domínios de conformidade** e os **controlos** de segurança dos serviços de networking Azure. Pode atribuir os incorporados a um controlo de **segurança** individualmente para ajudar a tornar os seus recursos Azure conforme com o padrão específico.
