# <a name="feature-comparison-azure-sql-database-versus-sql-server"></a>Comparação de recursos: Banco de dados SQL do Azure versus SQL Server

O banco de dados SQL do Azure compartilha uma base de código comum com SQL Server. Os recursos do SQL Server com suporte do banco de dados SQL do Azure dependem do tipo de banco de dados SQL do Azure que você criar. Com o banco de dados SQL do Azure, você pode criar um banco de dados como parte de uma [instância gerenciada](sql-database-managed-instance.md), como um banco de dados individual ou como parte de um pool elástico.

A Microsoft continua a adicionar recursos ao banco de dados SQL do Azure. Visite a página da Web atualizações de serviço do Azure para obter as atualizações mais recentes usando estes filtros:

- Filtrado para o [serviço da Base de Dados SQL](https://azure.microsoft.com/updates/?service=sql-database).
- Filtrado para [anúncios](https://azure.microsoft.com/updates/?service=sql-database&update-type=general-availability) de Disponibilidade Geral (DG) para funcionalidades de Base de Dados SQL.

Se precisar de mais detalhes sobre as diferenças, você poderá encontrá-las nas páginas separadas para [um banco de dados individual e pools elásticos](sql-database-transact-sql-information.md) ou [instância gerenciada](sql-database-managed-instance-transact-sql-information.md).
