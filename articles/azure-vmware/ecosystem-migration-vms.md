# <a name="migration-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Soluções de migração para máquinas virtuais Azure VMware Solution (VMs)

Um dos casos de uso mais comuns para a utilização da Solução VMware Azure é a evacuação do data center.  Permite-lhe continuar a maximizar os seus investimentos em VMware, porque a Azure VMware Solution estará sempre atualizada. Além disso, pode melhorar as suas cargas de trabalho com toda a gama de serviços nativos da Azure.  Um passo fundamental inicial neste processo é a migração do seu legado ambiente baseado em VMware para a Azure VMware Solution.

Os nossos parceiros de migração têm soluções de migração líderes no setor em ambientes baseados em VMware. Os clientes de todo o mundo têm usado estas soluções para as suas migrações tanto para a Azure como para a Azure VMware Solution.

Não é obrigado a usar o VMware HCX como uma ferramenta de migração, o que significa que também pode migrar cargas de trabalho físicas para a Solução VMware Azure. Além disso, as migrações para o seu ambiente Azure VMware Solution não precisam de uma ligação ExpressRoute se não estiver disponível dentro do seu ambiente de origem.  As migrações podem ser feitas em vários locais se decidir hospedar essas cargas de trabalho em várias regiões do Azure.

Para mais informações sobre estas soluções, consulte [o RiverMeadow.](https://www.rivermeadow.com/migrating-to-vmware-on-azure)
