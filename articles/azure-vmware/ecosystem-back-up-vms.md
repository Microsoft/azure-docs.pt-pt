# <a name="backup-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Soluções de backup para máquinas virtuais Azure VMware Solution (VMs)

Um princípio chave da Azure VMware Solution é permitir-lhe continuar a utilizar os seus investimentos e as suas soluções VMware favoritas em execução no Azure. O suporte tecnológico independente do fornecedor de software (ISV), validado com a Azure VMware Solution, é uma parte importante desta estratégia. 

Os nossos parceiros de backup têm soluções de backup líderes no setor e restauram soluções em ambientes baseados em VMware. Os clientes adotaram amplamente estas soluções para as suas implementações no local. Agora estes parceiros alargaram as suas soluções para a Azure VMware Solution, utilizando o Azure para fornecer um repositório de backup e um alvo de armazenamento para retenção e arquivo a longo prazo.

O tráfego de rede de backup entre os VMs da Azure VMware Solution e o repositório de backup em Azure viaja por uma ligação de alta largura de banda e baixa latência. O tráfego de replicação através das regiões viaja através da rede interna de backplane Azure, o que reduz os custos de largura de banda para os utilizadores.
