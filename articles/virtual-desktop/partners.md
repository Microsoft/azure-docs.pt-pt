# <a name="windows-virtual-desktop-partner-integrations"></a>Integrações de parceiros de área de Trabalho Virtual do Windows

Este artigo apresenta uma lista de parceiros do Windows Virtual da área de trabalho em ordem alfabética.

## <a name="active-cypher"></a>Cifras Active Directory

Cifras Active Directory é um fornecedor de serviços de valor acrescentado de área de Trabalho Virtual do Windows que dota IT com a tecnologia de encriptação para ajudar a proteger a empresa contra violações e perda de dados. Com Cypher Active Directory, a tecnologia criptografia inovadoras, oferece suporte a estratégia geral para a proteção de ficheiros para executar a área de Trabalho Virtual do Windows no Azure.

Quando utilizado com a área de Trabalho Virtual do Windows, Active Directory Cypher fornece capacidades adicionais para ativar a acessibilidade e a mobilidade de dados de utilizador, enquanto impede a perda de dados acidentais. Com Cypher Active Directory, as políticas de segurança de grupo existentes e as permissões no Azure Information Protection podem ser utilizadas para encriptar os ficheiros de utilizador ao utilizar a área de Trabalho Virtual do Windows.

[Aceda ao site de parceiro.](https://activecypher.com/)

## <a name="cloudjumper"></a>CloudJumper

CloudJumper é um provedor de valor agregado de serviços de área de Trabalho Virtual do Windows que dota os fornecedores de soluções e TI empresariais com o software holística aprovisionar e gerir ambientes de área de Trabalho Virtual do Windows. Com o software de CloudJumper, o departamento de TI pode gerir todas as camadas de uma implantação de área de Trabalho Virtual do Windows. Entrega de aplicações e cargas de trabalho é automatizada, garantindo que os utilizadores podem aceder rapidamente sua área de trabalho em qualquer lugar em qualquer dispositivo.

Software do CloudJumper, área de trabalho de Cloud Management Suite expande o valor de área de Trabalho Virtual do Windows, ao simplificar a implantação e tarefas de administração contínua no Azure. A partir de um único painel de vidro, profissionais de TI podem aprovisionar, gerir e otimizar a infra-estrutura para áreas de trabalho do utilizador. Simples o Acionamento do mecanismo do CloudJumper de Script integra-se a plataformas de serviços IT para automatizar tarefas envolvidas na área de Trabalho Virtual do Windows de aprovisionamento. Além disso, as APIs de CloudJumper permitem mais extensibilidade e integração com outros sistemas empresariais, como o ServiceNow e BMC pronto.

[Aceda ao site de parceiro.](https://cloudjumper.com/wvd/)

## <a name="lakeside-software"></a>Lakeside Software

Software de Lakeside é um fornecedor de serviços de valor acrescentado de área de Trabalho Virtual do Windows que dota as equipas de TI com o software para monitorizar o desempenho e avaliar a preparação da migração do Azure de cargas de trabalho do utilizador. Com este software, os ganhos IT visibilidade mais clara sobre o consumo de recursos e a utilização da aplicação para simplificar o processo de migração. Lakeside Software recolhe dados em cada área de trabalho para criar um relatório abrangente em ambientes de usuário, permitindo a rápida resolução de problemas e otimização de ativos.

Experiência de digital do Software Lakeside solução, SysTrack, de monitorização pode ajudar a proporcionar uma experiência de utilizador fantástica ao controlar o desempenho e identificar cargas de trabalho ideais para a migração. SysTrack funciona para estender o valor de área de Trabalho Virtual do Windows através de avaliações de redimensionamento e monitorização contínua de ambientes de usuário.

[Aceda ao site de parceiro.](https://www.lakesidesoftware.com/assessments/wvd)

## <a name="liquidware"></a>Liquidware

Liquidware é um provedor de valor agregado de serviços de área de Trabalho Virtual do Windows que fornece software que gere e otimiza a implantação de área de Trabalho Virtual do Windows. O conjunto de Liquidware Essentials fornece a entrega de aplicativos por meio de disposição em camadas, gestão de ambiente de usuário e visibilidade de experiência de utilizador de chave e diagnósticos. Com as soluções para avaliar a preparação da migração e analisar as métricas de utilização, Liquidware fornece uma experiência de área de trabalho virtual totalmente integrada para os utilizadores finais.

Liquidware Essentials estende o valor da área de Trabalho Virtual do Windows, com eficiência coleta perfis de utilizador e a recolha de dados de chave do utilizador para simplificar a migração de ambientes de usuário para o Azure. Além disso, Liquidware Essentials simplifica o gerenciamento de imagens ao unificar os perfis de utilizador e aplicações de disposição em camadas com base nos direitos configuráveis as definições de gestão.

[Aceda ao site de parceiro.](https://www.liquidware.com/solutions/solutions-platform/microsoft)

## <a name="nerdio"></a>Nerdio

Nerdio é uma plataforma de automação de TI do Azure torna mais fácil de implementar e gerir a área de Trabalho Virtual do Windows. Nerdio fornece o conhecimento e tecnologia para implementar, preço, empacotar, gerir e otimizar as implementações de clientes do Azure — com área de Trabalho Virtual do Windows frente e no centro.

Nerdio expande o valor de área de Trabalho Virtual do Windows, tornando fácil de aprovisionar recursos do Azure e simplificar a implantação. Com Nerdio para Azure, profissionais de TI podem implementar e gerir um ambiente do Azure completo, incluindo a área de Trabalho Virtual de Windows, em duas horas automaticamente.

[Aceda ao site de parceiro.](https://getnerdio.com/academy/windows-virtual-desktop-explained-for-msps/)

## <a name="thinprint"></a>ThinPrint

ThinPrint é um provedor de valor agregado de serviços de área de Trabalho Virtual do Windows que oferece a cloud simple e segura, impressão a partir de área de Trabalho Virtual do Windows. Com os seus serviços e software, a infraestrutura de impressa existente pode ser utilizada para imprimir documentos a partir da cloud. ThinPrint permite a ligação para ambos os locais e impressoras de rede, tornando mais fácil para os utilizadores imprimam enquanto estão no escritório ou de trabalho remoto.

Solução de ezeep do ThinPrint expande o valor de área de Trabalho Virtual do Windows, permitindo a conexão com a infraestrutura de impressão empresarial existente. ezeep proporciona aos utilizadores controlo sobre impressão na empresa onde quer que estejam. Utilizar ezeep, os utilizadores podem preencher a lacuna entre a área de Trabalho Virtual do Windows e o hardware de impressão.

[Aceda ao site de parceiro.](http://www.ezeep.com/wvd-printing)

## <a name="workspot"></a>Workspot

Workspot é um provedor de valor agregado de serviços de área de Trabalho Virtual do Windows que dota as empresas com áreas de trabalho de elevado desempenho e estações de trabalho no Azure. Com Workspot, provisionamento da infraestrutura é automatizado, que significa que os usuários podem acessar seus ambientes de área de Trabalho Virtual do Windows a partir de qualquer lugar em todo o mundo com elevada disponibilidade.

Workspot expande o valor de área de Trabalho Virtual do Windows, ao simplificar o processo de aprovisionamento da infraestrutura de ambiente de trabalho de nuvem. Com Workspot, recursos podem ser facilmente dimensionados e reduzir verticalmente para satisfazer as necessidades de diferentes casos de utilizadores e os utiliza. Workspot pode otimizar as implementações de alto desempenho GPU estações de trabalho necessárias para CAD e engenharia utilizadores, bem como aplicativos do Windows e áreas de trabalho do Windows 10 para todos os utilizadores empresariais.

[Aceda ao site de parceiro.](https://www.workspot.com/wvd)
