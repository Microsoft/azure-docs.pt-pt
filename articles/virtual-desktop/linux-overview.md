# <a name="thin-client-support"></a>Suporte fino do cliente

Pode aceder aos recursos de Desktop Virtual do Windows a partir dos seus dispositivos de cliente fino com o [cliente web](connect-web.md) ou os seguintes clientes suportados, fornecidos pelos nossos parceiros. Estamos a trabalhar com vários parceiros para permitir clientes de Desktop Virtual do Windows suportados noutras plataformas.

## <a name="connect-with-your-thin-client-device"></a>Conecte-se com o seu dispositivo de cliente fino

Os seguintes parceiros aprovaram clientes do Windows Virtual Desktop.

|Parceiro|Documentação do parceiro|Apoio a parceiros|
|:------|:--------------------|:--------------|
|10ZiG |[Documentação do cliente 10ZiG](https://www.10zig.com/about/microsoft-windows-virtual-desktop)|[Suporte 10ZiG](https://www.10zig.com/resources/support_faq)|
|Dell |[Documentação do cliente de Dell](https://www.delltechnologies.com/en-us/collaterals/unauth/data-sheets/products/thin-clients/dell-thinos-9-for-microsoft-wvd.pdf)|[Suporte de Dell](https://www.dell.com/support)|
|HP |[Documentação do cliente HP](https://h20195.www2.hp.com/v2/GetDocument.aspx?docname=c07051097)|[Suporte HP](https://support.hp.com/us-en/products/workstations-thin-clients)|
|IGEL |[Documentação do cliente IGEL](https://www.igel.com/igel-solution-family/windows-virtual-desktop/)|[Suporte IGEL](https://www.igel.com/support/)|
|NComputing |[Documentação do cliente NComputing](https://www.ncomputing.com/microsoft)|[Suporte NComputing](https://www.ncomputing.com/support/support-options)|
|Estratodesco |[Documentação do cliente de Stratodesk](https://www.stratodesk.com/kb/Microsoft_Windows_Virtual_Desktop_(WVD))|[Apoio a Stratodesk](https://www.stratodesk.com/support/)|
