# <a name="additional-documentation"></a>Documentação adicional 

Alguns recursos fora deste site docs.microsoft.com podem ajudá-lo a entender e trabalhar com a cache do filtro de borda do Microsoft Azure FXT híbrida.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT para documentação da cache do Azure

[Avere vFXT para o Azure](https://docs.microsoft.com/azure/avere-vfxt/) é um produto de cache de cluster semelhante que utiliza nós virtuais no Microsoft Azure. Ele é criado com o portal do Azure, mas utiliza o mesmo software de configuração do painel de controle como a cache de hardware de ficheiros do Azure FXT Edge. 

## <a name="legacy-hardware-documentation"></a>Documentação do hardware herdado 

Essa tecnologia foi utilizada anteriormente em produtos de sistemas de Avere. Documentação do utilizador para esses produtos mais antigos está disponível na [site de documentação de legado Avere](https://azure.github.io/Avere/). 

Especificamente, estes documentos podem ter detalhes úteis:

* [Guia de configuração de cluster](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) – inclui informações de descrição geral sobre tarefas comuns de configuração e o gerenciamento de cluster e tem detalhadas explicações das páginas de definições do cluster 

* [Guia de dashboard](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) -explica as funcionalidades do painel de controlo **Dashboard** separador

* [Guia de criação do Cluster FXT](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) -guia de criação do Cluster dos produtos anteriores
