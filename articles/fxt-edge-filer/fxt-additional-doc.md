# <a name="additional-documentation"></a>Documentação adicional 

Alguns recursos fora deste site docs.microsoft.com podem ajudá-lo a entender e trabalhar com a sua cache híbrida Microsoft Azure FXT Edge Filer.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT para documentação de cache Azure

[Avere vFXT para Azure](https://docs.microsoft.com/azure/avere-vfxt/) é um produto de cache de cluster semelhante que utiliza nós virtuais no Microsoft Azure. É criado com o portal Azure, mas utiliza o mesmo software de configuração de painel de controlo que o cache de hardware do Ficheiro de Borda Azure FXT. 

## <a name="legacy-hardware-documentation"></a>Documentação de hardware legado 

Esta tecnologia foi usada anteriormente em produtos da Avere Systems. A documentação do utilizador para os produtos mais antigos está disponível no site da [Avere Legacy Documentation.](https://azure.github.io/Avere/) 

Especificamente, estes documentos podem ter detalhes úteis:

* [Guia de Configuração](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) de Cluster - Inclui informações sobre tarefas comuns de configuração e gestão de clusters, e tem explicações detalhadas das páginas de configurações do cluster 

* Guia do Painel de [Instrumentos](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - Explica as características do **separador painel** de controlo Dashboard

* [FXT Cluster Creation Guide](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Guia de criação de clusters de produtos anteriores
