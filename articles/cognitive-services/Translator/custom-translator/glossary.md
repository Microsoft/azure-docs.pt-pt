# <a name="custom-translator-glossary"></a>Glossário tradutor personalizado

O glossário [custom Tradutor](https://portal.customtranslator.azure.ai) explica termos que pode encontrar enquanto trabalha com o Tradutor Personalizado.

| **Palavra ou Frase**       | **Definição**                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Língua fonte          | A linguagem fonte é a linguagem com a que está a começar e quer converter-se para outra língua (o "alvo").                                                                                                                                                                                                                                                                                                                                                         |
| Língua alvo          | A linguagem-alvo é a linguagem que pretende que a tradução da máquina forneça depois de receber a língua fonte.                                                                                                                                                                                                                                                                                                                                               |
| Arquivo monolingual         | Um ficheiro monolingual tem uma única linguagem que não é emparelhada com outro arquivo de uma língua diferente.                                                                                                                                                                                                                                                                                                                                                                 |
| Arquivos Paralelos           | Um ficheiro paralelo é a combinação de dois ficheiros com texto correspondente. Um ficheiro tem a linguagem de origem. O outro tem a linguagem alvo.                                                                                                                                                                                                                                                                                                                                         |
| Alinhamento de frases       | O conjunto de dados paralelos deve ter frases alinhadas com frases que representam o mesmo texto em ambas as línguas. Por exemplo, num ficheiro paralelo de fonte, a primeira frase deve, em teoria, mapear a primeira frase no ficheiro paralelo alvo.                                                                                                                                                                                                                               |
| Texto alinhado             | Um dos passos mais importantes de validação de ficheiros é alinhar as frases nos documentos paralelos. As coisas são expressas de forma diferente em diferentes línguas. Também línguas diferentes têm ordens de palavras diferentes. Este passo faz o trabalho de alinhar as frases com o mesmo conteúdo para que possam ser usadas para a formação. Um alinhamento de frases baixas indica que pode haver algo errado com um ou ambos os ficheiros. |
| Quebra de palavras/ Inquebrando | Quebrar palavras é a função de marcar os limites entre as palavras. Muitos sistemas de escrita usam um espaço para denotar a fronteira entre as palavras. Palavra ininterrupta refere-se à remoção de qualquer marcador visível que possa ter sido inserido entre palavras num passo anterior.                                                                                                                                                                                                  |
| Delimitadores               | Delimitadores são as formas como uma frase é dividida em segmentos ou delimitam a margem entre frases. Por exemplo, nos espaços ingleses delimitam as palavras, os cólons e os semi-cólons delimitam cláusulas e os períodos delimitam as frases.                                                                                                                                                                                                                                         |
| Arquivos de Formação           | Um ficheiro de formação é usado para ensinar o sistema de tradução automática a mapear de uma língua (a fonte) para uma língua-alvo (o alvo). Quanto mais dados puder fornecer, melhor será o desempenho do sistema na tradução.                                                                                                                                                                                                               |
| Arquivos de Afinação             | Estes ficheiros são frequentemente derivados aleatoriamente do conjunto de treino (se não selecionar qualquer conjunto de afinação). As frases selecionadas automaticamente são utilizadas para afinar o sistema e certificar-se de que está a funcionar corretamente. Caso decida criar os seus próprios ficheiros Tuning, certifique-se de que são um conjunto aleatório de frases em domínios se pretender criar um modelo de tradução para fins gerais.                                                                                 |
| Arquivos de Teste            | Estes ficheiros são frequentemente ficheiros derivados, selecionados aleatoriamente a partir do conjunto de treino (se não selecionar qualquer conjunto de teste). O objetivo destas frases é avaliar a exatidão do modelo de tradução. Estas são frases que pretende certificar-se de que o sistema traduz com precisão. Assim, pode querer criar um conjunto de testes e carregá-lo ao tradutor para garantir que estas frases são usadas na avaliação do sistema (a geração de uma pontuação BLEU).   |
| Arquivo combo               | Um tipo de ficheiro em que a fonte e as frases traduzidas estão contidas no mesmo ficheiro. Formatos de ficheiros suportados (".tmx", ".xliff", ".xlf", ".lcl", ".xlsx").                                                                                                                                                                                                                                                                                                                       |
| Arquivo             | Um ficheiro que contém outros ficheiros. Formatos de ficheirosuportados (zip, gz, tgz).                                                                                                                                                                                                                                                                                                                                                                                                |
| Classificação de BLEU               | [A BLEU](what-is-bleu-score.md) é o método padrão da indústria para avaliar a "precisão" ou precisão do modelo de tradução. Embora existam outros métodos de avaliação, o Microsoft Tradutor conta com o método BLEU para reportar a precisão aos Proprietários do Projeto.
