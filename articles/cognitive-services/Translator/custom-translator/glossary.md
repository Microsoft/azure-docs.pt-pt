# <a name="glossary"></a>Glossário

O [Translator personalizado](https://portal.customtranslator.azure.ai) glossário explica certos termos que poderá encontrar ao trabalhar com o tradutor personalizado.

| **Palavra ou expressão**       | **Definição**                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Idioma de origem          | O idioma de origem é o idioma estiver a iniciar e pretende converter em outro idioma (o "destino").                                                                                                                                                                                                                                                                                                                                                         |
| Idioma de Destino          | O idioma de destino é o idioma que pretende que a tradução automática para fornecer depois de receber o idioma de origem.                                                                                                                                                                                                                                                                                                                                               |
| Ficheiro monolíngüe         | Um ficheiro monolíngüe tem um único idioma que não se encontra emparelhado com outro ficheiro de um idioma diferente.                                                                                                                                                                                                                                                                                                                                                                 |
| Ficheiros paralelos           | Um arquivo paralelo é a combinação de dois ficheiros com o texto correspondente. Um ficheiro tem o idioma de origem. O outro tem o idioma de destino.                                                                                                                                                                                                                                                                                                                                         |
| Alinhamento de frase       | Conjunto de dados paralelo tem alinhado frases para as frases que representam o mesmo texto em ambas as linguagens. Por exemplo, num arquivo paralelo de origem a primeira frase deve, em teoria, mapear para a primeira frase no arquivo paralelo de destino.                                                                                                                                                                                                                               |
| Texto alinhado             | Uma das etapas mais importantes de validação do arquivo é alinhar as frases nos documentos paralelas. As coisas são expressas de forma diferente em diferentes idiomas. Também diferentes idiomas têm diferentes ordens de word. Este passo faz o trabalho de alinhar as frases com o mesmo conteúdo, para que eles podem ser usados para treinamento. Um alinhamento de sentença baixa indica que poderá haver um problema com um ou ambos os ficheiros. |
| Word recentes / Unbreaking | Separação de palavras é a função de marcar os limites entre as palavras. Muitos sistemas de escrita utilizam um espaço para denotar o limite entre palavras. Word unbreaking refere-se para a remoção de qualquer marcador visível que foi inserido entre as palavras num passo anterior.                                                                                                                                                                                                  |
| Delimitadores               | Delimitadores são as formas que uma frase é dividida em segmentos ou delimitar a margem entre frases. Por exemplo, nos espaços em inglês delimitar palavras, vírgula, ponto e vírgula delimitar cláusulas e períodos delimitar frases.                                                                                                                                                                                                                                         |
| Arquivos de treinamento           | Um ficheiro de treinamento é utilizado para ensinar o sistema de tradução automática como mapear de um idioma (a origem) para um idioma de destino (o destino). Quanto mais dados que pode fornecer melhor o sistema irá efetuar a tradução.                                                                                                                                                                                                               |
| Otimização de ficheiros             | Estes ficheiros aleatoriamente, muitas vezes, são derivados de conjunto (se não selecionar qualquer conjunto de otimização) de preparação. O autoselected frases são usados para otimizar o sistema e certifique-se de que está a funcionar corretamente. Caso decida criar seus próprios arquivos de ajuste, certifique-se de que são um conjunto aleatório de frases entre domínios se pretender criar um modelo de tradução para fins gerais.                                                                                 |
| Teste de ficheiros            | Estes ficheiros, muitas vezes, são ficheiros derivados, selecionados aleatoriamente entre o conjunto de treinamento (se não selecionar qualquer conjunto de teste). A finalidade dessas frases é avaliar exatidão o modelo de tradução. Estas são as frases que pretende certificar-se de que o sistema se traduz com precisão. Portanto, pode pretender criar um conjunto de teste e carregue-o para o Microsoft translator para garantir que essas frases são utilizadas no período de avaliação do sistema (a geração de uma pontuação de BLEU).   |
| Ficheiro de combinação               | Um tipo de ficheiro no qual a origem e de frases traduzidas por estão contidas no mesmo ficheiro. Formatos de ficheiro suportados (".tmx", ".xliff", "xlf", ".lcl", ". xlsx").                                                                                                                                                                                                                                                                                                                       |
| Ficheiro de arquivo             | Um ficheiro que contém outros ficheiros. Suporte de formatos de arquivo (zip, gz, tgz).                                                                                                                                                                                                                                                                                                                                                                                                |
| Classificação de BLEU               | [BLEU](what-is-bleu-score.md) é o método padrão do setor para avaliar a "precisão" ou a precisão do modelo de tradução. Apesar de existirem outros métodos de avaliação, o Microsoft Translator baseia-se método BLEU a exatidão do relatório para os proprietários do projeto.
