# <a name="quickstart-use-the-translator-text-api-to-transliterate-text"></a>Quickstart: Use a API de texto tradutor para transliterar texto

Neste arranque rápido, aprenderá a transliterar (converter) texto de um script para outro usando a API de Repouso de Texto tradutor. No exemplo apresentado, o japonês é transliterado para utilizar o alfabeto latino.

Este início rápido requer uma [conta dos Serviços Cognitivos do Azure](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) com um recurso de Tradução de Texto. Se não tiver uma conta, pode utilizar a [avaliação gratuita](https://azure.microsoft.com/try/cognitive-services/) para obter uma chave de subscrição.
