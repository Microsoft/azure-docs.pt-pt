# <a name="how-to-sign-up-for-the-translator-text-api"></a>Como inscrever-se para a API de texto tradutor

## <a name="sign-in-to-the-azure-portal"></a>Iniciar sessão no portal do Azure

- Não tem uma conta? Pode criar um [conta gratuita](https://azure.microsoft.com/free/) para testar, sem encargos.
- Já tem uma conta? [A iniciar sessão](https://ms.portal.azure.com/)

## <a name="create-a-subscription-to-the-translator-text-api"></a>Criar uma subscrição para a API de texto tradutor

Depois de iniciar sessão no portal, pode criar uma subscrição para a API de texto tradutor da seguinte forma:

1. Selecione **+ Criar um recurso**.
1. No **procurar no Marketplace** caixa de pesquisa, introduza **tradutor texto** e, em seguida, selecione-o na lista de resultados.
1. Selecione **criar** definir detalhes da subscrição.
1. Do **escalão de preço** lista, selecione o escalão de preço que melhor se adequa às suas necessidades.
    1. Cada subscrição tem um escalão gratuito. O escalão gratuito tem as mesmas funções e funcionalidades como os planos pagas e não expira.
    1. Pode ter apenas uma subscrição gratuita para a sua conta.
1. Selecione **criar** para concluir a criação da subscrição.

## <a name="authentication-key"></a>Chave de autenticação

Quando se inscreve tradutor texto, receberá uma chave de acesso personalizado exclusivo para a sua subscrição. Esta chave é necessária em cada chamada à API de texto do conversor.

1. Obter a sua chave de autenticação, primeiro, selecionando a subscrição adequada.
1. Selecione **chaves** no **gestão de recursos** secção de detalhes da sua subscrição.
1. Copie qualquer uma das chaves listadas para a sua subscrição.
