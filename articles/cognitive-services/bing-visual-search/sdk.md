# <a name="bing-visual-search-sdk"></a>SDK de pesquisa Visual do Bing

Pesquisa Visual do Bing fornece os SDKs que pode utilizar para testar a API de pesquisa Visual do cliente seguinte.

* [Exemplos .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pacote NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Bibliotecas .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) para dependências e definições
* [Amostras de Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Bibliotecas de node. js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) para dependências e definições
* [Exemplos de Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Bibliotecas Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) para dependências e definições
* [Amostras de Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Bibliotecas Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) para dependências e definições

Amostras SDK para cada idioma incluem um arquivo Leiame com pré-requisitos para utilizar o SDK juntamente com os passos para instalar e executar os exemplos.

Para esses exemplos, terá de iniciar uma subscrição no escalão de preço S9, conforme mostrado na [Cognitive Services preços - API de pesquisa Bing](https://azure.microsoft.com/en-us/pricing/details/cognitive-services/search-api/). 

Para iniciar uma subscrição no Portal do Azure:
1. Introduza "BingSearchV7" na caixa de texto na parte superior do Portal do Azure que diz `Search resources, services, and docs`.  
2. Em Marketplace na lista pendente, selecione `Bing Search v7`.
3. Introduza `Name` do novo recurso.
4. Selecione `Pay-As-You-Go` subscrição.
5. Selecione `S9` escalão de preço.
6. Clique em `Enable` para iniciar a subscrição.

### <a name="sdk-tools"></a>Ferramentas do SDK
Ferramentas do SDK para explorar as capacidades de pesquisa Visual do Bing podem ser encontradas neste repositório de Git: [ferramentas de pesquisa Visual do Bing](https://github.com/Azure-Samples/bing-visualsearch-tools)
