# <a name="how-to-sign-up-and-upload-your-first-video"></a>Como inscrever-se e carregar o seu primeiro vídeo

Este tutorial de introdução curto mostra como iniciar sessão para o Video Indexer e como carregar o primeiro vídeo.

> [!Note]
> Ao criar uma conta do Video Indexer, pode escolher uma conta de avaliação gratuita (onde pode obter um determinado número de minutos de indexação gratuitos) ou uma opção paga (em que não está limitado pela quota). <br/>Com a avaliação gratuita, Video Indexer fornece até 600 minutos de indexação gratuito para os utilizadores do Web site e até 2400 minutos de gratuitos de indexação para os utilizadores de API. <br/>Com uma opção paga, cria uma conta do Video Indexer que seja [ligado à sua subscrição do Azure e uma conta de Media Services do Azure](connect-to-azure.md). Paga minutos indexados, bem como a conta de suporte de dados relacionados com custos. 

## <a name="sign-up-and-sign-in"></a>Inscrever-se e iniciar sessão

Para começar a desenvolver com o indexador de vídeo, deve primeiro inscrever e início de sessão para o [Video Indexer](https://api-portal.videoindexer.ai/) portal. 
     
Para obter mais informações, consulte [começar a utilizar APIs](video-indexer-use-apis.md).

## <a name="upload-a-video-using-the-video-indexer-portal"></a>Carregar um vídeo com o portal do Video Indexer

1. Inicie sessão no [https://www.videoindexer.ai/](https://www.videoindexer.ai/)
2. Para carregar o vídeo, prima o botão de carregamento ou uma ligação.

 ![Carregar](./media/video-indexer-get-started/video-indexer-upload.png)

Quando o vídeo foi carregado, o indexador de vídeos inicia indexação e analisar o vídeo.

![Carregado](./media/video-indexer-get-started/video-indexer-uploaded.png) 

Depois de fazer o Video Indexer, analisar, receberá uma notificação com uma ligação para o seu vídeo e uma breve descrição do que foi encontrada no seu vídeo. Por exemplo: as pessoas, tópicos, OCRs.

## <a name="next-steps"></a>Passos Seguintes

Agora, pode utilizar o [portal do Video Indexer](video-indexer-view-edit.md) ou o [API](video-indexer-use-apis.md) para ver as informações do vídeo.
