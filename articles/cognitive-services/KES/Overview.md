# <a name="knowledge-exploration-service"></a>Serviço de Exploração do Conhecimento

Bem-vindo ao Serviço de Exploração de Conhecimento da Microsoft.  O Serviço de Exploração de Conhecimento (KES) permite adicionar pesquisa interativa e refinamento às aplicações de forma rápida e eficaz.  Com o KES, pode criar um índice comprimido a partir de dados estruturados, criar uma gramática que interpreta consultas em linguagem natural e permite formular consultas interativas com sugestões de preenchimento automático.

Veja a [Introdução](GettingStarted.md) para obter um exemplo detalhado sobre como utilizar o KES para criar uma interface de pesquisa interativa para um domínio de publicações académicas.

![PreenchimentoAutomáticoAcadémico](AutoComplete.png)

O KES foi movido da Pré-visualização de Serviços Cognitivos para os Laboratórios dos Serviços Cognitivos. A nova home page do projeto é [http://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration](https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
