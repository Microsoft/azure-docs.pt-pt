## <a name="prerequisites"></a>Pré-requisitos

* Utilize uma assinatura Azure. Se não tiver uma subscrição do Azure, crie uma [conta gratuita][free-account] antes de começar.
* Instale o [Azure CLI][azure-cli] (az).
* [Docker motor][docker-engine] e validar que o Docker CLI funciona numa janela de consola.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: /cli/azure/install-azure-cli
[docker-engine]: https://www.docker.com/products/docker-engine
