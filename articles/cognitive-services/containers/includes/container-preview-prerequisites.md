## <a name="prerequisites"></a>Pré-requisitos

* Utilize uma subscrição do Azure. Se não tiver uma subscrição do Azure, crie uma [conta gratuita][free-account] antes de começar.
* Instalar o [CLI do Azure][azure-cli] (az).
* [Motor do docker][docker-engine] e validar que a CLI do Docker funciona numa janela de consola.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
