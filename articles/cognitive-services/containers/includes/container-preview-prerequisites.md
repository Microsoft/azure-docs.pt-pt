## <a name="prerequisites"></a>Pré-requisitos

* Use uma assinatura do Azure. Se não tiver uma subscrição do Azure, crie uma [conta gratuita][free-account] antes de começar.
* Instale o [CLI do Azure][azure-cli] (AZ).
* O [mecanismo do Docker][docker-engine] e valida que a CLI do Docker funciona em uma janela de console.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
