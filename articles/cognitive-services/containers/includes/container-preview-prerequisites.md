## <a name="prerequisites"></a>Pré-requisitos

* Utilize uma assinatura Azure. Se não tiver uma subscrição Azure, crie uma [conta gratuita][free-account] antes de começar.
* Instale o [Azure CLI][azure-cli] (az).
* [Docker motor][docker-engine] e valide que o Docker CLI funciona numa janela de consola.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
