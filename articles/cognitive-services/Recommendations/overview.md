# <a name="recommendations-api-discontinued"></a>API de Recomendações (descontinuada)

A pré-visualização da API de Recomendações já não está em permanente desenvolvimento e foi oficialmente extinta no dia 15 de fevereiro. 

Para uma alternativa, considere o novo código-fonte aberto [modelo Solução de Recomendações](https://aka.ms/recopcs), que permite alojar um motor de recomendações na sua própria subscrição do Azure. Suporta os principais cenários da API de Recomendações, além de fornecer também a liberdade para implementar em qualquer região do Azure e controlar mais estreitamente os recursos informáticos utilizados para preparar modelos e disponibilizar recomendações. 

Saiba mais sobre o modelo Solução de Recomendações ao ler [a documentação](https://github.com/Microsoft/Product-Recommendations).
