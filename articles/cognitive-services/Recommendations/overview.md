# <a name="recommendations-api-discontinued"></a>Recomendações de API (descontinuado)

A pré-visualização de API de recomendações já não está em desenvolvimento Active Directory e foi oficialmente extinguida em 15 de Fevereiro. 

Para uma alternativa, considere o novo open source [modelo de solução de recomendações](http://aka.ms/recopcs), que lhe permite alojar um motor de recomendações na sua própria subscrição do Azure. Suporta os cenários principais da API recomendações, enquanto também que lhe confere a liberdade para implementar em qualquer região do Azure e mais controlam rigorosamente os recursos informáticos utilizados para modelos de formação e fornecer recomendações. 

Saiba mais sobre o modelo de solução de recomendações o lendo [a documentação](https://github.com/Microsoft/Product-Recommendations).
