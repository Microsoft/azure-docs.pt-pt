# <a name="project-url-preview-use-and-display-requirements"></a>Utilização de pré-visualização do URL do projeto e apresentar requisitos

Utilize apenas os dados de pré-visualização do URL de projeto para apresentar os fragmentos de pré-visualização e imagens em miniatura com hiperligação aos respetivos sites de origem, no URL iniciada pelo utilizador final partilha em de redes sociais, chat bot ou ofertas semelhantes. Não deve copiar, armazenar ou colocar em cache todos os dados recebidos da pré-visualização do URL de projeto. Deve honrar quaisquer pedidos para desativar pré-visualizações poderá a receber do Web site ou os proprietários de conteúdo.
