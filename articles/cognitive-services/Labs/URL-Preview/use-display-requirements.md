# <a name="project-url-preview-use-and-display-requirements"></a>Requisitos de exibição e uso da versão prévia da URL do projeto

Use apenas os dados da visualização da URL do projeto para exibir trechos de código de visualização e imagens em miniatura hiperlinks para seus sites de origem, no compartilhamento de URL iniciada pelo usuário em mídia social, bot de bate-papo ou ofertas semelhantes. Você não deve copiar, armazenar ou colocar em cache todos os dados recebidos da versão prévia da URL do projeto. Você deve honrar as solicitações para desabilitar as visualizações que você pode receber do site ou dos proprietários do conteúdo.
