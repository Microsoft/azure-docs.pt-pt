Você precisará de uma chave e ponto final de um recurso Text Analytics para ligar a sua aplicação. Obtenha-os criando um dos recursos abaixo:

* <a href="https://azure.microsoft.com/try/cognitive-services/my-apis/?api=text-analytics" title="Criar um recurso experimental de <span class="docon docon-navigate-external x-hidden-focus"></span> análise de texto" target="_blank">Criar um recurso experimental</a>
    * Nenhuma assinatura do Azure é necessária. 
    * Válido por sete dias, gratuitamente. Depois de se inscrever, uma chave de avaliação e um ponto de extremidade estarão disponíveis no [site do Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/). 
    * Essa é uma ótima opção se você quiser experimentar a API de Análise de Texto, mas não tem uma assinatura do Azure.
    
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesTextAnalytics"  title="Criar um recurso de Análise <span class="docon docon-navigate-external x-hidden-focus"></span> de Texto"  target="_blank">Criar um recurso de Análise de Texto</a>
    * Disponível até que você exclua o recurso.
    * Use o tipo de preço gratuito para experimentar o serviço e atualizar mais tarde para uma camada paga para produção.
     
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne"  title="Criar um recurso para múltiplos serviços <span class="docon docon-navigate-external x-hidden-focus"></span>" target="_blank">Criar um recurso multi-serviço</a>
    * Disponível até que você exclua o recurso.  
    * Use a mesma chave e ponto de extremidade para acessar vários serviços cognitivas.
