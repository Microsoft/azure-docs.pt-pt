# <a name="microsoft-speech-client-samples"></a>Exemplos de reconhecimento de voz de cliente da Microsoft

Serviço de reconhecimento de voz da Microsoft fornece exemplos de ponto-a-ponto que mostra como utilizar a API de reconhecimento de voz Microsoft em casos de utilização diferentes, por exemplo reconhecimento de comando, reconhecimento contínuo e deteção de intenção. Todas as amostras estão disponíveis no GitHub e pode ser transferidas pelas seguintes ligações: O README.md em cada repositório, bem como o [bibliotecas de cliente](GetStarted/GetStartedClientLibraries.md) página fornecem detalhes sobre como criar e executar os exemplos.

- [Exemplos de REST API](https://github.com/Azure-Samples/SpeechToText-REST)

- [Exemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Exemplos de cliente de ambiente de trabalho c#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Exemplos de serviço do c#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Java amostras Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC amostras de iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos os Microsoft cognitivos serviços SDKs e amostras são licenciadas com a licença MIT. Para obter mais informações, consulte [licença](https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript/blob/master/LICENSE.md).
