# <a name="bing-speech-client-samples"></a>Exemplos de cliente Fala do Bing

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-bing-speech-api-deprecation-note.md)]

O Fala do Bing fornece amostras de ponta a ponta para demonstrar diferentes casos de uso, por exemplo, reconhecimento de comando, reconhecimento contínuo e detecção de intenção. Todos os exemplos estão disponíveis no GitHub e podem ser baixados pelos links a seguir: O README.md em cada repositório, bem como a página [bibliotecas de cliente](GetStarted/GetStartedClientLibraries.md) , fornece detalhes sobre como criar e executar os exemplos.

- [Exemplos de API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Exemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [C#Exemplos de cliente de desktop](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [C#exemplos de serviço](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Exemplos de Java em Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC em exemplos do iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos os exemplos e SDKs dos Serviços Cognitivos da Microsoft estão licenciados com a Licença do MIT. Para obter mais informações, consulte [License](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE).
