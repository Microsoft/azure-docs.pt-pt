# <a name="natural-language-support-for-azure-cognitive-services"></a>Suporte de linguagem natural para os serviços cognitivos do Azure

Serviços cognitivos do Azure permite-lhe criar aplicações que ver, ouvem, falam com e compreender seus usuários. Entre estes serviços, são suportadas mais de três dezenas de linguagens, permitindo que os usuários comunicar com a sua aplicação de forma natural.

Este artigo é dividido em duas seções. O primeiro apresenta uma lista de idiomas de núcleo normalmente suportados em todos os serviços cognitivos do Azure. A segunda seção se concentra no suporte de idiomas adicionais para cada um dos serviços. Disponibilidade de linguagem pode variar por país/região e o mercado.

## <a name="core-languages"></a>Idiomas principais

Essas linguagens core são suportadas em todos os serviços cognitivos do Azure:

* Chinês
* Português
* Francês
* Alemão
* Italiano
* Japonês
* ¹ coreano
* Português
* Espanhol

> [!NOTE]
> ¹ LUIS e Video Indexer: Não há suporte para a conversão de fala em texto.

## <a name="additional-language-availability-by-service"></a>Disponibilidade de idioma adicionais por serviço

Estas tabelas realçam a disponibilidade de idioma por categoria de serviço; idiomas de núcleo excluídos. Use os links abaixo para saber mais sobre o suporte de idiomas adicionais e a disponibilidade de país/região e o mercado.
