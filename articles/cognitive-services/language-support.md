# <a name="natural-language-support-for-azure-cognitive-services"></a>Suporte a idioma natural para serviços cognitivas do Azure

Os serviços cognitivas do Azure permitem criar aplicativos que veem, ouvem, falam e entendem seus usuários. Entre esses serviços, há suporte para mais de três dúzias de idiomas, permitindo que os usuários se comuniquem com seu aplicativo de maneiras naturais.

Este artigo é dividido em duas seções. A primeira lista os idiomas principais com suporte em serviços cognitivas do Azure. A segunda seção concentra-se no suporte a idiomas adicionais para cada um dos serviços. A disponibilidade da linguagem pode variar por país/região e mercado.

## <a name="core-languages"></a>Idiomas principais

Esses idiomas principais têm suporte nos serviços cognitivas do Azure:

* Chinês
* Português
* Francês
* Alemão
* Italiano
* Japonês
* ¹ coreano
* Português
* Espanhol

> [!NOTE]
> Não há suporte para ¹ LUIS, Video Indexer, Análise de Texto e Speech-to-Text.
