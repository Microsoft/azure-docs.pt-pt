## <a name="clean-up-resources"></a>Limpar recursos

Se desejar implementar o seu próprio projeto de deteção de objetos (ou, em vez disso, experimentar um projeto de [classificação de imagens](../quickstarts/image-classification.md)), deve eliminar o projeto de deteção de bifurcação/tesouras deste exemplo. Um teste gratuito permite dois projetos de Visão Personalizada.

No site da [Custom Vision,](https://customvision.ai)navegue para **Projetos** e selecione o caixote do lixo no âmbito do My New Project.
