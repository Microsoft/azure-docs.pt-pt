## <a name="clean-up-resources"></a>Limpar recursos
Se desejar implementar o seu próprio projeto de classificação de imagens (ou, em vez disso, experimentar um projeto de [deteção de objetos](../csharp-tutorial-od.md)), deve eliminar o projeto de identificação de árvores deste exemplo. Uma avaliação gratuita permite dois projetos de Visão Personalizada.

No [Site de Visão Personalizada](https://customvision.ai), navegue para **Projetos** e selecione o caixote do lixo em O Meu Novo Projeto.
