## <a name="clean-up-resources"></a>Limpar recursos

Se desejar implementar o seu próprio projeto de classificação de imagens (ou, em vez disso, experimentar um projeto de [deteção de objetos](../quickstarts/object-detection.md)), deve eliminar o projeto de identificação de árvores deste exemplo. Uma subscrição gratuita permite dois projetos de Visão Personalizada.

No site da [Custom Vision,](https://customvision.ai)navegue para **Projetos** e selecione o caixote do lixo no âmbito do My New Project.
