## <a name="september-2019"></a>Setembro de 2019

### <a name="suggested-tags"></a>Etiquetas sugeridas

* A ferramenta Smart Labeler no [website Da Visão Personalizada](https://www.customvision.ai/) gera etiquetas sugeridas para as suas imagens de treino. Isto permite-lhe rotular um grande número de imagens mais rapidamente ao treinar um modelo de Visão Personalizada. Para obter instruções sobre como utilizar esta função, consulte [as etiquetas sugeridas](./suggested-tags.md).
