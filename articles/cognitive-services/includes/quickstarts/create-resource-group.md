## <a name="create-a-resource-group"></a>Criar um grupo de recursos

Antes de criar um recurso de Serviços Cognitivos, a sua conta deve ter um grupo de recursos Azure para conter o recurso. Se ainda não tiver um grupo de recursos, crie um no [portal Azure](https://ms.portal.azure.com/) antes de continuar.
