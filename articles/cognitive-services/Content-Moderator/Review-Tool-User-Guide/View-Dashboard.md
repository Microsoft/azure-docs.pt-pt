# <a name="view-dashboard"></a>Ver Dashboard #

Clique em de **Dashboard** SEPARADOR para ver as métricas-chave para todos os revisão conteúdo feito na ferramenta. Pode ver as revisões totais, concluídas e em curso para imagens e texto. Pode ver as métricas para a sua equipa de predefinição e para a sua subteams.
