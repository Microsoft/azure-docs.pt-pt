# <a name="view-dashboard"></a>Ver Dashboard

Clique nas **Dashboard** SEPARADOR para ver as métricas chave para todos os revisão de conteúdo concluído dentro da ferramenta. Pode ver as revisões de total, concluídas e em curso de imagens e texto. Pode ver as métricas para a sua equipa de padrão e para sua subequipas.
