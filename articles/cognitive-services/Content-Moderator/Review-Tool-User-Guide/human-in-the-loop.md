# <a name="human-in-the-loop"></a>Humanos no ciclo

Obter os melhores resultados quando as máquinas e humans trabalham em conjunto em conteúdo moderação de interrupção. Máquinas eficazmente aumentar revisões humanos em que a confiança de predição tem assistido ou tempered dentro de um contexto do mundo real. O resultado é um processo de moderação de interrupção de conteúdo de híbridos que executa a melhor do que o se o humans ou máquinas estava a trabalhar individualmente.

## <a name="how-the-review-tool-helps"></a>Como a ferramenta de revisão ajuda ##

A ferramenta de revisão humana quando utilizada em conjunto com a moderação ' assistida por máquina interrupção APIs permite-lhe realizar estas tarefas importantes em relação a moderação de interrupção de conteúdo do ciclo de vida.

1. Automatizar a criação de revisões humanos dos resultados de moderação de interrupção API subjacente
2. Utilize uma ferramenta (ferramenta de revisão e API) para moderada vários formatos (texto, imagem e o vídeo - brevemente)
3. Atribuir ou escalar conteúdos revisões para várias equipas de revisão organizadas por nível de categoria ou a experiência de conteúdo.
4. Utilizar predefinição fluxos de trabalho ou definir fluxos de trabalho personalizados com regras flexíveis e sem escrever qualquer código.
5. Adicionar revisão humano para qualquer API ou a empresa processos ao simplesmente criar um conector.
6. Utilize os conectores predefinido para rever os resultados da Microsoft PhotoDNA, análise de texto e APIs de rostos em.
7. Obter métricas chave de desempenho no seus processos de moderação de interrupção de conteúdo.
