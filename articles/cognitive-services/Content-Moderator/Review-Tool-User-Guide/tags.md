# <a name="about-tags"></a>Sobre etiquetas #

Para além das etiquetas predefinidos (um isracy – isadult e r –), pode criar etiquetas personalizadas para direcionados mais análise. Estas etiquetas personalizadas, em seguida, estão disponíveis para os revisores humanos atribuir a imagens ou texto.

## <a name="create-tags"></a>Criar etiquetas ##

1.  Selecione o separador Definições etiquetas.

  ![Etiquetas de moderação de interrupção de conteúdo](images/tags-1.png)

2.  Introduza um código de curto de duas letras da etiqueta.
3.  Introduza um nome da etiqueta. Mantenha o nome curto e descritivo. Por exemplo, "isNudity".
4.  Introduza uma descrição.
5.  Clique em Adicionar.
6.  Quando tiver terminado de criar etiquetas, clique em Guardar.

![Definir as etiquetas de moderação de interrupção de conteúdo](images/tags-2-define.png)

## <a name="using-custom-tags"></a>Utilizar etiquetas personalizadas ##

Etiquetas personalizadas são utilizadas durante a revisão humano. São apresentam a pré-visualização e o revisor seleciona-lo ao clicar no mesmo.

![Utilizando as etiquetas de moderação de interrupção de conteúdo](images/tags-3-use.png)

Pode desativar etiquetas diferentes para diferentes revisões, a verificação ou desmarcando está visível.
 
![Desativar etiquetas de moderação de interrupção de conteúdo](images/tags-4-disable.png)

Enquanto não é possível eliminar o duas etiquetas predefinidas, isadult e isracy, pode eliminar quaisquer etiquetas personalizadas que foram definidos. Clique na pode de libertação da memória junto a etiqueta que pretende eliminar.

![Eliminar conteúdo moderação de interrupção etiquetas](images/tags-5-delete.png)
