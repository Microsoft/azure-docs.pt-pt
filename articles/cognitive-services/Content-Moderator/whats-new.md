# <a name="whats-new-in-content-moderator"></a>Quais as novidades no Moderador de Conteúdo

Saiba o que há de novo no serviço. Estes itens podem divulgar notas, vídeos, posts de blog e outros tipos de informação. Marque esta página para manter-se atualizado com o serviço.

## <a name="august-2020"></a>Agosto de 2020

### <a name="new-video-moderation-ui"></a>Nova IA de moderação de vídeo

* A página de moderação de vídeo da ferramenta Review foi atualizada com muitas novas ferramentas e funcionalidades para facilitar a marcação e processamento de vídeos e partes de vídeos. Consulte o [guia de moderação de Vídeo](./video-moderation-human-review.md) para saber mais.

### <a name="new-dashboard"></a>Novo dashboard

* O painel de ferramentas Review foi atualizado para mostrar mais informações sobre avaliações abertas, pendentes e concluídas. Consulte o guia da [ferramenta Review](./Review-Tool-User-Guide/human-in-the-loop.md) para obter um passeio pelas novas funcionalidades.
