# <a name="bing-web-search-sdk-samples"></a>Amostras do SDK de pesquisa Web Bing

O SDK de pesquisa Web Bing está disponível em Python, node. js, c# e Java. Exemplos de código, pré-requisitos e instruções de compilação são fornecidas no GitHub. São abordados os seguintes cenários:

* Consultar uma única palavra e imprimir o nome e o URL do primeiro resultado para páginas da web, imagens, notícias e vídeos.
* Uma expressão de consulta, verifique se o número de resultados e imprimir o nome e o URL do primeiro resultado.
* Consultar um termo de pesquisa com filtros de resposta definidos como `news` e imprimir os detalhes dos resultados de notícias.
* Consultar um termo de pesquisa com `answerCount` e `promote` parâmetros, em seguida, imprima detalhes de resultados.
