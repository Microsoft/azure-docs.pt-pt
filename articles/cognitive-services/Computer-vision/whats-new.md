## <a name="january-2020"></a>Janeiro de 2020

### <a name="read-api-30-preview"></a>Leia a pré-visualização da API 3.0

Tem agora a opção de utilizar a versão 3.0 da API de leitura para extrair texto impresso ou manuscrito a partir de imagens. Em comparação com as versões anteriores, 3.0 fornece:
* precisão melhorada
* novo formato de saída
* pontuação de confiança para cada palavra extraída
* apoio tanto às línguas espanhola como inglesa com o parâmetro linguístico adicional

Siga um [texto de extrato rápido](https://docs.microsoft.com/azure/cognitive-services/computer-vision/quickstarts/csharp-hand-text?tabs=version-3) para começar a usar a API 3.0.
