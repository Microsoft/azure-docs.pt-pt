# <a name="get-the-cognitive-services-speech-devices-sdk"></a>Obter os dispositivos de voz dos serviços cognitivos SDK

O SDK de dispositivos de voz está em pré-visualização restrita e tem de estar inscritos no programa. Atualmente, a Microsoft prefere grandes empresas como candidatos para o acesso para este produto.

## <a name="request-access"></a>Pedir acesso

Para obter acesso para o SDK de dispositivos de voz:

1. Vá para o SDK de dispositivos do Microsoft Speech [formulário de inscrição](https://aka.ms/sdsdk-signup).
1. Leitura a [contrato de licença](speech-devices-sdk-license.md).
1. Se concordar com os termos do contrato de licença, selecione **concordo**.
1. Responda às perguntas no formulário.
1. Submeta o formulário. 
1. Se seu endereço de e-mail já não faz parte do Azure Active Directory (Azure AD), receberá um e-mail de convite semelhante ao seguinte exemplo quando está aprovada para acesso. Se seu endereço de e-mail já no Azure AD, receberá uma mensagem de e-mail da equipa de voz do Microsoft quando está aprovada para acesso e pode avançar diretamente para [transferir o SDK de dispositivos de voz](#download-the-speech-devices-sdk).

## <a name="approval-e-mail"></a>E-mail de aprovação

```
From: Microsoft Speech Team from Microsoft (via Microsoft) <invites@microsoft.com> 
Subject: You're invited to the Microsoft organization 
```

![mensagem de e-mail](media/speech-devices-sdk/get-sdk-1.png)

## <a name="accept-access"></a>Aceitar o acesso

Conclua os passos seguintes para aderir ao Azure AD com o endereço de e-mail que indicou durante o registo. Este processo lhe concede acesso para o SDK de dispositivos de voz [site de download](https://shares.datatransfer.microsoft.com/).

1. Na mensagem de e-mail que recebeu, selecione **começar**. Se sua organização já é cliente do Office 365, lhe for pedido para iniciar sessão e pode avançar diretamente para o passo 8.

2. Na janela do browser que se abre, selecione **seguinte**.

    ![janela de autenticação](media/speech-devices-sdk/get-sdk-2.png)

3. Se ainda não tiver uma, crie uma conta Microsoft. Introduza o mesmo endereço de e-mail em que recebeu o e-mail de convite.

    ![Criar uma conta Microsoft](media/speech-devices-sdk/get-sdk-3.png)

4. Selecione **seguinte** para criar uma palavra-passe.

5. Quando lhe for pedido para verificar seu email, obter o código de verificação do e-mail de convite que recebeu.
 
7. Cole ou introduza o código de segurança da mensagem de e-mail na caixa de diálogo. Neste exemplo, o código de segurança é **8406**. Selecione **Seguinte**.

    ![Verificar e-mail](media/speech-devices-sdk/get-sdk-6.png)
 
8. Quando vir o aplicativo de painel de acesso no navegador, confirmar que o seu endereço de correio eletrónico faz parte do Azure AD. Agora tem acesso ao site de download do SDK de dispositivos de voz.

## <a name="download-the-speech-devices-sdk"></a>Transferir os SDK de dispositivos de voz

Vá para o [site de download do SDK de dispositivos de voz](https://shares.datatransfer.microsoft.com/). Inicie sessão com a conta da Microsoft que criou anteriormente. 

![Site de download do SDK](media/speech-devices-sdk/get-sdk-7.png)

Para transferir a voz dispositivos SDK, associados código de exemplo e material de referência:

1. Baixe e instale a ferramenta Aspera ligar quando lhe for pedido no browser.

    ![Baixe o Aspera ligar](media/speech-devices-sdk/get-sdk-8.png)
 
1. Selecione **Sim** para mudar de aplicações para ligar do Aspera.

    ![Mude para ligar do Aspera](media/speech-devices-sdk/get-sdk-9.png)
 
1. Selecione **permitir** para confirmar a transferir os ficheiros com o Aspera ligar.

    ![Transferir com Aspera ligar](media/speech-devices-sdk/get-sdk-10.png)
 
1. Feche a janela de transferências de ligar Aspera depois dos ficheiros são transferidos.

    ![Janela de transferências de ligar Aspera](media/speech-devices-sdk/get-sdk-11.png)
 
Por predefinição, os ficheiros são transferidos para o seu **Downloads** pasta. Pode terminar sessão neste site agora.
