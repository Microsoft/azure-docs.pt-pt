Uma das características centrais do serviço de fala é a capacidade de reconhecer e transcrever a fala humana (muitas vezes referida como discurso-a-texto). Nesta amostra, aprende-se a usar o Speech SDK nas suas apps e produtos para realizar conversão de fala a texto de alta qualidade.

## <a name="react-sample-on-github"></a>Reagem à amostra no GitHub

Vá à [amostra reagir](https://github.com/Azure-Samples/AzureSpeechReactSample) no GitHub para aprender a usar o SDK do discurso num ambiente JavaScript baseado no navegador. Esta amostra mostra exemplos de padrão de design para a troca e gestão de símbolos de autenticação e captura de áudio de um microfone ou ficheiro para conversões de voz para texto.

Além disso, os padrões de design usados no [Node.js quickstart](../../../get-started-speech-to-text.md?pivots=programming-language-nodejs&tabs=script%2cbrowser%2cwindowsinstall) também podem ser usados em um ambiente de navegador.
