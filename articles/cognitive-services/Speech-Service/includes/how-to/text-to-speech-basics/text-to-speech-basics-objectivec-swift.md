Pode sintetizar o discurso a partir de texto utilizando o SDK do discurso para Swift e Objective-C.

## <a name="prerequisites"></a>Pré-requisitos

As seguintes amostras assumem que tem uma conta Azure e subscrição do serviço de fala. Se não tiver uma conta e subscrição, [experimente gratuitamente o serviço Desemação](../../../overview.md#try-the-speech-service-for-free).

## <a name="install-speech-sdk-and-samples"></a>Instalar SDK de discurso e amostras

O [Discurso dos Serviços Cognitivos SDK](https://github.com/Azure-Samples/cognitive-services-speech-sdk) contém amostras escritas em Swift e Objective-C para iOS e Mac. Clique num link para ver as instruções de instalação de cada amostra:

* [Sintetizar o discurso no Objective-C sobre o macOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/macos/text-to-speech)
* [Sintetizar o discurso em Swift sobre o macOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/macos/text-to-speech)
* [Sintetizar o discurso no Objective-C no iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/ios/text-to-speech)
* [Sintetizar o discurso em Swift no iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/ios/text-to-speech)
* [Amostras adicionais para o Objective-C no iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/samples/objective-c/ios)

Também fornecemos um [SDK de discurso on-line para referência objective-C](/objectivec/cognitive-services/speech/).
