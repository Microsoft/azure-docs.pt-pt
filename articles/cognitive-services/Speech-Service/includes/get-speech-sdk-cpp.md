#### <a name="c-nuget-package"></a>Pacote C++ NuGet

O SDK de discurso C++ pode ser instalado a partir do Gestor de **Pacotes** com o seguinte `Install-Package` comando.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
