Terá de substituir os seguintes valores:

 - `YourSubscriptionKey`: encontrado na página **Chaves** do portal Azure para o recurso Speech
 - `YourServiceRegion`: encontrado na página **geral** do portal Azure para o recurso Speech
 - `YourFileUrl`: encontrado na página **blob /contentores** do portal Azure para o recurso da conta de armazenamento
     - Selecione o recipiente apropriado
     - Selecione a bolha desejada
     - Copiar o **URL** na página **Propriedades**
