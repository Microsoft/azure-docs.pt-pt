Terá de substituir os seguintes valores:

 - `YourSubscriptionKey`: encontrado na página **Chaves** do portal Azure para o recurso Discurso
 - `YourServiceRegion`: encontrado na página **geral** do portal Azure para o recurso Discurso
 - `YourFileUrl`: encontrado sob a página **blob /Containers** do portal Azure para o recurso de conta de armazenamento
     - Selecione o recipiente apropriado
     - Selecione a bolha desejada
     - Copie o **URL** na página **Propriedades**
