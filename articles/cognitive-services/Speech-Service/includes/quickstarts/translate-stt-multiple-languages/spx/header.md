Neste arranque rápido, utiliza a ferramenta SPX da linha de comando para converter a fala de uma entrada de microfone para texto em várias outras línguas.
Após uma configuração única, a ferramenta SPX permite traduzir a fala utilizando comandos da linha de comando.
