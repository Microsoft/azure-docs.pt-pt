Neste arranque rápido, utiliza a ferramenta SPX da linha de comando para reconhecer a fala gravada num ficheiro de som e produz uma transcrição de texto. É fácil usar a ferramenta SPX para executar tarefas comuns de reconhecimento, como transcrever conversas. Após uma configuração única, a ferramenta SPX permite transcrever o áudio para o texto interactivamente com um microfone ou a partir de ficheiros utilizando um script de lote.
