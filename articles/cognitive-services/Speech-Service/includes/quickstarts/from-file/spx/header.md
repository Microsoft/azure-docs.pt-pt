Neste arranque rápido, você usa o CLI da fala da linha de comando para reconhecer a fala gravada num ficheiro áudio, e produzir uma transcrição de texto. É fácil usar o TALK CLI para executar tarefas comuns de reconhecimento, como transcrever conversas. Após uma configuração única, o Speech CLI permite transcrever o áudio para o texto interactivamente com um microfone ou a partir de ficheiros utilizando um script de lote.
