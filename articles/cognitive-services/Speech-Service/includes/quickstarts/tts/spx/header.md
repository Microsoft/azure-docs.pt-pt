Neste arranque rápido, utiliza a ferramenta SPX da linha de comando para converter texto para discurso que ouve do altifalante áudio do computador. O serviço de texto-a-fala fornece muitas opções para vozes sintetizadas, sob apoio da [linguagem texto-a-fala](../../../../language-support.md#text-to-speech). Após uma configuração única, a ferramenta SPX permite sintetizar a fala a partir de texto utilizando comandos da linha de comando.
