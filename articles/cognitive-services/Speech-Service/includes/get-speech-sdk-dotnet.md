#### <a name="c-nuget-package"></a>Pacote NuGet C#

O SDK de discurso .NET pode ser instalado a partir do **CLI .NET Core** com o seguinte `dotnet add` comando.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

O .NET Speech SDK pode ser instalado a partir do Gestor de **Pacotes** com o seguinte `Install-Package` comando.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
