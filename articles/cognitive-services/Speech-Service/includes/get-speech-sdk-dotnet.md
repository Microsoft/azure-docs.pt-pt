#### <a name="c-nuget-package"></a>Pacote C# NuGet

O .NET Speech SDK pode ser instalado a partir `dotnet add` do **CLI .NET Core** com o seguinte comando.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

O .NET Speech SDK pode ser instalado a `Install-Package` partir do Gestor de **Pacotes** com o seguinte comando.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
