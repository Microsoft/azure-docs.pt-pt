# <a name="sapi-phone-set-usage"></a>Utilização do conjunto de telefone SAPI

A Microsoft define o conjunto de telefone SAPI para sete idiomas (en-US, fr-FR, de-DE, es-ES, ja-JP, zh-CN, zh-TW). O conjunto de telefoneS SAPI é relativamente fácil de usar e lembre-se especialmente para o altifalante nativo. Por exemplo, ja-JP use kana como telefone, zh-CN usar PinYin como conjunto de telefone. Mas às vezes não é poderoso o suficiente, por exemplo, o conjunto telefónico SAPI de fr-FR não poderia representar a ligação francesa.
