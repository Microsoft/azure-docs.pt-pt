# <a name="supported-languages-for-speech-service"></a>Idiomas suportados para o serviço de reconhecimento de voz

São suportados idiomas diferentes para diferentes funções de serviços de reconhecimento de voz. As tabelas a seguir resumem o suporte de idiomas.

## <a name="speech-to-text"></a>Conversão de Voz em Texto

O reconhecimento de voz Microsoft API suporta os seguintes idiomas. 

| Código | Idioma | Código | Idioma |
|-----|-----|-----|-----|
| ar ex | Árabe (Egypt) padrão moderna | It-IT | Italiano (Itália) |
| Alemanha da Alemanha | Alemão (Alemanha) | ja-JP | Japonês (Japão) |
| pt-PT | Inglês (Estados Unidos) | pt-BR | Português (Brasil) |
| es-ES | Espanhol (Espanha) | ru-RU | Russo (Rússia) |
| FR-FR | Francês (França) | zh-CN | Chinês (Mandarim, simplificado) |

### <a name="customization"></a>Personalização

Personalizada **reconhecimento de voz do texto para** tipos de letra de voz suportam inglês (en-US) e chinês (zh-CN).

## <a name="speech-translation"></a>Tradução de Voz

O **tradução de reconhecimento de voz** API suporta idiomas diferentes para tradução de reconhecimento de voz-para-reconhecimento de voz e o texto de reconhecimento de voz. O idioma de origem tem de ser sempre da tabela de idioma de reconhecimento de voz abaixo. Os idiomas de destino disponíveis dependem se o destino de tradução é voz ou texto.

Pode obter a lista de idiomas ou através de programação ou utilizar o ponto final de idiomas da API REST. A lista apresenta cada código de idioma, bem como o nome de idioma inglês — ou qualquer outro idioma suportado. Esta lista é atualizada pelo serviço Microsoft Translator sempre que um novo idioma fica disponível.
