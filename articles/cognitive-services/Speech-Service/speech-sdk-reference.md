# <a name="speech-sdk-api-reference"></a>Referência da API do SDK de voz

A conversão de voz Software Development Kit (SDK) proporciona o acesso de aplicações para as funções do serviço de voz, tornando-o mais fácil de desenvolver software habilitado para fala. Atualmente, os SDKs fornecem acesso ao **conversão de voz em texto**, **tradução por voz**, e **reconhecimento da intenção**. Uma visão geral sobre as capacidades e plataformas suportadas pode ser encontrada na documentação do [página de entrada](http://aka.ms/csspeech).
