# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Como inscrever-se para a API de reconhecimento de voz tradutor Microsoft

Siga estes passos para inscrever-se para a API de reconhecimento de voz do conversor de Microsoft.

## <a name="1-sign-into-azure"></a>1: Inicie sessão no Azure   
- Não tem uma conta? [Inscreva-se uma conta do Microsoft Azure](http://azure.com/)
- Já tem uma conta? [A iniciar sessão](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: subscrever tradutor da Microsoft     
- Depois de se inscrever no Azure, vá para o [serviços cognitivos](https://portal.azure.com/#create/Microsoft.CognitiveServices) secção
- Em "Tipo de API", selecione a API de reconhecimento de voz tradutor
- Na secção de escalão de preço, selecione o escalão de preço que melhor se adequa às suas necessidades
    - Cada subscrição tem um escalão gratuito. O escalão gratuito tem as mesmas funções e funcionalidades como os planos pagas e não tem uma data de expiração.
    - Pode ter apenas uma subscrição gratuita na sua conta
- Preencha o resto do formulário e selecione o botão para criar
- A subscrição está pronta para utilização imediata

## <a name="3-authentication-key"></a>3: chave de autenticação    
- Obter a sua chave de autenticação ao aceder ao menu de todos os recursos > clique na sua subscrição
- O valor de chave é utilizado para autenticação, mantenha o valor de chave confidencial. Terá do valor da chave quando desenvolver a sua aplicação
