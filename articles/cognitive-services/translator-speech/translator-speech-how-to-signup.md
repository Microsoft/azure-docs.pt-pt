# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Como inscrever-se para a API de voz do tradutor

Siga estes passos para se inscrever para a API de voz do Translator.

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-translator-speech-deprecation-note.md)]

## <a name="1-sign-into-azure"></a>1: iniciar sessão no Azure

- Não tem uma conta? [Inscreva-se uma conta do Microsoft Azure](http://azure.com/)
- Já tem uma conta? [Iniciar sessão](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: subscrever o Microsoft Translator

- Depois de iniciar sessão no Azure, vá para o [serviços cognitivos do Azure](https://portal.azure.com/#create/Microsoft.CognitiveServices) secção
- Em "Tipo de API", selecione a API de voz do tradutor
- Na secção de escalão de preço, selecione o escalão de preço que melhor se adequa às suas necessidades
    - Cada subscrição tem um escalão gratuito. O escalão gratuito tem as mesmas funções e funcionalidades como os planos pagos e não tem uma data de expiração.
    - Pode ter apenas uma subscrição gratuita na sua conta
- Preencha o resto do formulário e selecione o botão Criar
- A subscrição está pronta para uso imediato

## <a name="3-authentication-key"></a>3: chave de autenticação

- Obter a sua chave de autenticação ao aceder ao menu de todos os recursos > clique na sua subscrição
- O valor de chave é utilizado para autenticação, mantenha o valor de chave confidenciais. Tem o valor da chave de quando desenvolver a sua aplicação
