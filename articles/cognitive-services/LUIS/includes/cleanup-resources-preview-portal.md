Quando já não precisar, elimine a aplicação LUIS. Para isso, selecione **Minhas aplicações** do menu no topo esquerdo. Selecione o nome da aplicação na lista de aplicações e, em seguida, selecione **Eliminar** a partir do menu de contexto. No dialog pop-up **Eliminar a aplicação**, selecione **Ok**.
