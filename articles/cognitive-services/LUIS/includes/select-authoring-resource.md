## <a name="select-subscription-and-authoring-resource"></a>Selecione recurso de subscrição e autoria

No portal LUIS, para ver uma lista de apps ou para criar uma app, selecione o recurso de subscrição e autoria. As aplicações são acedidas pelo recurso de autoria LUIS.
