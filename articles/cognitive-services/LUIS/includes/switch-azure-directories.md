Se não vir uma aplicação criada por si ou partilhada consigo, poderá ter de mudar para um diretório Azure diferente.

1. Clique no avatar no canto superior direito do ecrã. Em seguida, clique **em Mudar para um diretório Azure diferente**. 
2. Na janela que aparece, não se esqueça de selecionar o diretório Azure que contém o recurso LUIS que foi partilhado consigo.
