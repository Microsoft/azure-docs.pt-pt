# <a name="authoring-api-changes"></a>Autor de alterações da API 

As APIs de autoria são atualizadas, para V3, para permitir a [decomposição do modelo com entidades aprendidas por máquinas.](luis-concept-data-extraction.md) 

Pode ser autor no V2 ou pré-visualização V3 APIS. A mesma aplicação também pode optar por utilizar respostas de ponto final de previsão V2 ou V3. Não é preciso ter a mesma versão da API para a autoria e previsão da resposta final.

O portal LUIS utiliza exclusivamente as APIs de autoria v3.
