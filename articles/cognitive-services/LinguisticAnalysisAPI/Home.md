# <a name="what-is-the-linguistic-analysis-api-preview"></a>O que é a API de Análise Linguística? (Pré-visualização)

> [!IMPORTANT]
> A pré-visualização da API de Análise Linguística foi desativada no dia 9 de agosto de 2018. Recomendamos que utilize os [módulos de análise de texto do Azure Machine Learning](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) para o processamento e a análise de texto.

A API de Análise Linguística proporciona acesso a ferramentas de processamento de linguagem natural (NLP) que identificam a estrutura do texto. A versão atual oferece três tipos de análise:

- [Separação e atomização de frases](Sentences-and-Tokens.md)
- [Identificação de classe gramatical](POS-tagging.md)
- [Análise de constituintes](Constituency-Parsing.md)

## <a name="getting-started"></a>Introdução

Assim que estiver familiarizado com os principais tipos de análise indicados acima, poderá examinar o ponto final para [obter a lista dos analisadores disponíveis](AnalyzersMethod.md) e saber como são [designados](Analyzer-Names.md)e, em seguida, o ponto final para [analisar a nova entrada](AnalyzeMethod.md).
