# <a name="troubleshooting-the-multivariate-api"></a>Resolução de problemas da API multivariada

Este artigo fornece orientações sobre como resolver problemas e remediar mensagens de erro HTTP comuns ao utilizar a API multivariada.

### <a name="multivariate-error-codes"></a>Códigos de erro multivariados

| Método                                       | Código de erro HTTP | Mensagem de erro                                                                                                            | Medidas a tomar                                                                                                                                                                                                         |
|----------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The 'source' field is required in the request.`                                                                         | A palavra-chave "fonte" não foi especificada corretamente. O formato deve ser `"{\"source\": \|" <SAS   URL>\"}"`                                                                                          |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The source field must be a valid sas blob url`                                                                          | O campo de origem deve ser um recipiente blob válido.                                                                                                                                                             |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The 'startTime' field is required in the request.`                                                                      | Adicionar startTime no pedido                                                                                                                                                                                         |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The 'endTime' field is required in the request.`                                                                        | Adicione endTime no pedido.                                                                                                                                                                                          |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `Invalid Timestamp format.`                                                                                              | O tempo que se prevê no ficheiro csv fechado no url de origem encontra-se em formato inválido ou "startTime", "endTime" em formato inválido.                                                                                 |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The displayName length exceeds maximum allowed length 24.`                                                              | O DisplayName é um parâmetro opcional a ser usado para os utilizadores distinguirem diferentes modelos. Um nome de exibição válido deve ser inferior a 24 caracteres.                                                                    |
| Treine um modelo de deteção de anomalias multivariadas | 400             | `The 'slidingWindow' field must be an integer between 28 and 2880.`                                                      | A janela deslizante deve estar num intervalo válido.                                                                                                                                                                             |
| Treine um modelo de deteção de anomalias multivariadas | 401             | `Unable to download blobs on the Azure Blob storage account.`                                                            | A URL não tem as permissões certas. A bandeira da lista não está definida.   O cliente deve recriar o URL SAS e certificar-se de que as bandeiras de leitura e lista são verificadas (por exemplo, utilizando o Storage Explorer) |
| Treine um modelo de deteção de anomalias multivariadas | 413             | `Unable to process the dataset. Number of variables exceed the limit (300).`                                             | Os dados no recipiente blob excedem o limite de 300 variáveis atualmente. O cliente tem de apontar para reduzir o tamanho variável.                                                                                |
| Treine um modelo de deteção de anomalias multivariadas | 413             | `Valid Timestamps in the dataset exceeds the limit (1 million points), please change   startTime or endTime parameters.` | O número máximo de pontos pode ser usado para treinar 1 milhão. Os clientes podem reduzir o tamanho variável ou alterar o startTime ou endTime                                                                                     |
| Treine um modelo de deteção de anomalias multivariadas | 413             | `Unable to process dataset. Size of dataset exceeds size limit (2GB).`                                                   | Os dados no recipiente blob excedem o limite de 4 MB atualmente. O cliente tem de apontar para uma bolha com dados menores.                                                                                          |
| Detetar anomalia multivariada                  | 404             | `The model does not exist.`                                                                                              | O ID do modelo é inválido. Os clientes precisam de treinar um modelo antes de o utilizarem.                                                                                                                                            |
| Detetar anomalia multivariada                  | 400             | `The model is not ready yet.`                                                                                            | O modelo ainda não está pronto. Os clientes precisam de ligar para a Get Multivariate Model api para verificar o estado do modelo.                                                                                                               |
| Detetar anomalia multivariada                  | 400             | `The 'source' field is required in the request.`                                                                         | A palavra-chave "fonte" não foi especificada corretamente. O formato deve ser `"{\"source\": \|" <SAS   URL>\"}"`                                                                                          |
| Detetar anomalia multivariada                  | 400             | `The source field must be a valid sas blob url`                                                                          | O campo de origem deve ser um recipiente blob válido.                                                                                                                                                             |
| Detetar anomalia multivariada                  | 400             | `The 'startTime' field is required in the request.`                                                                      | Adicionar startTime no pedido                                                                                                                                                                                         |
| Detetar anomalia multivariada                  | 400             | `The 'endTime' field is required in the request.`                                                                        | Adicione endTime no pedido.                                                                                                                                                                                          |
| Detetar anomalia multivariada                  | 400             | `Invalid Timestamp format.`                                                                                              | O tempo que se prevê no ficheiro csv fechado no url de origem encontra-se num formato inválido ou "startTime", "endTime" em formato inválido.                                                                                 |
| Detetar anomalia multivariada                  | 400             | `The corresponding file of the variable does not exist.`                                                                 | Uma variável foi utilizada no comboio, mas não pode ser encontrada quando o cliente utiliza o modelo correspondente para fazer a deteção. Os clientes precisam adicionar esta variável e, em seguida, submeter o pedido de deteção.         |
| Detetar anomalia multivariada                  | 413             | `Unable to process the dataset. Number of variables exceed the limit (300).`                                             | Os dados no recipiente blob excedem o limite de 300 variáveis atualmente. O cliente tem de apontar para reduzir o tamanho variável.                                                                                |
| Detetar anomalia multivariada                  | 413             | `The limit timestamps of one detection request is 2880, please change startTime or endTime parameters.`                | Os sinais tempotais máximos a serem detetados num pedido de deteção é de 2880, os clientes precisam de alterar o startTime ou endTime e, em seguida, apresentar pedido de deteção.                                                           |
| Detetar anomalia multivariada                  | 413             | `Unable to process dataset. Size of dataset exceeds size limit (2GB).`                                                   | Os dados no recipiente blob excedem o limite de 4 MB atualmente. O cliente tem de apontar para uma bolha com dados menores.                                                                                          |
| Obter modelo multivariado                       | 404             | `Model with 'id=<input model ID>' not found.`                                                                            | O ID não é um ID de modelo válido. Utilize modelos GET para encontrar todos os IDs de modelo válidos.                                                                                                                               |
| Obter modelo multivariado                       | 404             | `Model with id=<input model ID>' not found.`                                                                            | O ID não é um ID de modelo válido. Utilize modelos GET para encontrar todos os IDs de modelo válidos.                                                                                                                               |
| Obtenha resultado de deteção de anomalias multivariadas    | 404             | `Result   with 'id=<input result ID>' not found.`                                                                          | O ID não é um resultado válido. Reenviar o seu pedido de deteção.                                                                                                                                   |
| Eliminar modelo multivariado                    | 404             | `Location   for model with 'id=<input model ID>' not found.`                                                               | O ID não é um ID de modelo válido. Utilize modelos GET para encontrar todos os IDs de modelo válidos.                                                                                                                               |
