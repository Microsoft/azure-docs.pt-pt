Neste arranque rápido, aprende-se a detetar anomalias num lote de dados da série temporal utilizando o serviço de Detetor de Anomalias e o CURL.

Para uma visão de alto nível dos conceitos do Detetor de Anomalias, consulte o artigo [de visão geral.](../../overview.md)

## <a name="prerequisites"></a>Pré-requisitos

- Subscrição Azure - [Crie uma gratuitamente](https://azure.microsoft.com/free/cognitive-services)
- Assim que tiver a subscrição do Azure, <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAnomalyDetector"  title=" crie um recurso de Detetor de "  target="_blank"> Anomalias crie um recurso de Detetor de <span class="docon docon-navigate-external x-hidden-focus"></span> </a> Anomalias no portal Azure para obter a sua chave e ponto final. Aguarde que seja implantado e selecione o botão **Go para o** recurso.
    - Necessitará do endereço chave e ponto final do recurso que criar para utilizar a API REST. 
    Pode utilizar o nível de preços gratuitos `F0` para experimentar o serviço e fazer upgrade mais tarde para um nível pago para produção.

## <a name="detect-anomalies-for-an-entire-series"></a>Detetar anomalias para uma série inteira

Com um pedido de comando, executar o seguinte comando. Terá de inserir os seguintes valores no comando.
- A chave de subscrição do detetor de anomalias.
- O endereço do detetor de anomalias. 
- Um ficheiro JSON válido de dados da série de tempo para testar anomalias. Se não tiver o seu próprio ficheiro, pode criar uma sample.jsno ficheiro a partir da [amostra do corpo](https://westus2.dev.cognitive.microsoft.com/docs/services/AnomalyDetector/operations/post-timeseries-entire-detect)do Pedido .

```curl
curl -v -X POST "https://{endpointresourcename.cognitive.microsoft.com}/anomalydetector/v1.0/timeseries/entire/detect"
-H "Content-Type: application/json"
-H "Ocp-Apim-Subscription-Key: {subscription key}"
-d "@{path_to_file.json}" 
```

Para um exemplo com todos os valores povoados:

```curl
curl -v -X POST "https://my-resource-name.cognitiveservices.azure.com/anomalydetector/v1.0/timeseries/entire/detect" -H  "Content-Type: application/json" -H "Ocp-Apim-Subscription-Key:1111112222222ed333333ab333333333" -d "@test.json"
```
