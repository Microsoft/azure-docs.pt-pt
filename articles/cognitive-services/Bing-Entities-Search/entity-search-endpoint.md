# <a name="entity-search-endpoints"></a>Pontos finais de pesquisa de entidade
O **API de pesquisa de entidade** inclui um ponto final.

## <a name="endpoint"></a>Ponto Final
Para pedir os resultados da pesquisa de entidade, envie um pedido para o seguinte ponto final. Utilize os parâmetros de URL e cabeçalhos para definir mais especificações.

Ponto final `GET`: 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

São necessários os seguintes parâmetros de URL:
- mkt. Mercado onde os resultados provenientes. 
- q. A consulta de pesquisa de entidade.
