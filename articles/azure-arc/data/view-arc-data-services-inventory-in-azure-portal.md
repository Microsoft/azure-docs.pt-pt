# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Veja o inventário das suas instâncias no portal Azure

Depois de fazer o upload das suas [métricas, registos](upload-metrics-and-logs-to-azure-monitor.md)ou [utilização,](view-billing-data-in-azure.md)pode ver a sua instância como um recurso Azure a partir do portal Azure. Para ver o seu recurso no portal Azure, vá ao URL especial [https://portal.azure.com](https://portal.azure.com) e complete estes passos:

1. Vá a **Todos os serviços.**
1. Procure o tipo de exemplo da sua base de dados.
1. Adicione o tipo aos seus favoritos.
1. No menu esquerdo, selecione o tipo de instância.
1. Veja as suas instâncias na mesma vista que os outros recursos de hiperescala Azure SQL ou Azure PostgreSQL (utilize filtros para uma vista granular).
