# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Veja o inventário das suas instâncias no portal Azure

Depois de ter carregado as suas [métricas, registos](upload-metrics-and-logs-to-azure-monitor.md)ou [utilização,](view-billing-data-in-azure.md)pode ver a sua instância como um recurso Azure a partir do portal Azure. Para fazê-lo navegue para o URL especial [https://portal.azure.com](https://portal.azure.com) e siga as imagens abaixo

1. Ir a Todos os serviços
1. Procure o seu tipo de tipos de casos de base de dados
1. Adicione aos seus favoritos
1. Clique no tipo de instância a partir do navegador esquerdo
1. Veja os seus casos na mesma vista que os seus outros recursos de hiperescala Azure SQL ou Azure PostgreSQL (use filtros para obter uma visão granular)
