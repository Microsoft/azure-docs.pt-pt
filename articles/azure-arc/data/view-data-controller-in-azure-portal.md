# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Ver recurso do controlador de dados Azure Arc no portal Azure

Depois de ter feito o upload das suas [primeiras métricas ou registos para o Azure](upload-metrics-and-logs-to-azure-monitor.md) ou [o uso carregado,](view-billing-data-in-azure.md)poderá ver que o recurso de controlador de dados Azure Arc é criado em Azure no portal Azure.

Abra o portal Azure utilizando o URL especial:  [https://portal.azure.com](https://portal.azure.com) .

Procure o seu controlador de dados pelo nome na barra de pesquisa e clique nele.
