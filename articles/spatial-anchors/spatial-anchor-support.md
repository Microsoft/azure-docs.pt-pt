# <a name="azure-spatial-anchors-support-options"></a>Opções de suporte do Azure âncoras espaciais

## <a name="open-a-tech-support-ticket"></a>Abra um pedido de suporte técnico

Para abrir um pedido de suporte técnico no Portal do Azure para as âncoras espaciais do Azure:

1. Com o [portal do Azure](https://azure.microsoft.com/account/) abrir, selecione o ícone de ajuda da barra de menus superior, em seguida, selecione a **ajuda + suporte** botão. 

![Ajuda do portal do Azure e suporte](./media/spatial-anchor-support.png)

2. Com a ajuda + suporte aberto, selecione de página **+ novo pedido de suporte**.

![Pedido de suporte de novos de portal do Azure](./media/spatial-anchor-support2.png)

3. Quando concluir os campos de pedido de suporte: 

- Tipo de problema: Técnico
- Serviço: Âncoras espaciais

![Campos de pedido de suporte do portal do Azure](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Suporte da comunidade

Para obter suporte da Comunidade do utilizador, consulte [âncoras espaciais do Azure no Stack Overflow](https://stackoverflow.com/questions/tagged/azure-spatial-anchors). 

## <a name="provide-content-feedback"></a>Fornecer comentários de conteúdo

Na parte inferior de cada artigo conteúdo, há uma oportunidade de abrir um problema do GitHub e forneça comentários sobre o conteúdo de documentação do Azure de âncora espaciais. 

## <a name="provide-product-feedback"></a>Fornecer comentários sobre o produto

Para fornecer comentários, partilhar uma ideia ou sugestão para o serviço de âncoras espaciais do Azure ou votar em ideias que outras pessoas submeter, visite o [fórum de comentários do Azure geográficos âncoras](https://feedback.azure.com/forums/919252-azure-spatial-anchors).
