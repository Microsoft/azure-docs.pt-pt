# <a name="azure-spatial-anchors-support-options"></a>Opções de suporte de âncoras espaciais Azure

## <a name="open-a-tech-support-ticket"></a>Abra um bilhete de apoio técnico

Para abrir um bilhete de apoio técnico dentro do Portal Azure para Âncoras Espaciais Azure:

1. Com o [portal Azure](https://azure.microsoft.com/account/) aberto, selecione o ícone de ajuda da barra de menu superior e, em seguida, selecione o botão **de suporte Ajuda +.**

![Ajuda e apoio do portal Azure](./media/spatial-anchor-support.png)

1. Com a página de apoio Ajuda + aberta, selecione **+ Novo pedido**de suporte .

![Novo pedido de apoio do portal Azure](./media/spatial-anchor-support2.png)

1. Ao completar os campos de bilhetes de apoio:

- Tipo de problema: técnico
- Serviço: Âncoras Espaciais

![Campos de bilhetes de apoio ao portal Azure](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Suporte da comunidade

Para obter apoio da comunidade utilizadora, consulte [as âncoras espaciais Azure no Transbordo de Pilhas](https://stackoverflow.com/questions/tagged/azure-spatial-anchors).
