# <a name="building-in-unity-with-azure-spatial-anchors"></a>Edifício em Unidade com Âncoras Espaciais Azure

Os desenvolvedores podem escolher a Unidade para criar e implementar aplicações de realidade mista que usam âncoras espaciais Azure. As seguintes plataformas são suportadas:

**Unidade para HoloLens**

[Quickstart: Criar uma app HoloLens de unidade que usa âncoras espaciais Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-hololens)

**Unidade para Android**

[Quickstart: Criar uma app Android de unidade que usa âncoras espaciais Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-android)

**Unidade para iOS**

[Quickstart: Criar uma app iOS de unidade que usa âncoras espaciais Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-ios)
