# <a name="what-is-the-azure-education-hub"></a>O que é o Azure Education Hub?

O Microsoft Azure [Education Hub](https://portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/quickstart) é uma ferramenta que ajuda os utilizadores académicos a providenciar e gerir o crédito em nuvem em muitas subscrições diferentes do Azure. Isto é útil quando os membros do corpo docente precisam de gerir muitos projetos de estudantes baseados na nuvem. O Education Hub também é útil para fins de pesquisa quando as suas necessidades de crédito Azure ainda não são conhecidas.

Os utilizadores podem facilmente ajustar o montante do crédito atribuído dentro de cada subscrição ou a granel. As subscrições também podem ser suspensas — por exemplo, quando um projeto de semestre ou trimestre está concluído — para maximizar os créditos disponíveis.

## <a name="prerequisites"></a>Pré-requisitos

Para aceder ao Centro de Educação, já deveria ter recebido uma notificação por e-mail de que foi aprovado para uma bolsa académica. O e-mail contém o seu valor de crédito aprovado e um link para o [Centro de Educação.](https://aka.ms/startedu)

Para utilizar a sua bolsa académica, tem de criar uma sala de aula no Education Hub e utilizar subscrições dentro da sala de aula que aceda ao seu crédito Azure.
