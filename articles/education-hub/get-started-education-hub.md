# <a name="getting-started-with-azure-education-hub"></a>Começando com o Azure Education Hub

A página Education Hub Get Started fornece ligações rápidas ao aterrar pela primeira vez no Education Hub. Lá, pode encontrar informações sobre como configurar o seu curso, aprender sobre diferentes serviços através do Microsoft Learn, ou implementar facilmente os seus primeiros serviços através de Quickstart Templates.

:::image type="content" source="media/get-started-education-hub/get-started-page.png" alt-text="A página Get Started no Azure Education Hub." border="false":::

Após os logins subsequentes, verá a página 'Vista Geral' por predefinição.
