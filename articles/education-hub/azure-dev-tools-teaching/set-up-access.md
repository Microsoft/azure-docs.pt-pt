# <a name="setting-up-access-for-azure-dev-tools"></a>Criação de acesso para ferramentas Azure Dev

Existem duas formas de aceder à sua subscrição para que possa implementar software para os seus alunos e equipar os seus laboratórios:
1. Ao descarregar software e chaves do Portal de Subscrição do Estúdio Visual.
1. Concessão de acesso para os alunos à Loja do Centro de Educação.

Para mais informações, visite o [site Azure for Education.](https://azureforeducation.microsoft.com)

## <a name="for-administrators-only"></a>Apenas para administradores  
Os administradores acedem a ferramentas dev Azure através do [Portal de Subscrição](https://my.visualstudio.com/)do Estúdio Visual.

Para solicitar acesso à sua Assinatura visual do Estúdio, pode fazê-lo através do [Portal de Gestão.](https://azureforeducation.microsoft.com/account/Subscriptions) Uma vez aprovado, terá acesso ao Portal de Assinaturas do Estúdio Visual. Se esta for a primeira vez que aceda ao Portal de Subscrição do Estúdio Visual, será solicitado que ligue uma Conta Microsoft com o endereço de e-mail que foi fornecido quando se inscreveu. Você precisará do seu ID do Assinante ou Número de Acesso a Benefícios, o primeiro e último nome de um administrador, e o e-mail dado durante a inscrição para ligar estas contas. Depois de ligar as suas contas, poderá aceder ao Portal do Assinante do Estúdio Visual.

## <a name="for-students-faculty-and-administrators"></a>Para estudantes, docentes e administradores
As studences acedem a ferramentas dev Azure através do [Education Hub](https://aka.ms/devtoolsforteaching).

Tanto os estudantes como os docentes podem ter acesso a todos os benefícios de descarregamento de software através do Education Hub. O Education Hub é construído dentro do portal Azure e proporciona aos seus alunos fácil acesso a todo o catálogo de software, bem como acesso a todo o catálogo da [Microsoft Learn.](/learn/)
