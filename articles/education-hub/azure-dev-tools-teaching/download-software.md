# <a name="help-students-download-software-developer-tools-from-the-azure-education-hub"></a>Ajude os estudantes a descarregar ferramentas de desenvolvimento de software do Azure Education Hub

Através do programa Azure Dev Tools for Teaching, os alunos que tenham uma subscrição atual e ativa do Azure for Students podem descarregar gratuitamente determinadas ferramentas de desenvolvimento de software. O Azure for Students fornece acesso a todos os produtos Azure que se destinam expressamente a apoiar a educação ou o ensino, a investigação não comercial ou os esforços para conceber, desenvolver, testar e demonstrar aplicações de software para estes fins. Os alunos descarregam as ferramentas do Azure Education Hub.

## <a name="instructions-for-students-how-to-download-software-developer-tools"></a>Instruções para os alunos: como descarregar ferramentas de desenvolvedores de software

Faça com que os seus alunos sigam este procedimento para descarregar as ferramentas de desenvolvimento de software:

1. Abra um browser e vá ao [Azure Education Hub.](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)
1. Inscreva-se com as suas credenciais de subscrição Azure for Students.
1. Selecione qualquer uma das ferramentas de desenvolvedor de software no Education Hub para abrir o painel de descarregamento.
1. Leia as informações do software para se certificar de que o software é compatível com o seu sistema informático.
1. Depois de confirmar que o software é compatível com o seu sistema, selecione **Download** .
1. Siga as instruções no ecrã para descarregar e instalar o software.

   Azure for Students dá-lhe 100 dólares de crédito por 12 meses. Inclui acesso a mais de 25 serviços gratuitos, incluindo computação, rede, armazenamento e bases de dados. Quaisquer encargos incorridos durante este período são deduzidos do crédito.
