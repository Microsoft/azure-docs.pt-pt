# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Alunos matriculados em Azure Dev Tools for Teaching
Após a compra da Azure Dev Tools for Teaching, todos os alunos localizados no domínio da sua instituição terão automaticamente acesso a esta oferta. Durante o processo de compra, será solicitado que insira o endereço de e-mail do Administrador de Subscrição. Este domínio verificado pela universidade irá desbloquear os estudantes com o mesmo domínio de e-mail e conceder-lhes a capacidade de iniciar sessão e resgatar os seus benefícios de descarregamento de software. Se os estudantes ainda não tiverem uma Conta Microsoft (MSA) ou fizerem parte de um Diretório Ative universitário, serão solicitados a criar um que corresponda ao seu e-mail universitário. Se os alunos estiverem com problemas de acesso ao Centro de Educação, faça-os olhar através do nosso Guia de Resolução de Problemas abaixo ou contacte-nos em [https://aka.ms/studentsupport](https://aka.ms/studentsupport) .

## <a name="managing-access-for-students"></a>Gerir o acesso aos alunos
Você pode gerir o acesso para estudantes com base no domínio que eles usam para iniciar sessão em Azure Dev Tools for Teaching. Aceda ao seu [Portal de Gestão](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) de Ferramentas de Azure Dev para O Ensino. Lá, verá todas as suas assinaturas. Escolha a subscrição para a qual deseja gerir o acesso.
