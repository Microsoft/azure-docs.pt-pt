# <a name="frequently-asked-questions-about-the-education-hub"></a>Perguntas frequentes sobre o Centro de Educação

Este artigo aborda questões frequentes sobre Azure for Students, Azure for Students Starter e Azure Dev Tools for Teaching.

## <a name="azure-for-students"></a>Azure for Students

### <a name="what-happens-after-i-use-my-100-credit-or-im-at-the-end-of-12-months"></a>O que acontece depois de usar o meu crédito de 100 dólares ou estar no fim de 12 meses?

Depois de esgotar o seu crédito disponível ou chegar ao final de 12 meses, a sua subscrição Azure fica desativada. Para continuar a utilizar [o Azure,](https://account.azure.com/)pode fazer upgrade para uma subscrição pay-as-you-go no portal da Conta Azure. Para obter termos de uso detalhados para Azure para Estudantes, visite os termos da [oferta.](https://azure.microsoft.com/offers/ms-azr-0170p/)

### <a name="can-i-get-azure-for-students-again-next-year"></a>Posso ter a Azure para estudantes de novo no próximo ano?

Obrigado pelo seu interesse em Azure para estudantes! Estamos felizes que tenha gostado de usá-lo. Neste momento, não anunciamos renovações. Consulte aqui para obter informações.

### <a name="what-are-subscriptions-and-how-do-they-relate-to-azure-for-students"></a>O que são as assinaturas e como se relacionam com a Azure for Students?

As subscrições fornecem acesso aos serviços Azure. Azure for Students dá-lhe 100 dólares de crédito por 12 meses. Inclui acesso a mais de 25 serviços gratuitos, incluindo computação, rede, armazenamento e bases de dados. Quaisquer encargos incorridos durante este período são deduzidos do crédito. Para continuar a utilizar os serviços Azure ao final de 12 meses ou depois de esgotar o seu crédito de 100 dólares, o que ocorrer primeiro, tem de fazer upgrade para uma subscrição pay-as-you-go.

### <a name="what-happens-with-my-azure-services-if-i-dont-upgrade"></a>O que acontece com os meus serviços de Azure se eu não atualizar?

Se decidir não fazer upgrade ao final de 12 meses ou depois de esgotar o seu crédito de 100 dólares, o que ocorrer primeiro, quaisquer produtos que tenha implementado são desativados e não poderá aceder-lhes. Tem 90 dias a partir do fim da sua subscrição gratuita para fazer upgrade para uma subscrição pay-as-you-go.

### <a name="how-do-i-know-how-much-of-the-100-credit-i-have-left"></a>Como é que sei quanto do crédito de 100 dólares que me restam?

Pode ver o seu crédito restante no [portal Azure Sponsorships.](https://www.microsoftazuresponsorships.com/)

### <a name="how-do-i-download-the-software-developer-tools"></a>Como faço o download das ferramentas de desenvolvimento de software?

A subscrição do Azure for Students dá-lhe acesso a determinadas ferramentas de desenvolvimento de software. Você deve ter uma subscrição Azure ative atuais para estudantes para aceder e descarregar as ferramentas de desenvolvedor de software. Vá ao  [Education Hub](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software) para descarregar as ferramentas de desenvolvedor de software utilizando a subscrição do Azure for Students.

### <a name="what-is-microsoft-learn"></a>O que é o Microsoft Learn?

[O Microsoft Learn](/learn/) é uma plataforma de aprendizagem online gratuita que lhe permite aprender tecnologias Azure ao seu próprio ritmo. Os Caminhos de Aprendizagem combinam módulos que permitem começar com o básico e depois avançar para métodos avançados que abordam os desafios do mundo real.

### <a name="can-azure-for-students-be-used-for-production-or-only-for-development"></a>O Azure para estudantes pode ser utilizado para produção ou apenas para desenvolvimento?

O Azure for Students fornece acesso a todos os produtos Azure que se destinam expressamente a apoiar a educação ou o ensino, a investigação não comercial ou os esforços para conceber, desenvolver, testar e demonstrar aplicações de software para estes fins.

### <a name="can-i-apply-any-of-my-100-credit-toward-azure-marketplace-offers"></a>Posso aplicar algum dos meus 100 dólares de crédito para as ofertas do Azure Marketplace?

Não. Não pode aplicar o seu crédito às ofertas do Azure Marketplace. No entanto, muitos parceiros do Azure Marketplace oferecem testes gratuitos ou planos de nível livre para as suas soluções.

## <a name="azure-for-students-starter"></a>Azure for Students Starter

### <a name="what-is-azure-for-students-starter"></a>O que é Azure para estudantes iniciado?

Azure for Students Starter começa com os produtos Azure que precisa para desenvolver na nuvem. Não tem nenhum custo. Este benefício proporciona-lhe acesso a um nível gratuito dos seguintes serviços:

- Serviço de Aplicações do Azure
- Funções do Azure
- Hubs de Notificação do Microsoft Azure
- Base de Dados do Azure para MySQL
- Application Insights
- Serviços Azure DevOps (anteriormente Visual Studio Team Services)

O Azure for Students Starter está disponível para estudantes verificados sem custos e sem compromisso ou prazo limite. Consulte a [Oferta de Arranque Azure para Estudantes](https://azure.microsoft.com/offers/ms-azr-0144p/) para obter termos de utilização detalhados.

### <a name="who-is-eligible-for-azure-for-students-starter"></a>Quem é elegível para Azure para estudantes iniciado?

O Azure for Students Starter está disponível apenas para estudantes que satisfaçam os seguintes requisitos:

- Deve afirmar que tem 13 anos ou mais se reside nos Estados Unidos.
- Deve afirmar que tem 16 anos ou mais se reside num país que não seja os Estados Unidos.
- Tem de verificar o seu estado académico através do endereço de e-mail da sua organização. Você também pode usar Shibboleth se for apoiado pela sua organização.

Esta oferta não está disponível para uso num curso online aberto massivo (MOOC) ou em outras formações profissionais de organizações com fins lucrativos.

Esta oferta está limitada a uma subscrição Azure for Students Starter por cliente elegível. É intransmissível e não pode ser combinado com qualquer outra oferta, a menos que seja permitido em contrário pela Microsoft.

### <a name="will-i-have-to-pay-something-at-any-point-in-time"></a>Terei que pagar algo a qualquer momento?

Um cartão de crédito não é necessário para a oferta de Azure para estudantes iniciador. A oferta dá-lhe acesso a um conjunto limitado de serviços Azure. Mas, a qualquer momento, você pode atualizar para uma subscrição pay-as-you-go para ter acesso a todos os serviços Azure.

### <a name="how-do-i-download-the-software-developer-tools"></a>Como faço o download das ferramentas de desenvolvimento de software?

A subscrição do Azure for Students dá-lhe acesso a determinadas ferramentas de desenvolvimento de software que estão disponíveis para descarregar gratuitamente.

Tem de ter uma subscrição Azure para Estudantes atuais e ativas para aceder às ferramentas de desenvolvimento de software.

Você pode baixar este software no [Education Hub](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).

### <a name="what-is-microsoft-learn"></a>O que é o Microsoft Learn?

[O Microsoft Learn](/learn/) é uma plataforma de aprendizagem online gratuita que permite aprender tecnologias Azure ao seu próprio ritmo. Os caminhos de aprendizagem combinam módulos que permitem começar com o básico e, em seguida, mover-se para métodos avançados que abordam os desafios do mundo real.

## <a name="azure-academic-grant"></a>Bolsa Académica de Azure

### <a name="how-do-i-start-using-my-azure-course-credits"></a>Como começo a usar os meus créditos do curso de Azure?

Pode aceder aos créditos do seu curso Azure criando uma nova subscrição do Microsoft Azure Academic Grant. Selecione o botão  **Ativar** no e-mail de aprovação do patrocínio.

Também pode converter uma subscrição existente na Oferta de Patrocínio do Microsoft Azure para aceder aos seus créditos. Os detalhes sobre como converter a sua subscrição estão na próxima pergunta.

### <a name="can-i-associate-my-course-credits-with-an-existing-subscription"></a>Posso associar os meus créditos de curso a uma subscrição existente?

Pode associar os créditos do seu curso a uma subscrição existente na conta que tem direito à oferta. Contacte [o Azure Support](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) para associar os créditos do seu curso.

### <a name="why-do-i-see-a-0-balance-in-the-azure-portal"></a>Por que vejo um saldo de $0 no portal Azure?

Quando for aos detalhes da subscrição no portal Azure, verá $0,00. A oferta coloca um desconto de 100% em todos os serviços. Portanto, o portal mostra-lhe o que será cobrado durante o seu período de utilização mensal, que deve ser $0,00.

Para ver o seu saldo e informações de patrocínio, vá aos [Patrocínios da Azure](https://www.microsoftazuresponsorships.com/balance) e inscreva-se na sua conta.

### <a name="can-i-apply-my-course-credits-to-an-existing-enterprise-agreement-ea"></a>Posso aplicar os meus créditos de curso a um Acordo Empresarial existente (EA)?

Não é possível associar a sua oferta de créditos de curso a qualquer subscrição que esteja numa conta ao abrigo de um Acordo Empresarial (EA).

Para aplicar os créditos do curso, você deve criar uma nova conta que está fora da EA, que podemos então dar direito.

Uma vez terminada a parte do período de patrocínio, pode associar essa subscrição de volta à EA.

> [!WARNING]
> Se associar a conta à sua EA antes do fim do patrocínio, todos os fundos de patrocínio serão encerrados. Consulte os termos e condições da Oferta de Patrocínio da [Azure](https://azure.microsoft.com/offers/ms-azr-0143p/) para mais informações.

### <a name="can-i-apply-my-course-credits-to-my-csp-subscription"></a>Posso aplicar os meus créditos de curso à minha assinatura CSP?

Não, não é possível associar a sua oferta de créditos de curso a qualquer subscrição do Cloud Solution Provider (CSP).

### <a name="can-i-move-my-course-credits-to-another-account"></a>Posso transferir os meus créditos para outra conta?

Sim, pode mover um curso de créditos de direito a outra conta. Para tal, contacte [o Suporte Azure](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

### <a name="are-azure-marketplace-applications-covered-by-course-credits"></a>As aplicações do Azure Marketplace estão cobertas por créditos de curso?

Não, as candidaturas de terceiros não são cobertas por créditos de curso. Serão cobrados no cartão de crédito da conta. Para obter mais informações:

- Ver serviços de marketplace no [Azure Marketplace.](https://azuremarketplace.microsoft.com/marketplace/)
- Consulte os termos e condições da [oferta de patrocínio da Azure.](https://azure.microsoft.com/offers/ms-azr-0143p/)

### <a name="i-have-a-previous-balance-due-can-i-pay-it-off-with-my-course-credits"></a>Tenho um saldo anterior, posso pagar com os meus créditos de curso?

Os créditos do curso Azure cobrem apenas o uso a partir do momento em que ativar a oferta de patrocínio da Azure. É responsável por todas as despesas antes da data de início da oferta.

### <a name="how-do-i-know-if-my-subscription-is-on-the-azure-course-credit-offer"></a>Como sei se a minha assinatura está na oferta de crédito do curso de Azure?

Se olhar para uma subscrição específica na lâmina de Subscrições no seu portal Azure, verá **o Nome da Oferta** como uma das propriedades. O Nome da Oferta dirá **patrocínio da Azure** se estiver ligado aos créditos do seu curso. Se **o Patrocínio Azure** não apresentar, contacte o suporte para convertê-lo.

### <a name="how-do-i-download-software"></a>Como faço o download de software?

A sua subscrição Microsoft Azure Dev Tools for Teaching fornece-lhe acesso a determinadas ferramentas de desenvolvimento de software. Estas ferramentas estão disponíveis para descarregar gratuitamente.

Você pode baixar este software na [secção De Software Education Hub](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software).

### <a name="how-do-we-distribute-software-to-our-students"></a>Como distribuímos software aos nossos alunos?

Como assinante da Azure Dev Tools for Teaching, a sua escola ou instituição tem acesso à nossa Loja de Centros educativos. Os seus alunos acedem aos seus serviços de nuvem na [Loja Do Centro educativo,](https://azureforeducation.microsoft.com/devtools)que está no [portal Azure.](https://portal.azure.com/)

Os alunos inscrevem-se no portal Azure com as suas credenciais escolares (ou Azure Dev Tools for Teaching). Em seguida, os alunos abrem a Education Hub Store e acedem aos downloads de software disponíveis.

### <a name="is-azure-dev-tools-for-teaching-available-internationally"></a>A Azure Dev Tools for Teaching está disponível internacionalmente?

Sim, está disponível nos mais de 140 países/regiões onde a Azure está comercialmente disponível.

### <a name="which-languages-are-available-in-the-software-to-end-users"></a>Que idiomas estão disponíveis no software para os utilizadores finais?

A Education Hub Store está disponível nas seguintes línguas: Árabe, Chinês Simplificado, Tradicional Chinês, Dinamarquês, Holandês, Inglês, Francês, Alemão, Hebraico, Italiano, Japonês, Coreano, Português, Russo, Espanhol, Sueco e Turco.

### <a name="if-our-students-download-software-through-the-azure-dev-tools-for-teaching-program-do-they-get-unlimited-use-of-the-software"></a>Se os nossos alunos descarregarem software através do programa Azure Dev Tools for Teaching, eles recebem uso ilimitado do software?

Sim. Os estudantes recebem uso ilimitado de software para promover os seus esforços de aprendizagem e pesquisa.

### <a name="if-our-students-create-viable-apps-and-products-using-azure-dev-tools-for-teaching-software-can-they-sell-them-commercially"></a>Se os nossos alunos criarem aplicações e produtos viáveis utilizando o software Azure Dev Tools for Teaching, podem vendê-los comercialmente?

Geralmente, não. Os estudantes não podem vender aplicativos e produtos feitos usando ferramentas Azure Dev para ensinar. No entanto, graças a uma parceria entre as equipas Azure Dev Tools for Teaching e Microsoft App Store, os estudantes podem criar jogos e aplicações para vender na Windows Store.

### <a name="do-i-have-unlimited-use-of-the-software-through-the-azure-dev-tools-for-teaching-program"></a>Tenho uso ilimitado do software através do programa Azure Dev Tools for Teaching?

Sim. Se um docente estiver matriculado num curso aprovado, é elegível para instalar o software Azure Dev Tools for Teaching no seu computador pessoal para uso não comercial.

### <a name="how-do-i-access-my-visual-studio-enterprise-benefit"></a>Como acesso ao meu benefício visual Studio Enterprise?

Como administrador da subscrição Azure Dev Tools for Teaching, pode aceder à subscrição da Visual Studio Enterprise solicitando acesso através do [portal Azure Dev Tools for Teaching Management](https://azureforeducation.microsoft.com/account/Subscriptions).

Uma vez aprovado, poderá iniciar sôms no [portal Visual Studio](https://my.visualstudio.com/) e resgatar benefícios adicionais.

### <a name="does-microsoft-azure-dev-tools-for-teaching-include-microsoft-office"></a>As ferramentas de dev do Microsoft Azure Dev incluem o Microsoft Office?

Não. O foco da Microsoft Azure Dev Tools for Teaching é fornecer aos departamentos, docentes e alunos as ferramentas necessárias para expandir especificamente o seu estudo de desenvolvimento e teste de software. Por isso, fornecemos tecnologias como Windows Server, Visual Studio .NET, SQL Server e Platform SDK.

### <a name="does-azure-dev-tools-for-teaching-include-azure-credit"></a>As ferramentas Azure Dev para o ensino incluem o Crédito Azure?

Não, a sua subscrição microsoft Azure Dev Tools for Teaching não inclui o crédito Azure. Mas pode inscrever-se no Azure for Students, o que lhe dá crédito de $100USD para usar para pagar os serviços da Azure. Vá para [começar a construir o futuro com Azure para estudantes!](https://aka.ms/student) para obter mais informações.

### <a name="do-students-need-an-office-365-or-active-directory-account-to-access-azure-dev-tools-for-teaching"></a>Os alunos precisam de uma conta do Office 365 ou do Ative Directory para aceder às Ferramentas de Ensino do Azure Dev?

Não. Os estudantes não precisam de uma conta do Escritório 365. Se os alunos tiverem acesso à sua conta Ative Directory, usam as mesmas credenciais para iniciar scontabilidade no software.

Se os estudantes não utilizarem o Ative Directory, devem criar uma [Conta Microsoft](https://account.microsoft.com/account) (se ainda não tiverem uma) utilizando o mesmo endereço de e-mail que os fornece.

### <a name="why-arent-my-sign-in-credentials-recognized-when-i-sign-in-to-azure-dev-tools-for-teaching"></a>Porque é que o meu sinal de credenciais não é reconhecido quando inscrevo a Azure Dev Tools for Teaching?

Certifique-se de que está a tentar inscrever-se no Azure Dev Tools for Teaching com as suas credenciais escolares. Pode ajudar a abrir uma sessão de janela de navegação privada.

Se ainda não conseguir iniciar sôs- ão, contacte a sua administração de subscrição. Para encontrar o seu administrador de subscrição, [contacte-nos.](https://aka.ms/adt4tsupport)

### <a name="how-do-i-find-my-subscriber-id"></a>Como encontro o meu ID assinante?

- **Quando se inscreve pela primeira vez no programa: o** número de ID do assinante está no e-mail de boas-vindas por subscrição que recebe.
- **Se renovou a sua subscrição:** O seu ID de Assinante está no e-mail de renovação que o administrador de subscrição recebeu.

O seu ID assinante também está no portal de subscrição do Estúdio Visual. Depois de iniciar sôm, procure na **página 'A Minha** **Conta'** a minha assinatura.

Se precisar de ajuda para localizar o seu ID assinante, [contacte-nos](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="are-we-automatically-enrolled-in-azure-dev-tools-for-teaching-if-we-receive-it-as-part-of-our-academic-volume-licensing-agreement"></a>Estamos automaticamente inscritos na Azure Dev Tools for Teaching se a recebermos como parte do nosso acordo de licenciamento de volume académico?

Não, a Microsoft não o inscreve automaticamente se tiver um acordo de licenciamento de volume académico. O contrato de licença de volume académico inclui:

- Inscrições para Soluções educativas (EES)
- Acordo de Assinatura de Valor Aberto para Soluções educativas (OVS-ES)
- Acordo do Campus
- Acordo Escolar

Deve inscrever-se no Azure Dev Tools for Teaching utilizando os códigos promocionais apropriados a partir do e-mail de boas-vindas por subscrição que recebe para a sua licença de volume académico.

Também deve renovar a sua subscrição quando expirar. Não se renova automaticamente.

Se não conseguir localizar o seu código promocional, [contacte-nos](https://azureforeducation.microsoft.com/institutions/Contact).

### <a name="how-and-when-do-we-renew-our-azure-dev-tools-for-teaching-subscription"></a>Como e quando renovamos a nossa assinatura Azure Dev Tools for Teaching?

Sessenta dias antes da sua adesão expirar, começará a receber lembretes de e-mail para renovar a sua subscrição. Num e-mail de renovação, selecione o [link de renovação](https://portal.azureforeducation.microsoft.com/).

Se não receber estes e-mails de lembrete e estiver preocupado que a sua subscrição esteja prestes a expirar, [contacte-nos](https://aka.ms/adt4tsupport).

Em alternativa, vá ao [Azure Dev Tools for Teaching - Renovar a página](https://portal.azureforeducation.microsoft.com/)e complete o processo de renovação.

Para verificar a data de validade da sua subscrição, aceda ao [portal Azure Dev Tools for Teaching Management](https://azureforeducation.microsoft.com/account/Subscriptions)e consulte as **Assinaturas** .

### <a name="what-if-i-need-more-help"></a>E se eu precisar de mais ajuda?

[Contacte-nos](https://azureforeducation.microsoft.com/institutions/Contact) indo à nossa página de Apoio às Assinaturas e localizando a sua região.

A [Declaração de Privacidade da Microsoft](https://privacy.microsoft.com/PrivacyStatement) descreve os dados pessoais que a Microsoft recolhe, como processa esses dados e porque partilha esses dados.

Esta declaração de privacidade abrange uma série de produtos da Microsoft, incluindo as suas aplicações, dispositivos, servidores, serviços, software e websites. Também fornece informações específicas do produto e detalha a sua política de utilização de cookies.
