# <a name="what-is-azure-dev-tools-for-teaching"></a>O que é o Azure Dev Tools for Teaching?

O Microsoft Azure Dev Tools for Teaching coloca ferramentas profissionais de desenvolvimento, software e serviços da Microsoft nas mãos de docentes e estudantes com subscrições de baixo custo. Os estudantes recebem ferramentas de desenvolvimento sem custos -- tudo o que é necessário para criar apps, jogos e websites -- para que possam perseguir os seus sonhos, criar o próximo grande avanço na tecnologia, ou começar a sua carreira.

Como administrador da subscrição Azure Dev Tools for Teaching, poderá:

- Gerir e aceder a software através de uma localização online centralizada que elimina a necessidade de manter uma WebStore ou um site interno.

- Dar aos utilizadores académicos acesso a software, materiais de formação/aprendizagem gratuitos e à plataforma cloud Azure através do mesmo portal online.

## <a name="program-details"></a>Detalhes do programa

Desenvolvemos o Azure Dev Tools for Teaching para ensino orientado para STEM. Qualquer programa curricular de curso orientado para ciência, tecnologia, engenharia ou matemática é elegível para utilizar o Azure Dev Tools for Teaching para ajudar os professores a ensinar e os estudantes a aprender mais eficazmente. 

A sua subscrição Microsoft Azure Dev Tools for Teaching fornece-lhe acesso a determinadas ferramentas de desenvolvimento de software. Estas ferramentas estão disponíveis para descarregar gratuitamente. Se for um docente matriculado num curso aprovado, pode instalar o software Azure Dev Tools para ensino no seu computador pessoal para uso não comercial.

A Azure Dev Tools for Teaching está disponível nos mais de 140 países/regiões onde o Azure está comercialmente disponível. A Education Hub Store está disponível nas seguintes línguas: Árabe, Chinês Simplificado, Tradicional Chinês, Dinamarquês, Holandês, Inglês, Francês, Alemão, Hebraico, Italiano, Japonês, Coreano, Português, Russo, Espanhol, Sueco e Turco.

## <a name="student-benefits"></a>Benefícios para os estudantes

Todo o software é acessível através de um [Education Hub](https://azureforeducation.microsoft.com/devtools) (integrado no [portal Azure)](https://portal.azure.com/)que alberga todos os produtos e serviços disponíveis para os utilizadores académicos. Os alunos acedem ao software simplesmente iniciando sessão no portal Azure com as mesmas credenciais que usam para aceder à sua subscrição Imagine.

Os estudantes não precisam de se inscrever para uma subscrição do Azure para aceder aos seus benefícios de software. No entanto, a Microsoft oferece USD$100 em crédito Azure mais acesso a serviços gratuitos de Azure para estudantes: os estudantes podem inscrever-se na [oferta do Azure for Students](azure-students-program.md) sem precisar de um cartão de crédito.
