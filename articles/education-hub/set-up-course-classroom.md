# <a name="quickstart-set-up-a-course-and-create-a-classroom"></a>Quickstart: Crie um curso e crie uma sala de aula

Este quickstart explica como configurar um curso e sala de aula no Microsoft Azure Education Hub, incluindo detalhes de subscrição.

## <a name="prerequisites"></a>Pré-requisitos

- Uma subvenção académica com um valor de crédito aprovado
- Uma subscrição dentro da sala de aula que aceda ao seu crédito Azure

### <a name="subscriptions"></a>Subscrições

A cada aluno é dada uma assinatura ligada a um limite monetário de crédito atribuído pelo professor. O termo *limite monetário* descreve a quantia de US$ de um patrocínio académico. Por exemplo, um limite monetário de $1.000 fornece ao destinatário um crédito Azure de USD$1.000 usando [as taxas WebDirect publicadas.](https://azure.microsoft.com/pricing/calculator/)

O educador pode optar por atribuir um limite às subscrições para evitar a utilização não intencional e, em seguida, fixar uma data de validade. Alguns exemplos:

- **Valor fixo por turma**: Cada aluno recebe *$x* para gerir durante todo o trimestre ou semestre.

- **Montante fixo por grupo de projeto**: Uma equipa de projeto recebe $*x* para financiar um projeto de grupo.

Ao nível da subscrição, pode aumentar ou diminuir a tampa e as alterações entrarão em vigor com a latência mínima. Quando a classe ou o projeto termina, pode realocar a tampa não reutilizada para outras subscrições antes da data de validade.

## <a name="create-a-course-and-classroom"></a>Criar um curso e sala de aula

Siga estes passos para criar um curso e sala de aula utilizando um de dois métodos diferentes:

1. Selecione a página **cursos** no Azure Education Hub para abrir a ferramenta que utiliza para criar e gerir cursos. Uma mesa abre mostrando todos os seus cursos existentes.

    :::image type="content" source="media/set-up-course-classroom/education-hub-courses.png" alt-text="Página de cursos do Hub de Educação Azure" border="false":::

1. Selecione o ícone **+ Adicione** o ícone no canto superior esquerdo da tabela para iniciar o fluxo de trabalho da criação.

    :::image type="content" source="media/set-up-course-classroom/courses-add.png" alt-text="Adicione um curso ao Azure Education Hub" border="false":::

1. Você pode criar uma lista de cursos usando dois métodos: carregando uma lista, ou usando um código de convite.
    - **Lista**: Se já tiver os nomes e logins de todos os alunos, pode preencher e carregar um ficheiro de lista. Para descarregar um ficheiro de amostra do ficheiro .csv necessário para fazer o upload da lista, selecione o link **de ficheiro de amostra de Descarregamento** no canto superior direito.
    - **Código de convite**: Se optar por utilizar um código de convite, decida quantos códigos podem ser resgatados e quando expirarão. Enviará aos seus alunos o seguinte link para resgatar o código: https://aka.ms/joinedu .

      :::image type="content" source="media/set-up-course-classroom/enter-invitation-code.png" alt-text="Insira o seu código de convite no Azure Education Hub" border="false":::

1. Selecione **Criar Sala de Aula** no canto inferior direito. Isto pode levar alguns momentos para ser concluído.

   :::image type="content" source="media/set-up-course-classroom/create-classroom.png" alt-text="Criar uma sala de aula no Azure Education Hub" border="false":::
