Todos os serviços do Azure incluem determinados limites e quotas predefinidos para os recursos e funcionalidades. Este artigo detalha as quotas predefinidas e os limites do Azure Container Instances.

A disponibilidade de recursos do Azure Container Instances e recursos em regiões do Azure, veja [de disponibilidade de recursos do Azure Container Instances](container-instances-region-availability.md).
