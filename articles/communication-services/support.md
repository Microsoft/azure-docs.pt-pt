# <a name="help-and-support-options-for-azure-communication-services"></a>Opções de ajuda e apoio para serviços de comunicação da Azure

Se tem um problema ou uma questão que envolva os Serviços de Comunicação Azure, então tem muitas opções.

## <a name="create-an-azure-support-request"></a>Criar um pedido de apoio Azure
Com o Azure, existem muitas [opções de suporte e planos](https://azure.microsoft.com/support/plans/) disponíveis, que você pode explorar e rever. Pode criar um bilhete de apoio no [portal Azure.](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview)

## <a name="post-a-question-to-microsoft-qa"></a>Publique uma pergunta ao Microsoft Q&A

Para respostas rápidas e fiáveis a produtos ou questões técnicas que possa ter sobre os Serviços de Comunicação Azure da Microsoft Engineers, Azure Most Valuable Professionals (MVPs), ou da nossa comunidade, envolva-se connosco no [Microsoft Q&A](/answers/products/azure). 

Se não conseguir encontrar uma resposta para o seu problema pesquisando pode, envie uma nova pergunta ao Microsoft Q&A. Ao criar uma pergunta, certifique-se de utilizar a [Etiqueta dos Serviços de Comunicação Azure](/answers/topics/azure-communication-services.html).

## <a name="post-a-question-on-stack-overflow"></a>Publique uma pergunta sobre Stack Overflow

Você também pode tentar fazer a sua pergunta sobre Stack Overflow, que tem uma grande comunidade de desenvolvedores comunitários e ecossistema. A Azure Communication Services também tem uma [etiqueta dedicada.](https://stackoverflow.com/questions/tagged/azure-communication-services)
