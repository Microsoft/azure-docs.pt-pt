## <a name="create-a-resource-group"></a>Criar um grupo de recursos

1. No ecrã principal do portal Azure, selecione **Criar um recurso**. O ecrã do Marketplace aparecerá.
1. Selecione **Grupos de recursos**.  Aparecerá uma lista de grupos de recursos.
1. Selecione **Adicionar**. Aparecerá o ecrã de grupo de recursos Create.
1. Selecione a subscrição que pretende utilizar para este grupo de recursos.
1. Introduza o nome do grupo de recursos no campo **grupo de Recursos.**
1. Selecione a **Região** para o grupo de recursos.
1. Selecione **Rever + criar**.
