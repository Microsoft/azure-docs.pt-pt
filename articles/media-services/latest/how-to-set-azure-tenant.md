# <a name="how-to-find-your-azure-tenant-id"></a>Como encontrar a sua ID do seu inquilino Azure

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Encontre a sua ID de inquilino no portal

1. Inicie sessão no [portal do Azure](https://portal.azure.com).
1. Selecione o ícone Azure Ative Directory sob os serviços Azure. A página do Diretório Ativo será aberta.
1. A identificação do seu inquilino está localizada em **Informação Básica.**
1. Copie a identificação do inquilino e cole-a num documento de texto à sua escolha para utilização posterior.
