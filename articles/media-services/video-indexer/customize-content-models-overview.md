# <a name="customizing-content-models-in-video-indexer"></a>Personalizando modelos de conteúdo no Video Indexer

O Video Indexer permite-lhe personalizar alguns dos seus modelos para ser adaptado ao seu caso de uso específico. Estes modelos incluem [marcas](customize-brands-model-overview.md), [linguagem](customize-language-model-overview.md), e [pessoa](customize-person-model-overview.md). Pode personalizar facilmente estes modelos com o indexador de vídeo ou do site da API.

Este artigo fornece ligações para os artigos conceptuais que explicam os benefícios de cada tipo de personalização. Este artigo também contém ligações para guias de procedimentos que mostram como pode implementar a personalização de cada modelo.
