# <a name="customizing-content-models-in-video-indexer"></a>Personalizar modelos de conteúdo no Video Indexer

O Video Indexer permite-lhe personalizar alguns dos modelos para se adaptarem ao seu caso de utilização específico. Estes modelos incluem [marcas,](customize-brands-model-overview.md) [linguagem](customize-language-model-overview.md)e [pessoa.](customize-person-model-overview.md) Pode personalizar facilmente estes modelos através da API ou do site do Video Indexer.

Este artigo dá links a artigos que explicam os benefícios de cada tipo de personalização. O artigo também se liga a guias de como fazer que mostram como pode implementar a personalização de cada modelo.
