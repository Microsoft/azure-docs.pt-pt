# <a name="troubleshoot-azure-migrate"></a>Resolver problemas do Azure Migrate

[A Azure Migrate](migrate-services-overview.md) fornece um centro de ferramentas para avaliação e migração, bem como ofertas de fornecedores de software independentes de terceiros (ISV). Este artigo ajuda-o a resolver problemas com Azure Migrate, Azure Migrate Server Assessment e Azure Migrate Server Migration.

## <a name="how-do-i-create-or-find-a-project"></a>Como posso criar ou encontrar um projeto?

Reveja o [guia de resolução de problemas do projeto Azure Migrate](troubleshoot-project.md).

## <a name="i-cant-get-the-appliance-working"></a>Não consigo pôr o aparelho a funcionar.

Reveja [as respostas a questões comuns](troubleshoot-appliance-discovery.md) com a colocação do aparelho.

## <a name="machines-arent-discovered"></a>As máquinas não são descobertas.

Reveja questões comuns de descoberta.

## <a name="app-discovery-isnt-working"></a>A descoberta de aplicativos não está a funcionar

A descoberta de apps, funções e funcionalidades em execução em máquinas no local é atualmente apenas suportada para VMware VMs. [Reveja erros comuns](troubleshoot-appliance-discovery.md#common-app-discovery-errors) para a descoberta de aplicações.

## <a name="assessment-isnt-working"></a>A avaliação não está a resultar.

[Reveja](troubleshoot-assessment.md) questões e erros comuns de avaliação.
