# <a name="azure-analysis-services-samples"></a>Amostras de Serviços de Análise Azure

Use os seguintes recursos de amostra para ajudá-lo a aprender e testar serviços de análise para o seu ambiente.

## <a name="code-samples"></a>Exemplos de código

O repositório [de Serviços](https://github.com/Microsoft/Analysis-Services) de Análise no GitHub inclui amostras de código de código de código de código de código aberto e projetos comunitários. 

## <a name="tabular-model-project-and-database-samples"></a>Modelo tabular e amostras de base de dados

[Adventure Works for Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) on GitHub é o projeto de modelo tabular de amostra mais utilizado. Você pode baixar um projeto VS ou uma base de dados de modelos tabulares de amostra concluída.

## <a name="sample-database-on-azure"></a>Base de dados de amostras em Azure

A Azure Synapse Analytics fornece uma base de dados AdventureWorksDW que pode ser incluída num recurso aprovisionado. Para saber mais, consulte [Quickstart: Criar e consultar uma piscina SQL dedicada em Azure Synapse Analytics ](../synapse-analytics/sql-data-warehouse/create-data-warehouse-portal.md).

## <a name="sample-databases-on-github"></a>Bases de dados de amostras no GitHub

Estas bases de dados de amostras no GitHub podem ser usadas para criar e testar os seus próprios modelos. 

[Bases de dados de amostras adventure works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks)

[Bases de dados de amostras de importadores mundiais](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Modelo de amostra de vendas de Internet Adventure Works em portal Azure

Se tiver um servidor de Serviços de Análise no portal Azure, pode criar de forma rápida e fácil um modelo de amostragem. Na página geral do seu servidor no portal Azure, clique em **Novo modelo** e, em seguida, em Escolha uma fonte **de dados**, selecione **dados de amostra**.
