# <a name="azure-network-security"></a>Segurança de rede do Azure
## <a name="abstract"></a>Abstrato
Os serviços de rede do Azure maximizam a flexibilidade, a disponibilidade, a resiliência, a segurança e a integridade por design. Este white paper fornece detalhes sobre as funções de rede do Azure. Ele também descreve como os clientes podem usar os recursos de segurança nativos no Azure para ajudar a proteger seus ativos de informações.
Os públicos-alvo para este white paper incluem:
- Gerentes técnicos, administradores de rede e desenvolvedores que estão procurando soluções de segurança que estão disponíveis e têm suporte no Azure.
- SMEs ou executivos de processos de negócios que desejam uma visão geral de alto nível das tecnologias e dos serviços do Azure relacionados à segurança de rede na nuvem pública do Azure.
