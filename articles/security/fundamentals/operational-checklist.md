# <a name="azure-operational-security-checklist"></a>Lista de verificação de segurança operacional azure
A implementação de uma aplicação no Azure é rápida, fácil e rentável. Antes de implementar uma aplicação em nuvem em produção útil para ter uma lista de verificação para ajudar na avaliação da sua aplicação contra uma lista de ações de segurança operacional essenciais e recomendadas para que possa considerar.

## <a name="introduction"></a>Introdução

O Azure disponibiliza um conjunto de serviços de infraestrutura que pode utilizar para implementar as suas aplicações. A Segurança Operacional Azure refere-se aos serviços, controlos e funcionalidades disponíveis para os utilizadores para proteger os seus dados, aplicações e outros ativos no Microsoft Azure.

-   Para obter o máximo benefício da plataforma cloud, recomendamos que aproveite os serviços Azure e siga a lista de verificação.
-   As organizações que investem tempo e recursos que avaliam a prontidão operacional das suas aplicações antes do lançamento têm uma taxa de satisfação muito maior do que as que não o fazem. Ao realizar este trabalho, as listas de verificação podem ser um mecanismo inestimável para garantir que as aplicações são avaliadas de forma consistente e holisticamente.
-   O nível de avaliação operacional varia consoante o nível de maturidade da nuvem da organização e a fase de desenvolvimento da aplicação, as necessidades de disponibilidade e os requisitos de sensibilidade de dados.

Esta lista de verificação destina-se a ajudar as empresas a pensar através de várias considerações de segurança operacional, à medida que implementam aplicações empresariais sofisticadas no Azure. Também pode ser usado para ajudá-lo a construir uma estratégia segura de migração e operação em nuvem para a sua organização.

## <a name="conclusion"></a>Conclusão
Muitas organizações implementaram e operaram com sucesso as suas aplicações em nuvem no Azure. Recomendamos vivamente estas considerações operacionais e estratégicas para as suas implementações de aplicações existentes e novas no Azure.
