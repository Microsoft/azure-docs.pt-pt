# <a name="azure-operational-security-checklist"></a>Lista de verificação de segurança operacional Azure
Implementar uma aplicação no Azure é rápido, fácil e rentável. Antes de implementar a aplicação em nuvem na produção útil para ter uma lista de verificação para ajudar na avaliação da sua aplicação contra uma lista de ações de segurança operacionais essenciais e recomendadas para que possa considerar.

## <a name="introduction"></a>Introdução

O Azure disponibiliza um conjunto de serviços de infraestrutura que pode utilizar para implementar as suas aplicações. A Azure Operational Security refere-se aos serviços, controlos e funcionalidades disponíveis para os utilizadores para proteger os seus dados, aplicações e outros ativos no Microsoft Azure.

-   Para obter o máximo benefício da plataforma cloud, recomendamos que aproveite os serviços Azure e siga a lista de verificação.
-   As organizações que investem tempo e recursos avaliando a prontidão operacional das suas aplicações antes do lançamento têm uma taxa de satisfação muito maior do que aquelas que não o fazem. Ao realizar este trabalho, as listas de verificação podem ser um mecanismo inestimável para garantir que as aplicações são avaliadas de forma consistente e holisticamente.
-   O nível de avaliação operacional varia consoante o nível de maturidade na nuvem da organização e a fase de desenvolvimento da aplicação, necessidades de disponibilidade e requisitos de sensibilidade aos dados.

Recomendamos vivamente estas considerações operacionais e estratégicas para as suas implementações existentes e novas aplicações no Azure.
