# <a name="a-practical-guide-to-designing-secure-health-care-solutions-in-azure"></a>Um guia prático para a criação de soluções dos cuidados de saúde seguras no Azure
Start-UPS do Estado de funcionamento da indústria, integradores de sistemas (SIs), fornecedores independentes de software (ISVs) e as organizações de cuidados de saúde Considerando uma mudança para o Azure está procurando por diretrizes que ajuda a incorporam controlos de segurança para satisfazer as suas obrigações reguladoras de conformidade.

[Um guia prático para criar soluções de cuidados de saúde seguras no Microsoft Azure](https://aka.ms/azureindustrysecurity) ajudam a compreender como pode aumentar a segurança para suas soluções com o serviços do Azure e as funcionalidades que pode configurar com base nos seus requisitos.
O conteúdo é dividido em três seções principais:

1. Orientações de considerações para a utilização de tecnologia de cloud, incluindo gestão de riscos, partilhado responsabilidade, estabelecimento de um sistema de gestão de segurança de informações, a indústria de compreensão e a legislação local e o estabelecimento de procedimentos operacionais padrão.
2. Princípios de segurança de chaves que são alinhados com um padrão de gestão de segurança de informações padrão, como a ISO 27001 e os processos de desenvolvimento padrão, por exemplo, da Microsoft Security Development Lifecycle (SDL).
3. A aplicar os princípios fundamentais para casos de utilização com a demonstração de alinhamento de uma perspectiva do arquiteto de solução, onde os requisitos para as soluções estão alinhados para a gestão de segurança de informações padrão.

Esperamos que encontre [um guia prático para criar soluções seguras de cuidados de saúde](https://aka.ms/azureindustrysecurity) útil e se tiver dúvidas ou sugestões, informe-por deixar um comentário abaixo.
