# <a name="azure-security-documentation"></a>Documentação da Segurança do Azure

A segurança integra todos os aspetos do Azure, sem exceção. O Azure oferece vantagens de segurança exclusivas que derivam de inteligência de segurança global, sofisticados controlos orientados para o cliente e de uma infraestrutura protegida segura. Esta poderosa combinação ajuda a proteger os seus dados e aplicações, confere suporte aos seus esforços de conformidade e garante segurança económica para organizações de todas as dimensões.
