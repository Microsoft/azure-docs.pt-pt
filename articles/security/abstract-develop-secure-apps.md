# <a name="develop-secure-applications-on-azure"></a>Desenvolver aplicações seguras no Azure
## <a name="abstract"></a>Abstrato
Este documento é um guia Geral para as perguntas de segurança e controles, que deve considerar em cada fase do ciclo de vida de desenvolvimento de software durante o desenvolvimento de aplicativos para a nuvem. Implementar esses conceitos, antes de lançar seu produto pode ajudá-lo a criar softwares mais seguros. As recomendações apresentadas neste artigo são fornecidos a nossa experiência com a segurança do Azure e as experiências dos nossos clientes.

Este documento destina-se para ser um recurso para designers de software, desenvolvedores e testadores em todos os níveis que criar e implementar soluções do Azure seguras.
