# <a name="azure-operational-security"></a>Segurança operacional do Azure
## <a name="abstract"></a>Abstrato
Segurança operacional do Microsoft Azure refere-se a serviços, controles e recursos disponíveis aos utilizadores para proteger seus dados, aplicativos e outros recursos no Azure. Segurança operacional do Azure baseia-se uma estrutura que incorpore os dados recolhidos através de vários recursos que são exclusivos da Microsoft, incluindo o Microsoft Security Development Lifecycle (SDL), o programa Microsoft Security Response Center, e o conhecimento profundo do campo das ameaças de cibersegurança. Este white paper descreve como pode abordar a segurança operacional ao utilizar o Azure. Ele aborda vários serviços do Azure, incluindo:
* Azure Log Analytics
* Azure Backup
* Centro de Segurança do Azure
* Azure Monitor
* Observador de rede do Azure
* Análise do Armazenamento do Azure
* Azure Active Directory
