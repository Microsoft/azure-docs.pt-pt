# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Versão de atualização da Threat Modeling Tool 7.1.51023.1 - 11-01-2018

Como originalmente notado nas notas de lançamento da [GA,](threat-modeling-tool-releases-71509112.md)lançámos uma atualização (7.1.51023.1) para a Ferramenta de Modelação de Ameaças da Microsoft que exigirá aos utilizadores da versão Preview (clientes de pré-visualização com versão < 7.1.50911.2) para atualizar para o lançamento da GA suportada. Esta versão não contém nenhuma nova funcionalidade ou correções.

- Os utilizadores da versão Preview descarregarão automaticamente a atualização quando o cliente for aberto. Se optar por não instalar a nova atualização, a versão 'Preview' da ferramenta encerrará.
- Os utilizadores da versão GA da ferramenta serão solicitados a escolher se querem ou não fazer o upgrade.
- Os utilizadores da versão de 2016 da ferramenta não serão afetados.

## <a name="feature-changes"></a>Alterações de funcionalidades

- Nenhum

## <a name="system-requirements"></a>Requisitos de sistema

- Sistemas Operativos Suportados
  - Microsoft Windows 10
- versão .NET necessária
  - .NET 3.5.2
- Requisitos Adicionais
  - É necessária uma ligação à Internet para receber atualizações da ferramenta, bem como modelos.
