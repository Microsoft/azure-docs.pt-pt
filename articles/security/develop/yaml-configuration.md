# <a name="yaml-configuration-options-to-customize-the-build-tasks"></a>Opções de configuração YAML para personalizar as tarefas de construção

Este artigo lista todas as opções de configuração YAML disponíveis em cada uma das tarefas de construção. O artigo começa com as tarefas para ferramentas de análise de códigode segurança. Termina com as tarefas pós-processamento.

|------------|---------------|-----------------------|----------|---------------------------|----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| InputType | pickList | sempre | Verdadeiro | Básico | Básico, Personalizado | 
| ScanType | pickList | Tipo de entrada = Básico | Verdadeiro | CustomScan | CustomScan, FullSystemScan, QuickScan, YourConfiguredScan | O tipo de digitalização a utilizar para a varredura AntiMalware.
