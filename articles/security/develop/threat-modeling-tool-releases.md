# <a name="threat-modeling-tool-releases"></a>Lançamentos de ferramentas de modelação de ameaças

A Ferramenta de Modelação de Ameaças da Microsoft é atualmente lançada como uma aplicação de [clique para download](https://aka.ms/threatmodelingtool) gratuita para o Windows. Este mecanismo de entrega permite-nos impulsionar as mais recentes melhorias e correções de bugs para os clientes sempre que abrem a ferramenta.

## <a name="system-requirements"></a>Requisitos de Sistema

- Sistemas Operativos Suportados
  - Atualização de Aniversário do Microsoft Windows 10 ou mais tarde
- .VERSÃO NET Necessária
  - .NET 4.7.1 ou mais tarde
- Requisitos Adicionais
  - É necessária uma ligação à Internet para receber atualizações da ferramenta e dos modelos.
