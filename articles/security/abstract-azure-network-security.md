# <a name="azure-network-security"></a>Segurança de rede do Azure
## <a name="abstract"></a>Abstrato
Serviços de rede do Azure maximizar a flexibilidade, disponibilidade, resiliência, segurança e integridade por predefinição. Este white paper fornece detalhes sobre as funções de sistema de rede do Azure. Ele também descreve como os clientes podem utilizar as funcionalidades de segurança nativo no Azure para ajudar a proteger seus ativos de informações.
Os públicos-alvo previstos deste white paper incluem:
- Gestores de técnicos, os administradores de rede e os desenvolvedores que estiver à procura de soluções de segurança que estão disponíveis e suportam no Azure.
- Os SMEs ou os executivos de processo de negócios que pretendem uma visão geral das tecnologias do Azure e serviços que estão relacionadas com segurança de rede na cloud pública do Azure.
