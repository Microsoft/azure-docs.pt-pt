# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Mapeamento do Benchmark benchmark de segurança Azure v2 e CIS Microsoft Azure Foundations Benchmark

O CIS Microsoft Azure Foundations Benchmark é a orientação de segurança fornecida pelo Center for Internet Security para estabelecer uma configuração de base segura para o Azure. O âmbito do benchmark é estabelecer o nível de segurança da fundação enquanto adota a Azure Cloud. Este benchmark está em alinhamento com o [Azure Security Benchmark v2](overview.md).

Fornecemos um mapeamento entre a versão mais recente do [CIS Microsoft Azure Foundations Benchmark v1.3.0](https://www.cisecurity.org/benchmark/azure/) e [Azure Security Benchmark v2](overview.md). Isto irá ajudá-lo a entender como as recomendações para ambos os parâmetros estão alinhadas. Você pode baixar o mapeamento completo [aqui.](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)
