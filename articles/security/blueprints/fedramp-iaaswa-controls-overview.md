# <a name="fedramp-overview"></a>Descrição geral de FedRAMP

O [risco Federal e o programa de gestão de autorização (FedRAMP)](https://www.fedramp.gov), é um programa de toda a government E.U.A. que fornece uma abordagem padronizada para avaliação de segurança, autorização e monitorização contínua para produtos de nuvem e serviços. FedRAMP define um processo de avaliação de segurança com base num conjunto padronizado de requisito derivado [National Institute of Standards e Technology (NIST) SP de publicação especial 800-53](https://csrc.nist.gov/publications/detail/sp/800-53/rev-4/final) controlos de segurança.

Procure as famílias de controlo abaixo para ver como esta segurança do Azure e a conformidade Blueprint automatização está alinhada com os requisitos de controlo.
