# <a name="azure-data-factory-whitepapers"></a>Papel branco da Azure Data Factory 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

Os whitepapers permitem-lhe explorar a Azure Data Factory a um nível mais profundo. Este artigo fornece-lhe uma lista de papels brancos disponíveis para Azure Data Factory.

| **Papel branco** | **Descrição** |
| --- | --- |
|[Azure Data Factory — Integração de Dados na Nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Este trabalho descreve como a Azure Data Factory pode permitir-lhe construir um moderno armazém de dados, permitir análises avançadas para conduzir aplicações inteligentes de SaaS e elevar os seus pacotes de Serviços de Integração de Servidores SQL para o Azure.|
|[Migração de dados do Armazém de Dados relacionais no local para a Azure utilizando a Azure Data Factory](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | Este artigo aborda a complexidade da migração de dezenas de dados de Tuberculose do armazém de dados relacionais existentes no local (por exemplo, Netezza, Oracle, Teradata, servidor SQL) para Azure (por exemplo, Blob Storage ou Azure Data Lake Storage) usando a Azure Data Factory. Os desafios e boas práticas são ilustrados em torno da resiliência, desempenho, escalabilidade, gestão e segurança para a viagem de big data ingestion a Azure by Azure Data Factory. |
|[Integração de dados SAP utilizando a Azure Data Factory](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | Este trabalho resume o suporte atual da Azure Data Factory na integração de dados SAP, incluindo cenário-alvo, opções de conector SAP e comparação para diferentes requisitos, e introdução em cada conector SAP na Data Factory. | 
|[Fábrica de Dados Azure: SSIS na Nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Este artigo explica por que razão pretende migrar as suas cargas de trabalho SSIS existentes para a Azure Data Factory e abordar considerações e preocupações comuns. Em seguida, iremos acompanhá-lo através dos detalhes técnicos da criação de um Azure-SSIS IR e depois mostrar-lhe como carregar, executar e monitorizar os seus pacotes através da Azure Data Factory usando as ferramentas que provavelmente está familiarizado com o SQL Server Management Studio (SSMS). |
|[Fábrica de dados Azure – Parâmetros de passagem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Este artigo percorre como passar parâmetros entre um oleoduto e atividade, bem como entre as atividades. |
|[Fábrica de dados Azure –DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Este papel branco cobre algumas das melhores práticas em torno da integração contínua e implantação da Azure Data Factory. |
