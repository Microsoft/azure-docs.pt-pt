# <a name="azure-data-factory---customer-case-studies"></a>Azure Data Factory - Estudos de caso de cliente
Data Factory é um serviço de gestão de informação baseado na nuvem que automatiza o movimento e transformação de dados. Os clientes de muitas indústrias utilizam a Data Factory e outros serviços Azure para construir os seus oleodutos de análise e resolver os seus problemas de negócio.  Aprenda diretamente com os nossos clientes como e por que estão a usar a Data Factory.
