# <a name="azure-powershell-samples-for-azure-data-factory"></a>Amostras da Azure PowerShell para a Azure Data Factory

A tabela seguinte inclui links para provar scripts Azure PowerShell para a Azure Data Factory.

| |  |
|---|---|
|**Copiar dados**||
|[Copie bolhas de uma pasta para outra pasta num Armazenamento De Blob Azure](scripts/copy-azure-blob-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Este script PowerShell copia bolhas de uma pasta no Armazenamento De Blob Azure para outra pasta no mesmo Blob Storage. |
