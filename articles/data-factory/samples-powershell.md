A tabela a seguir inclui links para scripts de Azure PowerShell de exemplo para Azure Data Factory.

| |  |
|---|---|
|**Copiar dados**||
|[Copiar blobs de uma pasta para outra pasta em um armazenamento de BLOBs do Azure](scripts/copy-azure-blob-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Este script do PowerShell copia blobs de uma pasta no armazenamento de BLOBs do Azure para outra pasta no mesmo armazenamento de BLOBs. |
