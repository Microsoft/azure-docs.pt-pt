# <a name="jenkins-plugins-for-azure"></a>Plug-ins de Jenkins para Azure

Os plug-ins Jenkins seguintes suportam várias funcionalidades para utilização com o Azure.
  
| Plug-in Jenkins | Descrição                                   |
|------------------------------------------------------------------------------|
| [Plug-in do serviço de aplicações do Azure](https://plugins.jenkins.io/azure-app-service)     | Um plug-in do Jenkins para implementar um serviço de aplicações do Azure (atualmente suporta apenas a aplicação Web). | 
| [Plug-in do Azure AD](https://plugins.jenkins.io/azure-ad)                       | Um plug-in do Jenkins que suporta a autenticação e autorização através do Azure Active Directory. | 
| [Plug-in de Agentes de Contentor do Azure](https://plugins.jenkins.io/azure-container-agents) | Plugin de agentes de contentor do Azure pode ajudá-lo para executar um contentor como um agente no Jenkins | 
| [Plug-in do Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Um plug-in do Jenkins para implementar as configurações para o Azure Container Service (AKS). | 
| [Plug-in de Credencial do Azure](https://plugins.jenkins.io/azure-credentials)      | Plug-in Jenkins para gerir as credenciais do Azure. | 
| [Plug-in de Função do Azure](https://plugins.jenkins.io/azure-function)           | Para utilizar este plug-in para implementar a função do Azure, primeiro tem de ter um Principal de serviço do Azure na sua instância do Jenkins. | 
| [Plug-in do Service Fabric do Azure](https://plugins.jenkins.io/service-fabric)     | Um plug-in do Jenkins para projetos de Azure Service Fabric do Linux. |
| [Plug-in do Armazenamento do Azure](https://plugins.jenkins.io/windows-azure-storage)     | Um plug-in para o carregamento de artefactos de compilação para ou baixar dependências de compilação do armazenamento de Blobs do Microsoft Azure. | 
| [Plugin de agentes VM do Azure](https://plugins.jenkins.io/azure-vm-agents)         | Um plug-in do Jenkins para criar agentes do Jenkins nas máquinas virtuais do Azure (por meio do modelo do Azure Resource Manager). | 
| [Plug-in do conjunto de dimensionamento de máquina virtual do Azure](https://plugins.jenkins.io/azure-vmss)           | Define um plug-in do Jenkins para implementar imagens de VM para o dimensionamento de máquina virtual do Azure. |
