# <a name="jenkins-plugins-for-azure"></a>Plugins Jenkins para Azure

Os seguintes plugins Jenkins suportam várias funcionalidades para utilização com o Azure.
  
| Plugin Jenkins | Descrição                                   |
|------------------------------------------------------------------------------|
| [Plugin do Serviço de Aplicações Azure](https://plugins.jenkins.io/azure-app-service)     | O plugin jenkins para implementar um Serviço de Aplicações Azure (atualmente suporta apenas web app). | 
| [Plug-in do Azure AD](https://plugins.jenkins.io/azure-ad)                       | O plugin Jenkins que suporta a autenticação & autorização via Azure Ative Directory. | 
| [Plugin do Gestor de Artefactos Azure](https://plugins.jenkins.io/azure-artifact-manager) | O plugin Azure Artifact Manager é um Gestor de Artefactos que lhe permite armazenar os seus artefactos no Armazenamento De Blob Azure. O plugin Azure Artifact Manager trabalha de forma transparente para o Jenkins e para os seus trabalhos, é como o Gestor de Artefactos padrão. | 
| [Plug-in de Agentes de Contentor do Azure](https://plugins.jenkins.io/azure-container-agents) | O plugin dos agentes de contentores Azure pode ajudá-lo a gerir um contentor como agente em Jenkins | 
| [Plugin de tarefa de registo de contentores azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | O plugin jenkins para enviar um pedido de construção de estivadores para o Registo de [Contentores Azure.](/azure/container-registry/container-registry-tasks-overview) |
| [Plug-in do Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Plugin Jenkins para implementar configurações para o Serviço de Contentores Azure (AKS). | 
| [Plug-in de Credencial do Azure](https://plugins.jenkins.io/azure-credentials)      | Jenkins plugin para gerir credenciais Azure. | 
| [Plug-in de Função do Azure](https://plugins.jenkins.io/azure-function)           | O plugin jenkins para implementar uma Função Azure. | 
| [Plugin Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | O plugin Azure IoT Edge facilita a instalação de um pipeline CI/CD para desenvolver uma solução IoT Edge em Jenkins. | 
| [Plug-in do Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Plugin Jenkins para projetos de Tecido de Serviço Linux Azure. |
| [Plug-in do Armazenamento do Azure](https://plugins.jenkins.io/windows-azure-storage)     | plugin para carregar artefactos de construção para, ou descarregar dependências de construção do armazenamento Microsoft Azure Blob. | 
| [Plugin de agentes Azure VM](https://plugins.jenkins.io/azure-vm-agents)         | Plugin Jenkins para criar agentes Jenkins em máquinas virtuais Azure (via modelo de Gestor de Recursos Azure). | 
| [Plugin de conjunto de escala de máquina virtual Azure](https://plugins.jenkins.io/azure-vmss)           | O plugin jenkins para implantar imagens VM em conjuntos de escala de máquinas virtuais Azure. |
