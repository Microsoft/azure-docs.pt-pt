# <a name="jenkins-plugins-for-azure"></a>Plug-ins de Jenkins para Azure

Os plug-ins Jenkins seguintes suportam várias funcionalidades para utilização com o Azure.
  
| Plug-in Jenkins | Descrição                                   |
|------------------------------------------------------------------------------|
| [Plug-in do serviço de aplicações do Azure](https://plugins.jenkins.io/azure-app-service)     | Plug-in do Jenkins para implementar um serviço de aplicações do Azure (atualmente suporta apenas a aplicação Web). | 
| [Plug-in do Azure AD](https://plugins.jenkins.io/azure-ad)                       | Plug-in Jenkins que suporta a autenticação e autorização através do Azure Active Directory. | 
| [Plug-in do Gestor de artefactos do Azure](https://plugins.jenkins.io/azure-artifact-manager) | Plug-in do Gestor de artefactos do Azure é um Gestor de artefacto que permite que armazenar os artefactos no armazenamento de Blobs do Azure. Funciona de Azure Plug-in de Gestor de artefacto de forma transparente para o Jenkins e as tarefas, é como a predefinição do Gestor de artefacto. | 
| [Plug-in de Agentes de Contentor do Azure](https://plugins.jenkins.io/azure-container-agents) | Plugin de agentes de contentor do Azure pode ajudá-lo para executar um contentor como um agente no Jenkins | 
| [Plug-in de tarefa de registo de contentor do Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Plug-in do Jenkins para enviar uma solicitação de compilação do docker para [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Plug-in do Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Plug-in Jenkins para implementar as configurações para o Azure Container Service (AKS). | 
| [Plug-in de Credencial do Azure](https://plugins.jenkins.io/azure-credentials)      | Plug-in Jenkins para gerir as credenciais do Azure. | 
| [Plug-in de Função do Azure](https://plugins.jenkins.io/azure-function)           | Plug-in Jenkins para implementar uma função do Azure. | 
| [Plug-in do IoT Edge do Azure](https://plugins.jenkins.io/azure-iot-edge)           | Plug-in do IoT Edge do Azure torna fácil de configurar um pipeline de CI/CD para o desenvolvimento de uma solução de IoT Edge no Jenkins. | 
| [Plug-in do Service Fabric do Azure](https://plugins.jenkins.io/service-fabric)     | Plug-in Jenkins para projetos de Azure Service Fabric do Linux. |
| [Plug-in do Armazenamento do Azure](https://plugins.jenkins.io/windows-azure-storage)     | Plug-in para o carregamento de artefactos de compilação para ou baixar dependências de compilação do armazenamento de Blobs do Microsoft Azure. | 
| [Plugin de agentes VM do Azure](https://plugins.jenkins.io/azure-vm-agents)         | Plug-in Jenkins para criar agentes do Jenkins nas máquinas virtuais do Azure (por meio do modelo do Azure Resource Manager). | 
| [Plug-in do conjunto de dimensionamento de máquina virtual do Azure](https://plugins.jenkins.io/azure-vmss)           | Define o plug-in do Jenkins para implementar imagens de VM para o dimensionamento de máquina virtual do Azure. |
