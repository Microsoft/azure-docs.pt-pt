# <a name="avs-node-quota-limits"></a>Limites de quota do nó AVS

Quatro nomes é a quantidade padrão disponível para compra quando a sua subscrição está ativada para o serviço AVS. Pode comprar qualquer [tipo de nó](cloudsimple-node.md) no portal Azure. Pelo menos três nódoas do mesmo SKU são necessárias para criar uma Nuvem Privada AVS. Se tiver comprado os nós, poderá ver um erro quando tentar comprar nós adicionais.

## <a name="quota-increase"></a>Aumento da quota

Pode aumentar a quota do nó submetendo um pedido de apoio. A equipa de operações de serviço irá avaliar o pedido e trabalhar consigo para aumentar a quota do nó. Selecione as seguintes opções quando abrir um novo bilhete:

* Tipo de emissão: **Técnico**
* Subscrição: **O seu ID de subscrição**
* Tipo de serviço: **Solução VMware por AVS**
* Tipo de problema: **Quota de nódedicados dedicados**
* Subtipo de problemas: **Aumentar a quota de nódosos dedicados**
* Objeto: **Aumento de quota**

Nos detalhes do bilhete de apoio, forneça o número necessário de nós e nó SKU.

* Nó SKU
* Número de nós adicionais para os quais está a solicitar o aumento da quota
