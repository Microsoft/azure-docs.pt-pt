# <a name="cloudsimple-node-quota-limits"></a>Limites de quota de nó CloudSimple

Quatro nós é a quantidade padrão disponível para compra quando a sua subscrição está ativada para o serviço CloudSimple.  Pode adquirir qualquer [tipo de nó](cloudsimple-node.md) no portal Azure.  Pelo menos três nós do mesmo SKU são necessários para criar uma Nuvem Privada.  Se tiver comprado os nós, poderá ver um erro quando tentar comprar nós adicionais.

## <a name="quota-increase"></a>Aumento da quota

Pode aumentar a quota de nó submetendo um pedido de apoio. A equipa de operações de serviço avaliará o pedido e trabalhará consigo para aumentar a quota de nó.  Selecione as seguintes opções quando abrir um novo bilhete:

* Tipo de emissão: **Técnico**
* Subscrição: **O seu ID de subscrição**
* Tipo de serviço: **Solução VMware by CloudSimple**
* Tipo de problema: **Quota de nó dedicado**
* Subtipo de problema: **Aumentar quota de nosdes dedicados**
* Objeto: **Aumento de quota**

Nos detalhes do bilhete de apoio, forneça o número necessário de nós e nó SKU.

* Nó SKU
* Número de nós adicionais para os quais está a solicitar o aumento da quota
