# <a name="azure-vmware-solution-by-cloudsimple"></a>Solução VMware no Azure da CloudSimple

Bem-vindo à Solução VMware no Azure da CloudSimple. Encontrará detalhes sobre como utilizar a CloudSimple e ativar soluções no serviço. Pesquise artigos ou navegue no índice.

- Veja a [Descrição geral](cloudsimple-vmware-solutions-overview.md) para saber mais sobre a Solução VMware no Azure da CloudSimple.
- Selecione [Descrição geral da cloud privada da CloudSimple](cloudsimple-private-cloud.md) para saber mais sobre a solução de cloud privada da CloudSimple.
- Selecione [Nós](cloudsimple-node.md) para saber mais sobre os nós da CloudSimple.
- Selecione [Inicializar e aprovisionar capacidade](quickstart-create-cloudsimple-service.md) para começar a utilizar a Solução VMware no Azure da CloudSimple.

| Síntese de conceitos     | Recurso                    | Objetivo                                                                       | Gestão         |
|--------------------------|-----------------------------|-------------------------------------------------------------------------------|--------------------|
| **Criação de Serviços**     | Serviço CloudSimple         | Configurar a Rede Edge de Serviço da CloudSimple                                    | Portal do Azure       |
|                          | Nó da CloudSimple            | Aprovisionar capacidade para utilizar no serviço da CloudSimple                          | Portal do Azure       |
| **Consumo de Serviços**  | Cloud Privada               | Criar ambientes VMware                                                    | Portal da CloudSimple |
|                          | VLAN/Sub-rede                 | Criar redes subjacentes para o consumo do VMware                               | Portal da CloudSimple |
|                          | Tabela de Firewall              | Configurar regras de segurança norte/sul e este/oeste em redes subjacentes | Portal da CloudSimple |
|                          | Endereço IP público           | Obter endereços destinados ao público para cargas de trabalho do VMware                           | Portal da CloudSimple |
|                          | Ligação do Azure            | Estabelecer conectividade a redes no local ou a Redes Virtuais no Azure         | Portal da CloudSimple |
| **Gestão Unificada**   | Máquina Virtual do CloudSimple | Gerir Máquinas Virtuais VMware no portal do Azure                                | Portal do Azure       |
