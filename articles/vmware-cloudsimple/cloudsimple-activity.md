# <a name="activity-management-overview"></a>Visão geral do gerenciamento de atividades

A AVS acompanha toda a atividade que pode afetar o funcionamento do seu ambiente AVS Private Cloud. As atividades incluem alertas, eventos, tarefas e atividade de auditoria. As [páginas de atividade](monitor-activity.md) resumem toda a atividade atual e permitem que você faça uma busca detalhada para obter detalhes adicionais.

## <a name="events"></a>Eventos

Os eventos rastreiam a atividade do utilizador e do sistema no portal AVS. Os eventos mostram a atividade associada a um recurso específico e a gravidade do impacto.  Pode ver os eventos a partir do portal AVS.

## <a name="alerts"></a>Alertas

Os alertas são notificações de qualquer atividade significativa no seu ambiente AVS. Os eventos que afetam a cobrança ou o acesso do usuário são mostrados como alertas. Pode reconhecer alertas do portal AVS.

## <a name="tasks"></a>Tarefas

As tarefas rastreiam qualquer operação de usuário que leva mais de 30 segundos para ser concluída. Pode monitorizar o progresso de uma tarefa a partir do portal AVS. Para tarefas concluídas, as informações incluem o tempo total para a conclusão.

## <a name="audit"></a>Auditoria

Os logs de auditoria controlam as operações do usuário. Os logs de auditoria contêm os parâmetros fornecidos para a operação pelo usuário. Você pode usar os logs de auditoria para monitorar a atividade do usuário para todos os usuários.
