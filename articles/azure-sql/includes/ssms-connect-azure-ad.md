> [!NOTE]
> Em dezembro de 2021, os lançamentos de SSMS antes do dia 18.6 deixarão de autenticar através do Azure Ative Directy com o MFA.
>
> Para continuar a utilizar a autenticação do Azure Ative Directory com MFA, necessita de [SSMS 18.6 ou posterior](/sql/ssms/download-sql-server-management-studio-ssms).
