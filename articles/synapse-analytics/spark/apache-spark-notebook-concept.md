# <a name="azure-synapse-analytics-notebooks"></a>Cadernos Azure Synapse Analytics

Um notebook Do Synapse Studio é uma interface web para criar ficheiros que contenham código ao vivo, visualizações e texto narrativo. Os cadernos são um bom local para validar ideias e usar experiências rápidas para obter informações dos seus dados. 

Com um caderno Azure Synapse Studio, você pode:

* Começa com nenhum esforço de preparação.
* Mantenha os dados seguros com funcionalidades de segurança da empresa incorporadas.
* Analise os dados em formatos brutos (CSV, txt, JSON, etc.), formatos de ficheiros processados (parquet, Delta Lake, ORC, etc.), e ficheiros de dados tabulares SQL contra Spark e SQL.
* Seja produtivo com capacidades de autoria melhoradas e visualização de dados incorporados.

Esta secção contém artigos sobre mistura de línguas, criando visualizações de dados, cadernos parametrizantes, oleodutos de construção, entre outros. Também contém referências e tutoriais sobre como pode começar com o desenvolvimento do seu caderno.
