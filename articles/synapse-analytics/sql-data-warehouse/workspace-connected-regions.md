# <a name="enabling-synapse-workspace-features-on-a-dedicated-sql-pool-formerly-sql-dw-in-your-region"></a>Ativar as funcionalidades do espaço de trabalho da Synapse numa piscina SQL dedicada (anteriormente SQL DW) na sua região.

## <a name="regions-not-supported"></a>Regiões não apoiadas 
O quadro seguinte resume por região onde não é suportada a capacidade de ativar as características do espaço de trabalho Synapse numa piscina SQL dedicada existente (anteriormente SQL DW). Volte a verificar quando a sua região fica disponível.

\* indica que um calendário específico para a região está atualmente indisponível.

| Austrália Central 2 |\* |
| China Leste 2 |\* |
| China Norte 2 |\* |
| Índia do Sul |\*|
| Oeste da Índia |\*|
| Sul da Coreia do Sul |\* |
| Suíça Oeste |\* |
| Uae Norte |\*|
 
 
> [!NOTE]
> A lista de regiões acima referida diz apenas respeito às regiões onde não existem atualmente funcionalidades de espaço de trabalho da Sinapse.
