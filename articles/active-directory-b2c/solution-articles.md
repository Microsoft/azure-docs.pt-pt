# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Soluções e Formação para o Diretório Ativo Azure B2C

O Azure Ative Directory B2C (Azure AD B2C) permite que as organizações implementem soluções de negócio que as ajudem a conectar-se com os seus clientes. Os seguintes guias de solução e formação são documentos transferíveis que irão acompanhá-lo através destas soluções.

| Título | Descrição |
| ----- | ----------- |
| [Gestão de Identidade do Cliente com Azure AD B2C](https://channel9.msdn.com/Shows/On-NET/Customer-Identity-Management-with-Azure-AD-B2C) | Nesta visão geral do serviço, parakh[@jainparakh](https://twitter.com/jainparakh)Jain ( ) da equipa Azure AD B2C fornece-nos uma visão geral de como o serviço funciona, e também mostra como podemos ligar rapidamente o B2C a uma aplicação ASP.NET Core. |
| [Benefícios da utilização do Azure AD B2C](https://aka.ms/b2coverview) | Compreenda os benefícios e cenários comuns do Azure AD B2C e como a sua aplicação pode alavancar este serviço CIAM. |
| [Adquirir Experiência em Azure AD B2C: Um curso para desenvolvedores](https://aka.ms/learnAADB2C) | Este curso de ponta a ponta leva os desenvolvedores através de uma jornada completa no desenvolvimento de aplicações com o Azure AD B2C como mecanismo de autenticação. Dez módulos aprofundados com laboratórios cobrem tudo, desde a criação de uma subscrição Azure até à criação de políticas personalizadas que definam as viagens que envolvem os seus clientes. |
| [Permitindo parceiros, fornecedores e clientes acederem a aplicações com diretório ativo Azure](https://aka.ms/aadexternalidentities) | O sucesso de cada organização, independentemente da sua dimensão, indústria ou postura de conformidade e segurança, depende da capacidade organizacional de colaborar com outras organizações e de se conectar com os clientes.<br><br>Reunindo a Azure AD, AD B2C e a Azure AD B2B Collaboration, este guia detalha o valor do negócio e a mecânica da construção de uma aplicação ou experiência web que proporciona uma experiência de autenticação consolidada adaptada aos contextos da sua colaboradores, parceiros de negócios e fornecedores, e clientes. |
| [Autenticação de aplicação migratória para Azure AD B2C em um ambiente híbrido](https://aka.ms/MigratetoAADB2C) | Nas organizações modernas de hoje, a transformação digital e a mudança para a nuvem acontecem por etapas, exigindo que a maioria das organizações operem temporariamente num ambiente de identidade híbrida.<br><br>Este guia centra-se na criação do plano de migração para mover a sua primeira aplicação para O AD B2C do Azure, e cobre as considerações para fazê-lo enquanto está num ambiente de identidade híbrida. |
| [Considerações gerais do Regulamento Geral de Proteção de Dados (RGPD) para aplicações viradas para o cliente](https://aka.ms/AADB2CandGDPR) | Para qualquer cliente que se depara com aplicações, o RGPD deve ser tomado em consideração por todas as organizações que embarcam em projetos que possuam dados pessoais e servem os cidadãos da UE.<br><br>Este guia de soluções centra-se na forma como o Azure AD B2C pode ser usado como uma componente flexível da sua abordagem global de conformidade com o RGPD, incluindo como os componentes Azure AD B2C suportam cada um dos principais direitos do RGPD para os indivíduos. |
