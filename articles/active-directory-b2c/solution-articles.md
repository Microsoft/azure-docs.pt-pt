# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Soluções e treinamento para Azure Active Directory B2C

O Azure Active Directory B2C (Azure AD B2C) permite que as organizações implementem soluções de negócios que os ajudem a se conectar com seus clientes. Os guias de solução e o treinamento a seguir são documentos que podem ser baixados, que o orientarão nessas soluções.

| Cargo | Descrição |
| ----- | ----------- |
| [Gerenciamento de identidade do cliente com Azure AD B2C](https://channel9.msdn.com/Shows/On-NET/Customer-Identity-Management-with-Azure-AD-B2C) | Nesta visão geral do serviço, Parakh Jain ([@jainparakh](https://twitter.com/jainparakh)) da equipe de Azure ad B2C nos fornece uma visão geral de como o serviço funciona e também mostra como podemos conectar o B2C rapidamente a um aplicativo ASP.NET Core. |
| [Benefícios do uso do Azure AD B2C](https://aka.ms/b2coverview) | Entenda os benefícios e cenários comuns do Azure AD B2C e como seus aplicativos podem aproveitar esse serviço CIAM. |
| [Obtendo experiência em Azure AD B2C: Um curso para desenvolvedores](https://aka.ms/learnAADB2C) | Este curso de ponta a ponta leva os desenvolvedores por uma jornada completa sobre o desenvolvimento de aplicativos com Azure AD B2C como o mecanismo de autenticação. Dez módulos detalhados com laboratórios abrangem tudo, desde a configuração de uma assinatura do Azure até a criação de políticas personalizadas que definem as jornadas que envolvem seus clientes. |
| [Permitindo que parceiros, fornecedores e clientes acessem aplicativos com o Azure Active Directory](https://aka.ms/aadexternalidentities) | O sucesso de cada organização, independentemente de seu tamanho, setor ou postura de conformidade e segurança, depende da capacidade organizacional de colaborar com outras organizações e conectar-se aos clientes.<br><br>Reunindo o Azure AD, o Azure AD B2C e a colaboração B2B do Azure AD, este guia detalha o valor comercial e a mecânica de criação de um aplicativo ou experiência na Web que fornece uma experiência de autenticação consolidada adaptada aos contextos de seu funcionários, parceiros de negócios e fornecedores e clientes. |
| [Migrando a autenticação de aplicativo para Azure AD B2C em um ambiente híbrido](https://aka.ms/MigratetoAADB2C) | Nas organizações modernas atuais, a transformação digital e a mudança para a nuvem ocorrem em estágios, exigindo que a maioria das organizações opere temporariamente em um ambiente de identidade híbrida.<br><br>Este guia concentra-se na criação do plano de migração para mover seu primeiro aplicativo para Azure AD B2C e aborda as considerações para fazer isso enquanto estiver em um ambiente de identidade híbrida. |
| [Considerações gerais sobre normas de proteção de dados (GDPR) para aplicativos voltados para o cliente](https://aka.ms/AADB2CandGDPR) | Para qualquer aplicativo voltado para o cliente, o GDPR deve ser levado em consideração por todas as organizações que embarcam em projetos que contêm dados pessoais e servem cidadãos da UE.<br><br>Este guia de solução concentra-se em como Azure AD B2C pode ser usado como um componente flexível da abordagem de conformidade geral do GDPR, incluindo como os componentes do Azure AD B2C dão suporte a cada um dos principais direitos do GDPR para indivíduos. |
