# <a name="azure-percept-studio-overview"></a>Visão geral do estúdio Azure Percept

O Azure Percept Studio é o único ponto de lançamento para criar modelos e soluções de IA de ponta. O estúdio Azure Percept permite-lhe descobrir e completar fluxos de trabalho guiados que facilitam a integração de hardware capaz de IA de borda e poderosos serviços de nuvem Azure AI e IoT.

## <a name="features"></a>Funcionalidades

No estúdio, pode ver os dispositivos capazes de AI de edge como pontos finais para recolher dados de treino iniciais e em curso, bem como alvos de implementação para iterações de modelos. Ter acesso a dispositivos e dados de formação permite um rápido desenvolvimento de modelos de AI edge e iterativos para cenários de [visão](./tutorial-nocode-vision.md) e [fala.](./tutorial-no-code-speech.md)

![Fluxo de projeto da página inicial no Azure Percept Studio.](./media/overview-azure-percept-studio/percept-studio-flow.png)

Os fluxos de trabalho no Azure Percept Studio integram muitos serviços subjacentes a Azure AI e IoT, como o Azure IoT Hub, Custom Vision, Speech Studio e Azure ML Services, para que possa utilizar estes serviços para criar uma solução de ponta a ponta, sem conhecimentos pré-existentes significativos. Se já está familiarizado com estes serviços Azure, também pode ligar e modificar os recursos existentes fora do Azure Percept Studio.

![Fluxo de dispositivo no Azure Percept Studio.](./media/overview-azure-percept-studio/device-flow.png)

Independentemente de ser um principiante ou um programador de soluções e modelos de IA mais avançados, trabalhando num protótipo ou movendo-se para uma solução de produção, para a fala ou visão Edge AI, o Azure Percept Studio oferece acesso a fluxos de trabalho que pode usar para reduzir o atrito em torno da construção de soluções Edge AI.

![Fluxo de captura de imagem no Azure Percept Studio.](./media/overview-azure-percept-studio/image-flow.png)
