# <a name="azure-percept-studio-overview"></a>Visão geral do estúdio Azure Percept

[O Azure Percept Studio](https://go.microsoft.com/fwlink/?linkid=2135819) é o único ponto de lançamento para criar modelos e soluções de IA de ponta. O Azure Percept Studio permite-lhe descobrir e completar fluxos de trabalho guiados que facilitam a integração de hardware capaz de IA e poderosos serviços de nuvem Azure AI e IoT.

No Estúdio, pode ver os seus dispositivos capazes de IA de borda como pontos finais para recolher dados de treino iniciais e em curso, bem como alvos de implementação para iterações de modelos. Ter acesso a dispositivos e dados de formação permite um rápido desenvolvimento de modelos de IA de ponta e de borda iterativa para cenários de [visão](./tutorial-nocode-vision.md) e [fala.](./tutorial-no-code-speech.md)

Os fluxos de trabalho no Azure Percept Studio integram muitos serviços Azure AI e IoT, como Azure IoT Hub, Custom Vision, Speech Studio e Azure ML, para que possa utilizar estes serviços para criar uma solução de ponta a ponta sem conhecimentos pré-existentes significativos. Se já está familiarizado com estes serviços Azure, também pode ligar e modificar os recursos de serviço Azure existentes fora do Azure Percept Studio.

Independentemente de ser um principiante ou um avançado modelo de IA e desenvolvedor de soluções, trabalhando num protótipo, ou movendo-se para uma solução de produção, o Azure Percept Studio oferece acesso a fluxos de trabalho que pode usar para reduzir o atrito em torno de soluções de IA de construção.
