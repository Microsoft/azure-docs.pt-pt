# <a name="advanced-development-with-azure-percept"></a>Desenvolvimento avançado com Azure Percept
Com a Azure Percept, os desenvolvedores de software e cientistas de dados são capazes de usar fluxos de trabalho de código avançados para a gestão do ciclo de vida da IA. Através de uma biblioteca de código aberto em crescimento, podem usar amostras para começar com a sua jornada de desenvolvimento de IA e construir soluções prontas para a produção.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Começar com os tutoriais de desenvolvimento avançado

Este [caderno Jupyter](https://github.com/microsoft/Project-Santa-Cruz-Private-Preview/blob/main/Sample-Scripts-and-Notebooks/Official/Machine%20Learning%20Notebooks/Transferlearningusing_SSDLiteV2%20Model.ipynb) realiza a aprendizagem de transferência, utilizando um modelo tensorFlow pré-treinado (MobileNetSSDV2Lite) em AzureML em Python com um conjunto de dados personalizado para detetar tigelas. Existem duas opções tutoriais para guiá-lo através do trabalho com o caderno para treinar e implementar o seu detetor de taças para o projeto Santa Cruz devkit:

- [Desenvolvimento](./advanced-development-cloud.md)em nuvem : neste tutorial, você executará o caderno no [Portal de Aprendizagem automática Azure](https://ml.azure.com) com uma instância de computação remota.

- [Desenvolvimento local](./advanced-development-local.md): neste tutorial, você executará o caderno localmente dentro do Código VS com uma instância de computação remota.
