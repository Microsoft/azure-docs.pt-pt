# <a name="advanced-development-with-azure-percept"></a>Desenvolvimento avançado com Azure Percept
Com a Azure Percept, os desenvolvedores de software e cientistas de dados são capazes de usar fluxos de trabalho de código avançados para a gestão do ciclo de vida da IA. Através de uma biblioteca de código aberto em crescimento, podem usar amostras para começar com a sua jornada de desenvolvimento de IA e construir soluções prontas para a produção.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Começar com os tutoriais de desenvolvimento avançado

Conheça todos os [modelos Azure Percept AI](./overview-ai-models.md)disponíveis.

Consulte o [Azure Percept DK desenvolvimento avançado GitHub](https://github.com/microsoft/azure-percept-advanced-development) para orientação atualizada, tutoriais e exemplos para coisas como:

* Trazer um modelo de IA personalizado para o dispositivo
* Atualizar um modelo que já apoiamos com a aprendizagem de transferências
* E mais

## <a name="next-steps"></a>Passos seguintes

Conheça todos os [modelos Azure Percept AI](./overview-ai-models.md)disponíveis. Se nenhum destes modelos se adequar às suas necessidades, sinta-se à vontade para usar a viagem de código avançada para levar o seu próprio modelo ou pipeline de visão computacional ao DK percept, e se sentir que ajudaria os outros, abra um pedido de puxar também!
