# <a name="advanced-development-with-azure-percept"></a>Desenvolvimento avançado com Azure Percept

Com a Azure Percept, os desenvolvedores de software e cientistas de dados podem usar fluxos de trabalho de código avançados para a gestão do ciclo de vida da IA. Através de uma biblioteca de código aberto em crescimento, podem usar amostras para começar com a sua jornada de desenvolvimento de IA e construir soluções prontas para a produção.

## <a name="get-started-with-advanced-development"></a>Começar com desenvolvimento avançado

Consulte o desenvolvimento avançado do [Azure Percept DK GitHub](https://github.com/microsoft/azure-percept-advanced-development) para obter orientações atualizadas, tutoriais e exemplos para coisas como:

- Implementando um modelo de IA personalizado para o seu Azure Percept DK
- Atualizar um modelo apoiado com aprendizagem de transferência
- E mais

## <a name="next-steps"></a>Passos seguintes

Saiba mais sobre os [modelos Azure Percept AI](./overview-ai-models.md)disponíveis. Se nenhum destes modelos se adequar às suas necessidades, utilize a viagem de código avançada para levar o seu próprio modelo ou pipeline de visão computacional ao Percept DK. Se tiver uma contribuição que acha que ajudaria os outros, sinta-se livremente a abrir um pedido de atração também.
