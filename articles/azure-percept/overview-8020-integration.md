# <a name="azure-percept-dk-8020-integration-overview"></a>Visão geral da integração de Azure Percept DK 80/20

O Azure Percept DK e o Audio Accessory foram concebidos para se integrarem com o [sistema de construção de alumínio de 80/20 T-slot](https://8020.net/).

## <a name="8020-features"></a>Características de 80/20

A placa de porta-aviões Azure Percept DK, o dispositivo Azure Percept Vision e o acessório Azure Percept Audio são fabricados com conectores 80/20 1010 incorporados, que permitem configurações de montagem intermináveis com trilhos de 80/20. Esta integração permite que clientes e construtores de soluções aumentem mais facilmente a sua prova de conceitos para ambientes de produção.
