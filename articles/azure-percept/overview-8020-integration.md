# <a name="azure-percept-dk-8020-integration-overview"></a>Visão geral da integração de Azure Percept DK 80/20

O Azure Percept DK e o Audio Accessory foram concebidos para se integrarem com o [sistema de gradeamento 80/20](https://8020.net/).

## <a name="8020-features"></a>Características de 80/20

Cada componente de hardware é construído com os entalhes e as saliências para caber no tipo de extrusão 1010. Esta integração permite que clientes e construtores de soluções aumentem mais facilmente a sua prova de conceitos para ambientes de produção.
