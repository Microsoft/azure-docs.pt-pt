# <a name="azure-percept-dk-overview"></a>Visão geral de Azure Percept DK

Azure Percept DK é um kit de desenvolvimento de IA de borda projetado para desenvolver soluções de IA de visão e áudio com [o Azure Percept Studio](./overview-azure-percept-studio.md). Azure Percept DK está disponível para compra na loja online da [Microsoft.](https://go.microsoft.com/fwlink/p/?LinkId=2155270)

> [Comprar agora](https://go.microsoft.com/fwlink/p/?LinkId=2155270)

## <a name="key-features"></a>Principais funcionalidades

- Executar AI na borda. Com a aceleração de hardware incorporada, o dev kit pode executar modelos de IA sem uma ligação à nuvem.

- Raiz de hardware da segurança de confiança incorporada. Saiba mais sobre [a segurança da Azure Percept.](./overview-percept-security.md)

- Integração perfeita com [o Azure Percept Studio](https://go.microsoft.com/fwlink/?linkid=2135819) e outros serviços Azure, tais como Azure IoT Hub, Azure Cognitive Services e [Live Video Analytics](https://docs.microsoft.com/azure/media-services/live-video-analytics-edge/overview).

- Compatível com [a Azure Percept Audio,](./overview-azure-percept-audio.md)um acessório opcional para a construção de soluções de áudio de IA.

- Suporte para ferramentas de IA de terceiros, tais como ONNX e TensorFlow.

- Integração com o sistema de gradeamento 80/20, que permite configurações intermináveis de montagem de dispositivos. Saiba mais sobre a [integração 80/20](./overview-8020-integration.md).

## <a name="hardware-components"></a>Componentes de hardware

- Placa de porta-aviões Azure Percept DK:
    - Processador NXP iMX8m
    - Módulo de Plataforma Fidedigna (TPM) versão 2.0
    - conectividade Wi-Fi e Bluetooth
    - Para mais informações, consulte a [folha de dados do Azure Percept DK](./azure-percept-dk-datasheet.md)

- Azure Percept Vision sistema-on-module (SoM):
    - Intel Movidius Myriad X (MA2085) unidade de processamento de visão (VPU)
    - Sensor de câmara RGB
    - Para mais informações, consulte a folha de dados da [Azure Percept Vision](./azure-percept-vision-datasheet.md)
