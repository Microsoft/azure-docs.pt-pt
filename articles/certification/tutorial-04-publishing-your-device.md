# <a name="tutorial-publish-your-device"></a>Tutorial: Publique o seu dispositivo

Parabéns por ter certificado com sucesso o seu dispositivo! O seu produto está a juntar-se a um ecossistema de dispositivos excecionais que funcionam muito bem com o Azure. Agora que o seu dispositivo foi certificado, pode publicar opcionalmente os detalhes do seu dispositivo no [catálogo Azure Certified Device](https://devicecatalog.azure.com) para um mundo de clientes descobrirem e comprarem.

Neste tutorial, ficará a saber como:

> [!div class="checklist"]
> * Publique o seu dispositivo no catálogo de Dispositivos Certificados Azure

## <a name="prerequisites"></a>Pré-requisitos

- Deverá fazer o seu contrato e ter um projeto **aprovado** para o seu dispositivo no [portal Azure Certified Device](https://certify.azure.com). Se não tiver um dispositivo certificado, pode ver este [tutorial](tutorial-01-creating-your-project.md) para começar.

## <a name="publishing-your-device"></a>Publicar o seu dispositivo

A publicação do seu dispositivo é um processo simples que ajudará a trazer clientes para o seu produto a partir do catálogo de Dispositivos Certificados Azure.

1. Para publicar o seu dispositivo, clique `Publish to Device Catalog` na página de resumo do projeto.

    ![Publicar no Catálogo](./media/images/publish-to-catalog.png)

1. Confirme a publicação na janela pop-up

    ![Publicar para confirmação do Catálogo](./media/images/publish-to-catalog-confirm.png)

1. Receberá a notificação para o endereço de e-mail no seu perfil da empresa assim que o dispositivo tiver sido processado no catálogo de Dispositivos Certificados Azure.

## <a name="next-steps"></a>Passos seguintes

Parabéns! O seu dispositivo certificado faz agora parte do catálogo Azure Certified Device, onde os clientes podem fazer compras para os seus produtos com confiança no seu desempenho com a Azure! Obrigado por fazer parte do nosso ecossistema de produtos IoT certificados. Vai notar que a sua página do projeto está agora apenas para ler. Se desejar estivar alguma informação sobre o seu dispositivo, consulte o nosso guia.
