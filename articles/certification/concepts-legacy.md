# <a name="legacy-devices-on-the-azure-certified-device-catalog"></a>Dispositivos legados no catálogo de dispositivos certificados Azure

Pode ter reparado no catálogo de Dispositivos Certificados Azure que alguns dispositivos não têm o contorno azul ou a etiqueta "Azure Certified Device". Estes dispositivos (apelidados de "dispositivos legados") foram previamente certificados ao abrigo do programa legado.

## <a name="certified-for-azure-iot-program"></a>Certificado para o programa Azure IoT

Antes do lançamento do programa Azure Certified Device, os parceiros de hardware poderiam previamente certificar os seus produtos ao abrigo do programa Certified for Azure IoT. O programa de certificação Azure Certified Device reorienta a sua missão de cumprir as promessas dos clientes em vez das capacidades técnicas do dispositivo.

Os dispositivos certificados como um "dispositivo certificado IoT Hub" aparecem no catálogo de dispositivos certificados Azure como um "dispositivo legado". Esta etiqueta indica que os dispositivos já se qualificaram anteriormente através do programa agora reformado, mas não foram certificados através do programa Azure Certified Device atualizado. Estes dispositivos são claramente notados no catálogo pela sua falta de contorno azul, e podem ser encontrados através do filtro "IoT Hub Certified (legado)".
