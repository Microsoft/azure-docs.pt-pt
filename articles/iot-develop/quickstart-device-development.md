# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Começando com o desenvolvimento de dispositivos incorporados Azure IoT

**Aplica-se a:** [Desenvolvimento de dispositivos embutidos](about-iot-develop.md#embedded-device-development)

Este guia de arranque contém um conjunto de quickstarts que lhe mostram como começar a trabalhar com dispositivos incorporados e Azure IoT. 

Em cada arranque rápido, completa-se as seguintes tarefas básicas:
* Instale um conjunto de ferramentas de desenvolvimento incorporadas para programar um dispositivo específico em C
* Construa uma imagem que inclua componentes e amostras de RTOS Azure e, em seguida, flash um dispositivo
* Ligue um dispositivo de forma segura ao Azure IoT
* Ver telemetria do dispositivo, ver propriedades e invocar métodos nuvem-a-dispositivo
