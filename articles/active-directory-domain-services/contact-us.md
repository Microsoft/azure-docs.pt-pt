# <a name="azure-ad-domain-services---contact-us"></a>Serviços de domínio do Azure AD - contacte-nos

## <a name="feedback"></a>Comentários
Se tiver comentários para nossa perguntas não técnicas ou serviço, pode partilhar o seu pedido com o grupo de produtos do serviço de domínio do Azure AD através do envio-nos para: [Comentários dos serviços de domínio do Azure AD](mailto:aaddsfb@microsoft.com). Mensagens de correio eletrónico para este endereço são revistas. Membros do grupo de produto **poderá** responder para solicitar mais informações. Pedidos de suporte técnico para este endereço de e-mail não obterão respostos.

## <a name="technical-assistance"></a>Assistência técnica
Se tiver problemas com o seu domínio gerido, certifique-se os passos descritos no [guia de resolução de problemas](troubleshoot.md) resolver o problema. Se ainda tiver problemas, inicie sessão no portal do Azure. Abra o **serviços de domínio do Azure AD** home page e selecione **novo pedido de suporte** do painel de ação na **suporte + resolução de problemas**.

![Novo pedido de suporte](./media/contact-us/supportRequest.png) 

Preencha cada seção do pedido de suporte de forma mais precisa possível. Ao escolher um serviço, certifique-se de que seleciona **do Azure Active Directory Domain Services (VM - controladores de domínio)** para encaminhar o pedido para o profissional de suporte adequada.

> [!IMPORTANT]
> Selecione **diretórios do Active Directory do Azure, domínios e objetos** listas de serviços para obter ajuda com a validação do domínio com o Azure Active Directory.
>
>

Os membros do grupo de produto de rever estes pedidos.
