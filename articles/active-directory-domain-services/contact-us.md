# <a name="azure-ad-domain-services---contact-us"></a>Serviços de Domínio Azure AD - Contacte-nos

## <a name="feedback"></a>Comentários
Se tiver feedback para o nosso serviço ou questões não técnicas, pode partilhar o seu pedido com o grupo de produtos Do Serviço de Domínio Azure AD enviando-nos um e-mail para: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com). São revistos os e-mails para este endereço. Os membros do grupo de produtos **podem** responder a solicitar mais informações. Os pedidos de apoio técnico a este endereço de e-mail não serão respondidos.

## <a name="technical-assistance"></a>Assistência técnica
Se tiver problemas com o seu domínio gerido, verifique se os passos descritos no Guia de Resolução de [Problemas](troubleshoot.md) resolvem o problema. Se ainda tiver problemas, inscreva-se no portal Azure. Abra a página inicial dos Serviços de **Domínio Azure AD** e selecione **Novo pedido** de suporte do painel de ação sob suporte **+ resolução de problemas**.

![novo pedido de apoio](./media/contact-us/supportRequest.png) 

Preencha cada secção do pedido de apoio com a maior precisão possível. Ao escolher um serviço, certifique-se de selecionar serviços de domínio de **diretório ativo Azure (VM - Controladores** de domínio) para encaminhar o seu pedido para o profissional de suporte adequado.

> [!IMPORTANT]
> Selecione **Diretórios, Domínios e Objetos Ativos azure** das listas de serviços para ajuda com validação de domínio com Diretório Ativo Azure.
>
> 

## <a name="feature-requests"></a>Pedidos de funcionalidades
Tem alguma ideia de como podemos melhorar os Serviços de Domínio de Diretório Ativo Azure? Gostaríamos de saber. Vá ao Azure Feedback Forums, navegue para a categoria [De serviços](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) de domínio e partilhe a sua ideia. Os membros do grupo de produtos analisam estes pedidos.
