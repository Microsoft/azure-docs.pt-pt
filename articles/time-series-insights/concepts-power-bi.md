# <a name="connect-time-series-insights-to-power-bi"></a>Conecte insights da série de tempo ao Power BI  

Está à procura de uma solução que ajude a correlacionar os seus dados da série de tempo, criar visualizações vívidas e partilhar insights em toda a sua organização?

O Azure Time Series Insights integra-se agora perfeitamente com [o Power BI,](https://powerbi.microsoft.com/)fornecendo-lhe capacidades de visualização e dashboarding mais poderosas sobre os seus dados de streaming e permite-lhe partilhar insights e resultados em toda a sua organização.

Ao ligar os seus dados ao Power BI, pode agora:

* Execute correlações com outras fontes de dados suportadas pelo Power BI e aceda a uma série de diferentes opções de visualização de dados. 
* Crie painéis e relatórios Power BI utilizando os seus dados time series Insight e partilhe-os com a sua organização. 
* Desbloqueie os cenários de interop de dados de uma forma simples e fácil de usar, e obtenha insights mais rapidamente do que nunca.
* Modificar os dados da Série De Tempo insights dentro do Power BI utilizando o poderoso Editor Avançado.

## <a name="native-connector"></a>Conector nativo

A ligação dos seus dados demora menos de um minuto. Ao exportar rapidamente a sua visão do Time Series Insight Explorer para o Power BI, qualquer pessoa na sua organização pode aceder intuitivamente e transformar dados visualmente.

As opções de configuração incluem escolher entre dados **de nível** **bruto** e agregado e selecionar um intervalo de tempo desejado. 

Uma vez que os dados estejam ligados, todas as ricas ferramentas de visualização e gráfico do Power BI estão disponíveis para melhorar ainda mais as suas capacidades de partilha de informação. 

   [![Exemplos de gráficos no Power BI](./media/concepts-connect-power-bi/power-bi-tsi-example.png)](./media/concepts-connect-power-bi/power-bi-tsi-example.png#lightbox)

Leia [Como ligar insights de séries de tempo ao Power BI](./how-to-connect-power-bi.md) para obter especificidades passo a passo.

## <a name="flexible-workflow"></a>Fluxo de trabalho flexível
 
Utilize o conector de qualquer lugar do seu fluxo de trabalho:

* Se estiver no Time Series Insights Explorer, pode exportar rapidamente a sua visão para Power BI. 
* Se já estiver no Power BI, aumente os seus cenários de visualização existentes utilizando dados de consulta de Time Series Insights.

Para uma funcionalidade avançada de consulta e edição dentro do Power BI, utilize o Editor Avançado do Power BI.
