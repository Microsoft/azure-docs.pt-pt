# <a name="discover-experiments-in-cortana-intelligence-gallery"></a>Detetar as experimentações na galeria da Cortana Intelligence
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="experiments-for-machine-learning-studio"></a>Experimentações de Machine Learning Studio
A Galeria tem uma ampla variedade de [experimentações](https://gallery.cortanaintelligence.com/experiments) que tenham sido desenvolvidos no [Azure Machine Learning Studio](https://studio.azureml.net). Intervalo de experimentações de rápida experimentações de prova de conceito que demonstram um específico de machine learning técnica, programadas totalmente as soluções de problemas do complexos de machine learning.

> [!NOTE]
> Um ***experimentação*** é uma tela no Machine Learning Studio, pode utilizar para construir um modelo de Análise Preditiva. Criar o modelo através da ligação de dados com vários módulos analíticos. Pode tente ideias diferentes, não executa de avaliação e, eventualmente, implementar o seu modelo como um serviço web no Azure. Para obter um exemplo de como criar uma experimentação básica, consulte [tutorial do Machine learning: crie a sua primeira experimentação no Azure Machine Learning Studio](create-experiment.md). Para obter instruções mais abrangente de como criar uma solução de Análise Preditiva, consulte [instruções: desenvolver uma solução de Análise Preditiva para a avaliação de riscos de crédito no Azure Machine Learning](walkthrough-develop-predictive-solution.md).
>
>

## <a name="discover"></a>Descobrir
Para procurar experimentações [na galeria](http://gallery.cortanaintelligence.com), na parte superior da home page do galeria, selecione **experimentações**.

O  **[experimentações](https://gallery.cortanaintelligence.com/experiments)**  página apresenta uma lista de experimentações recentemente adicionadas e populares. Para ver todas as experimentações, selecione o **ver todos os** botão. Para procurar uma experimentação específica, selecione **ver todos os**e os critérios de filtro, em seguida, selecione. Também pode introduzir termos de pesquisa no **pesquisa** caixa na parte superior da página da galeria.

Pode obter mais informações sobre uma experimentação na página de detalhes de experimentação. Para abrir uma página de detalhes da experimentação, selecione a experimentação. Numa experimentação detalhes da página, além de **comentários** secção, pode comente, fornecer comentários ou fazer perguntas sobre a experimentação. Ainda pode partilhar a experimentação com amigos ou colegas no Twitter ou LinkedIn. Também pode correio uma ligação à página de detalhes da experimentação, para convidar outros utilizadores para ver a página.

![Partilhar este item com amigos](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Adicionar os seus comentários](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Transferência
Pode transferir uma cópia de quaisquer experimentação na Galeria para a sua área de trabalho do Machine Learning Studio. Em seguida, pode modificar a cópia para criar as suas próprias soluções.

Galeria da Cortana Intelligence oferece duas formas para importar uma cópia de uma experimentação:

* **Na galeria do**. Se encontrar uma experimentação que pretender na galeria, pode transferir uma cópia e, em seguida, abra-o na sua área de trabalho do Machine Learning Studio.
* **A partir do Machine Learning Studio**. No Machine Learning Studio, pode utilizar qualquer experimentação na galeria como um modelo para criar uma nova experimentação.

### <a name="from-the-gallery"></a>Na galeria do

1. Na galeria, abra a página de detalhes de experimentação.
2. Selecione **abrir no Studio**.

    ![Abra experimentação da galeria do](./media/gallery-experiments/open-experiment-from-gallery.png)

Quando seleciona **abrir no Studio**, abre a experimentação na sua área de trabalho do Machine Learning Studio. (Se ainda não tem sessão iniciada Machine Learning Studio, lhe for pedido para primeiro início de sessão utilizando a sua conta Microsoft.)

### <a name="from-within-machine-learning-studio"></a>A partir do Machine Learning Studio

1. No Machine Learning Studio, selecione **novo**.
2. Selecione **experimentação**. Pode escolher entre uma lista de experimentações de galeria, ou localizar uma experimentação específica utilizando o **pesquisa** caixa.
3. Apontar o rato para uma experimentação e, em seguida, selecione **abrir no Studio**. (Para ver informações sobre a experimentação, selecione **ver na galeria**. Isto leva-o para a página de detalhes da experimentação na galeria.)

    ![Abra Galeria experimentação de dentro de Machine Learning Studio](./media/gallery-experiments/open-experiment-from-studio.png)

Pode personalizar, iterar e implementar uma experimentação transferida como quaisquer outra experimentação que criar no Machine Learning Studio.

![Experimentação abrir no Studio](./media/gallery-experiments/experiment-open-in-studio.png)

## <a name="contribute"></a>Contribuir
Quando iniciar sessão para a galeria, pode tornar-se um membro da Comunidade da galeria. Como um membro da Comunidade, pode contribuir as seus próprios experimentações, para que outros utilizadores podem beneficiar das soluções que já descobriu.

### <a name="publish-your-experiment-to-the-gallery"></a>Publicar a sua experimentação para a Galeria

1. Inicie sessão no Machine Learning Studio, utilizando a sua conta Microsoft.
2. Criar a sua experimentação e volte a executar.
3. Quando estiver pronto para publicar a sua experimentação na galeria, na lista de ações abaixo a tela de experimentação, selecione **publicar para a galeria**.

    ![Selecione "publicar Galeria"](./media/gallery-experiments/publish-experiment-to-gallery.png)
4. No **experimentação Descrição** página, introduza um título e etiquetas. Tornar o título e etiquetas descritivo. Realce as técnicas que utilizou ou o problema do mundo real resolver. Um exemplo de um título descritivo experimentação é "classificação binária: análise de dados de sentimento do Twitter."

    ![Introduza o título e etiquetas para publicação](./media/gallery-experiments/experiment-description.png)
5. No **resumo** box, introduza um resumo da sua experimentação. Descreva resumidamente o problema que a experimentação resolve e, como o approached.
6. No **descrição detalhada** caixa, descrevem os passos que demorou cada parte da sua experimentação. Alguns tópicos útil para incluir são:
   * Captura de ecrã de gráfico de experimentação
   * Explicação e origens de dados
   * Processamento de dados
   * Com engenharia
   * Descrição do modelo
   * Resultados e avaliação de desempenho do modelo

   Pode utilizar markdown para formatar a descrição. Para ver como é que as suas entradas na página de descrição da experimentação serão apresentada quando a experimentação for publicada, selecione **pré-visualização**.

   ![Selecione "Preview" para ver o texto será aspeto](./media/gallery-experiments/preview-markdown-text.png)

   > [!TIP]
   > As caixas de texto fornecidas para pré-visualização e de edição de markdown são pequenos. Recomendamos que escrever documentação da sua experimentação num editor de markdown, copiá-lo e, em seguida, cole a documentação concluída na caixa de texto na galeria. Depois de publicar a experimentação, pode efetuar qualquer correção utilizando ferramentas baseadas na web padrão que markdown de utilização para editar e pré-visualizar.

7. No **seleção da imagem** página, escolha uma imagem em miniatura para sua experimentação. Na imagem em miniatura aparece na parte superior da página de detalhes de experimentação e no mosaico de experimentação. Outros utilizadores verão a imagem em miniatura quando procurarem a galeria. Pode carregar uma imagem do seu computador ou selecionar uma imagem as cotações da galeria.
    </br>
    ![Carregar ou selecionar uma imagem para a Galeria](./media/gallery-experiments/select-gallery-image.png)
8. No **definições** página **visibilidade**, escolha se pretende publicar o seu conteúdo publicamente (**pública**) ou para o acessível apenas a utilizadores que têm uma ligação para o (página **Não listados**).

    ![Escolha se pretende publicar publicamente ou como não listados](./media/gallery-experiments/choose-public-or-unlisted.png)

    <!-- -->

   > [!TIP]
   > Se pretender certificar-se a sua documentação de procura correta antes de libertar publicamente, pode publicar a experimentação como primeiro **Unlisted**. Mais tarde, pode alterar a definição de visibilidade para **pública** na página de detalhes de experimentação.
   >
   >
9. Para publicar a experimentação para a galeria, selecione o **OK** marca de verificação.

    ![Selecione a marca de verificação OK para publicar a experimentação](./media/gallery-experiments/ok-checkmark.png)

Para sugestões sobre como publicar uma experimentação de Galeria de alta qualidade, consulte [sugestões para documentar e publicar a sua experimentação](#tips-for-documenting-and-publishing-your-experiment).

É-lo - tudo terminar.

Pode agora ver a sua experimentação na galeria e partilhe a ligação com outros utilizadores. Se publicou sua experimentação, utilizando o **pública** visibilidade definição, a experimentação aparece nos resultados de pesquisa e procurar na galeria. Pode editar a documentação de experimentação na página de detalhes da experimentação sempre que tem sessão iniciada para a galeria.

Para ver a lista das suas contribuições, selecione a imagem no canto superior direito da página qualquer da galeria. Em seguida, selecione o nome para abrir a página de conta.

![Selecione o nome de conta](./media/gallery-experiments/click-account-name.png)

### <a name="update-your-experiment"></a>Atualizar a sua experimentação
Se for necessário, pode efetuar alterações ao fluxo de trabalho (módulos, parâmetros e assim sucessivamente) numa experiência que publicou para a galeria. No Machine Learning Studio, efetue as alterações que pretende efetuar a experimentação e, em seguida, publicar novamente. A experimentação publicada será atualizada com as suas alterações.

Pode alterar qualquer uma das seguintes informações para a sua experimentação diretamente na galeria:

* Nome da experimentação
* Resumo ou descrição
* Etiquetas
* Imagem
* Definição de visibilidade (**pública** ou **Unlisted**)

Também pode eliminar a experimentação da galeria do.

Pode efetuar estas alterações ou eliminar a experimentação, da página de detalhes da experimentação ou a partir de página na galeria do seu perfil.


#### <a name="from-the-experiment-details-page"></a>Na página de detalhes da experimentação
Na página de detalhes da experimentação, para alterar os detalhes para a sua experimentação, selecione **editar**.

![Selecione Editar para editar a sua experimentação](./media/gallery-experiments/edit-button.png)

A página de detalhes introduz o modo de edição. Para efetuar alterações, selecione **editar** junto do nome da experimentação, resumo ou etiquetas. Quando tiver terminado de efetuar alterações, selecione **feito**.

![Selecione "Editar" para editar os detalhes e selecione "Concluído" quando terminar](./media/gallery-experiments/edit-details-page.png)

Para alterar as definições de visibilidade para a experimentação (**pública** ou **Unlisted**), ou para eliminar a experimentação da galeria, selecione o **definições** ícone.

![Selecione "Definições" para alterar a visibilidade ou para eliminar a experimentação](./media/gallery-experiments/settings-button.png)

#### <a name="from-your-profile-page"></a>Na página do seu perfil
Na página do seu perfil, selecione a seta para baixo para a experimentação e, em seguida, selecione **editar**. Isto leva-o para a página de detalhes para a sua experimentação, no modo de edição. Quando terminar de efetuar alterações, selecione **feito**.

Para eliminar a experimentação da galeria, selecione **eliminar**.

![Selecione "Editar" ou "Eliminar"](./media/gallery-experiments/edit-delete-buttons.png)

### <a name="tips-for-documenting-and-publishing-your-experiment"></a>Sugestões para documentar e publicar a sua experimentação
* Pode assumir que o leitor tem experiência de ciência de dados anterior, mas pode ser útil utilizar linguagem simple. Explica as coisas detalhadamente sempre que possível.
* Cortana Intelligence Suite é relativamente nova. Nem todos os leitores estiver familiarizados com como utilizá-la. Fornece suficiente informações e uma explicação passo a passo para ajudar a leitores, navegue até a sua experimentação.
* Os visuais que podem ser útil para os leitores a interpretar e utilize a documentação da sua experimentação corretamente. Os visuais que incluem gráficos de experimentação e capturas de ecrã dos dados. Para obter mais informações sobre como incluir imagens na documentação da sua experimentação, consulte o [coleção diretrizes de publicação e exemplos](https://gallery.cortanaintelligence.com/Collection/Publishing-Guidelines-and-Examples-1).
* Se incluir um conjunto de dados na sua experimentação (ou seja, que está a não importar o conjunto de dados através do módulo de importar dados), o conjunto de dados faz parte da sua experimentação e é publicado na galeria. Certifique-se de que o conjunto de dados que publicar tem os termos que permitam a partilha e a transferência por qualquer pessoa de licenciamento. Contribuições de galeria são abordadas do Azure [os termos de utilização](https://azure.microsoft.com/support/legal/website-terms-of-use/).

## <a name="frequently-asked-questions"></a>Perguntas mais frequentes
**Quais são os requisitos para submeter ou editar uma imagem para a minha experimentação?**

As imagens que submetem com a sua experimentação são utilizadas para criar um mosaico de experimentação para a contribuição. Recomendamos que imagens pode ser inferior a 500 KB, com um aspect ratio de 3:2 e uma resolução de 960 &#215; 640.

**O que acontece ao conjunto de dados que posso utilizado na experimentação? O conjunto de dados também é publicado na Galeria?**

Se o seu conjunto de dados faz parte da sua experimentação e não está a ser importado através do módulo de importar dados, o conjunto de dados é publicado na galeria como parte da sua experimentação. Certifique-se de que o conjunto de dados que publicam com a sua experimentação tem os termos de licenciamento adequados. Os termos de licenciamento devem permitir a partilha e transferir os dados.

**Tenho uma experimentação que utiliza um módulo de importar dados para retirar dados do Azure HDInsight ou SQL Server. Utiliza as minhas credenciais para obter os dados. Pode publicar este tipo de experimentação? Como pode posso ter a garantia que as minhas credenciais não irão ser partilhados?**

Atualmente, não é possível publicar uma experimentação que utiliza as credenciais na galeria.

**Como posso introduzir várias etiquetas?**

Depois de introduzir uma etiqueta, introduza outra tag, prima a tecla de tabulação.
