# <a name="discover-and-share-tutorials-in-azure-ai-gallery"></a>Detetar e partilhar os tutoriais na Galeria de IA do Azure

[Galeria de IA do Azure](http://gallery.cortanaintelligence.com) oferece várias [tutoriais](https://gallery.cortanaintelligence.com/tutorials) que pode ajudá-lo por meio de conceitos e tecnologias do machine learning. Tutoriais também poderão descrevem métodos para resolver problemas do machine learning.

## <a name="discover"></a>Descobrir
Para procurar tutoriais [na galeria](http://gallery.cortanaintelligence.com), em **mais**, selecione **tutoriais**.

![Selecione os tutoriais na home page da Galeria](./media/gallery-tutorials/select-tutorials-in-gallery.png)

O **[tutoriais](https://gallery.cortanaintelligence.com/tutorials)** página apresenta uma lista de tutoriais recentemente adicionados e populares. Para ver todos os tutoriais, selecione o **ver todas as** botão. Para procurar um tutorial específico, selecione **ver todas as**e os critérios de filtro, em seguida, selecione. Também pode introduzir termos da procura na **pesquisa** caixa na parte superior da página da galeria.

Pode obter mais informações sobre um tutorial sobre a página de detalhes de tutorial. Para abrir uma página de detalhes do tutorial, selecione o tutorial. Na **comentários** secção numa página de detalhes do tutorial, pode comentar, fornecer comentários ou fazer perguntas sobre o tutorial. Pode até compartilhar o tutorial com amigos ou colegas no Twitter ou LinkedIn. Também pode enviar um e-mail uma ligação para a página de detalhes do tutorial, convidar outros utilizadores para ver a página.

![Compartilhar este item com amigos](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Adicione os seus próprios comentários](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Contribuir
Pode adicionar um tutorial para a Galeria para ajudar outros utilizadores a resolver um problema ou obtenha um conceito.

### <a name="create-a-tutorial"></a>Criar um tutorial

1. Inicie sessão para a galeria com sua conta Microsoft.

2. Selecione a sua imagem na parte superior da página e, em seguida, selecione o seu nome.
  
    ![Selecione o nome da sua conta](./media/gallery-tutorials/click-account-name.png)

3. Selecione **Novo Item**.
  
    ![Selecione "Novo Item"](./media/gallery-collections/click-new-item.png)

4. Sobre o **Descrição** página, para **tipo de ITEM**, selecione **Tutorial**. Introduza um nome para o tutorial, um breve resumo, uma descrição detalhada e todas as marcas que podem ajudar outros utilizadores a encontrar seu tutorial.
  
    ![Introduza as informações para o tutorial novo](./media/gallery-tutorials/create-tutorial-page-1.png)
5. Sobre o **seleção de imagem** , selecione uma imagem que é apresentada com a coleção. Pode carregar seu próprio ficheiro de imagem ou selecionar uma imagem de estoque. Escolha uma imagem que pode ajudar os utilizadores a identificar o conteúdo e o objetivo do tutorial.
  
    ![Introduza as informações para o tutorial novo](./media/gallery-tutorials/create-tutorial-page-2.png)

6. Sobre o **definições** página, para **visibilidade**, selecione se a coleção é **pública** (qualquer pessoa pode ver o tutorial) ou **Unlisted** ( apenas as pessoas com uma ligação direta para o tutorial podem vê-lo).
  
    > [!IMPORTANT]
    > Depois de definir um tutorial **pública** e, em seguida, selecione **Create**, mais tarde não é possível alterar o tutorial para **Unlisted**.
    > 
    > 
  
    ![Selecione "Public" ou "Não listados"](./media/gallery-tutorials/create-tutorial-page-3.png)

7. Selecione **Criar**.

O tutorial está agora na Galeria de IA do Azure. Os tutoriais são listados na sua página de conta, o **itens** separador.
