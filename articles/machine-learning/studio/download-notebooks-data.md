# <a name="download-notebookspreview-data"></a>Baixar dados de Cadernos (pré-visualização)

**APLICA-SE A:** ![ sim ](../../../includes/media/aml-applies-to-skus/yes.png) Machine Learning Studio (clássico) no ![ ](../../../includes/media/aml-applies-to-skus/no.png)[Azure Machine Learning](../compare-azure-ml-to-studio-classic.md)  

A funcionalidade Cadernos (pré-visualização) foi removida no dia *13 de abril de 2020*. A remoção do separador Portátil (pré-visualização) e dos ficheiros de cadernos de utilizador está atualmente a ser lançada para as regiões do Azure em todo o mundo.

Este artigo fornece instruções passo a passo sobre como descarregar dados de Portátil (pré-visualização).

1. Vá ao **separador Notebooks** no Azure Machine Learning Studio (clássico).
    ![Navegue para o separador Cadernos!](./media/download-notebooks/notebooks-list.png)

1. Selecione o caderno que pretende descarregar. Vai abrir em Jupyter.
    ![Selecione um portátil ](./media/download-notebooks/select-notebook.png) Descarregar vários cadernos ao mesmo tempo não está suportado.

1. Vá a **Ficheiros**  ->  **Descarregar como**, e selecione uma opção de formato.
    ![Opções de descarregamento de portátil](./media/download-notebooks/download-options.PNG)

1. Depois de selecionar uma opção, o ficheiro do portátil começará a ser descarregado. Ignore quaisquer erros de 500 ou 502 que apareçam, eles não impactarão o seu download.
