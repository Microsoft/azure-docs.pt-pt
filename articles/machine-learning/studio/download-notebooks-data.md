# <a name="download-notebookspreview-data"></a>Baixar dados de Cadernos (pré-visualização)

A função Cadernos (pré-visualização) será removida no dia 13 de abril de *2020.* Após essa data, o separador Cadernos (pré-visualização) desaparecerá e os dados dos cadernos não podem ser restaurados. Faça o download dos seus dados de cadernos antes de 13 de abril de 2020.

Este artigo fornece instruções passo a passo sobre como descarregar dados de Cadernos (pré-visualização).

1. Vá ao separador **De portátil** no Azure Machine Learning Studio (clássico).
    ![Navegar para o separador Cadernos!](./media/download-notebooks/notebooks-list.png)

1. Selecione o caderno que pretende descarregar. Abrirá em Jupyter.
    ![Selecione um caderno](./media/download-notebooks/select-notebook.png) Descarregando vários cadernos ao mesmo tempo não é suportado.

1. Vá a **Ficheiros** -> **Descarregar como**, e selecione uma opção de formato.
    ![opções de descarregamento do Caderno](./media/download-notebooks/download-options.PNG)

1. Depois de selecionar uma opção, o ficheiro portátil começará a ser descarregado. Ignore quaisquer erros de 500 ou 502 erros que apareçam, não afetarão o seu download.
