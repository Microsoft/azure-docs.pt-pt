# <a name="download-notebookspreview-data"></a>Baixar dados de Cadernos (pré-visualização)

A funcionalidade Cadernos (pré-visualização) foi removida no dia 13 de abril de *2020.* A remoção do separador Notebooks (pré-visualização) e dos ficheiros de cadernos de utilizadores está atualmente a ser lançada para as regiões de Azure em todo o mundo.

Este artigo fornece instruções passo a passo sobre como descarregar dados de Cadernos (pré-visualização).

1. Vá ao separador **De portátil** no Azure Machine Learning Studio (clássico).
    ![Navegue para o separador Cadernos!](./media/download-notebooks/notebooks-list.png)

1. Selecione o caderno que pretende descarregar. Abrirá em Jupyter.
    ![Selecione](./media/download-notebooks/select-notebook.png) um caderno Descarregando vários cadernos ao mesmo tempo não é suportado.

1. Vá a **Ficheiros** -> **Descarregue como**, e selecione uma opção de formato.
    ![Opções de descarregamento de cadernos](./media/download-notebooks/download-options.PNG)

1. Depois de selecionar uma opção, o ficheiro portátil começará a ser descarregado. Ignore quaisquer erros de 500 ou 502 erros que apareçam, não afetarão o seu download.
