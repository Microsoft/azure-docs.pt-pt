# <a name="discover-and-share-collections-in-azure-ai-gallery"></a>Detetar e partilhar coleções na Galeria de AI do Azure
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="collections"></a>Coleções
Pode utilizar um [coleção](https://gallery.cortanaintelligence.com/collections) para agrupar as experimentações, APIs e outros itens de galeria que utiliza para uma solução específica ou um conceito. Pode agrupá-los a utilizar, para guardar para futura referência ou para partilhar.

## <a name="discover"></a>Descobrir
Para procurar coleções [na galeria](http://gallery.cortanaintelligence.com), em **mais**, selecione **coleções**.

![Selecione as coleções na home page da Galeria](./media/gallery-collections/select-collections-in-gallery.png)

O **[coleções](https://gallery.cortanaintelligence.com/collections)** página apresenta uma lista de coleções recentemente adicionadas e populares. Para ver todas as coleções, selecione **ver todos os**. Para procurar uma coleção específica, selecione **ver todos os**e os critérios de filtro, em seguida, selecione. Também pode introduzir termos de pesquisa no **pesquisa** caixa na parte superior da página da galeria.

Pode obter mais informações sobre uma coleção na página de detalhes da coleção. Para abrir uma página de detalhes de coleção, selecione a coleção. Uma coleção de detalhes de página no **comentários** secção, pode comente, fornecer comentários ou fazer perguntas sobre a coleção. Ainda pode partilhar a coleção com amigos ou colegas no Twitter ou LinkedIn. Também pode enviar por e-mail uma ligação para a página de detalhes de coleção, para convidar outros utilizadores para ver a página.

![Partilhar este item com amigos](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Adicionar os seus comentários](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Contribuir
Pode criar uma coleção que tenha itens que é proprietário ou itens que foram contribuíram pela Comunidade. Um item numa coleção pode ser qualquer item de galeria que uma solução específica ou um conceito de endereços.

Por exemplo, pode utilizar uma coleção para agrupar itens de um tópico específico ou pode agrupar uma experimentação de vários passo que resolve um problema complexo. As coleções iniciais contribuídas pela Microsoft é constituída por vários passo de machine learning modelos de experimentação. Pode utilizar os modelos para ajudar a resolver problemas do mundo real, tais como a deteção de fraudes online, classificação de texto, a previsão de revenda ou manutenção preditiva.

### <a name="create-a-collection"></a>Criar uma coleção

1. Inicie sessão para a galeria com a conta Microsoft.

2.  Selecione a imagem na parte superior da página e, em seguida, selecione o nome.
  
    ![Selecione o nome de conta](./media/gallery-collections/click-account-name.png)

3. Selecione **Novo Item**.
   
    ![Selecione o "novo Item de"](./media/gallery-collections/click-new-item.png)
4. No **Descrição** página, para **tipo de ITEM**, selecione **coleção**. Introduza um nome para a coleção, um breve resumo, uma descrição detalhada e as etiquetas que podem ajudar a outros utilizadores a encontrar a coleção.
   
    ![Introduza as informações para uma nova coleção](./media/gallery-collections/create-collection-page-1.png)
5. No **seleção da imagem** página, selecione uma imagem que será apresentada com a coleção. Pode carregar o seu próprio ficheiro de imagem ou selecionar uma imagem as cotações. Escolha uma imagem que pode ajudar os utilizadores a identificar o conteúdo e o objetivo da coleção.
   
    ![Introduza as informações para uma nova coleção](./media/gallery-collections/create-collection-page-2.png)
6. No **definições** página, para **visibilidade**, selecione se a coleção é **pública** (qualquer pessoa pode ver a coleção) ou **Unlisted** (apenas as pessoas com uma ligação direta para a coleção podem visualizá-lo).
   
   > [!IMPORTANT]
   > Depois de definir uma coleção para **pública** e, em seguida, selecione **criar**, mais tarde não é possível alterar a coleção para **Unlisted**.
   > 
   > 
   
    ![Selecione "Público" ou "Não listados"](./media/gallery-collections/create-collection-page-3.png)
7. Selecione **Criar**.

A coleção está agora na Galeria de AI do Azure. As coleções são listadas na sua página de conta, no **coleção** separador.

### <a name="add-items-to-a-collection"></a>Adicionar itens a uma coleção
Para adicionar itens à sua coleção, abra a coleção, selecione **editar**e, em seguida, selecione **Adicionar Item**.

![Adicionar itens a uma coleção da página de detalhes](./media/gallery-collections/add-to-collection-from-details-page.png)

Uma lista de itens que contribuíram para a galeria é apresentada na sua página de conta. Também pode procurar na Galeria para itens para adicionar. Selecione os itens que pretende adicionar. Todos os itens que selecionou está incluído no conjunto de itens a adicionar. O **adicionar** botão indica o número de itens que selecionou.

![Adicionar à coleção](./media/gallery-collections/add-to-collection.png)

Se encontrar um item que pretende adicionar enquanto está a navegar na galeria, para abrir o item, selecione **adicionar à coleção**. Selecione a coleção que pretende adicionar o item a.

![Adicionar itens a uma coleção da página de detalhes](./media/gallery-collections/add-to-collection-from-item-details.png)

Para alterar o resumo, descrição ou as etiquetas para a coleção, abra a coleção e, em seguida, selecione **editar**. 

Enquanto estiver a editar a coleção, pode alterar a ordem dos itens na coleção. Utilize os botões de seta junto a um item movê-lo para cima ou para baixo na lista de ordem. 

Para adicionar notas para os itens na sua coleção, selecione o canto superior direito de um item e, em seguida, selecione **adicionar/editar nota**. 

Para remover um item da sua coleção, selecione **remover**.

![Detalhes de alteração para uma coleção](./media/gallery-collections/change-collection-details.png)
