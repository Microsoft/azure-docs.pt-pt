# <a name="discover-and-share-collections-in-azure-ai-gallery"></a>Detetar e partilhar coleções na Galeria de IA do Azure

Pode utilizar um [recolha](https://gallery.cortanaintelligence.com/collections) na [Galeria de IA do Azure](http://gallery.cortanaintelligence.com) para agrupar experimentações, APIs e outros itens de galeria que utiliza para uma solução específica ou um conceito. Pode agrupá-las para utilizar, para guardar para referência posterior ou para partilhar.

## <a name="discover"></a>Descobrir
Para procurar coleções [na galeria](http://gallery.cortanaintelligence.com), em **mais**, selecione **coleções**.

![Selecione as coleções na home page da Galeria](./media/gallery-collections/select-collections-in-gallery.png)

O **[coleções](https://gallery.cortanaintelligence.com/collections)** página apresenta uma lista de coleções recentemente adicionadas e populares. Para ver todas as coleções, selecione **ver todas as**. Para pesquisar por uma coleção específica, selecione **ver todas as**e os critérios de filtro, em seguida, selecione. Também pode introduzir termos da procura na **pesquisa** caixa na parte superior da página da galeria.

Pode obter mais informações sobre uma coleção na página de detalhes de coleção. Para abrir uma página de detalhes de coleção, selecione a coleção. Numa coleção de detalhes da página, além da **comentários** seção, pode comentar, fornecer comentários ou fazer perguntas sobre a coleção. Pode até compartilhar a coleção com amigos ou colegas no Twitter ou LinkedIn. Também pode enviar um e-mail uma ligação para a página de detalhes de coleção, convidar outros utilizadores para ver a página.

![Compartilhar este item com amigos](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Adicione os seus próprios comentários](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Contribuir
Pode criar uma coleção que tenha os itens que é proprietário ou itens que contribuíram pela Comunidade. Um item numa coleção pode ser qualquer item de galeria que resolve uma solução específica ou um conceito.

Por exemplo, pode usar uma coleção para agrupar itens num tópico específico, ou pode agrupar uma experimentação de vários passo que resolve um problema complexo. As coleções iniciais enviadas pela Microsoft são compostos por experimentação modelos de aprendizagem automática com vários passo. Pode utilizar os modelos para ajudar a resolver problemas do mundo real, como deteção de fraudes online, classificação de texto, previsão de varejo ou de manutenção preditiva.

### <a name="create-a-collection"></a>Criar uma coleção

1. Inicie sessão para a galeria com sua conta Microsoft.

2.  Selecione a sua imagem na parte superior da página e, em seguida, selecione o seu nome.
  
    ![Selecione o nome da sua conta](./media/gallery-collections/click-account-name.png)

3. Selecione **Novo Item**.
   
    ![Selecione "Novo Item"](./media/gallery-collections/click-new-item.png)
4. Sobre o **Descrição** página, para **tipo de ITEM**, selecione **coleção**. Introduza um nome para a sua coleção, um breve resumo, uma descrição detalhada e todas as marcas que podem ajudar outros utilizadores a encontrar a sua coleção.
   
    ![Introduza as informações para uma nova coleção](./media/gallery-collections/create-collection-page-1.png)
5. Sobre o **seleção de imagem** , selecione uma imagem que é apresentada com a coleção. Pode carregar seu próprio ficheiro de imagem ou selecionar uma imagem de estoque. Escolha uma imagem que pode ajudar os utilizadores a identificar o conteúdo e o objetivo da coleção.
   
    ![Introduza as informações para uma nova coleção](./media/gallery-collections/create-collection-page-2.png)
6. Sobre o **definições** página, para **visibilidade**, selecione se a coleção é **pública** (qualquer pessoa pode ver a coleção) ou **Unlisted** (apenas as pessoas com uma ligação direta para a coleção podem vê-lo).
   
   > [!IMPORTANT]
   > Depois de definir uma coleção **pública** e, em seguida, selecione **Create**, mais tarde não é possível alterar a coleção que pretende **Unlisted**.
   > 
   > 
   
    ![Selecione "Public" ou "Não listados"](./media/gallery-collections/create-collection-page-3.png)
7. Selecione **Criar**.

A coleção está agora na Galeria de IA do Azure. Suas coleções são listadas na sua página de conta, o **coleção** separador.

### <a name="add-items-to-a-collection"></a>Adicionar itens a uma coleção
Para adicionar itens à sua coleção, abra a coleção, selecione **edite**e, em seguida, selecione **Add Item**.

![Adicionar itens a uma coleção a partir da página de detalhes](./media/gallery-collections/add-to-collection-from-details-page.png)

É apresentada uma lista de itens que contribuíram para a galeria na sua página de conta. Também pode procurar na Galeria de itens a adicionar. Selecione os itens que pretende adicionar. Todos os itens que selecionou está incluído no conjunto de itens a serem adicionados. O **adicionar** botão indica o número de itens que selecionou.

![Adicionar à coleção](./media/gallery-collections/add-to-collection.png)

Se encontrar um item que pretende adicionar enquanto está a aceder a galeria, para abrir o item, selecione **adicionar à coleção**. Selecione a coleção que pretende adicionar o item para.

![Adicionar itens a uma coleção a partir da página de detalhes](./media/gallery-collections/add-to-collection-from-item-details.png)

Para alterar o resumo, descrição ou etiquetas da coleção, abra a coleção e, em seguida, selecione **editar**. 

Enquanto estiver a editar a coleção, pode alterar a ordem dos itens na coleção. Utilize os botões de seta junto a um item para movê-lo para cima ou para baixo na lista ordem. 

Para adicionar notas para os itens na sua coleção, selecione o canto superior direito de um item e, em seguida, selecione **adicionar/editar nota**. 

Para remover um item de sua coleção, selecione **remover**.

![Detalhes da alteração para uma coleção](./media/gallery-collections/change-collection-details.png)
