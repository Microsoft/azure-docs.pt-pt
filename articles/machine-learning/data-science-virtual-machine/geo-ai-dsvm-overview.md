# <a name="what-is-the-geo-artificial-intelligence-data-science-virtual-machine"></a>O que é a Máquina Virtual geo artificial de dados de dados?

A Máquina Virtual geo artificial data data Science (também conhecida como Geo AI Data Science VM, ou Geo-DSVM) fornece capacidades de análise geoespacial da Máquina Virtual de Ciência de Dados Azure (DSVM). Especificamente, este VM alarga os kits de ferramentas de IA e ciência de dados no [DSVM,](overview.md) adicionando o sistema de informação geográfica [ArcGIS Pro](https://www.esri.com/arcgis/products/arcgis-pro/overview) líder no mercado da ERSI.

 Geo AI Data Science VM é uma oferta de produto que surgiu da colaboração entre a ESRI e a Microsoft. O resultado é um ambiente potente, que fornece a capacidade de:

- Explore dados com as ferramentas de IA e ciência dos dados mais populares
- Aplicar análise estatística a dados com aspetos geográficos ou espaciais
- Treine modelos para análisepre, utilizando machine learning e algoritmos de aprendizagem profunda
- Criar visualizações ricas

Geo AI Data Science VM é suportado no DSVM do Windows 2016. As interfaces Python e R para arcGIS também são reconfiguradas no Geo-DSVM, permitindo o acesso programático à análise geoespacial dentro das suas aplicações de IA. Além disso, são fornecidas amostras de Caderno Jupyter para ajudá-lo a começar a construir aplicações de IA infundidas com inteligência geoespacial.


## <a name="why-geo-ai-data-science-vm"></a>Por que VM de ciência de dados de IA geográfica? 

Integrar a geografia e a informação de localização nos seus dados traz uma nova dimensão poderosa para compreender um mundo cada vez mais instrumentado com dispositivos IoT. Existe agora um vasto leque de aplicações em praticamente todos os segmentos da sociedade civil, sejam comerciais, governamentais, académicos ou sem fins lucrativos. A análise geoespacial fornece ferramentas robustas para a recolha, gestão e análise de dados geográficos e baseados em localização. Também oferecem uma visualização poderosa que pode permitir insights únicos sobre a importância de tais dados.

A Máquina Virtual de Ciência de [Dados (DSVM)](overview.md) da Microsoft é um ambiente popular de experimentação e modelação no Azure, que fornece uma série de IA, Machine Learning e ferramentas de ciência de dados. Estas plataformas e ferramentas são todas convenientemente configuradas para ajudá-lo a obter produtivo imediatamente.

Geo AI Data Science VM combina as capacidades preditivas do DSVM com a escalabilidade da nuvem Azure e o software de análise geoespacial da [ESRI.](https://www.esri.com) Esta integração pioneira fornece aos cientistas de dados e analistas geoespaciais uma plataforma produtiva a partir da qual obter novas informações a partir dos seus dados, e identificar novas oportunidades.
