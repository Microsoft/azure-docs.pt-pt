# <a name="get-support-and-training-for-azure-machine-learning-services"></a>Obter assistência e formação para serviços do Azure Machine Learning

Este artigo fornece informações sobre como obter mais informações sobre os serviços do Azure Machine Learning, bem como obter suporte para os problemas e perguntas. 

## <a name="learn-more-about-azure-machine-learning"></a>Saiba mais sobre o Azure Machine Learning

O [Centro de Documentação do Azure Machine Learning](./overview-what-is-azure-ml.md) aloja tutoriais em vídeo e manuais de procedimentos. Estes guias passo a passo introduzem os serviços e explicam o ciclo de vida de ciência de dados. Adicionamos materiais novos ao Centro do Machine Learning continuamente. 

Além disso, existem [vídeos, pode ver para obter mais informações](https://azure.microsoft.com/resources/videos/index/?services=machine-learning)

Pode **submeter pedidos** para os materiais de aprendizagem adicional através da ligação de comentários na parte inferior do artigo.

## <a name="get-support-for-azure-machine-learning-services"></a>Obter o suporte para serviços do Azure Machine Learning

Pode obter suporte para serviços do Azure Machine Learning das seguintes formas:

+ **O suporte técnico**: suporte técnico do Azure de contacto, visitando [Azure suporta](https://azure.microsoft.com/support/options/) e selecionando o Machine Learning. 

+ **Fórum MSDN utilizador**: colocar perguntas, responda às perguntas e estabelecer ligação com outros utilizadores no [fórum de suporte de serviços do Azure Machine Learning no MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Capacidade excedida da pilha**: aceda a Comunidade do Azure Machine Learning no [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) marcados com "Azure-Machine-Learning".

+ **Partilhar sugestões de produto** e funcionalidade pedidos no nosso [canal de comentários do Azure Machine Learning](https://feedback.azure.com/forums/257792-machine-learning), que também é acessível através da ligação na parte inferior de cada artigo.
