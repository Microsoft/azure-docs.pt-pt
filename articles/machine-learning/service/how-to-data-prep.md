# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Preparar dados com o SDK de preparação de dados do Azure Machine Learning para Python
 
Preparação de dados é uma parte importante de um fluxo de trabalho do machine learning. Os modelos será mais preciso e eficiente se tiverem acesso ao limpar dados num formato mais fácil de consumir. Pode utilizar o [SDK do Azure Machine Learning Data Prep](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) para carregar dados de vários formatos, transformá-lo para ser mais utilizável e gravá-los para uma localização para os seus modelos aceder.

2. [Transformar dados](how-to-transform-data.md )
