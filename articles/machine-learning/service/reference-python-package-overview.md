# <a name="python-packages-for-azure-machine-learning"></a>Pacotes de Python para o Azure Machine Learning

Saiba mais sobre o os pacotes de Python proprietários da Microsoft para o Azure Machine Learning. Pode utilizar estas bibliotecas e as funções em combinação com outros pacotes open source ou de terceiros, mas a utilizar os pacotes proprietários, o Python código tem de executar em relação a um serviço ou num computador que fornece os interpreters.

Os pacotes do Azure Machine Learning são **extensões pip instalável do Python para o Azure Machine Learning** que permitem cientistas de dados e programadores de AI para rapidamente criar e implementar altamente exata de aprendizagem e a configuração avançada modelos para vários domínios.

<a name="amlpcv"></a>
## <a name="azure-ml-package-for-computer-vision"></a>Pacote de ML do Azure para problemas de visão do computador

Com o Azure ML pacote de imagem digitalizada do computador, pode criar, otimizar, implementar modelos de profunda learning para classificação de imagem, a deteção de objeto e semelhança de imagem.

Repita estes passos para este pacote:
1. [Transferir](https://aka.ms/aml-packages/vision/download) o pacote.
1. Leia o [instalar docs](https://aka.ms/aml-packages/vision).
1. [Criar e implementar um modelo de visão computador](how-to-build-deploy-image-classification-models.md) com um bloco de notas do Jupyter.
1. Explorar o pacote [documentação de referência](https://aka.ms/aml-packages/vision).

<a name="amlrealtimeai"></a>
## <a name="amlrealtimeai-fpga-acceleration"></a>amlrealtimeai (aceleração FPGA)

Com o pacote de aceleração de Hardware do Azure Machine Learning, cientistas de dados e AI os programadores podem featurize imagens com uma versão de ResNet 50 quantized, dar formação classificadores com base nessas funcionalidades e, em seguida, implementar modelos para um [campo programável As matrizes (FPGA) de porta](concept-accelerate-with-fpgas.md) no Azure para inferencing ultra-baixa latência.

Repita estes passos para este pacote:
1. [Transferir](https://aka.ms/aml-real-time-ai-package) o pacote.
1. Leia o [instalar docs](reference-fpga-package-overview.md).
1. [Implementar um modelo para um FPGA](how-to-deploy-fpga-web-service.md).
