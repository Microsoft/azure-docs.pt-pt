# <a name="pricing-and-billing-for-azure-machine-learning-services"></a>Preços e faturação para serviços do Azure Machine Learning

Para obter detalhes completos, ou para ver uma fatura de exemplo, visite o [do Azure de página de preços](https://azure.microsoft.com/pricing/details/machine-learning-services/).  

Seguem-se algumas perguntas mais comuns em torno de faturação e preços.

## <a name="can-i-use-azure-machine-learning-for-free-during-preview"></a>Posso utilizar o Azure Machine Learning gratuitamente durante a pré-visualização?    

A aplicação do Azure Machine Learning Workbench está disponível gratuitas para subscritores do Azure.

O serviço de experimentação e gestão de modelo de oferecem livres camadas, para além dos escalões pagos, disponíveis com um desconto durante a pré-visualização pública.

## <a name="am-i-charged-based-on-how-many-experiments-i-run"></a>Estou posso cobrado com base numa experimentações quantos posso executar?

Não, o serviço Experimentação permite quaisquer experimentações necessárias e são cobrados custos apenas com base no número de utilizadores. Os recursos de computação de Experimentação são cobrados em separado.  Aconselhamo-lo para efetuar várias experimentações para chegar ao modelo de melhor para a sua solução. 

## <a name="am-i-charged-based-on-how-many-times-my-web-services-is-called"></a>Estou posso cobrado com base na quantidade de vezes que os meus serviços web é denominado?

Não. Serviços Web podem ser chamados como muitas vezes, conforme necessário, sem qualquer implicações de faturação de gestão de modelo. Tem total controlo para dimensionar as implementações para satisfazer as necessidades das suas aplicações.

## <a name="how-can-i-scale-the-units-i-purchase-in-the-azure-machine-learning-model-management"></a>Como dimensionar a unidades que posso adquirir na gestão de modelo do Azure Machine Learning?

Pode alterar o número de unidades, ou reduza verticalmente, utilizando o portal do Azure ou a CLI. 

## <a name="what-does-a-bill-look-like"></a>O que como um aspeto fatura?

Faturas são produzidas diariamente. Para efeitos de faturação, um dia começa à meia-noite UTC. As faturas são geradas mensalmente. São cobradas taxas separadas para os serviços do Azure utilizados em conjunto com o Azure Machine Learning. Os encargos podem incluir, mas não estão limitados a: 
- Encargos de computação
- HDInsight
- Azure Container Service
- Registo de Contentores do Azure 
- Armazenamento de Blobs do Azure
- Application Insights
- Azure Key Vault
- Visual Studio Team Services
- Hub de eventos do Azure
- Azure Stream Analytics para o obter mais detalhes ou, para ver uma fatura de exemplo, visite o [do Azure de página de preços](https://azure.microsoft.com/pricing/details/machine-learning-services/).
