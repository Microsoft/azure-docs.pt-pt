# <a name="team-data-science-process-for-developer-operations"></a>Processo de ciência de dados de equipa para operações de desenvolvedor

Este artigo explora as funções de Developer Operations (DevOps) específicas para uma implementação de solução de Analítica Avançada e Serviços Cognitivos. Estes materiais de formação implementam o Processo de Ciência de Dados de Equipa (TDSP) e a Microsoft e software e kits de ferramentas de código aberto, úteis para visualizar, executar e fornecer soluções de ciência de dados. Refere temas que cobrem o DevOps Toolchain que é específico para projetos e soluções de Data Science e IA.
