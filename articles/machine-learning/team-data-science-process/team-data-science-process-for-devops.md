# <a name="team-data-science-process-for-developer-operations"></a>Processo de Ciência de Dados de Equipa para Operações de Desenvolvimento

Estes materiais de formação implementam o Processo de Ciência de Dados da Equipa (TDSP) e a Microsoft e software e kits de ferramentas de código aberto, úteis para a visualização, execução e entrega de soluções de ciência de dados. Refere tópicos que cobrem o Cadeia de Ferramentas DevOps que é específico dos projetos e soluções e soluções da Data Science e DaI.
