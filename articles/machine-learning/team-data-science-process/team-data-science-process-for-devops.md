# <a name="team-data-science-process-for-developer-operations"></a>Processo de ciência de dados de equipa para operações de programador

Este artigo explora as funções de operações de desenvolvedor (DevOps) que são específicas para uma implementação de solução de análise avançada e os serviços cognitivos. Esses materiais de treinamento relacionados com o processo de ciência de dados de equipa (TDSP) e Microsoft e software open-source e kits de ferramentas, o que são úteis para a previsão, executar e fornecer soluções de ciência de dados. Ele faz referência a tópicos que abrangem a coleção de ferramentas de DevOps que é específico para soluções e projetos de ciência de dados e IA.
