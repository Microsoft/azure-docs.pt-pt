# <a name="team-data-science-process-for-developer-operations"></a>Processo de ciência de dados de equipa para operações de programador

Este artigo explora as funções de operações de desenvolvedor (DevOps) que são específicas para uma implementação de solução de análise avançada e os serviços cognitivos. Estes materiais de formação implementam o Processo de Ciência de Dados da Equipa (TDSP) e a Microsoft e software e kits de ferramentas de código aberto, úteis para a visualização, execução e entrega de soluções de ciência de dados. Ele faz referência a tópicos que abrangem a coleção de ferramentas de DevOps que é específico para soluções e projetos de ciência de dados e IA.
