# <a name="team-data-science-process-for-developer-operations"></a>Processo de ciência de dados de equipe para operações do desenvolvedor

Este artigo explora as funções de operações de desenvolvedor (DevOps) que são específicas para uma implementação de solução de análise avançada e serviços cognitivas. Esses materiais de treinamento estão relacionados ao TDSP (processo de ciência de dados de equipe) e ao software e aos kits de programas da Microsoft e de código aberto, que são úteis para prever, executar e fornecer soluções de ciência de dados. Ele faz referência a tópicos que abrangem o DevOps ferramentas que é específico para projetos e soluções de ciência de dados e ia.
