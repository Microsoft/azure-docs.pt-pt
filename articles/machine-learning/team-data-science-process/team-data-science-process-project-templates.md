# <a name="team-data-science-process-project-planning"></a>Planeamento de projetos de processo de ciência de dados de equipa

O Processo de Ciência de Dados de Equipa (TDSP) fornece um ciclo de vida para estruturar o desenvolvimento dos seus projetos de ciência de dados. Este artigo fornece links para modelos do Microsoft Project e do Excel que o ajudam a planear e gerir estas fases do projeto.

O ciclo de vida descreve as principais fases que os projetos normalmente executam, muitas vezes iterativamente:

- Compreensão empresarial
- Aquisição e Compreensão de Dados
- Modelação
- Implementação
- Aceitação do Cliente

Para descrições de cada uma destas fases, consulte o ciclo de [vida do Processo de Ciência de Dados da Equipa](https://docs.microsoft.com/azure/machine-learning/team-data-science-process/lifecycle).

 
## <a name="microsoft-project-template"></a>Modelo de projeto da Microsoft

O modelo do Projeto Microsoft para o Processo de Ciência de Dados de Equipa está disponível a partir daqui: [modelo do Microsoft Project](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.mpp) 

Quando abrir o plano, clique no link para a esquerda para o TDSP. Altere o nome e a descrição e adicione os outros recursos da equipa que precisar. Estimeça as datas necessárias da sua experiência.

![1](./media/team-data-science-process-project-templates/ms-project-templates.png)

Cada tarefa tem uma nota. Abra essas tarefas para ver que recursos já foram criados para si.

![2](./media/team-data-science-process-project-templates/ms-project-template-task.png)


## <a name="excel-template"></a>Modelo de excel

Se não tiver acesso ao Microsoft Project, uma folha de cálculo do Excel com todos os mesmos dados também está disponível para download aqui: [Modelo do Excel](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.xlsx) Podes descodê-lo em qualquer ferramenta que prefiras usar.

Use estes modelos por sua conta e risco. [Aplicam-se as isenções de responsabilidade habituais.](https://www.gnu.org/licenses/gpl-3.0.en.html)

## <a name="repository-template"></a>Modelo de repositório

Use este [repositório de modelo de projeto](https://github.com/Azure/Azure-TDSP-ProjectTemplate) para apoiar a execução e colaboração eficientes do projeto. Este repositório dá-lhe uma estrutura de diretório padronizada e modelos de documentos que pode usar para o seu próprio projeto TDSP.
