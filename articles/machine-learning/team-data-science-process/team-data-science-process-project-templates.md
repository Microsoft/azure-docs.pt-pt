# <a name="team-data-science-process-project-planning"></a>Planeamento do projeto de processo de ciência de dados da equipa

O Team Data Science Process (TDSP) fornece um ciclo de vida para estruturar o desenvolvimento dos seus projetos de ciência de dados. Este artigo fornece links para os modelos Microsoft Project e Excel que o ajudam a planear e gerir estas fases do projeto.

O ciclo de vida descreve as principais etapas que os projetos normalmente executam, muitas vezes iterativamente:

- Compreensão de Negócios
- Aquisição e Compreensão de Dados
- Modelação
- Implementação
- Aceitação do cliente

Para descrições de cada uma destas etapas, consulte [o ciclo](https://docs.microsoft.com/azure/machine-learning/team-data-science-process/lifecycle)de vida do Processo de Ciência de Dados da Equipa .

 
## <a name="microsoft-project-template"></a>Modelo do Projeto Microsoft

O modelo do Projeto Microsoft para o Processo de Ciência de Dados da Equipa está disponível a partir daqui: [modelo do Projeto Microsoft](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.mpp) 

Quando abrir o plano, clique no link para a esquerda para o TDSP. Mude o nome e a descrição e adicione quaisquer outros recursos da equipa que precisar. Calcule as datas exigidas pela sua experiência.

![1](./media/team-data-science-process-project-templates/ms-project-templates.png)

Cada tarefa tem uma nota. Abra essas tarefas para ver que recursos já foram criados para si.

![2](./media/team-data-science-process-project-templates/ms-project-template-task.png)


## <a name="excel-template"></a>Modelo excel

Se não tiver acesso ao Microsoft Project, uma folha de cálculo do Excel com todos os mesmos dados também está disponível para download aqui: [Modelo Excel](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.xlsx) Pode puxá-lo para qualquer ferramenta que prefira usar.

Use estes modelos por sua conta e risco. Aplicam-se as [isenções de responsabilidade habituais.](https://www.gnu.org/licenses/gpl-3.0.en.html)

## <a name="repository-template"></a>Modelo de repositório

Utilize este [repositório de modelo seletiva](https://github.com/Azure/Azure-TDSP-ProjectTemplate) para apoiar a execução e colaboração eficientes do projeto. Este repositório dá-lhe uma estrutura de diretório padronizada e modelos de documentoque pode usar para o seu próprio projeto TDSP.
