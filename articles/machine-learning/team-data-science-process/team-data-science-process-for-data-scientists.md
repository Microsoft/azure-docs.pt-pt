# <a name="team-data-science-process-for-data-scientists"></a>Processo de Ciência de Dados da Equipa para cientistas de dados

Este artigo fornece orientação para um conjunto de objetivos que são normalmente usados para implementar soluções abrangentes de ciência de dados com tecnologias Azure. É guiado através de:

- compreender uma carga de trabalho analítica
- usando o Processo de Ciência de Dados da Equipa
- usando azure machine learning
- as bases da transferência e armazenamento de dados
- fornecendo documentação de fonte de dados
- usando ferramentas para processamento de análise

Estes materiais de formação estão relacionados com o Processo de Ciência de Dados da Equipa (TDSP) e microsoft e software e kits de ferramentas de código aberto, que são úteis para a visualização, execução e entrega de soluções de ciência de dados.
