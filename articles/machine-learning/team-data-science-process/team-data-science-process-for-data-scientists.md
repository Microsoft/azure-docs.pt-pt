# <a name="team-data-science-process-for-data-scientists"></a>Processo de ciência de dados da equipa para cientistas de dados

Este artigo fornece orientação para um conjunto de objetivos que são normalmente usados para implementar soluções abrangentes de ciência de dados com tecnologias Azure. Você é guiado através de:

- compreender uma carga de trabalho analítica
- usando o processo de ciência de dados de equipa
- usando Azure Machine Learning
- as fundações da transferência e armazenamento de dados
- fornecendo documentação de fonte de dados
- usando ferramentas para o processamento de analíticas

Estes materiais de formação estão relacionados com o Processo de Ciência de Dados de Equipa (TDSP) e com a Microsoft e com software e kits de ferramentas de código aberto, que são úteis para visualizar, executar e fornecer soluções de ciência de dados.
