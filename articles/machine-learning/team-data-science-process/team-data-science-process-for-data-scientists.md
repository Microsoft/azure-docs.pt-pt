# <a name="team-data-science-process-for-data-scientists"></a>Processo de ciência de dados de equipe para cientistas de dados

Este artigo fornece orientação para um conjunto de objetivos que normalmente são usados para implementar soluções de ciência de dados abrangentes com as tecnologias do Azure. Você é orientado por meio de:

- compreendendo uma carga de trabalho de análise
- usando o processo de ciência de dados de equipe
- usando Azure Machine Learning
- as bases da transferência de dados e do armazenamento
- fornecendo documentação de fonte de dados
- usando ferramentas para processamento de análise

Esses materiais de treinamento estão relacionados ao TDSP (processo de ciência de dados de equipe) e ao software e aos kits de programas da Microsoft e de código aberto, que são úteis para prever, executar e fornecer soluções de ciência de dados.
