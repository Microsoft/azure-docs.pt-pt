# <a name="azure-data-lake-data-science-walkthroughs-using-u-sql"></a>Azure Data Lake data science walkthroughs usando U-SQL

Estes walkthroughs usam U-SQL com Azure Data Lake para fazer análises preditivas. Seguem os passos delineados no Processo de Ciência de Dados da Equipa. Para uma visão geral do processo de ciência de dados da equipa, consulte o [Processo de Ciência de Dados.](overview.md) Para uma introdução ao Lago de Dados Azure, consulte [a visão geral da Azure Data Lake Store.](../../data-lake-store/data-lake-store-overview.md)

As análises adicionais da ciência dos dados que executam o Processo de Ciência de Dados de Equipa são agrupadas pela **plataforma** que utilizam. Consulte [Walkthroughs executando o Processo de Ciência de Dados de Equipa](walkthroughs.md) para uma itemização destes exemplos.


## <a name="predict-taxi-tips-using-u-sql-with-azure-data-lake"></a>Prever dicas de táxi usando U-SQL com Azure Data Lake

O [Use Azure Data Lake para](data-lake-walkthrough.md) a revista data science mostra como usar o Lago de Dados Azure para fazer tarefas de exploração de dados e classificação binária.  Os dados são uma amostra do conjunto de dados de táxi de NYC.  A tarefa é prever se uma gorjeta é ou não paga por um cliente.
