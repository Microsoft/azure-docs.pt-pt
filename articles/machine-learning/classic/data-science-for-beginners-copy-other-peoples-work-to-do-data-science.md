## <a name="find-examples-in-the-azure-ai-gallery"></a>Encontre exemplos na Galeria Azure AI

A Microsoft tem um serviço baseado na nuvem chamado [Azure Machine Learning Studio (clássico)](https://azure.microsoft.com/services/machine-learning-studio/). Ele fornece-lhe um espaço de trabalho onde você pode experimentar com diferentes algoritmos de aprendizagem automática, e, quando você tem a sua solução trabalhada, você pode lançá-lo como um serviço web.

Parte deste serviço é algo chamado **[Galeria Azure AI.](https://gallery.azure.ai/)** Contém recursos, incluindo uma coleção de experiências (clássicas) do Azure Machine Learning Studio, ou modelos, que as pessoas construíram e contribuíram para que outros usassem. Estas experiências são uma ótima maneira de aproveitar o pensamento e o trabalho árduo dos outros para começar as suas próprias soluções. Todos são bem-vindos a navegar através dele.

![Galeria de IA do Azure](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/azure-ai-gallery.png)

Se clicar em Experiências no topo, verá **algumas** das experiências mais recentes e populares na galeria. Pode pesquisar através do resto das experiências clicando em **Navegar tudo** na parte superior do ecrã, e aí pode introduzir termos de pesquisa e escolher filtros de pesquisa.

## <a name="find-and-use-a-clustering-algorithm-example"></a>Encontre e use um exemplo de algoritmo de clustering
Então, por exemplo, digamos que quer ver um exemplo de como o agrupamento funciona, então você procura experiências **de "agrupamento".**

![Pesquisa de experiências de agrupamento](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/search-for-clustering-experiments.png)

Aqui está uma interessante que alguém contribuiu para a galeria.

![Experiência de agrupamento](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/clustering-experiment.png)

Clique nessa experiência e obtém uma página web que descreve o trabalho que este colaborador fez, juntamente com alguns dos seus resultados.

![Página de descrição de experiência de clustering](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/clustering-experiment-description-page.png)

Note o link que diz **Open in Studio (clássico)**.

![Abrir em botão studio (clássico)](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/open-in-studio.png)

Posso clicar nisso e leva-me diretamente ao **Azure Machine Learning Studio (clássico)**. Cria uma cópia da experiência e coloca-a no meu próprio espaço de trabalho. Isto inclui o conjunto de dados do contribuinte, todo o processamento que fizeram, todos os algoritmos que usaram e como guardaram os resultados.

![Abra uma experiência de Galeria no Machine Learning Studio (clássico) - exemplo de algoritmo de clustering](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/cluster-experiment-open-in-studio.png)

E agora tenho um ponto de partida. Posso trocar os dados deles pelos meus e fazer o meu próprio ajuste do modelo. Isto dá-me um começo de corrida, e permite-me basear-me no trabalho de pessoas que realmente sabem o que estão a fazer.

## <a name="find-experiments-that-demonstrate-machine-learning-techniques"></a>Encontre experiências que demonstrem técnicas de aprendizagem automática
Existem outras experiências na [Galeria Azure AI](https://gallery.azure.ai) que foram contribuidas especificamente para fornecer exemplos de como fazer para pessoas novas para a ciência dos dados. Por exemplo, há uma experiência na galeria que demonstra como lidar com valores em falta[(Métodos para lidar com valores em falta).](https://gallery.azure.ai/Experiment/Methods-for-handling-missing-values-1) Percorre 15 maneiras diferentes de substituir valores vazios, e fala sobre os benefícios de cada método e quando usá-lo.

![Experiências de galeria abertas no Machine Learning Studio (clássico) - métodos para valores em falta](./media/data-science-for-beginners-copy-other-peoples-work-to-do-data-science/experiment-methods-for-handling-missing-values.png)

[A Azure AI Gallery](https://gallery.azure.ai) é um local para encontrar experiências de trabalho que pode usar como ponto de partida para as suas próprias soluções.

Certifique-se de conferir mais vídeos em "Data Science for Beginners" do Microsoft Azure Machine Learning Studio (clássico).
