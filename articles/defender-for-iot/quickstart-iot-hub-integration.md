# <a name="get-started-with-built-in-iot-hub-integration"></a>Começar com integração IoT Hub incorporada

Esta opção permite-lhe utilizar o serviço sem utilizar o Defender para agentes de segurança IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Permitir a integração do IoT Hub incorporado

Para permitir monitorizar a gestão da identidade do seu dispositivo, dispositivo para nuvem e nuvem para padrões de comunicação do dispositivo, faça o seguinte para iniciar o serviço:

1. Abra o seu **Hub IoT**.
1. Selecione o menu **de visão geral de segurança.**
1. Clique **em Secure your IoT solution** and complete the onboarding form.

Parabéns! Completou a habilitação do Serviço Defender para IoT no seu Hub IoT.
