# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Quickstart: Prepare a sua rede para o Azure Defender para implantação de IoT

## <a name="choose-your-hardware"></a>Escolha o seu hardware
O Azure Defender for IoT suporta vários aparelhos certificados, incluindo:
- Aparelhos pré-configurados certificados, nos quais o software já está instalado
- Aparelhos certificados não configurados nos quais pode descarregar e instalar software de sensores
- Especificações do aparelho de consola de gestão no local. A consola de gestão no local não está disponível como um aparelho pré-configurado. 

Para mais informações, consulte o [Guia de Especificações de Hardware](https://aka.ms/AzureDefenderforIoTBareMetalAppliance)


## <a name="install-components"></a>Instalar componentes
Depois de adquirir os seus componentes de hardware e descarregar o seu pacote de software a partir do portal Azure Defender para IoT, pode instalar software de componentes. Para mais informações, consulte o [Guia de Instalação.](https://aka.ms/AzureDefenderforIoTInstallSensorISO)

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Sensores a bordo do Azure Defender para portal IoT 
 Este artigo fornece uma visão geral da implementação, embarque e processos de atualização para Azure Defender para sensores IoT e a consola de gestão no local. Este processo é realizado seguindo instruções na página Welcome to Azure Defender para página IoT e páginas subsequentes.

![Screenshot da vista da página de boas-vindas do Azure](media/updates/image4.png)

Para abrir a página de boas-vindas:

1. Faça login na sua conta Azure e procure por Azure Defender para IoT.

1. **A implementação do sensor** requer que você: Compre hardware pré-configurado a partir de Seta ou compre o seu próprio dispositivo e instale software de sensores nele.
1. **A implementação da consola de gestão no local** requer que compre o seu próprio dispositivo e instale o software de consola de gestão no local. Para obter mais informações.




## <a name="network-setup"></a>Configuração da rede
Depois de instalar o Seu Defender para componentes do sistema IoT, pode configurar o seu equipamento de rede para uma monitorização ótima da rede. Para obter mais informações, consulte o [guia de configuração da Rede.](https://aka.ms/AzureDefenderForIoTNetworkSetup)
