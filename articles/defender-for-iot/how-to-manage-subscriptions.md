# <a name="manage-a-subscription"></a>Gerir uma subscrição

As assinaturas são geridas mensalmente. Quando estiver a bordo de uma subscrição, será cobrado por essa subscrição até ao final do mês. Da mesma forma, se você desligar uma subscrição, você será cobrado pelo resto do mês em que você offboarded that subscription.

## <a name="onboard-a-subscription"></a>A bordo de uma subscrição

Para começar com o Azure Defender para IoT, você deve ter uma subscrição microsoft Azure. Se não tiver uma subscrição, pode inscrever-se numa conta gratuita. Se já tem acesso a uma subscrição do Azure, mas não está listada, verifique os dados da sua conta e confirme as suas permissões com o proprietário da subscrição.

Para bordo de uma subscrição:

1. Navegue para a página de preços do portal Azure. 

   :::image type="content" source="media/how-to-manage-subscriptions/no-subscription.png" alt-text="Navegue para a página de preços do portal Azure.":::

1. Selecione **a subscrição a bordo**.

1. Na janela **de subscrição** A bordo selecione a sua subscrição e o número de dispositivos comprometidos a partir dos menus suspensos. 

   :::image type="content" source="media/how-to-manage-subscriptions/onboard-subscription.png" alt-text="selecione a sua subscrição e o número de dispositivos comprometidos.":::

1. Selecione **a bordo**.

## <a name="offboard-a-subscription"></a>Offboard uma subscrição

As assinaturas são geridas mensalmente. Quando você não embarcar uma subscrição, você será cobrado para essa subscrição até o final do mês.

Desinstale todos os sensores associados à subscrição antes de iniciar a subscrição. Para obter mais informações sobre como eliminar um sensor, consulte [Eliminar um sensor](how-to-manage-sensors-on-the-cloud.md#delete-a-sensor). 

Para offboard uma subscrição:

1. Navegue para a página **de preços.**
1. Selecione a subscrição e, em seguida, selecione o ícone **de eliminação** :::image type="icon" source="media/how-to-manage-sensors-on-the-cloud/delete-icon.png" border="false"::: .
1. No popup de confirmação, selecione a caixa de verificação para confirmar que eliminou todos os sensores associados à subscrição.

    :::image type="content" source="media/how-to-manage-sensors-on-the-cloud/offboard-popup.png" alt-text="Selecione a caixa de verificação e selecione offboard para offboard o seu sensor.":::

1. Selecione o **botão Offboard.**
