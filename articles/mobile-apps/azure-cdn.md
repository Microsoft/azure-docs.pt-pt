# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Rede de distribuição de conteúdo no Azure com o serviço CDN do Azure
A [CDN do Azure](https://azure.microsoft.com/services/cdn/) oferece aos desenvolvedores uma solução global para fornecimento rápido de conteúdo de alta largura de banda, como vídeos, imagens ou arquivos estáticos a usuários, armazenando em cache seu conteúdo em centenas de pontos de presença (pop) em todo o mundo. A CDN do Azure também pode acelerar o conteúdo dinâmico, que não pode ser armazenado em cache e permite reduzir os tempos de carregamento, economizar largura de banda e aumentar a capacidade de resposta — independentemente de você estar desenvolvendo ou gerenciando sites ou aplicativos móveis, ou codificando e distribuindo streaming mídia, software de jogos, atualizações de firmware ou pontos de extremidade de IoT.

## <a name="azure-cdn-features"></a>Funcionalidades da CDN do Azure
  - Funciona **diretamente** com os serviços do Azure, incluindo aplicativos Web, serviços de mídia, armazenamento e serviços de nuvem.
  - **Manipule facilmente picos de tráfego e cargas pesadas**.
  - **Integração com a Akamai e a Verizon** para criar e ajustar a estratégia de distribuição de conteúdo.
  - Aceleração de site dinâmico.
  - Controlar o comportamento de cache da CDN do Azure com **regras de cache**.
  - **Suporte a domínio personalizado**HTTPS.
  - **Logs de diagnóstico do Azure** para exibir a análise principal e exportá-los para o armazenamento de BLOBs, hubs de eventos ou Azure monitor.
  - Melhorar o desempenho **compactando arquivos**.
  - **Filtragem geográfica** para restringir o conteúdo da CDN do Azure por país/região.
