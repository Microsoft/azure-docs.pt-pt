# <a name="cloud-hosted-source-code-management-services"></a>Serviços de gestão de códigos de origem hospedados na nuvem
Se tem equipas de desenvolvimento com vários membros da equipa que trabalham na mesma base de código, precisa de encontrar um lugar adequado para hospedar o seu código. Armazenar os dados na nuvem e ter um repositório centralizado permite que todos carreguem, editem e gerem os ficheiros de código. Também podem interagir com outros desenvolvedores em projetos. O código pode ser facilmente acessível, não importa onde esteja, desde que tenha uma ligação à Internet.

Hospedagem em nuvem é muito mais fácil do que opções no local. Requer menos configuração de hardware e permite que as organizações completem o processo de implementação de uma forma mais ágil.

## <a name="benefits-of-hosting-source-code-in-the-cloud"></a>Benefícios de hospedar código fonte na nuvem
- **Armazenamento centralizado em nuvem:** Veja e gere os seus dados a partir de qualquer lugar.
- **Melhor colaboração e código mais limpo:** Acompanhe o código dentro das equipas e gere projetos para garantir a entrega contínua de excelente software.
- **Mais fácil de envolver:** Contribua facilmente para os seus projetos.
- Ciclo de **libertação mais rápido:** Trabalhe mais rápido nas suas equipas e facilmente contribua para os seus projetos.
- **Reduzir o custo:** Não te preocupes em manter o teu próprio hardware, servidores, VPN, e assim por diante.

Utilize os seguintes serviços para alojar os dados da sua aplicação na nuvem.

## <a name="github"></a>GitHub
[GitHub](https://github.com/) é um serviço de hospedagem de repositório de código aberto que acolhe projetos de código fonte em várias línguas de programação diferentes. GitHub acompanha as várias alterações feitas a cada iteração.

**Principais funcionalidades**
- Use o hospedeiro de código para manter todo o seu código num só lugar. Os repositórios privados, públicos e de código de lançamento estão equipados com ferramentas para o ajudar a hospedar, versão e código de lançamento.
- Reveja o código e utilize as ferramentas de revisão incorporadas para fazer da revisão de código uma parte essencial do processo de qualquer equipa:
    - Proteja os balcões, proponha alterações e solicite revisões.
    - Diferenças de manchas, comentários no contexto e obter feedback claro.
- Coordene cedo, mantenha-se alinhado e faça mais com as ferramentas de gestão de projetos do GitHub:
    - Veja o panorama geral do projeto.
    - Utilize quadros de tarefas que estejam mesmo ao lado do seu código dentro do GitHub.
    - Arraste cartões para atribuir problemas ou fazer pedidos aos membros da equipa.
    - Estabeleça marcos para organizar e acompanhar o progresso.
    - Escreva notas para capturar ideias que possam ser úteis, mas que não pertençam a um determinado problema ou a um pedido de puxar.
- Descubra facilmente e escolha as ferramentas certas para uma melhor comunicação e automatização do trabalho comprando aplicações do [GitHub Marketplace.](https://github.com/marketplace)
- Gerir e cultivar equipas utilizando: 
    - Funções do utilizador para ajudar a organizar equipas e aceder a permissões.
    - Discutir ferramentas de linha para manter as conversas sobre o tópico e a equipa focadas.
    - As diretrizes comunitárias para criar rapidamente novos membros da equipa com uma conta.
- Navegue e navegue projetos populares para segui-los.
- Rede e aprender com os outros da indústria.

## <a name="azure-devops"></a>DevOps do Azure
[A Azure DevOps](https://azure.microsoft.com/services/devops/) apoia o [Azure Repos](https://azure.microsoft.com/services/devops/repos/) e o [Team Foundation Version Control (TFVC)](https://docs.microsoft.com/azure/devops/repos/tfvc/index?view=azure-devops) como opções de controlo de fontes. Tem repositórios privados gratuitos ilimitados com revisões de códigocolaborativos, gestão avançada de ficheiros, pesquisa de códigos e políticas de sucursais para garantir código de alta qualidade. O Azure Repos é ótimo para pequenos projetos e grandes organizações que precisam de apoio nativo do Azure Ative Directory e políticas avançadas.
    
**Principais funcionalidades**
- Utilize um repositório ilimitado de código fonte Git hospedado em nuvem para os seus repositórios públicos e privados:
    - Obtenha apoio para qualquer cliente Git.
    - Use web hooks e integração DeAPi.
- Inicie a sua próxima construção a partir de um pedido de repo pull:
    - Colabore para construir um melhor código utilizando a discussão e a integração contínua para cada mudança.
    - Configurar a integração contínua/entrega contínua (CI/CD) para acionar automaticamente as construções, testes e implementações com todos os pedidos de pull concluídos. Pode utilizar os Gasodutos Azure ou as suas ferramentas.
    - Proteja a qualidade do seu código com as políticas de sucursais.
- Mantenha o controlo de versão centralizado com o Team Foundation Version Control, que inclui a revisão de códigos.
- Conecte-se ao seu código utilizando Xcode, Eclipse, IntelliJ, Android Studio, Visual Studio, Visual Studio, Visual Studio Code e muito mais.
- Use uma poderosa pesquisa de código semântico.
- Ganhe benefícios com funcionalidades prontas para a empresa. A Azure DevOps tem integração nativa com o Azure Ative Directory, o que simplifica o processo de gestão do acesso aos seus repositórios de código.
- Garantir a qualidade do código com políticas de sucursais, tais como um número mínimo de revisões de códigos, requisitos para construções bem sucedidas e aplicação de estratégias de fusão git.
- Conecte o seu ambiente de desenvolvimento favorito para aceder a repos e gerir o trabalho.
