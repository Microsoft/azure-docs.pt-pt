# <a name="build-real-time-applications-with-azure-signalr-service"></a>Construa aplicações em tempo real com o Serviço De Sinalização Azure

Com o [Serviço De Sinalização Azure,](https://azure.microsoft.com/services/signalr-service/)pode facilmente adicionar funcionalidade em tempo real às aplicações. Esta funcionalidade em tempo real permite ao serviço empurrar as atualizações de conteúdo para aplicações móveis conectadas. Como resultado, as aplicações são atualizadas sem a necessidade de fazer uma sondagem ao servidor ou enviar novos pedidos de http para atualizações.

## <a name="azure-signalr-service-features"></a>Funcionalidades do Serviço De Sinalização Azure
- Tem um amplo suporte de clientes para uma vasta gama de clientes, tais como navegadores web e móveis, aplicações de desktop, processo de servidor, dispositivos IoT e consolas de jogos.
- Permite que várias instâncias trabalhem em conjunto para escalar milhões de ligações de aplicações móveis. Também apoia várias regiões globais para fins de sharding, alta disponibilidade ou recuperação de desastres.
- Oferece APIs ricos para diferentes padrões de mensagens para enviar mensagens para uma determinada conexão, todas as ligações, ou um subconjunto de conexões que pertencem a um utilizador específico ou foram colocadas num grupo arbitrário.
