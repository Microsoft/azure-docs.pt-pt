# <a name="mobile-application-development-in-microsoft"></a>Desenvolvimento de aplicações móveis na Microsoft
Os decisores, as equipas de desenvolvimento e o departamento de TI precisam de criar uma estratégia de desenvolvimento de aplicações móveis sólida que consiga abordar vários aspetos:
- Visar os tipos de aplicações que correspondam aos objetivos de negócio, tais como Empresa-Negócio (B2E), Empresa-Empresa (B2B) ou Empresa-Consumidor (B2C).
- Decidir se deve investir e criar aplicações para iOS, Android ou Windows com linguagens para uma única plataforma ou se deve optar por uma abordagem para várias plataformas.
- Fornecer serviços de back-end que possam ser dimensionados.
- Integrar perfeitamente nos dados de back-end, seja online ou offline.
- Criar aplicações seguras com capacidades de gestão de autenticação e dispositivos.
- Fornecer de forma contínua e rápida.
- Ligar a sistemas de back-end, seja na cloud ou no local.
- Integrar nos processos e ferramentas existentes.
- Evoluir a aplicação com o negócio.
- Reduzir o tempo de comercialização e permitir aos programadores concentrar os esforços na criação de aplicações de elevada qualidade.

Esta série de documentação explora os **serviços fornecidos pela Microsoft que lhe permitem criar aplicações móveis com base na cloud seguras, fiáveis e dimensionáveis**.

**Público-alvo**

Esta série de documentação será útil tanto para os programadores que estejam a dar os primeiros passos na criação de aplicações móveis como para os especialistas nesta área que queiram saber mais sobre os serviços cloud que os ajudarão a garantir um maior êxito no desenvolvimento de aplicações. Nesta documentação, encontrarão uma descrição geral sucinta de todos os serviços, ferramentas e tecnologias da Microsoft que precisam de conhecer. Esta documentação não inclui exemplos de código nem demonstrações. Destina-se a dar-lhe uma descrição geral de alto nível do processo de desenvolvimento de aplicações e dos serviços associados que deve utilizar. Cada tópico está ligado a outros recursos para lhe dar informações mais aprofundadas. Cada parte da série pode ser lida de forma independente para que possa escolher os tópicos que sejam mais do seu interesse.
