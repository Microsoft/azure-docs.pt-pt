# <a name="mobile-application-development-in-microsoft"></a>Desenvolvimento de aplicações móveis na Microsoft
Os decisores, as equipas de desenvolvimento e os departamentos de TI precisam de criar uma estratégia de desenvolvimento de aplicações móveis sólida que:
- Escolha os tipos de aplicações que correspondam aos objetivos de negócio, tais como empresa-negócio (B2E), empresa-empresa (B2B) ou empresa-consumidor (B2C).
- Decida se deve investir e criar aplicações para iOS, Android ou Windows com linguagens para uma única plataforma ou se deve optar por uma abordagem para várias plataformas.
- Forneça serviços de back-end que possam ser dimensionados.
- Se integre perfeitamente nos dados de back-end, seja online ou offline.
- Crie aplicações seguras com capacidades de gestão de autenticação e dispositivos.
- Forneça de forma contínua e rápida.
- Ligue a sistemas de back-end, seja na cloud ou no local.
- Se integre nos processos e ferramentas existentes.
- Perita que a aplicação evolua com o negócio.
- Reduza o tempo de comercialização e permita aos programadores concentrar os esforços na criação de aplicações de elevada qualidade.

Esta série de documentação explora os serviços fornecidos pela Microsoft que pode utilizar para criar aplicações móveis com base na cloud seguras, fiáveis e dimensionáveis.

**Público-alvo**

Esta série de documentação será útil caso seja um programador que esteja a dar os primeiros passos na criação de aplicações móveis ou um especialista nesta área e queira saber mais sobre os serviços cloud que ajudam a garantir um maior êxito no desenvolvimento de aplicações. Nesta documentação, encontrará uma descrição geral sucinta de todos os serviços, ferramentas e tecnologias da Microsoft que precisa de conhecer. Esta documentação não inclui exemplos de código nem demonstrações. Destina-se a dar-lhe uma descrição geral de alto nível do processo de desenvolvimento de aplicações e dos serviços associados que deve utilizar. Cada artigo está ligado a outros recursos para lhe dar informações mais aprofundadas. Cada parte da série pode ser lida de forma independente para que possa escolher apenas os artigos que sejam do seu interesse.
