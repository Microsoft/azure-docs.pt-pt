# <a name="build-real-time-applications-with-azure-signalr-service"></a>Construir aplicações em tempo real com o Serviço Azure SignalR

Com [o Serviço Azure SignalR,](https://azure.microsoft.com/services/signalr-service/)pode adicionar facilmente funcionalidades em tempo real às aplicações. Esta funcionalidade em tempo real permite ao serviço empurrar atualizações de conteúdo para aplicações móveis conectadas. Como resultado, as aplicações são atualizadas sem a necessidade de pesquisar o servidor ou submeter novos pedidos HTTP para atualizações.

## <a name="azure-signalr-service-features"></a>Funcionalidades do Serviço Azure SignalR
- Possui um amplo suporte ao cliente para uma vasta gama de clientes, tais como navegadores web e móveis, aplicações de desktop, processo de servidor, dispositivos IoT e consolas de jogos.
- Permite que várias instâncias trabalhem em conjunto para escalar para milhões de ligações de aplicações móveis. Também apoia várias regiões globais para fins de sharding, alta disponibilidade ou recuperação de desastres.
- Oferece APIs ricos para diferentes padrões de mensagens para enviar mensagens para uma determinada ligação, todas as ligações, ou um subconjunto de conexões que pertencem a um utilizador específico ou foram colocadas em um grupo arbitrário.
