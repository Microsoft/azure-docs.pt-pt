# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Serviço de nuvem movido a IA com pesquisa cognitiva azure
[A Azure Cognitive Search](https://azure.microsoft.com/services/search/) é uma solução de nuvem de pesquisa como serviço que dá aos desenvolvedores APIs e ferramentas para adicionar uma rica experiência de pesquisa sobre conteúdo privado, heterogéneo em aplicações web, móveis e empresariais. O seu código personalizado invoca a ingestão de dados (indexação) para criar e carregar um índice. Por outro lado, o seu código de aplicação emite pedidos de consulta e lida com respostas. A experiência de pesquisa é definida no seu cliente utilizando a funcionalidade da Azure Cognitive Search, com execução de consulta sobre um índice persistido que cria, possui e armazena no seu serviço.

## <a name="azure-cognitive-search-features"></a>Funcionalidades de Pesquisa Cognitiva Azure
- **Pesquisa gratuita de texto**: Fornece pesquisa completa de texto, sintaxe de consulta simples e sintaxe de consulta Lucene.
- **Perfis de pontuação simples**: Relevância do modelo em função dos valores nos próprios documentos.
- **Geosearch**: Explora dados baseados na proximidade de um resultado de pesquisa para uma localização física.
- **Funcionalidades**de experiências do utilizador : Fornecer sugestões de pesquisa automática, de pesquisa, triagem e paging.
- **Pesquisa cognitiva**: Pode ser aplicado a um pipeline de indexação para extrair informações de texto de conteúdo bruto para análise de imagem e texto.
- **Loja de conhecimento**: Economiza os enriquecimentos criados durante a indexação.
- **Fontes de dados**: Extrair conteúdo pesquisável nas lojas de dados primários quando estiver ligado à Base de Dados Azure SQL, Azure Cosmos DB e armazenamento Azure Blob.
- **Importador de dados assistente**: Configures indexers. 
- **Designer de índices:** Ergue-se um índice. 
- **Explorador de pesquisa:** Testa consultas e refina perfis de pontuação.

Encontre uma descrição detalhada sobre [as funcionalidades](/azure/search/search-what-is-azure-search#feature-descriptions)de Pesquisa Cognitiva Azure .
