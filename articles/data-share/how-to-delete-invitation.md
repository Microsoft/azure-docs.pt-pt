# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share-preview"></a>Como excluir um convite para um destinatário na visualização do compartilhamento de dados do Azure

Este artigo explica como excluir um convite para um destinatário antes de aceitá-lo. Isso impedirá que eles criem uma assinatura de compartilhamento e nunca receberão nenhum dado compartilhado. Se você excluir o convite para um consumidor que já criou uma assinatura de compartilhamento, a exclusão de seu convite não terá nenhum impacto.

## <a name="navigate-to-a-sent-data-share"></a>Navegar até um compartilhamento de dados enviado

Na visualização do compartilhamento de dados do Azure, navegue até o compartilhamento enviado e selecione a guia **convites** . Aqui estão listados todos os destinatários de convites para esse compartilhamento de dados. Clique nas caixas ao lado dos destinatários cujos convites você gostaria de excluir e, em seguida, clique em **excluir**.
