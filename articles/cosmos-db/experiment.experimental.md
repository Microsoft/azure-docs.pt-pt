# <a name="azure-cosmos-db-documentation-experiment"></a>Experimentação de documentação do Cosmos BD do Azure

BD do Cosmos do Azure é uma base de dados globalmente distribuído, com vários modelo. Porque, pode fazê-lo, pelo que, muito com o Azure Cosmos DB - a partir de criação de um gráfico de modo a criar uma base de dados de documento ou migrar os dados de armazenamento MongoDB, Cassandra ou tabelas do Azure - organizar a documentação pode ser um desafio. Por isso, gostaríamos sua ajudá-lo.

Iremos estiver a testar a organização da tabela de base de dados do Azure Cosmos do conteúdo e [página de destino](https://docs.microsoft.com/en-us/azure/cosmos-db/), conforme mostrado nas seguintes imagens, e gostaríamos de obter os seus comentários sobre estas propostas de alterações.

## <a name="to-load-both-views"></a>Para ambas as vistas de carga

1. Para abrir a ambas as vistas lado a lado, navegue para o [página de destino](https://docs.microsoft.com/en-us/azure/cosmos-db/) do seu browser favorito. 
2. Utilizar imagens acima para determinar que versão vir, original ou experimental. 
3. Abra uma janela do browser inprivate ou incognito e carregue o [página de destino](https://docs.microsoft.com/en-us/azure/cosmos-db/) na nova janela do browser e utilizar novamente as imagens para determinar qual é a versão que vê. 

   Se não tiver ambas as vistas presentes, feche e volte a abrir uma nova janela inprivate ou incognito e a carga de [página de destino](https://docs.microsoft.com/en-us/azure/cosmos-db/) novamente. Pode demorar alguns tenta obter ambas as versões carregadas no windows separados. Atualizar a janela incognito não será carregado a outra versão, terá de abrir uma nova janela. 

4. Em seguida, pode redimensionar do windows para corrigir lado, ou mudar novamente e estabelecido entre vistas para comparar. 

5. Depois de ter tido oportunidade para se familiarizar com o esquema da experiência experimental e organização, clique em de **experimentação inquérito** botão abaixo para fornecer comentários. Submeta o formulário não pode ser posterior à meia-noite 4 de Abril de 2018 para que os comentários podem ser revistos e incorporados.
