|**Descrição**|A biblioteca de executores em massa permite que aplicativos cliente executem operações em massa em contas Azure Cosmos DB. a biblioteca de executores em massa fornece namespaces BulkImport e BulkUpdate. O módulo BulkImport pode ingerir documentos em massa de forma otimizada, de modo que a taxa de transferência provisionada para uma coleção seja consumida em sua extensão máxima. O módulo BulkUpdate pode atualizar dados existentes em massa em contêineres Cosmos do Azure como patches.|
