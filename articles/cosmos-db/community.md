# <a name="community-portal"></a>Portal da comunidade
## <a name="community-spotlight"></a>Spotlight de Comunidade
Informe-na promover o projeto! Mostrar-no projeto extraordinário que está a trabalhar na base de dados do Azure Cosmos, a próxima geração da pilha de DocumentDB, e iremos ajudar partilhar o seu genius com o mundo. Para submeter o seu projeto, envie-num e-mail em: [ askcosmosdb@microsoft.com ](mailto:askcosmosdb@microsoft.com).

### <a name="documentdb-lumenize"></a>documentdb-lumenize
*por Larry Maccherone*

As agregações (agrupar por, tabela dinâmica e cubo N-dimensional) e transformações de séries de tempo como procedimentos armazenados no DocumentDB do Azure Cosmos DB API.

A verificar terminar em [GitHub](https://github.com/lmaccherone/documentdb-lumenize) e [npm](https://www.npmjs.com/package/lumenize).

### <a name="documentdb-studio"></a>O DocumentDB Studio
*por Ming Liu*

Um cliente Visualizador/Explorador de gestão para o serviço de API de DocumentDB do Azure Cosmos DB.

A verificar terminar em [GitHub](https://github.com/mingaliu/DocumentDBStudio).

### <a name="doqmentdb"></a>DoQmentDB
*por Ariel Mashraki*

DoQmentDB é um cliente de baseado em promessa de Node.js, que fornece uma camada de como o MongoDB por cima da BD do Cosmos do Azure.

A verificar terminar em [GitHub](https://github.com/a8m/doqmentdb) e [npm](https://www.npmjs.com/package/doqmentdb).

### <a name="typescript-api"></a>TypeScript API
*por Jelmer Cormont*

Um wrapper em torno do cliente do Node.js escrito em TypeScript (funciona em JavaScript simples demasiado). Suporta `async/await` e uma API simplificada.

A verificar terminar em [GitHub](https://github.com/jcormont/documentdb-typescript) e [npm](https://www.npmjs.com/package/documentdb-typescript).

### <a name="swagger-rest-api-for-documentdb"></a>API de REST do swagger para o DocumentDB
*por Howard Edidin*

Um ficheiro Swagger de API REST do Azure Cosmos DB para a API do DocumentDB que pode ser facilmente implementado como uma aplicação API.

A verificar terminar em [GitHub](https://github.com/HEDIDIN/DocumentDB-REST/tree/master/DocumentDBRestApi).

### <a name="fluent-plugin-documentdb"></a>documentdb Fluent Plug-in
*por Yoichi Kawasaki*

Fluent Plug-in do documentdb é um plug-in de Fluentd para exportar a API de DocumentDB do Azure Cosmos DB.

A verificar terminar em [GitHub](https://github.com/yokawasa/fluent-plugin-documentdb) e [rubygems](https://rubygems.org/gems/fluent-plugin-documentdb).

*Localizar mais open source para projetos de base de dados do Azure Cosmos no [GitHub](https://github.com/search?p=4&q=documentdb&type=Repositories).*
