# <a name="startswith-azure-cosmos-db"></a>STARTSWITH (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

 Devolve um Boolean indicando se a primeira expressão de corda começa com a segunda.  
  
## <a name="syntax"></a>Sintaxe
  
```sql
STARTSWITH(<str_expr1>, <str_expr2> [, <bool_expr>])  
```  
  
## <a name="arguments"></a>Argumentos
  
*str_expr1*  
   É uma expressão de corda.
  
*str_expr2*  
   É uma expressão de corda a ser comparada com o início de *str_expr1*.

*bool_expr* Valor opcional para ignorar o caso. Quando definido como verdadeiro, STARTWITH fará uma pesquisa caso-insensível. Quando não especificado, este valor é falso.

## <a name="return-types"></a>Tipos de retorno
  
  Devolve uma expressão booleana.  
  
## <a name="examples"></a>Exemplos
  
O exemplo a seguir verifica se a cadeia "abc" começa com "b" e "A".  
  
```sql
SELECT STARTSWITH("abc", "b", false) AS s1, STARTSWITH("abc", "A", false) AS s2, STARTSWITH("abc", "A", true) AS s3
```  
  
 Aqui está o conjunto de resultados.  
  
```json
[
    {
        "s1": false,
        "s2": false,
        "s3": true
    }
]
```
