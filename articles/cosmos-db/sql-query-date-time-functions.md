# <a name="date-and-time-functions-azure-cosmos-db"></a>Funções de data e hora (Azure Cosmos DB)

As funções de data e hora permitem-lhe executar as operações de datetime e timetamp em Azure Cosmos DB.

## <a name="functions"></a>Funções

As seguintes funções de escalar permitem-lhe obter a data e hora atuais da UTC em duas formas; Um numérico de tempotaque cujo valor é a época Unix em milissegundos ou como uma cadeia que está em conformidade com o formato ISO 8601:

* [GetCurrentDateTime](sql-query-getcurrentdatetime.md)
* [GetCurrentTimestamp](sql-query-getcurrenttimestamp.md)
