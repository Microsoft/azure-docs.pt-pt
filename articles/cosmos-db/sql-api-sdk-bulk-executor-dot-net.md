Biblioteca de BulkExecutor fornece BulkImport e BulkUpdate espaços de nomes. BulkImport módulo pode efetuar em massa de inserção documentos de uma forma otimizada, de modo a que o débito aprovisionado para uma coleção é consumido para o respetivo limite máximo permitido. BulkUpdate módulo pode efetuar em massa atualizar os dados existentes na base de dados do Azure Cosmos contentores como patches.</td></tr>
