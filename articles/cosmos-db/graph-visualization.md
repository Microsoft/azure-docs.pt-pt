# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualize os dados armazenados na API do Azure Cosmos DB Gremlin com soluções de visualização de dados do gráfico

É possível visualizar os dados armazenados na API do Gremlin do Azure Cosmos DB ao utilizar várias soluções de visualização de dados. As seguintes soluções são recomendadas o [Comunidade do Apache Tinkerpop](http://tinkerpop.apache.org/#poweredby) gráfico para visualização de dados.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Logótipo de Linkurious](./media/graph-visualization/linkurious-logo.jpg)

[Linkurious Enterprise](https://linkurio.us/product/) utiliza tecnologia e visualização de dados para transformar os conjuntos de dados complexos em redes de visual interativas do gráfico. A plataforma liga-se às suas origens de dados e permite investigators navegar de forma totalmente integrada em biliões de entidades e relações. O resultado é uma nova capacidade de detetar relações suspeitas sem malabarismo com tabelas ou consultas.

A interface interativa do Linkurious Enterprise oferece uma forma fácil para investigar dados complexos. Pode procurar entidades específicas, expanda conexões para descobrir as relações ocultas e aplicar esquemas de sua escolha desvincular redes complexas. Linkurious Enterprise é agora compatível com a API do Azure Cosmos DB Gremlin. É adequado a cenários de visualização de gráfico-a-ponto e suporta a leitura e escrita capacidades da interface do usuário. Pode pedir um [demonstração de Linkurious com o Azure Cosmos DB](https://linkurio.us/contact/)

![Demonstração de Linkurious](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>Inteligência de Cambridge

![Logótipo de inteligência de Cambridge](./media/graph-visualization/ci-logo.png)

[Da inteligência de Cambridge](https://cambridge-intelligence.com/products/) Kits de ferramentas de visualização de gráfico suportam agora o Azure Cosmos DB. Os kits de ferramentas visualização seguintes dois são suportados pelo Azure Cosmos DB:

- [KeyLines para desenvolvedores de JavaScript](https://cambridge-intelligence.com/keylines/)

- [Criar um gráfico novamente para desenvolvedores do React](https://cambridge-intelligence.com/regraph/)

![Demonstração de inteligência de Cambridge](./media/graph-visualization/ci-demo-2.gif)

Esses Kits de ferramentas permitem-lhe criar a visualização de gráfico de alto desempenho e aplicações de análise para seu caso de utilização. Eles tire partido da composição de Library(WebGL) de gráficos de Web potente e código cuidadosamente elaborado para dar aos utilizadores uma experiência de visualização rápida e esclarecedores. Essas ferramentas são compatíveis com qualquer navegador, dispositivo, servidor ou base de dados e são fornecidos com os tutoriais passo a passo, APIs devidamente documentadas e demonstrações interativas.

![Demonstração de inteligência de Cambridge](./media/graph-visualization/ci-demo-1.gif)
