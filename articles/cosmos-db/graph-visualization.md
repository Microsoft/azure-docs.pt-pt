# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualizar dados de gráficos armazenados na API Azure Cosmos DB Gremlin com soluções de visualização de dados

Pode visualizar dados armazenados na API Azure Cosmos DB Gremlin utilizando várias soluções de visualização de dados. As seguintes soluções são recomendadas pela [comunidade Apache Tinkerpop](https://tinkerpop.apache.org/#poweredby) para visualização de dados gráficos.

## <a name="linkurious-enterprise"></a>Empresa Linkurious

![Logotipo linkurious](./media/graph-visualization/linkurious-logo.jpg)

[A Linkurious Enterprise](https://linkurio.us/product/) utiliza tecnologia de gráficos e visualização de dados para transformar conjuntos de dados complexos em redes visuais interativas. A plataforma conecta-se às suas fontes de dados e permite que os investigadores naveguem perfeitamente através de milhares de milhões de entidades e relacionamentos. O resultado é uma nova capacidade de detetar relações suspeitas sem fazer malabarismos com consultas ou mesas.

A interface interativa da Linkurious Enterprise oferece uma maneira fácil de investigar dados complexos. Você pode pesquisar por entidades específicas, expandir ligações para descobrir relacionamentos ocultos, e aplicar layouts da sua escolha para desembaraçar redes complexas. A Linkurious Enterprise é agora compatível com a Azure Cosmos DB Gremlin API. É adequado para cenários de visualização de gráficos de ponta a ponta e suporta capacidades de leitura e escrita a partir da interface do utilizador. Você pode solicitar uma [demonstração de Linkurious com Azure Cosmos DB](https://linkurio.us/contact/)

![Linkurious Demo](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>Inteligência de Cambridge

![Logotipo da Inteligência de Cambridge](./media/graph-visualization/ci-logo.png)

Os kits de ferramentas de visualização de gráficos [da Cambridge Intelligence](https://cambridge-intelligence.com/products/) suportam agora o Azure Cosmos DB. Os seguintes dois kits de ferramentas de visualização são suportados por Azure Cosmos DB:

- [KeyLines para desenvolvedores JavaScript](https://cambridge-intelligence.com/keylines/)

- [Re-Graph para desenvolvedores de reagir](https://cambridge-intelligence.com/regraph/)

![Demonstração de Inteligência de Cambridge](./media/graph-visualization/ci-demo-2.gif)

Estes kits de ferramentas permitem-lhe conceber aplicações de visualização e análise de gráficos de alto desempenho para o seu caso de utilização. Eles aproveitam a poderosa Web Graphics Library (WebGL) renderização e código cuidadosamente elaborado para dar aos utilizadores uma experiência de visualização rápida e perspicaz. Estas ferramentas são compatíveis com qualquer navegador, dispositivo, servidor ou base de dados, e vêm com tutoriais passo a passo, APIs totalmente documentados e demonstrações interativas.

![Demonstração de Inteligência de Cambridge](./media/graph-visualization/ci-demo-1.gif)
