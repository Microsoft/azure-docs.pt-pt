# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualize os seus dados de gráfico na API do Azure Cosmos DB Gremlin com soluções de visualização de dados

As seguintes soluções são recomendadas o [Comunidade do Apache Tinkerpop](http://tinkerpop.apache.org/#poweredby) para visualização de gráfico Gremlin.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Logótipo de Linkurious](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise utiliza a visualização de tecnologia e os dados de gráfico para transformar os conjuntos de dados complexos em redes de visual interativas. A plataforma liga-se às suas origens de dados e permite investigators navegar de forma totalmente integrada em biliões de entidades e relações. O resultado é uma nova capacidade de detetar relações suspeitas sem malabarismo com tabelas ou consultas.

A interface de visualização interativa do Linkurious Enterprise oferece uma forma fácil para investigar dados ligados complexos. Procurar entidades específicas, expanda conexões para descobrir as relações ocultas e aplicar esquemas de sua escolha desvincular redes complexas. Linkurious é agora compatível com a API do Azure Cosmos DB Gremlin. É adequado a cenários de visualização de gráfico-a-ponto e suporta a leitura e escrita capacidades da interface do usuário. 

![Demonstração de Linkurious](./media/graph-visualization/linkurious-demo.gif)

**Passos seguintes:** Pode pedir uma [demonstração de Linkurious com o Azure Cosmos DB](https://linkurio.us/contact/).

![Logótipo de inteligência de Cambridge](./media/graph-visualization/ci-logo.png)


Kits de ferramentas de visualização de gráfico de Cambridge do Intelligence suportam agora o Azure Cosmos DB:
- [KeyLines para desenvolvedores de JavaScript](https://cambridge-intelligence.com/keylines/) 
- [Criar um gráfico novamente para desenvolvedores do React](https://cambridge-intelligence.com/regraph/)

![Demonstração de inteligência de Cambridge](./media/graph-visualization/ci-demo-2.gif)

Os kits de ferramentas permitem-lhe criar a visualização de gráfico de alto desempenho e aplicações de análise para seu caso de utilização. Eles tire partido da composição de WebGL poderosas e código cuidadosamente elaborado para dar aos utilizadores uma experiência de visualização rápida e esclarecedores.

Estão compatíveis com qualquer navegador, dispositivo, servidor ou base de dados e são fornecidos com os tutoriais passo a passo, APIs devidamente documentadas e demonstrações interativas.
