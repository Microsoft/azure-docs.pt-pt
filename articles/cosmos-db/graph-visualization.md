# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualizar dados gráficos armazenados na API API da Azure Cosmos DB Gremlin com soluções de visualização de dados

Pode visualizar dados armazenados na Azure Cosmos DB Gremlin API utilizando várias soluções de visualização de dados. As seguintes soluções são recomendadas pela [comunidade Apache Tinkerpop](https://tinkerpop.apache.org/#poweredby) para visualização de dados gráficos.

## <a name="linkurious-enterprise"></a>Empresa Linkurious

:::image type="content" source="./media/graph-visualization/linkurious-logo.jpg" alt-text="Logotipo linkurious" border="false":::

[A Linkurious Enterprise](https://linkurio.us/product/) utiliza tecnologia de gráficos e visualização de dados para transformar conjuntos de dados complexos em redes visuais interativas. A plataforma conecta-se às suas fontes de dados e permite que os investigadores naveguem perfeitamente através de milhares de milhões de entidades e relacionamentos. O resultado é uma nova capacidade de detetar relações suspeitas sem fazer malabarismos com consultas ou tabelas.

A interface interativa da Linkurious Enterprise oferece uma maneira fácil de investigar dados complexos. Você pode pesquisar por entidades específicas, expandir conexões para descobrir relacionamentos ocultos, e aplicar layouts da sua escolha para desemargar redes complexas. A Linkurious Enterprise é agora compatível com a AZure Cosmos DB Gremlin API. É adequado para cenários de visualização de gráficos de ponta a ponta e suporta capacidades de leitura e escrita a partir da interface do utilizador. Você pode solicitar uma [demonstração de Linkurious com Azure Cosmos DB](https://linkurio.us/contact/)

## <a name="cambridge-intelligence"></a>Inteligência de Cambridge

:::image type="content" source="./media/graph-visualization/ci-logo.png" alt-text="Logotipo da Inteligência de Cambridge" border="false":::

[Os](https://cambridge-intelligence.com/products/) kits de ferramentas de visualização de gráficos da Cambridge Intelligence agora suportam Azure Cosmos DB. Os dois kits de ferramentas de visualização são suportados pela Azure Cosmos DB:

- [KeyLines para desenvolvedores JavaScript](https://cambridge-intelligence.com/keylines/)

- [Re-Graph para desenvolvedores react](https://cambridge-intelligence.com/regraph/)

:::image type="content" source="./media/graph-visualization/ci-demo-2.gif" alt-text="Demonstração de Inteligência de Cambridge" border="false":::

Estes conjuntos de ferramentas permitem-lhe conceber aplicações de visualização e análise de gráficos de alto desempenho para o seu caso de utilização. Eles aproveitam a poderosa renderização da Web Graphics Library (WebGL) e código cuidadosamente elaborado para dar aos utilizadores uma experiência de visualização rápida e perspicaz. Estas ferramentas são compatíveis com qualquer navegador, dispositivo, servidor ou base de dados, e vêm com tutoriais passo a passo, APIs totalmente documentados e demonstrações interativas.
