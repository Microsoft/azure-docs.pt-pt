# <a name="azure-cosmos-db-query-cheat-sheets"></a>Folhas de batota de consulta azure Cosmos DB

As folhas de **batota de consulta Do MB Azure** ajudam-no a escrever rapidamente consultas para os seus dados, exibindo consultas comuns de bases de dados, operações, funções e operadores em folhas de referência PDF fáceis de imprimir. As folhas de batota incluem informações de referência para as APIs SQL, MongoDB, Table e Gremlin. 

Escolha entre um download do tamanho de uma letra ou do tamanho De A3. 

## <a name="letter-sized-cheat-sheets"></a>Folhas de batota do tamanho de uma letra

Descarregue as folhas de batota de consulta do tamanho de uma carta do [Azure Cosmos DB](https://go.microsoft.com/fwlink/?LinkId=623215) se vai imprimir para papel de tamanho de letra (8,5" x 11").

![Folhas de batota de consulta Azure Cosmos DB - tamanho de letra, com SQL API, JavaScript, MongoDB, Gremlin e Tabela API consultas e funções](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Folhas de batota de grandes dimensões
Descarregue as folhas de batota de consulta do [tamanho de Azure Cosmos DB A3](https://go.microsoft.com/fwlink/?linkid=870413) se você vai imprimir usando um plotter ou impressora de grande escala em papel de tamanho A3 (11,7" x 16,5").

![Folhas de batota de consulta Azure Cosmos DB - A3-size, com SQL API, JavaScript, MongoDB, Gremlin e Tabela API consultas e funções](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
