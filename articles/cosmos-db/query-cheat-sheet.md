# <a name="azure-cosmos-db-query-cheat-sheets"></a>Azure Cosmos DB consulta folhas de batota

As **folhas de fraude de consulta Azure Cosmos DB** ajudam-no a escrever rapidamente consultas para os seus dados, exibindo consultas comuns de base de dados, operações, funções e operadores em folhas de referência PDF fáceis de imprimir. As folhas de batota incluem informações de referência para as APIs SQL, MongoDB, Table e Gremlin. 

Escolha entre um download do tamanho de uma letra ou A3. 

## <a name="letter-sized-cheat-sheets"></a>Folhas de batota do tamanho de uma letra

Baixe as [folhas de fraude do tamanho de uma letra do Azure Cosmos DB](https://go.microsoft.com/fwlink/?LinkId=623215) se for imprimir para papel do tamanho de uma letra (8,5" x 11").

:::image type="content" source="./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png" alt-text="Azure Cosmos DB consultas folhas de batota - tamanho de letra, com SQL API, JavaScript, MongoDB, Gremlin, e tabela API consultas e funções" border="false":::

## <a name="oversized-cheat-sheets"></a>Folhas de batota de grandes dimensões
Descarregue as [folhas de fraude do tamanho de Azure Cosmos DB A3](https://go.microsoft.com/fwlink/?linkid=870413) se for imprimir usando um plotter ou impressora em larga escala em papel de tamanho A3 (11,7" x 16,5").
