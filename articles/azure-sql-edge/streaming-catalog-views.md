# <a name="streaming-catalog-views-transact-sql"></a>Vistas de catálogo de streaming (Transact-SQL)

Esta secção contém as vistas e funções de catálogo disponíveis que estão relacionadas com o Streaming T-SQL.
  
## <a name="in-this-section"></a>Nesta secção  
  
|Vista|Descrição|  
|:---|:---|
|[sys.external_streams](sys-external-streams.md) |Devolve uma linha para cada objeto de fluxo externo criado no âmbito da base de dados.|
|[sys.external_streaming_jobs](sys-external-streaming-jobs.md) |Devolve uma linha para cada trabalho de streaming externo criado no âmbito da base de dados.|
|[sys.external_job_streams](sys-external-job-streams.md)|Devolve uma linha cada uma para o objeto de fluxo externo de entrada ou saída mapeado para um trabalho de streaming externo.|
