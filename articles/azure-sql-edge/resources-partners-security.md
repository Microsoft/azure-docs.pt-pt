# <a name="azure-sql-edge-security-partners"></a>Parceiros de segurança Azure SQL Edge

Este artigo destaca as empresas parceiras da Microsoft com soluções de segurança para fornecer soluções de segurança final para as suas implementações IoT Edge utilizando o Azure SQL Edge.

## <a name="security-partners"></a>Parceiros de segurança
 
| Parceiro| Description | Ligações |
|-----|-----|-----|
|![DH2i](media/resources/dh2i-logo.png)|A DH2i assume uma nova abordagem inovadora à conectividade de networking, permitindo às organizações com o seu Software Defined Perimeter (SDP) Always-Secure e Always-On infraestrutura de TI. O DxOdyssey para IoT estende-o a dispositivos de borda, permitindo o acesso sem emenda dos dispositivos de borda para o centro de dados e nuvem. Este módulo SDP funciona em qualquer dispositivo IoT em um recipiente na arquitetura x64 e arm64. Uma vez ativados, as organizações podem criar túneis seguros e privados de nível de aplicação entre dispositivos e hubs sem a exigência de uma VPN ou expor portas abertas e públicas. Este módulo SDP é construído de propósito para casos de utilização IoT onde os dispositivos de borda devem comunicar com quaisquer outros dispositivos, recursos, aplicações ou nuvens.
