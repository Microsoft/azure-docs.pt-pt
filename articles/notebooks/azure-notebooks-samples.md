# <a name="discover-and-use-sample-notebooks"></a>Descubra e use blocos de anotações de exemplo

A Internet tem muitas coleções de notebooks Jupyter interessantes e úteis que você pode facilmente colocar em Azure Notebooks. Este artigo fornece uma lista de exemplos notáveis.

Depois de identificar um bloco de anotações que você gostaria de explorar, aqui estão alguns detalhes que você desejará entender:

- Se um bloco de anotações já estiver em Azure Notebooks, use o botão **clonar** em seu painel de projeto para fazer uma cópia do projeto em sua própria conta.
- Alguns blocos de anotações exigem pacotes específicos para serem executados. Embora você possa usar o `!pip install` (Python) ou o `install.packages` (R) para instalar pacotes de um próprio bloco de anotações, também é possível [configurar um projeto de Azure notebooks com o ambiente apropriado](configure-manage-azure-notebooks-projects.md).

## <a name="great-sample-notebooks"></a>Excelentes blocos de anotações de exemplo

- [Azure notebooks conjunto de início](https://notebooks.azure.com/#sample-redirect) inclui introduções para Python, R e F#, juntamente com notebooks que demonstram acesso a dados, Azure Machine Learning e alguns exercícios de ciência de dados.
- [Introdução aos Jupyter notebooks](https://nbviewer.jupyter.org/github/jupyter/notebook/blob/master/docs/source/examples/Notebook/Notebook%20Basics.ipynb) (jupyter.org)
- [Introdução à ciência de dados](https://github.com/jakevdp/PythonDataScienceHandbook/tree/master/notebooks) da Jake Vanderplas.
- [Galeria de blocos de anotações interessantes](https://github.com/ipython/ipython/wiki/A-gallery-of-interesting-IPython-Notebooks) (projeto Jupyter no GitHub)
- [nbviewer.jupyter.org](https://nbviewer.jupyter.org)
- Instrução de física do [departamento de física na faculdade Imperial de Londres](https://notebooks.azure.com/cvanbreu/libraries)
- [Exemplos de introdução do Azure Machine Learning](https://notebooks.azure.com/azureml/projects/azureml-getting-started)
