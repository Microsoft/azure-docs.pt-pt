# <a name="discover-and-use-sample-notebooks-in-azure-notebooks-preview"></a>Descubra e use cadernos de amostras em Pré-visualização de Cadernos Azure

[!INCLUDE [notebooks-status](../../includes/notebooks-status.md)]

A Internet tem muitas coleções de cadernos Jupyter interessantes e úteis que você pode facilmente trazer para os Cadernos Azure. Este artigo fornece uma lista de amostras notáveis.

Uma vez identificado um caderno que gostaria de explorar, aqui estão alguns detalhes que vai querer entender:

- Se um caderno já estiver em Azure Notebooks, use o botão **Clone** no seu painel de projeto para fazer uma cópia do projeto na sua própria conta.
- Alguns cadernos requerem pacotes específicos para executar. Embora possa utilizar `!pip install` (Python) ou `install.packages` (R) para instalar pacotes a partir de um próprio caderno, também pode [configurar um projeto Azure Notebooks com o ambiente apropriado.](configure-manage-azure-notebooks-projects.md)

## <a name="great-sample-notebooks"></a>Grandes cadernos de amostra

- [O conjunto de iniciações Azure Notebooks](https://notebooks.azure.com/#sample-redirect) inclui introduções a Python, R e F#, juntamente com cadernos que demonstram acesso a dados, Azure Machine Learning e alguns exercícios de ciência de dados.
- [Introdução aos cadernos jupyter](https://nbviewer.jupyter.org/github/jupyter/notebook/blob/master/docs/source/examples/Notebook/Notebook%20Basics.ipynb) (jupyter.org)
- [Introdução à Ciência dos Dados](https://github.com/jakevdp/PythonDataScienceHandbook/tree/master/notebooks) por Jake Vanderplas.
- [Galeria de cadernos interessantes](https://github.com/ipython/ipython/wiki/A-gallery-of-interesting-IPython-Notebooks) (projeto Jupyter no GitHub)
- [nbviewer.jupyter.org](https://nbviewer.jupyter.org)
- Instrução física do [Departamento de Física do Imperial College de Londres](https://notebooks.azure.com/cvanbreu/libraries)
- [Azure Machine Learning recebendo amostras inicios](https://notebooks.azure.com/azureml/projects/azureml-getting-started)
