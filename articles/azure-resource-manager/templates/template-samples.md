# <a name="azure-resource-manager-templates-for-management-features"></a>Modelos de Gestor de Recursos Azure para funcionalidades de gestão

A tabela seguinte inclui links para os modelos do Gestor de Recursos Azure para as funcionalidades fornecidas pelo Gestor de Recursos.

| | |
|-|-|
|**Atribuições de funções**||
| [Designar função para grupo de recursos](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Atribui um papel incorporado a um utilizador para um grupo de recursos existente. |
| [Designar função para máquina virtual existente](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Atribui um papel incorporado a um utilizador para um VM existente. |
| [Designar função para várias máquinas virtuais](https://github.com/Azure/azure-quickstart-templates/tree/master/201-rbac-builtinrole-multipleVMs)| Atribui funções incorporadas aos utilizadores para mais do que uma máquina virtual. |
| [Atribuir papel para subscrição do Azure](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/subscription-role-assigment)| Atribui um papel a um utilizador para uma subscrição Do Azure. |
|**Definição de função**||
| [Criar definição de papel personalizado](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-role-def)| Cria uma nova definição de papel numa subscrição azure. |
|**Bloqueio de recursos**||
| [Grupo de recursos de bloqueio](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-rg-lock-role-assignment)| Cria um grupo de recursos e aplica um bloqueio **DoNotDelete** ao grupo de recursos. Atribui o papel de contribuinte a um utilizador. |
