# <a name="azure-resource-manager-templates-for-management-features"></a>Modelos de Gestor de Recursos Azure para funcionalidades de gestão

A tabela seguinte inclui links para modelos do Gestor de Recursos Azure para as funcionalidades fornecidas pelo Gestor de Recursos.

| | |
|-|-|
|**Atribuições de funções**||
| [Atribuir papel para grupo de recursos](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Atribui uma função incorporada a um utilizador para um grupo de recursos existente. |
| [Atribuir papel para máquina virtual existente](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Atribui uma função incorporada a um utilizador para um VM existente. |
| [Atribuir papel para várias máquinas virtuais](https://github.com/Azure/azure-quickstart-templates/tree/master/201-rbac-builtinrole-multipleVMs)| Atribui funções incorporadas aos utilizadores para mais de uma máquina virtual. |
| [Atribuir papel para subscrição do Azure](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/subscription-role-assignment)| Atribui uma função a um utilizador para uma subscrição Azure. |
|**Definição de função**||
| [Criar definição de função personalizada](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/create-role-def)| Cria uma nova definição de papel numa subscrição do Azure. |
|**Bloqueio de recursos**||
| [Grupo de recursos de bloqueio](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-deployments/create-rg-lock-role-assignment)| Cria um grupo de recursos e aplica um bloqueio **DoNotDelete** ao grupo de recursos. Atribui a função de contribuinte a um utilizador. |
