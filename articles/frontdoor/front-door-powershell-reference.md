# Módulo de AzureRM.FrontDoor
Os tópicos nesta secção documentam os cmdlets do PowerShell do Azure desde início do serviço do Azure que utilizam o Azure Resource Manager. Os cmdlets existe no espaço de nomes Microsoft.Azure.Commands.FrontDoor.

## Cmdlets de AzureRM.FrontDoor
### [Get-AzureRmFrontDoor](Get-AzureRmFrontDoor.md)
Obtenha o Balanceador de carga de porta de entrada

### [Get-AzureRmFrontDoorFireWallPolicy](Get-AzureRmFrontDoorFireWallPolicy.md)
Obter política de firewall de aplicações web

### [Novo AzureRmFrontDoor](New-AzureRmFrontDoor.md)
Criar um novo Balanceador de carga de porta de entrada do Azure

### [Novo AzureRmFrontDoorBackendObject](New-AzureRmFrontDoorBackendObject.md)
Criar um objeto de PSBackend

### [Novo AzureRmFrontDoorBackendPoolObject](New-AzureRmFrontDoorBackendPoolObject.md)
Criar um objeto de PSBackendPool para criação de porta de entrada

### [Novo AzureRmFrontDoorCustomRuleObject](New-AzureRmFrontDoorCustomRuleObject.md)
Criar o objeto de CustomRule WAF para criação de políticas

### [Novo AzureRmFrontDoorFireWallPolicy](New-AzureRmFrontDoorFireWallPolicy.md)
Criar política de WAF

### [Novo AzureRmFrontDoorFrontendEndpointObject](New-AzureRmFrontDoorFrontendEndpointObject.md)
Criar um objeto de PSFrontendEndpoint para criação de porta de entrada

### [Novo AzureRmFrontDoorHealthProbeSettingObject](New-AzureRmFrontDoorHealthProbeSettingObject.md)
Criar um objeto de PSHealthProbeSetting para criação de porta de entrada

### [Novo AzureRmFrontDoorLoadBalancingSettingObject](New-AzureRmFrontDoorLoadBalancingSettingObject.md)
Criar um objeto de PSLoadBalancingSetting para criação de porta de entrada

### [Novo AzureRmFrontDoorManagedRuleObject](New-AzureRmFrontDoorManagedRuleObject.md)
Criar o objeto de ManagedRule WAF para criação de políticas

### [Novo AzureRmFrontDoorMatchConditionObject](New-AzureRmFrontDoorMatchConditionObject.md)
Criar o objeto de MatchCondition WAF para criação de políticas

### [Novo AzureRmFrontDoorRoutingRuleObject](New-AzureRmFrontDoorRoutingRuleObject.md)
Criar um PSRoutingRuleObject para a criação de porta de entrada

### [Novo AzureRmFrontDoorRuleGroupOverrideObject](New-AzureRmFrontDoorRuleGroupOverrideObject.md)
Criar o objeto de RuleGroupOverride WAF para criação de políticas

### [Remove-AzureRmFrontDoor](Remove-AzureRmFrontDoor.md)
Remover Balanceador de carga de porta de entrada

### [Remove-AzureRmFrontDoorFireWallPolicy](Remove-AzureRmFrontDoorFireWallPolicy.md)
Remover política do WAF

### [Conjunto AzureRmFrontDoor](Set-AzureRmFrontDoor.md)
Atualizar um balanceador de carga de porta de entrada

### [Conjunto AzureRmFrontDoorFireWallPolicy](Set-AzureRmFrontDoorFireWallPolicy.md)
Atualizar a política de WAF
