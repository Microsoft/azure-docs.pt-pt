# <a name="rest-api-specification-for-azure-front-door-service"></a>Especificação de REST API do serviço de porta de entrada do Azure
O serviço de porta de entrada do Azure permite-lhe definir, gerir e monitorizar o roteamento global para o tráfego da web ao otimizar para melhor desempenho e a ativação pós-falha instantânea de global de elevada disponibilidade. Com a porta de entrada, pode transformar o seu consumo de global (várias regiões) e aplicações empresariais em aplicações modernas personalizadas robustas de alto desempenho, APIs e conteúdo que alcançam um público global com o Azure a pedido.


A tabela seguinte fornece uma referência para a API REST do serviço de porta de entrada do Azure.

Versão de API: 2018-08-01

| Grupo de operação | Descrição |
|-------------|------------|
| [Porta de entrada](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/frontdoor.json) | Define a forma como pode criar e gerir a sua configuração de porta de entrada |
| [Política de Firewall de aplicações Web](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/webapplicationfirewall.json) | Define como gerir a política de firewall de aplicações web para a porta de entrada |
