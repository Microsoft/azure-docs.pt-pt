# Get-AzureRmFrontDoorFireWallPolicy

Obter política de WAF

## SINTAXE

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIÇÃO
O **Get-AzureRmFrontDoorFireWallPolicy** cmdletGet obtém a política de WAF num grupo de recursos na subscrição atual

## EXEMPLOS

### Exemplo 1: Obter uma política de WAF chamada $Name no $resourceGroup
```powershell
PS C:\> Get-AzureRmFrontDoorFireWallPolicy -Name $Name -ResourceGroupName $resourceGroup

PolicyEnabledState : Enabled
CustomRules        : {Rule1}

Tags               :

                     osoft.Network/frontdoorwebapplicationfirewallpolicies/{Name}

Type               :
```

Obter uma política de WAF chamada $Name no $resourceGroup
