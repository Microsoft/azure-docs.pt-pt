# Get-AzureRmFrontDoor
Obter configuração de porta de entrada

## SINTAXE

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIÇÃO
O **Get-AzureRmFrontDoor** cmdletGet obtém todas as portas de Front existentes na subscrição atual essas informações de correspondências fornecidas.
