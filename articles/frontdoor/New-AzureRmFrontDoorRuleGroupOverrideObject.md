# Novo AzureRmFrontDoorRuleGroupOverrideObject
Criar o objeto de RuleGroupOverride WAF para criação de políticas

## SINTAXE

```
New-AzureRmFrontDoorRuleGroupOverrideObject -Override <PSRuleGroupOverride> -Action <PSAction>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIÇÃO
Criar o objeto de RuleGroupOverride WAF para criação de políticas

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\>  New-AzureRmFrontDoorRuleGroupOverrideObject -Override SqlInjection -Action Block

Action RuleGroupOverride
------ -----------------
 Block      SqlInjection
```

Criar um objeto de RuleGroupOverride
