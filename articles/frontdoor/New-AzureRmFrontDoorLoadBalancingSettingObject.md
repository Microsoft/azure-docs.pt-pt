# Novo AzureRmFrontDoorLoadBalancingSettingObject
Criar um objeto de PSLoadBalancingSetting para criação de porta de entrada

## SINTAXE

```
New-AzureRmFrontDoorLoadBalancingSettingObject -Name <String> [-SampleSize <Int32>]
 [-SuccessfulSamplesRequired <Int32>] [-AdditionalLatencyInMilliseconds <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIÇÃO
Criar um objeto de PSLoadBalancingSetting para criação de porta de entrada

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzureRmFrontDoorLoadBalancingSettingObject -Name "loadbalancingsetting1"

AdditionalLatencyMilliseconds : 0
SuccessfulSamplesRequired     : 2

Type                          :
```
