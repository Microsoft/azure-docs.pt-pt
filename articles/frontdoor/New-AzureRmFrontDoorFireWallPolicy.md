# Novo AzureRmFrontDoorFireWallPolicy
Criar política de WAF

## SINTAXE

```
New-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> -Name <String> [-EnabledState <PSEnabledState>]
 [-Mode <PSMode>] -Customrule <PSCustomRule[]> [-ManagedRule <PSManagedRule[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIÇÃO
O **New-AzureRmFrontDoorFireWallPolicy** cmdlet cria uma nova política de WAF de Azure no grupo de recursos especificado na subscrição atual

## EXEMPLOS

### Exemplo 1: Criar a política de WAF
