# <a name="create-an-azure-application-offer"></a>Criar uma oferta de aplicações do Azure

Este artigo descreve como criar e publicar uma oferta de candidatura Azure para o Mercado Azure. Cada oferta aparece como sua própria entidade no Azure Marketplace e é composta pelos seguintes agrupamentos de ativos e serviços de apoio: oferecer detalhes, SKUs, artefactos de marketing e informações de apoio.

## <a name="new-offer-form"></a>Formulário New Offer

Inscreva-se no Portal do [Parceiro cloud](https://cloudpartner.azure.com/)e, em seguida, selecione + **Nova oferta** na barra de menu seletiva. No menu New Offer, selecione **Aplicações Azure** para exibir o formulário **New Offer** e iniciar o processo de definição de ativos para uma nova oferta.

![Novo formulário de oferta](./media/azure-new-managedapp-offer.png)
