# <a name="iot-edge-module-offer-publishing-overview"></a>Módulo IoT Edge oferece visão geral da publicação

<table> <tr> <td>Esta secção explica como publicar uma nova oferta de módulos Azure IoT Edge ao Microsoft <a href="https://azuremarketplace.microsoft.com">Azure Marketplace</a>. Um módulo IoT Edge é um recipiente compatível com o Docker que é feito para funcionar num dispositivo IoT Edge. Os módulos Azure IoT Edge são a menor unidade de computação gerida pelo IoT Edge, e podem conter serviços Azure ou código de solução personalizado. </td> <td><img src="./media/iotedge-icon1.png"  alt="Azure IoT Edge module icon" /></td> </tr> </table>

## <a name="publishing-process"></a>Processo de publicação

Os passos de alto nível para a publicação de uma oferta de móduloIT Edge são:

1. Criar a oferta<br> Forneça informações detalhadas sobre a oferta. Estas informações incluem: descrição da oferta, materiais de marketing, informações de suporte e especificações de ativos.

2. Criar o negócio e os ativos técnicos<br> Criar os ativos empresariais (documentos legais e materiais de marketing) e os ativos técnicos para a solução associada (as imagens do módulo IoT Edge alojadas num Registo de Contentores Azure.

3. Criar o SKU<br> Crie os SKU(s) associados à oferta. É necessário um SKU único para cada imagem que planeia publicar.

4. Certificar e publicar a oferta <br>Após a oferta e os ativos técnicos estarem concluídos, pode submeter a oferta. Esta submissão inicia o processo de publicação. Durante este processo a solução é testada, validada, certificada e depois "vai ao vivo" no Azure Marketplace.
