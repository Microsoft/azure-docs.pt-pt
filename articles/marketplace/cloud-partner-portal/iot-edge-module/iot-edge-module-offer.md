# <a name="iot-edge-module-offer"></a>Oferta do módulo do IoT Edge

O [portal Cloud Partner](https://cloudpartner.azure.com/) já não suporta a criação e gestão das ofertas do módulo IoT Edge. Esta funcionalidade foi transferida para o Microsoft [Partner Center.](https://partner.microsoft.com/pcv/) Para mais informações, consulte [Create, configure e publique uma oferta de módulo IoT Edge no Azure Marketplace](../../partner-center-portal/azure-iot-edge-module-creation.md).
