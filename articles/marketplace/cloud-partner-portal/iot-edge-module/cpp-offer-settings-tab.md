# <a name="iot-edge-module-offer-settings-tab"></a>Separador de oferta de módulo SioT Edge

Os **Módulos IoT Edge > nova página de Oferta** abre com o foco no separador Definições de **Oferta.** 

![Nova página de Oferta para módulos IoT Edge](./media/iot-edge-module-offer-settings-tab.png)


## <a name="offer-identity-settings"></a>Oferecer definições de identidade

No âmbito da Identidade da **Oferta,** deve fornecer informações para os campos descritos na tabela seguinte. Um asterisco (*) anexado ao nome de campo indica que é necessário. 

|  **Campo**       |     **Descrição**                                                          |
|  ---------       |     ---------------                                                          |
| **ID da oferta\***       | Um identificador único (dentro de um perfil de editor) para a oferta. Este identificador será visível em URLs de produto e relatórios de insights. Tem um comprimento máximo de 50 caracteres, e pode usar caracteres alfanuméricos minúsculos e traços (-). (O identificador não pode terminar com um traço.) **Nota:** Este campo não pode ser mudado depois de uma oferta ir ao vivo. <br> Por exemplo, se a Contoso publicar uma oferta com um módulo de **iot-edge-iot-edge-** de oferta, é atribuído o URL `https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-iot-edge-module?tab=Overview`do Mercado Azure . |
| **Editora\***     | O identificador único da sua organização no Mercado Azure. Todas as suas ofertas devem estar associadas à sua identificação do editor. Este valor não pode ser alterado depois da oferta ser guardada. |
| **Nome\***          | O nome de exibição para a sua oferta. Este nome é apresentado no Azure Marketplace e no Portal do Parceiro cloud. Pode ter um máximo de 50 carateres. Recomendamos a utilização de uma marca reconhecível para o seu produto. Não inclua o nome da sua organização a menos que seja assim que o seu produto é comercializado. Se estiver a comercializar esta oferta noutros websites e publicações, certifique-se de que o nome é exatamente o mesmo em todas as publicações. |
|  |  |


Selecione **Guardar** para guardar as definições de oferta.
