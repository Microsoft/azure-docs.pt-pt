# <a name="containers"></a>Contentores

O [Cloud Partner Portal](https://cloudpartner.azure.com/) já não suporta a criação e gestão das ofertas do Azure Container. Esta funcionalidade foi transferida para o Microsoft [Partner Center.](https://partner.microsoft.com/pcv/) Para mais informações, consulte [Criar uma oferta de contentores Azure.](../../partner-center-portal/create-azure-container-offer.md)
