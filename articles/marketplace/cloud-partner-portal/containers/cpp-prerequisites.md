# <a name="container-publishing-prerequisites"></a>Pré-requisitos de publicação de contentores

Este artigo descreve os pré-requisitos para a publicação de uma oferta de contentores no Mercado Azure.  Se ainda não o fez, deve rever o Guia de Publicação da Oferta de [Contentores.](../../marketplace-containers.md)

## <a name="publishing-prerequisites"></a>Pré-requisitos de publicação

Para publicar uma nova imagem de contentor, tem de cumprir os seguintes pré-requisitos:

- Acesso ao Portal do Parceiro cloud. Para mais informações, consulte o guia de publicação Azure Marketplace e AppSource.
- Acordo com os Termos do Mercado Azure
- Acolde o seu ativo técnico de contentores num Registo de Contentores Azure.
- Tenha os metadados do seu recipiente prontos a utilizar. Por exemplo, a seguinte lista não exaustiva:
  - Um título
  - Uma descrição (em formato HTML)
  - Uma imagem de logotipo (em formato PNG) e nestes tamanhos de imagem fixos: 40x40 px, 90x90 px, 115x 115 px e 255x115 px.
- Um *Termos de Uso* e uma Declaração de Política de *Privacidade*
- Documentação da solução de contentores
- Contactos de suporte

## <a name="next-steps"></a>Passos seguintes

Se ainda não o fez, tem de [preparar os seus ativos técnicos](./cpp-create-technical-assets.md) de contentores antes de criar a sua oferta de [contentores.](./cpp-create-offer.md)
