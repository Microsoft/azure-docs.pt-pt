# <a name="consulting-services-offer"></a>Oferta de serviços de consultoria

O [portal Cloud Partner](https://cloudpartner.azure.com/) já não apoia a criação e gestão de ofertas de serviços de consultoria. Esta funcionalidade foi transferida para o Microsoft [Partner Center.](https://partner.microsoft.com/pcv/) Para mais informações, consulte os [pré-requisitos](../../partner-center-portal/consulting-service-prerequisites.md)do serviço de consultoria.
