# <a name="consulting-service-prerequisites"></a>Pré-requisitos do serviço de consultoria

Para submeter ofertas de serviço de consultoria para qualquer um dos produtos abaixo, tem de satisfazer primeiro uma ou mais dos requisitos correspondentes.

* **Dynamics 365 para envolvimento do cliente**: Parceiros têm de ter uma certificação Silver ou Gold para o [competência Cloud Customer Relationship Management](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency). 
* **Dynamics 365 para finanças e operações**: Parceiros têm de ter uma certificação Silver ou Gold para o [Enterprise Resource Planning, competência](https://partner.microsoft.com/membership/enterprise-resource-planning-competency)e uma mínimo de US $25 mil em operações de Cloud dentro de 12 meses que se à direita de receita.
* **Dynamics 365 business central**: Parceiros tem de ter agia como fornecedor de soluções Cloud (CSP) ou o parceiro de registo Digital (DPOR) para, pelo menos, um cliente.
    * **CSP**: Ver [introdução ao CSP](https://partner.microsoft.com/cloud-solution-provider/get-started) para como se tornar um CSP.
    * **DPOR**: Ver [descrição geral de DPOR](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1) para requisitos de DPOR.
* **Power BI**: Esteja listado no showcase de parceiros do Power BI. Ver [critérios de parceiro de solução](https://powerbi.microsoft.com/become-a-partner) para obter mais informações.
* **PowerApps**: Parceiros têm de ter uma solução publicada no [Showcase de parceiros](https://powerapps.microsoft.com/partner-showcase).

**Passos seguintes?**

Se forem cumpridos os requisitos acima, está pronto para [criar uma oferta de serviços de consultoria](./cpp-consulting-service-create-offer.md).
