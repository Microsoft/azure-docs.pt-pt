# <a name="consulting-service-prerequisites"></a>Pré-requisitos do serviço de consultoria

Reveja a descrição do e requisitos para esta oferta digitar [dos serviços de consultoria para o Azure Marketplace e AppSource](../../consulting-services.md).  Os seguintes tipos de serviço consultoria específicos têm requisitos adicionais:

|     Tipo de oferta   |    Requisito   |
|     ----------   |    -----------   |
| Dynamics 365 for Customer Engagement | Parceiros têm de ter uma certificação Silver ou Gold para o [competência Cloud Customer Relationship Management](https://partner.microsoft.com/membership/cloud-customer-relationship-management-competency). |
| Dynamics 365 para finanças e operações | Parceiros têm de ter uma certificação Silver ou Gold para o [Enterprise Resource Planning, competência](https://partner.microsoft.com/membership/enterprise-resource-planning-competency). Parceiros também têm de ter uma mínimo receita de US $25.000 em operações de Cloud dentro de 12 meses que se anterior. |
| Dynamics 365 Business Central | Parceiros têm de ter agia como um fornecedor de soluções Cloud (CSP) ou um parceiro de registo Digital (DPOR) para, pelo menos, um cliente. <br/> Para obter informações sobre como se tornar um CSP, consulte [começar a utilizar como um CSP](https://partner.microsoft.com/cloud-solution-provider/get-started). <br/> Para DPOR requisitos, consulte a [descrição geral DPOR](https://partner.microsoft.com/membership/digital-partner-of-record#Membership_Digital_Partner_of_Record_Navigated_Rich_Text_Node1). |
|  Power BI  | Esteja listado no showcase de parceiros do Power BI. Para obter mais informações, consulte [critérios de parceiro de solução](https://powerbi.microsoft.com/become-a-partner). |
|  PowerApps       | Parceiros têm de ter uma solução publicada no [Showcase de soluções de parceiros](https://powerapps.microsoft.com/partner-showcase). |
|  |  |
