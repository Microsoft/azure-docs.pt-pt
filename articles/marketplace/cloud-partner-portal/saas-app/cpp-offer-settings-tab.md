# <a name="saas-application-offer-settings-tab"></a>Separador de definições da oferta de aplicação de SaaS

Este artigo descreve como configurar as definições de oferta para a oferta.

O **aplicação SaaS > nova oferta** página abre-se com o foco no **oferecer definições** separador. 

Utilize o separador de definições da oferta para configurar o **oferecer identidade**, mostrado na captura de ecrã seguinte. Um asterisco (*) acrescentado ao nome do campo indica que é obrigatório.

![Separador Definições de Oferta](./media/saas-new-offer.png)

## <a name="offer-identity-settings"></a>Definições de identidade da oferta

Sob a identidade de oferta, tem de fornecer informações para os campos descrito na tabela seguinte. 


|  **Nome do campo**   |  **Descrição**  |
|  ---------------   |  ---------------  |
|    ID de Oferta  |  Um identificador exclusivo para a oferta de um perfil do publicador. Este ID será visível em URLs de produto e os relatórios de faturas. Apenas pode ser constituído por carateres alfanuméricos em minúsculas ou hífenes (-). O ID não pode terminar com um travessão e está limitado a um máximo de 50 carateres. Tenha em atenção que este campo está bloqueado assim que uma oferta entra no ar. Por exemplo, se um publicador, a Contoso, publica uma oferta com o ID de oferta exemplo-vm, ela aparecerá no Azure marketplace, como: https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-vm?tab=Overview.                 |
|  ID de publicador    |  O ID de publicador é o identificador exclusivo no Marketplace. Todas as suas ofertas devem ser ligado seu ID de publicador. Não é possível modificar o ID de publicador depois da oferta guardada....                |
|  Name      |   Este é o nome a apresentar para a sua oferta. Este é o nome que irá aparecer no Azure Marketplace e no Portal do Azure. Pode ter um máximo de 50 carateres. Inclua um nome de marca reconhecível para o produto. Não inclua o nome da sua empresa aqui, a menos que o que é a forma como é comercializada. Se estiver marketing esta oferta em seu próprio Web site, certifique-se de que o nome é exatamente como ele aparece no seu Web site.               |

Selecione **guardar** para guardar o seu progresso.
