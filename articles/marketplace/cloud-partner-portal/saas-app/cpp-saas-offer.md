# <a name="azure-saas-application-offer"></a>Oferta de candidatura azure SaaS

O [Portal do Parceiro Cloud](https://cloudpartner.azure.com/) já não suporta a criação e gestão das ofertas de aplicações SaaS.  Em vez disso, esta funcionalidade foi transferida para o portal Microsoft [Partner Center](https://partner.microsoft.com/pcv/).  Para mais informações, consulte [Criar uma nova oferta SaaS.](../../partner-center-portal/create-new-saas-offer.md) 


## <a name="next-steps"></a>Passos seguintes

Antes de criar uma nova oferta de aplicações SaaS, deve ler o Guia de Publicação de [Oferta de Aplicações SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) e rever os passos do portal Partner Center listados na lista de verificação de criação de [oferta.](../../partner-center-portal/offer-creation-checklist.md)
