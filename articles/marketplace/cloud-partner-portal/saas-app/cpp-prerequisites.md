# <a name="saas-application-publishing-prerequisites"></a>Pré-requisitos de publicação da aplicação SaaS

Este artigo descreve os pré-requisitos para a publicação de uma oferta de aplicação SaaS (aplicação) no Azure Marketplace. O Portal de parceiros da Cloud fornece acesso baseado em funções para o portal, permitindo que várias pessoas colaborar em direção à publicação de uma oferta. Para obter mais informações, consulte [gerir utilizadores](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-manage-users).

## <a name="publishing-prerequisites"></a>Publicação de pré-requisitos

Para publicar uma nova oferta de aplicação SaaS, tem de cumprir os seguintes pré-requisitos:
- Acesso para o [Portal de parceiros da Cloud](https://cloudpartner.azure.com/). Para obter mais informações, consulte os guias de publicação do Azure Marketplace e AppSource.
- Antes de uma oferta pode ser publicada em nome de uma conta de editor, um dos indivíduos com função de proprietário devem concordar em conformidade com o [termos de utilização](https://azure.microsoft.com/support/legal/website-terms-of-use/), [declaração de privacidade da Microsoft](https://www.microsoft.com/privacystatement/default.aspx), e [ Contrato do programa do Microsoft Azure Certified](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/).
- Todo o conteúdo que não o inglês precisa ser acompanhada com uma versão de idioma inglês. Este conteúdo inclui o texto de loja, documentos, capturas de ecrã, termos de utilização e a política de privacidade. É aceitável para fornecer um URL de ligação útil para um aplicativo que não o inglês. 
- Ter seus metadados pronto a utilizar. A lista seguinte (parcial) mostra um exemplo dos metadados:
  - Um título
  - Uma descrição (no formato HTML)
  - Uma imagem de logótipo (no formato PNG) e nestes corrigido tamanhos de imagem: 40 x 40 pixels, 90 x 90 pixels, 115 x 115 pixels e 255 x 115 pixels.
  - Termos de utilização e uma política de privacidade
  - Documentação
  - Contactos de suporte
