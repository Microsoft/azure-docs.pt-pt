# <a name="saas-application-channel-info-tab"></a>Separador de informações de canal de aplicações de SaaS

Utilize o separador informações de canal para fornecer informações de canal para a sua aplicação, incluindo materiais do go-to-market (GTM) e informações de contacto.

![Formulário de informações de canal](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Para configurar informações de canal

1. Sob **informações de canal > materiais GTM**, introduza o URL para o site que aloja os seus materiais GTM.
2. Sob **Manager contactar o canal**, indicar um contacto de manager Microsoft pode entrar em contacto com para problemas de suporte e de negócios. Introduza informações para estes campos: Nome, endereço de E-Mail e número de telefone.
3. Selecione **guardar** para concluir a configuração da segurança das informações de canal.
