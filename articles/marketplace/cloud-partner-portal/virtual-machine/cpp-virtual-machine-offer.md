# <a name="azure-virtual-machine-offer"></a>Oferta de Máquina Virtual Azure

O [Cloud Partner Portal](https://cloudpartner.azure.com/) já não suporta a criação e gestão das ofertas da Máquina Virtual Azure. Esta funcionalidade foi transferida para o Microsoft [Partner Center.](https://partner.microsoft.com/pcv/) Para mais informações, consulte Criar uma oferta de [Máquina Virtual Azure.](../../partner-center-portal/azure-vm-create-offer.md)
