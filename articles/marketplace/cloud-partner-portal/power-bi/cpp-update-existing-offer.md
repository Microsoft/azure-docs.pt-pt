# <a name="update-a-power-bi-app-offer"></a>Atualizar uma oferta de Aplicação Power BI

>[!Important]
>A partir de 13 de abril de 2020, começaremos a movimentar a gestão das suas ofertas de aplicações Power BI para partner Center. Após a migração, você vai criar e gerir suas ofertas no Partner Center. Siga as instruções na criação de [aplicações Power BI](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer) para gerir as suas ofertas migratórias.

Este artigo mostra-lhe como atualizar a sua oferta da Microsoft Power BI App no [Portal do Parceiro cloud](https://cloudpartner.azure.com/) e, em seguida, reeditar a oferta. Aqui estão algumas razões comuns para atualizar uma oferta:

- Para atualizar o conteúdo da aplicação no Power BI e obter um URL de instalação da aplicação recém-embalada
- Para atualizar os metadados do Azure Marketplace da oferta (informações de vendas, marketing ou suporte e ativos)
 
Utilize as funcionalidades **Compare** e **History** do portal para acompanhar alterações como estas.

## <a name="common-update-operations"></a>Operações comuns de atualização

Alguns atributos de uma oferta de Aplicação Power BI não podem ser alterados após a oferta ser publicada no AppSource. Alterações não permitidas incluem **ID de oferta** e ID da **editora**. Mas ainda pode mudar uma grande variedade de características. Aqui estão algumas mudanças comuns.

### <a name="update-app-content-in-power-bi"></a>Atualizar conteúdo de aplicativo sintetizá-lo em Power BI

As aplicações power BI são muitas vezes atualizadas periodicamente com novos conteúdos, patches de segurança, funcionalidades, e assim por diante. Quando uma aplicação for alterada, atualize o URL de instalação seguindo estes passos:

1.  Inscreva-se no Portal do [Parceiro cloud.](https://cloudpartner.azure.com/)
2.  Em **todas as ofertas,** encontre a oferta que pretende atualizar.
3.  No separador **Informação Técnica,** introduza um novo URL de instalação.
4.  Selecione **Publicar** para iniciar o fluxo de trabalho para publicar a nova versão da aplicação no AppSource.


### <a name="update-the-offers-marketplace-metadata"></a>Atualizar os metadados do Marketplace da oferta

Os metadados marketplace da sua oferta incluem o nome da sua empresa, logotipos e outras informações. Para atualizar os metadados, siga estes passos:

1.  Inscreva-se no Portal do [Parceiro cloud.](https://cloudpartner.azure.com/)
2.  Em **todas as ofertas,** encontre a oferta que pretende atualizar.
3.  No separador **Storefront Details,** altere os metadados seguindo as instruções no [separador Storefront Details](./cpp-storefront-details-tab.md)da aplicação Power BI .
4.  Selecione **Publicar** para iniciar o fluxo de trabalho para publicar as suas alterações.


## <a name="the-compare-feature"></a>A funcionalidade Compare

Ao atualizar uma oferta publicada, pode verificar as alterações que fez. Para utilizar a função **Compare:**

1.  Em qualquer ponto do processo de edição, selecione o botão **Compare** da oferta.

    ![Comparar botão](./media/compare-feature-button.png)

2.  Ver versões lado a lado de ativos de marketing e metadados.


## <a name="history-of-publishing-actions"></a>História das ações editoriais

Para ver o histórico de publicação da sua oferta, no lado esquerdo do Portal do Parceiro cloud, abra o separador **História.** Aqui você verá um histórico de ações carimbadas no tempo nas suas ofertas AppSource.
