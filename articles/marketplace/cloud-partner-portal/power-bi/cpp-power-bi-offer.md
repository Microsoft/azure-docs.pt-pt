Esta funcionalidade foi transferida para o Microsoft [Partner Center.](https://partner.microsoft.com/pcv/) Para mais informações, consulte [Create a Power BI para microsoft AppSource](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer).
