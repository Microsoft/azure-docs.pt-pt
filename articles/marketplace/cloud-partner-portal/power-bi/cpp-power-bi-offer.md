# <a name="power-bi-app-offer"></a>Oferta do Power BI aplicação

|              |                                |
|-|--------------------------------|
| Este artigo explica como publicar uma aplicação do Power BI da Microsoft [marketplace do AppSource](https://appsource.microsoft.com/). Uma aplicação do Power BI pacotes de conteúdos do Power BI e utiliza fluxos de dados para ligar relatórios e dashboards a dados no armazenamento de dados comuns. Em seguida, pode implementar a aplicação para outros inquilinos através do AppSource. | ![Ícone do Power BI](./media/powerbi-icon.png) |


Este artigo está dividido em três partes principais:

-   [Pré-requisitos](./cpp-prerequisites.md). Os requisitos técnicos e empresariais para criar e publicar uma oferta de aplicação do Power BI.
-   [Criar uma oferta de aplicação do Power BI](./cpp-create-offer.md). Como criar uma entrada de oferta de aplicação do Power BI com o [Cloud Partner Portal](https://cloudpartner.azure.com).
-   [Publicar uma oferta de aplicação do Power BI](./cpp-publish-offer.md). Como submeter no AppSource, uma nova oferta para publicação e como atualizar uma oferta existente.


## <a name="publishing-steps"></a>Passos de publicação

Aqui estão as etapas de alto nível para publicar uma oferta de aplicação do Power BI:

![Passos de publicação da oferta de aplicação Power BI](media/publishing-steps.png)

Esta é a oferta de aplicação do Power BI, processo de publicação:

1. Crie uma aplicação de serviço no Power BI. Esta ação gera um URL de instalação do pacote, que representa os recursos técnicos para a oferta. Para obter mais informações, consulte [o que é o Power BI?](https://go.microsoft.com/fwlink/?linkid=2028636).
2. Recolher ou criar os ativos empresariais, incluindo:
    - Materiais de marketing.
    - Legal e documentos de suporte.
3.  Crie a oferta. Utilize o Portal de parceiros da Cloud para configurar os detalhes da oferta, incluindo a descrição da oferta, materiais, informações legais, informações de suporte e as especificações dos ativos de marketing.
4.  Certificar e publicar a oferta. Esta ação inicia o processo de publicação.
    - A equipa de integração do AppSource testa, valida e certifica que seu aplicativo. 
    - Depois da oferta for certificada, examine-o em seu ambiente de teste e liberá-lo.

Depois de concluir estes passos, a sua oferta de aplicação do Power BI for executada no AppSource.

## <a name="next-steps"></a>Passos Seguintes

Antes de criar a sua oferta de aplicação do Power BI e publicá-la no AppSource, tem de cumprir os [requisitos](./cpp-prerequisites.md) para publicar uma aplicação do Power BI no AppSource.
