# <a name="create-a-power-bi-application-offer"></a>Criar uma oferta de aplicação do Power BI

Esta secção lista os passos necessários para criar uma nova oferta de aplicação do Power BI para [AppSource](https://appsource.microsoft.com). Cada oferta é apresentada como uma entidade própria no AppSource.  Quando cria uma nova oferta na [Cloud Partner Portal](https://cloudpartner.azure.com/), tem de fornecer quatro grupos de recursos para a sua oferta.

|   Grupo de recursos      | Descrição                                                                         |
| ----------------   | ----------------                                                                    |
| Definições da oferta     | Identificações primárias e o nome para a oferta                                      |
| Informações técnicas     | URL de instalador utilizado para instalar a aplicação na área de trabalho do cliente Power BI. Para obter mais informações sobre como gerar este URL, consulte [documentação de aplicação do Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).   |
| Vitrine detalhes | Contém recursos de gerenciamento de marketing, legais e provocar. Recursos de marketing incluem descrição da oferta e logotipos.  Ativos legais incluem uma política de privacidade e termos de utilização e outra documentação legal.  Política de gestão levar permite que especifique como lidar com leva a partir do portal de utilizador final do AppSource. |
| Contactos           | Contém informações de contacto e política de suporte                                     |
|    |     |


## <a name="new-offer-form"></a>Novo formulário de oferta

Assim que o seu início de sessão no Portal de parceiro de Cloud, clique nas **+ nova oferta** item na barra de menu à esquerda.  No menu de resultante, clique em **aplicações do Power BI** para apresentar o **nova oferta** formam e iniciar o processo de definição de ativos para uma nova oferta de aplicação.

![Item de menu de oferta do Power BI](./media/new-offer-menu.png)

> [!WARNING] 
> Se o **aplicações do Power BI** opção não está apresentada ou não está ativada, então a sua conta não tem permissão para criar este tipo de oferta. Certifique-se de que cumpriu todos os [pré-requisitos](./cpp-prerequisites.md) para este tipo de oferta, incluindo se registrar para uma conta de programador.
