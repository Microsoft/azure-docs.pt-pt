# <a name="create-a-power-bi-app-offer"></a>Criar uma oferta de Aplicação Power BI

>[!Important]
>A partir de 13 de abril de 2020, começaremos a movimentar a gestão das suas ofertas de aplicações Power BI para partner Center. Após a migração, você vai criar e gerir suas ofertas no Partner Center. Siga as instruções na criação de [aplicações Power BI](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer) para gerir as suas ofertas migratórias.

Este artigo lista os passos a seguir para criar uma nova oferta de Aplicação Power BI para [AppSource](https://appsource.microsoft.com). Cada oferta aparece como sua própria entidade no AppSource. Quando criar uma nova oferta no Portal do [Parceiro cloud,](https://cloudpartner.azure.com/)é obrigado a fornecer quatro grupos de ativos para a sua oferta.

Os grupos de ativos são descritos na tabela seguinte:

|   Grupo de ativos      | Descrição                                                                         |
| ----------------   | ----------------                                                                    |
| Oferta De Definições     | As principais identificações e o nome da oferta.                                      |
| Informação Técnica     | O URL instalador que utiliza para instalar a aplicação no espaço de trabalho power BI do cliente. Para obter mais informações sobre como gerar este URL, consulte a [documentação da Aplicação Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). |
| Detalhes da montra | Contém ativos de marketing, legais e de gestão de chumbo. Os ativos de marketing incluem uma descrição da oferta e logotipos. Os bens legais incluem uma política de privacidade, termos de uso e outra documentação legal. A política de gestão de chumbo permite especificar como lidar com os cabos do portal de utilizador AppSource. |
| Contactos           | Contém informações de contacto de suporte e políticas.                                     |

## <a name="new-offer-form"></a>Formulário New Offer

Depois de iniciar sessão no Portal do Parceiro cloud, selecione **New Offer** no painel esquerdo. Em seguida, para exibir o formulário New Offer e iniciar o processo de definição de ativos para uma nova oferta de aplicações, selecione **Power BI Apps**.

![Power BI oferecer item de menu](./media/new-offer-menu.png)

> [!NOTE] 
> Se a opção **Power BI Apps** não for mostrada ou não estiver ativada, a sua conta não tem permissão para criar este tipo de oferta. Verifique se cumpriu todos os [pré-requisitos](./cpp-prerequisites.md) para este tipo de oferta, incluindo o registo de uma conta de desenvolvedor.
