# <a name="create-power-bi-app-offer-technical-assets"></a>Criar Power BI aplicativo oferecer ativos técnicos

O principal ativo técnico que você precisa para esse tipo de oferta é um aplicativo Power BI. Um aplicativo Power BI é uma coleção de conjuntos de gráficos, relatórios ou dashboards primários e serviços conectados opcionais e conjuntos de aplicativos inseridos. Ele evoluiu de um tipo de ativo anteriormente conhecido como [pacote de conteúdo](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction). Para obter mais informações sobre como desenvolver esse tipo de aplicativo, consulte [o que são Power bi aplicativos de modelo?](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtain-an-installation-url"></a>Obter uma URL de instalação

Você pode criar um aplicativo Power BI somente dentro do [serviço do Power bi](https://powerbi.microsoft.com/). Para começar, você entra com uma [licença Power bi pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Depois de criar e testar seu aplicativo de serviço no Power BI, salve a URL de instalação do aplicativo que Power BI gera. Por fim, forneça a URL na guia [informações técnicas](./cpp-technical-info-tab.md) do formulário de envio de portal do Cloud Partner.
 

## <a name="next-steps"></a>Passos seguintes

Depois de criar e testar seu aplicativo Power BI e criar a oferta associada, você poderá [publicar a oferta de aplicativo Power bi](./cpp-publish-offer.md).
