# <a name="power-bi-apps-prerequisites"></a>Pré-requisitos do Power BI aplicações

Este artigo lista as técnicas e requisitos empresariais que tem de cumprir antes de podem publicar uma oferta de aplicação do Power BI Microsoft AppSource Marketplace.

## <a name="technical-requirements"></a>Requisitos técnicos

O recurso técnico principal, terá de para este tipo de oferta é uma [aplicação do Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Para obter mais informações, consulte [ativos técnicos do Power BI criar](./cpp-create-technical-assets.md).

Assim que a aplicação do serviço é criada e testada no Power BI, deve guardar o URL de instalação de aplicações que o Power BI gera. Irá precisar dele para [crie uma nova oferta de aplicação do Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Requisitos comerciais

Os requisitos comerciais incluem obrigações de procedimentos, contratuais e legais:

-   Tem de ser um publicador de Marketplace na nuvem registados. Se não estiver registrado ainda, siga os passos no artigo [se tornar um publicador de Marketplace da Cloud](../../become-publisher.md).

    > [!NOTE] 
    > Deve utilizar a mesma conta de registo do Microsoft Developer Center para iniciar sessão para o [Cloud Partner Portal](https://cloudpartner.azure.com). Deve ter apenas uma conta Microsoft para as ofertas de AppSource. Não deve ser específica de ofertas ou serviços individuais.
 
-   É responsável por disponibilizar suporte técnico aos clientes de forma que sejam comercialmente razoável.
-   Tem de fornecer conteúdo que cumpre os critérios para a sua oferta seja incluída no AppSource. Para obter mais informações, consulte [tem uma aplicação à lista no AppSource? Eis como](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how).
-   Deve estar em conformidade com o [declaração de privacidade do Microsoft](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Passos Seguintes

Depois de cumprir todos os requisitos, está pronto para [criar uma oferta do Power BI](./cpp-create-offer.md) no Portal de parceiros de nuvem.
