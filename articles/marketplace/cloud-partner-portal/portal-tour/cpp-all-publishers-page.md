# <a name="all-publishers-page"></a>Página com todos os editores

A página [**todos os Publicadores**](https://cloudpartner.azure.com/#publishers) exibe uma lista de todos os Publicadores de nuvem da Microsoft registrados, incluindo o nome exclusivo do editor e o nome de exibição. Você pode ordenar os resultados por coluna ou especificar uma cadeia de caracteres para filtrar os resultados.  Essa página também exibe um botão **Adicionar Publicador** , mas esse botão só é habilitado para administradores do Portal.  A seleção de um Publicador nesta lista exibirá o perfil para esse Publicador.  Para obter mais informações, consulte [Gerenciando o perfil do Publicador](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).
