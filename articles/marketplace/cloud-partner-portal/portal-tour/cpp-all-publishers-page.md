# <a name="all-publishers-page"></a>Página com todos os editores

O [ **todos os editores** ](https://cloudpartner.azure.com/#publishers) página apresenta uma lista de todos os registados Microsoft cloud editoras, inclusive seu nome exclusivo do publicador e o nome a apresentar. Pode ordenar os resultados por coluna ou especificar uma cadeia de caracteres para filtrar os resultados.  Esta página também apresenta uma **publicador adicionar** botão, mas este botão só está ativada para administradores do portal.  A seleção de um fabricante de nesta lista irá apresentar o perfil para esse editor.  Para obter mais informações, consulte [gerenciamento de perfil do publicador](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).
