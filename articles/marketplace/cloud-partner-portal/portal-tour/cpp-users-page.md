# <a name="users-page"></a>Página de utilizadores

O [ **usuários** ](https://cloudpartner.azure.com/#users) página apresenta uma lista de todos os utilizadores registados do Portal de parceiro de Cloud.  As colunas incluem o endereço de e-mail de cada utilizador, função de portal, o tipo de oferta e organização de publicação. Os resultados podem ser ordenados pela coluna ou filtrado com base numa cadeia especificada.  

Esta página também apresenta uma **adicionar utilizador** botão para adicionar utilizadores à sua organização de publicação.  Para obter mais informações, consulte [gestão de utilizadores no Portal de parceiros de nuvem](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
