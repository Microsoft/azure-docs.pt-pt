# <a name="new-offer-menu"></a>Novo menu de oferta

O menu **New offer** permite aos parceiros criar instâncias de tipos de oferta suportados pelo Portal do [Parceiro cloud.](https://cloudpartner.azure.com)  A seleção do item do menu **New offer,** na barra de menus de navegação do lado esquerdo, apresenta a seguinte lista de tipos de oferta.  A seleção de um destes tipos inicia o processo de criação e publicação de ofertas.

|    **Novo item de menu de oferta**     |     **Secção do doc correspondente**                       |
|    -----------------------     |     -----------------------------                       |
| ![Artigo de menu de aplicações Azure](./media/new-offer-menu1.png) |  [Oferta de candidatura azure](../azure-applications/cpp-azure-app-offer.md) |
| ![Artigo de menu de serviços de consultoria](./media/new-offer-menu2.png) | [Oferta de serviços de consultoria](./../../cloud-partner-portal-orig/cloud-partner-portal-consulting-services-publishing-offer.md) |
| ![Itens de menu de recipientes](./media/new-offer-menu3.png) | [Oferta de contentores](./../containers/cpp-containers-offer.md) |
| ![Item do menu Dynamics 365 Business Central](./media/new-offer-menu4.png) | [Oferta do Dynamics 365 Business Central](./../../cloud-partner-portal-orig/cpp-business-central-offer.md) |
| ![Dinâmica 365 para menu de envolvimento com o cliente](./media/new-offer-menu5.png) | [Oferta do Dynamics 365 for Customer Engagement](./../../cloud-partner-portal-orig/cpp-customer-engagement-offer.md) |
| ![Dinâmica 365 para menu de operações](./media/new-offer-menu6.png) | [Oferta do Dynamics 365 for Operations](./../../cloud-partner-portal-orig/cpp-dynamics-365-operations-offer.md) |
| ![Item do menu de módulos IoT Edge](./media/new-offer-menu7.png) | [Oferta de módulo IoT Edge](./../iot-edge-module/cpp-offer-process-parts.md) |
| ![Artigo de menu power BI App](./media/new-offer-pbi.png)   |  [Oferta da aplicação do Power BI](../power-bi/cpp-power-bi-offer.md)  |
| ![ Item do menu de aplicações SasS](./media/new-offer-menu8.png) | [Oferta de candidatura SasS](../saas-app/cpp-saas-offer.md) |
| ![ Item do menu de máquinas virtuais](./media/new-offer-menu9.png) | [Oferta de máquina virtual](./../virtual-machine/cpp-virtual-machine-offer.md) |
|  |  |

> [!NOTE]
> Como este menu é periodicamente atualizado para refletir novos tipos de oferta e oferecer alterações de nome, os itens que você vê podem não corresponder exatamente aos da tabela acima.


### <a name="next-steps"></a>Passos seguintes

Se estiver a criar uma nova oferta, pode usar a tabela anterior para navegar na secção de oferta para orientação na criação e publicação da sua oferta.  Caso contrário, o passeio do portal continua com um exame da [página All Offers.](./cpp-all-offers-page.md)
