# <a name="azure-marketplace-and-appsource-roadmap"></a>Mapa do Azure Marketplace e AppSource

Este documento fornece uma visão e o que se segue para o Azure Marketplace, AppSource e programas de parceria relacionados. Ele captura, alguns dos recursos significativos, que podemos ter-se comprometido e um intervalo de tempo aproximado para quando pode esperar para vê-los. Não é uma lista abrangente de todos os novos recursos, mas foi concebida para proporcionar visibilidade das nossas principais investimentos. Estes conjuntos de recursos e períodos de tempo de entrega estão atuais e sujeitas a alterações. Esta página é atualizada mensalmente.

Suas sugestões são bem-vindos! Participe da conversa [Comunidade de parceiros da Microsoft](https://www.microsoftpartnercommunity.com/) para obter as últimas atualizações sobre novos recursos, programas e eventos.
