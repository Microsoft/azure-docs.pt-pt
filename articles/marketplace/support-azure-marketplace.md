# <a name="support-for-microsoft-commercial-marketplace"></a>Suporte ao mercado comercial da Microsoft

Tem alguma dúvida sobre o início como editor de marketplace comercial da Microsoft? Aqui está uma lista de opções de apoio para o mercado comercial. Além dos seguintes recursos, também pode obter muitas das suas perguntas respondidas no [canal Marketplace do Fórum Comunitário C+AI.](https://www.microsoftpartnercommunity.com/t5/Marketplace/bd-p/2222)  

## <a name="onboarding"></a>Inclusão

Abra um bilhete com [o suporte da editora microsoft marketplace](https://aka.ms/marketplacepublishersupport) para problemas com o embarque e o início.

## <a name="partner-center"></a>Centro de Parceiros

| Canal de suporte | Descrição | Disponibilidade |  
|:--- |:--- |:--- |  
| Para assistência, visite a página criar um incidente localizado no [Marketplace Support](https://aka.ms/marketplacepublishersupport)</li> </ul> | Apoio ao Centro de Parceiros. | O suporte é fornecido 24x5. |
