# <a name="co-sell-partner-engagement"></a>Compromisso entre parceiros para venda conjunta

Se fores um parceiro, podes usar o programa de co-venda da Microsoft para trabalhar diretamente com a Microsoft em oportunidades de venda conjunta. O programa ajuda os nossos parceiros a alcançar uma vasta comunidade de clientes e a criar novas oportunidades de vendas que podem ajudar a acelerar o crescimento do negócio.

## <a name="partner-selling-motions"></a>Movimentos de venda de parceiros

A Microsoft oferece três movimentos de venda de parceiros:

* *Venda através de parceiro* – Pode revender a tecnologia microsoft através dos seus próprios vendedores.
* *Serviços co-vendem* - Os vendedores e parceiros colaboram para vender o seu projeto ou serviços geridos com a tecnologia Microsoft incorporada.
* *Co-venda IP* – Os vendedores e parceiros colaboram para vender a sua solução IP repetível e embalada.*

\*Algumas destas soluções de co-venda IP podem receber incentivos adicionais quando são vendidas e implementadas com sucesso com equipas de vendas de campo da Microsoft.
