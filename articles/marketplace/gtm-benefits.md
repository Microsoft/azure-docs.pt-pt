# <a name="gtm-benefits-on-the-marketplace"></a>Vantagens de GTM no marketplace   
Novo listagens no storefronts no mercado são elegíveis para receber os benefícios de Marketplace GTM livres. Depois de listagem, especialistas de marketing na Microsoft contactar o utilizador ou o contacto marketing à dê início às atividades. Não há nada para si, excepto para o envolvimento com a Microsoft quando são contactados.  

As atividades que a Microsoft disponibiliza diferem consoante o estado da sua solução no marketplace. Vantagens aumentam significativamente para listagens.  

| Storefront | Vantagens adicionais disponíveis  |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Parceiros do Microsoft Gold</td></tr> <tr><td>Aplicações de avaliação</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Transact ofertas</td></tr> <tr><td>Ofertas de avaliação</td></tr> </table> |  

Para maximizar o impacto destas atividades, Microsoft encoraja que tenha o seu plano de iniciação pronto para implementar. Pode querer utilizar a página de destino para muitos táticas de TI as iniciação.  

>[!NOTE]
>Um catálogo de parceiros comerciais (catálogo OCP) é uma vantagem da adesão ao Microsoft Partner Network.  

![Vantagens de GTM no marketplace](./media/marketplace-publishers-guide/gtmmay2018.png)  

O suporte de mercado aceda inclui modelos, conteúdo web, formação e ferramentas para promover a sua empresa.  
*   Forfor obter mais informações sobre o suporte de ir para o mercado, visite os serviços de ir para o mercado da Microsoft página localizada em [partner.microsoft.com/reach-customers/gtm](https://partner.microsoft.com/reach-customers/gtm).
