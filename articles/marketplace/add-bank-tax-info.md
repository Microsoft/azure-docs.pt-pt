# <a name="add-bank-and-tax-information"></a>Adicionar informações bank e impostos  
As informações de bank e impostos são necessárias para todas as ofertas comerciais utilizando transact listagem tipo.  
*   Se estiverem a publicar comerciais ofertas de compra, em seguida, tem de adicionar informações de dividendos e impostos e validação de pedido no Dev Center.  
    >[!IMPORTANT]
    >Para ofertas de comerciais (transact), tem de concluir as informações de bank e impostos antes de conseguir push as ofertas para produção.  

*   Se estiver apenas livre de publicação ou BYOL oferece, em seguida, não é necessário para adicionar as informações. Pode adicionar as informações mais tarde, mas a validação das informações dedução dos impostos demora algum tempo. Se pretender oferecer comerciais ofertas de compra, deve adicionar as informações logo que possível.  

## <a name="add-bank-information"></a>Adicionar informações de bank  
1.  Inicie sessão no Dev Center com o Microsoft ID.  
    *   Para iniciar sessão no Dev Center, aceda ao Windows Dev Center página localizada em [dev.windows.com](https://dev.windows.com).  
2.  Selecione a conta de dividendos no menu à esquerda.
3.  No `Choose payment method` página, selecione `Bank account` ou `PayPal`.  
    >[!IMPORTANT]
    >Se tiver comerciais ofertas aos clientes comprar no Marketplace, em seguida, dividendos a sua conta é onde recebe dividendos para as compras.  

4.  Introduza as informações de pagamento. Depois de ter verificado que as informações estão corretas, clique no botão guardar.  
    >[!IMPORTANT]
    >Se atualizar ou alterar a sua conta de dividendos, tem de seguir os mesmos passos substituir as informações atuais com as novas informações. Alterações à sua conta de dividendos podem atrasar pagamentos até o ciclo de um pagamento. Este atraso ocorre porque o Microsoft tem de verificar a alteração da conta, este é o mesmo quando configurar primeiro a conta de dividendos. Obter paga a quantidade total após a sua conta foi verificada. Qualquer pagamentos devido a partir de pagamento atual ciclo são adicionadas para o próximo ciclo de.  

4.  Clique no botão seguinte.

## <a name="add-tax-information"></a>Adicionar informações de dedução dos impostos  
1.  Inicie sessão no Dev Center com o Microsoft ID.  
    *   Para iniciar sessão no Dev Center, aceda ao Windows Dev Center página localizada em [dev.windows.com](https://dev.windows.com).  
2.  Selecione o perfil de dedução dos impostos no menu da esquerda. 
3.  No `Set up your tax form` página, selecione o país ou região onde tenha residency permanente.  
    
    Selecione o país ou região onde espera que aceita primária.  
    
    Clique no botão seguinte. 
4.  Introduza os detalhes da sua dedução dos impostos.  
5.  Clique no botão seguinte.
