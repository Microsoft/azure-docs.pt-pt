# <a name="add-bank-and-tax-information"></a>Adicionar informação bancária e fiscal  

O banco e a informação fiscal são necessários para todas as ofertas comerciais utilizando o tipo de listagem transacionada.  

## <a name="partner-center"></a>Centro de Parceiros

A informação bancária e fiscal é agora gerida no Partner Center. As instruções para configurar estas definições podem ser encontradas na página "Commercial Marketplace overview" da Conta De Gestão do [Centro de Parceiros](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#payout-account).
