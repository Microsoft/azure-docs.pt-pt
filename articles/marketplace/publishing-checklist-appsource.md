# <a name="publishing-checklist-for-appsource"></a>Publicar Lista de Verificação para o AppSource  

Reveja os componentes necessários antes de iniciar o processo de publicação.  

Os seguintes artefactos são necessários para publicar a sua aplicação no AppSource.
