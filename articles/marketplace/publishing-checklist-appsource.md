# <a name="publishing-checklist-for-appsource"></a>Lista de verificação de publicação para AppSource  

Examine os componentes necessários antes de iniciar o processo de publicação.  

Os seguintes artefatos são necessários para publicar seu aplicativo no AppSource.

| Tipo de listagem | Tipo de oferta | Artefato de publicação |   
|:--- |:--- |:--- |
