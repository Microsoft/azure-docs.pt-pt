# <a name="notifying-microsoft-regarding-the-publisher-agreement"></a>Notificando a Microsoft sobre o Acordo de Editor

Utilize um bilhete de apoio para submeter avisos legais à Microsoft sobre o Microsoft Publisher Agreement para garantir uma ação e rastreio adequados.

## <a name="submit-notice-regarding-the-publisher-agreement"></a>Subofim aviso sobre o Contrato de Editor

1. Aceda à secção [de questões contratuais ou legais](https://partner.microsoft.com/support/?stage=2&topicid=05a1a389-1256-d441-89c9-a140217de6b9) do Apoio ao Centro de Parceiros.

1. **Selecione Fornecer detalhes de emissão** (se necessário, inscreva-se primeiro no Partner Center).

    :::image type="content" source="media/notice-submit/recommended-solutions.png" alt-text="Mostra a janela &quot;Soluções Recomendadas&quot;.":::

1. Edite o **resumo do problema,** se desejar, até 200 caracteres.
1. Utilize **os seus ficheiros para arrastar os seus ficheiros aqui ou navegue** no link dos seus ficheiros para anexar ficheiros aplicáveis a este pedido de suporte.

    :::image type="content" source="media/notice-submit/create-support-ticket.png" alt-text="Mostra os campos na janela para criar um bilhete de apoio.":::

1. Fornecer **detalhes adicionais:**

    - **Tipo de oferta** (escolha "Não tenho a certeza" se este aviso não for específico para qualquer tipo de oferta em particular)
    - **Detalhes** do aviso
    - A data de **início** do seu aviso (se tiver uma data efetiva mais tarde do que hoje)
    - As suas **informações** e preferências de contacto

    Esta página de pedido de suporte também fornece orientações sobre como encontrar o seu ID editor e iD do vendedor.

1. Quando tiver fornecido todas as informações necessárias, **selecione Submeter-se** para criar o seu bilhete de suporte e enviar o aviso para a Microsoft.

## <a name="next-steps"></a>Passos seguintes

- Um agente de apoio entrará em contacto consigo se tivermos dúvidas.
- Reveja o [Microsoft Publisher Agreement](https://go.microsoft.com/fwlink/?LinkID=699560) (PDF).
