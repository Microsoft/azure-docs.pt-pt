# <a name="saas-fulfillment-apis-version-1-deprecated"></a>ApIs de cumprimento saas (precotado)

> [!WARNING]
> Esta versão inicial da API de Realização de SaaS é depreciada; em vez disso, utilize [SaaS Fulfillment API V2](./pc-saas-fulfillment-api-v2.md).  Esta versão inicial da API está atualmente a ser mantida apenas para servir os editores existentes. 

A documentação para esta versão da API já não está disponível. Se sentir que precisa de aceder a ele, contacte- o saas-api-v1@microsoft.com por favor.

Para todos os novos desenvolvimentos, mude para a [versão V2 da API](./pc-saas-fulfillment-api-v2.md) mesmo que já utilize v1. Esta versão será completamente depreciada no futuro.
