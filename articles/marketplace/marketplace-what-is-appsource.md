# <a name="what-is-appsource"></a>O que é o AppSource
AppSource permite que os parceiros da Microsoft tornar seus serviços e tecnologia detectáveis para os clientes através de uma loja online Microsoft com suporte. Como um publicador de AppSource, pode interagir com os utilizadores empresariais no AppSource para encontrar, experimentar e obter aplicações de SaaS de linha de negócio, bem como serviços de implementação para ajudar a garantir resultados de negócios e reduzir o tempo / valor: 

| Necessidade do cliente       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Está à procura de soluções de negócios que funcionam com produtos da Microsoft que já utilizam**    | Permite aos parceiros usar serviços e aplicações de terceiros para estender aplicativos de nuvem da Microsoft e tecnologias. |
| **Capacidade de localizar facilmente o serviço de solução ou implementação correto à direita.**     | Fornece um meio único para descobrir, avaliação e obter aplicações e serviços, suplementos e muito mais. |
| **Solução de linha de negócio específicos da indústria para resolver seus desafios de negócios específicas**   | Fornece soluções de concluir o procedimento de ponto-a-ponto de setor para ajudar a requisitos específicos do endereço vários setores. |
| **Aplicações para ajudar a melhorar a produtividade, eficiência e informações de negócio**       | Fornece aplicações de linha de negócio, incluindo suporte ao cliente, RH, operações e muito mais. |
| **Parceiro de implementação experiente para ajudar a adaptar-se aplicações à sua situação exclusiva**     | Fornece, um catálogo de ofertas de serviços de consultoria para soluções baseadas no Dynamics 365, Power BI, PowerApps e aplicações de terceiros de 3rd que estão disponíveis no AppSource para ajudar os utilizadores empresariais a encontrar serviços de consultoria concebidos para proporcionar resultados previsíveis. |

## <a name="appsource-publishing"></a>Publicação de AppSource
Através do AppSource, pode listar uma aplicação ou uma oferta de consultoria que ajuda a atender às necessidades do cliente durante o uso de produtos da Microsoft, como o Office 365, Dynamics 365, Power BI e Power Apps. As ofertas seguintes estão disponíveis no mercado do AppSource: 
*   **Dynamics 365 para operações e finanças**: um solução (ERP) de planeamento de recursos empresariais. As soluções diferenciadas são a Enterprise Edition para as empresas de médio a grande e o Business Edition para pequenas e médias empresas.
*   **Dynamics 365 for Customer Engagement**: uma solução de gestão (CRM) de relação de cliente que inclui a vendas, suporte ao cliente, o serviço de campo e aplicativos de automatização do serviço do projeto.
*   **Serviço de gerida do Dynamics NAV**: Microsoft **Dynamics NAV** é um pacote de software (ERP) para organizações de médio porte de planeamento de recursos de empresa. O serviço oferece funcionalidades especializadas para produção, distribuição, administração pública, varejo e outros setores.
*   **Power BI**: uma solução de visualização de análise de negócio.
*   **Ofertas de consultoria**: dos serviços fornecidos pela Microsoft, parceiros para ajudar os clientes a entender, experimentar e implementar especializados de soluções tecnológicas.
*   **Cortana Intelligence**: O Cortana Intelligence Suite é uma coleção de independentes, mas totalmente integrada a dados e ferramentas de plataforma de análise oferecidos pelo Azure.
*   **Office 365**: A área de trabalho moderna na cloud. Colabore gratuitamente com o Microsoft Word, PowerPoint, Excel e OneNote.

## <a name="azure-active-directory-integration"></a>Integração do Azure Active Directory
Alguns aplicativos vitrine do AppSource são necessários para integrar com o Azure Active Directory (Azure AD) para ser publicado. Integração de aplicações com o Azure AD está bem documentada; A Microsoft fornece vários SDKs e recursos adicionais para satisfazer os seus requisitos. 

| Nome da oferta | É necessária a integração do Azure AD | Notas |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Não | Uma oferta com base na licença não necessita de integração do Azure AD |
| Dynamics 365 para finanças | Não | Uma oferta com base na licença não necessita de integração do Azure AD |
| Dynamics 365 for Customer Engagement | Não | Uma oferta com base na licença não necessita de integração do Azure AD |
| Serviço gerido do Dynamics NAV | Não | Uma oferta com base na licença não necessita de integração do Azure AD |
| Power BI | Não | Uma oferta com base na licença não necessita de integração do Azure AD |
| Ofertas de consultoria | Não | Os serviços são coordenados pelo parceiro, não por meio de uma experiência na web |
| Cortana Intelligence | Sim | O Azure AD é uma prática recomendada para garantir que um cliente totalmente integrado, experiência e unidades que leva a mais alta qualidade |
| Aplicações SaaS (anteriormente conhecido como aplicações Web) | Preferencial | O Azure AD é uma prática recomendada para garantir que um cliente totalmente integrado, experiência e unidades que leva a mais alta qualidade |
