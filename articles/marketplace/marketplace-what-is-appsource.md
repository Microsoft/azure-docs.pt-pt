# <a name="what-is-appsource"></a>O que é AppSource
AppSource permite que os parceiros da Microsoft tornar as tecnologias e serviços detetável aos clientes através de um storefront suportada do Microsoft online. Como um publicador AppSource, pode interagir com os utilizadores empresariais no AppSource para localizar, tente e obter aplicações de SaaS de linha de negócio, bem como serviços de implementação para ajudar a resultados de negócios de unidade e reduza o valor de tempo: 

| Necessidade de cliente       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **A procurar soluções de negócio que funcionam com produtos Microsoft que já utilizam**    | Permite que os parceiros ao utilizar serviços e aplicações de terceiros para expandir as aplicações de nuvem da Microsoft e tecnologias. |
| **Capacidade para localizar facilmente o direito do serviço de solução ou implementação certo.**     | Fornece uma cafetaria paragem para detetar, versão de avaliação e obter aplicações e serviços, suplementos e muitos mais. |
| **Solução de linha de negócio específicos da indústria para abordar os desafios de negócio específicas**   | Fornece soluções de terminar a extremidade da indústria para ajudar a requisitos específicos do endereço em indústrias muitos. |
| **Aplicações para ajudar a melhorar a produtividade, eficiência e informações de negócio**       | Fornece as aplicações de linha de negócio, incluindo o serviço de cliente, HR, operações e muitos mais. |
| **Parceiro de experiência de implementação para o ajudar a adaptar aplicações para as respetivas situação exclusiva**     | Fornece, catálogo de consultadoria ofertas de serviços para soluções baseadas no Dynamics 365, Power BI, PowerApps e aplicações de terceiros 3rd que estão disponíveis no AppSource para ajudar os utilizadores empresariais localizar consultadoria serviços concebida para fornecer resultados previsíveis. |

## <a name="appsource-publishing"></a>Publicação de AppSource
Através de AppSource, pode listar uma aplicação ou a oferta consultoria que ajuda a satisfazer as necessidades do cliente com produtos Microsoft, tais como o Office 365, Dynamics 365, Power BI e aplicações de energia. As ofertas de seguintes estão disponíveis no mercado AppSource: 
*   **Dynamics 365 para operações e financeiro**: um recurso de empresa planeamento da solução (ERP). As soluções diferenciadas são a edição Enterprise para as empresas de média a grande e a edição de negócio para pequenas e médias empresas.
*   **Dynamics 365 para o Engagement cliente**: uma solução de gestão (CRM) de relação de cliente que inclua vendas, serviço de cliente, o serviço de campo e aplicações de automatização de serviço de projeto.
*   **Dynamics NAV gerido serviço**: Microsoft **Dynamics NAV** é um recurso de empresa planeamento suite de software (ERP) para organizações médias. O serviço oferece uma funcionalidade especializada para produção, distribuição, government, de revenda e outras indústrias.
*   **Power BI**: uma solução de visualização de análise de negócio.
*   **Consultadoria ofertas**: serviços fornecidos pela Microsoft parceiros para ajudar os clientes compreender, experimente e implementar especializada soluções de tecnologia.
*   **Cortana Intelligence**: O Cortana Intelligence Suite é uma coleção de independentes, mas e ferramentas de plataforma de análise de dados totalmente integrado oferecidas pelo Azure.
*   **Office 365**: moderna área de trabalho na nuvem. Colabore gratuitamente com o Microsoft Word, o PowerPoint, o Excel e o OneNote.

## <a name="azure-active-directory-integration"></a>Integração do Azure Active Directory
Algumas aplicações storefront AppSource são necessárias para integrar com o Azure Active Directory (Azure AD) para ser publicado. Integração de aplicações com o Azure AD está bem documento; A Microsoft fornece várias SDKs e recursos adicionais para satisfazer os seus requisitos. 

| Nome da oferta | Requer uma integração do Azure AD | Notas |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Não | Uma oferta baseada no licenciamento não necessita de integração do AD do Azure |
| Dynamics 365 financeiro | Não | Uma oferta baseada no licenciamento não necessita de integração do AD do Azure |
| Dynamics 365 para o Engagement de cliente | Não | Uma oferta baseada no licenciamento não necessita de integração do AD do Azure |
| Dynamics NAV de serviço geridas | Não | Uma oferta baseada no licenciamento não necessita de integração do AD do Azure |
| Power BI | Não | Uma oferta baseada no licenciamento não necessita de integração do AD do Azure |
| Consultadoria ofertas | Não | Os serviços são coordenados pelo parceiro, não através de uma experiência de web |
| Cortana Intelligence | Sim | Azure AD é uma melhor prática para garantir que um cliente totalmente integrado experiência e a mais alta qualidade servem como de unidades |
| Aplicações SaaS (anteriormente Web Apps) | Preferencial | Azure AD é uma melhor prática para garantir que um cliente totalmente integrado experiência e a mais alta qualidade servem como de unidades |
