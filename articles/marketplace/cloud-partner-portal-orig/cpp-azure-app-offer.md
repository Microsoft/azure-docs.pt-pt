# <a name="azure-application-offer"></a>Oferta de aplicação do Azure

<table> <tr> <td>Esta secção explica como publicar uma nova oferta de aplicação do Azure gerida para o <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a>.  Cada aplicação gerida contém um modelo do Azure Resource Manager que define todos os recursos técnicos usados pelo aplicativo, que normalmente inclui uma ou mais máquinas virtuais e outros serviços de suporte do Azure - ou baseado na Web. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>Descrição geral de publicação

O vídeo seguinte, [criando modelos de soluções e aplicações geridas para o Azure Marketplace](https://channel9.msdn.com/Events/Build/2018/BRK3603), é uma descrição geral sobre como criar um modelo do Azure Resource Manager para definir uma solução de aplicação do Azure e, em seguida, como em seguida, publicar a oferta da aplicação no Azure Marketplace.

## <a name="publishing-process"></a>Processo de publicação

Os artigos seguintes irão orientá-lo no processo de criar e publicar uma aplicação gerida do Azure:

1. [Registar uma conta do Microsoft Developer Center](./cloud-partner-portal-dev-center-accounts-registration.md) tornam-se aprovadas para publicação no Azure Marketplace.
2. [Publicar uma nova oferta de aplicação do Azure](./cloud-partner-portal-managed-app-publish.md) no Azure Marketplace.
3. [Tornar a sua oferta live](./cloud-partner-portal-managed-app-offer-go-live.md)!
