<a name="how-to-fill-out-the-offer-settings-form"></a>Como preencher o formulário de definições da oferta
=======================================

O formulário de definições da oferta é um formulário básico para especificar as definições da oferta.
Os campos obrigatórios estão descritos abaixo.

### <a name="offer-id"></a>ID de Oferta

`OfferId` é um identificador exclusivo para a oferta de um perfil do publicador.
Este ID será visível em URLs de produto. Apenas pode ser constituído por carateres alfanuméricos em minúsculas ou hífenes (-). O ID não pode terminar com um travessão e pode ter um máximo de 50 carateres. Este campo está bloqueado assim que uma oferta entra no ar.

Se, por exemplo, o parceiro "Contoso" Criar um ID de oferta chamado "a aplicação de exemplo-Web", que será apresentado no AppSource, como:

&emsp; `https://appsource.microsoft.com/marketplace/apps/contoso.sample-Web App?tab=Overview`


### <a name="publisher-id"></a>ID de publicador

Essa área suspensa permite-lhe escolher o perfil do publicador que pretende publicar esta oferta em. Este campo está bloqueado assim que uma oferta entra no ar.


### <a name="name"></a>Nome

Este é o nome a apresentar para a sua aplicação/oferta, que será apresentado no Microsoft [AppSource](https://appsource.microsoft.com/). Pode ter um máximo de 50 carateres.

> [!NOTE]
> O nome abreviado tem de ser o mesmo que o nome do publicador especificado no manifesto do aplicativo.

Clique em **guardar** para guardar o seu progresso. Próxima etapa seria adicionar informações técnicas para a sua oferta.
