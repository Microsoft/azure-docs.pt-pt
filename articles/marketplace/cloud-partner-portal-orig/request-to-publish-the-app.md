# <a name="request-to-publish-the-app"></a>Pedido para publicar a aplicação

![Clique para publicar](./media/CRMScreenShot18.png)

1. Clique no botão Publish para submeter a sua oferta para a nossa equipa de certificação. Será capaz de ver o estado da sua oferta no separador de estado.

![Reveja o estado de publicação](./media/CRMScreenShot19.png)

## <a name="republish-the-app"></a>Voltar a publicar a aplicação

 Se precisar de fazer um pedido para voltar a publicar uma aplicação, primeiro clique no botão Cancelar publicar no separador Estado (se o pedido já não estiver ativo, ignore este passo). Fazer as alterações à aplicação, atualizar a oferta da mesma forma, em seguida, clique no botão publicar para submeter novamente. # Considerações especiais para o Microsoft Dynamics 365 Business Edition observação que soluções criadas para as edições do Microsoft Dynamics 365 Business vão sofrer verificações adicionais durante a certificação para garantir que os limites de Business Edition sejam seguidas. Os limites de Business Edition estão documentados aqui. # Obter ajuda. Caso se depare com problemas, pode iniciar um pedido de suporte ao clicar no canto superior direito e vai.
