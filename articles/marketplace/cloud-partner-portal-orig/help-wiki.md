# <a name="dynamics-365-for-operations-help-wiki"></a>Wiki da ajuda do Dynamics 365 for Operations

O Microsoft Dynamics 365 for Operations é o aplicativo de negócios da Microsoft para empresas. Ele permite que as pessoas tomem decisões mais inteligentes rapidamente por meio de uma interface de usuário inteligente que fornece acesso a dados e informações em tempo real. Ele permite que as empresas transformem e inovassem fornecendo uma lógica de negócios comprovada que permite reprojetar seus processos de negócios com mais rapidez. Ele oferece às empresas a flexibilidade de crescer em seu próprio ritmo e fazer negócios praticamente em qualquer lugar, a qualquer momento, permitindo dimensionar suas operações globalmente para atender às necessidades dos negócios.

Este guia do usuário pode ajudá-lo a aprender a fazer o Dynamics 365 para operações trabalharem para sua empresa.

Consulte o [wiki do Dynamics 365 for Operations Help](https://docs.microsoft.com/dynamics365/operations/) para obter mais informações.
