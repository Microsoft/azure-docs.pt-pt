# <a name="dynamics-365-for-operations-help-wiki"></a>Dynamics 365 para operações de ajudar a Wiki

Microsoft Dynamics 365 para operações é Microsoft\'aplicativo de negócios s para as empresas. Ele permite que as pessoas rapidamente tome decisões mais informadas através de uma interface de utilizador inteligentes que fornece acesso a informações em tempo real e dados. Ele permite que as empresas transformar e inovar ao fornecer lógica de negócios comprovadas que permita que sejam reestruturar os processos empresariais mais rapidamente. Ele proporciona às empresas a flexibilidade para crescer em seu próprio ritmo e fazer negócios de praticamente qualquer lugar e em qualquer altura, permitindo que elas abranjam suas operações globalmente para atender às necessidades de negócio.

Este guia de utilizador pode ajudá-lo a saber como tornar do Dynamics 365 para operações de trabalho para a sua empresa.

Ver [Dynamics 365 para operações de Wiki de ajudar a](https://docs.microsoft.com/dynamics365/operations/) para obter mais informações.
