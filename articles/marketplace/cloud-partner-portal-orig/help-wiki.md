# <a name="dynamics-365-for-operations-help-wiki"></a>Dinâmica 365 para Operações Ajudam a Wiki

Microsoft Dynamics 365 para Operações é a aplicação comercial da Microsoft para empresas. Permite que as pessoas tomem rapidamente decisões mais inteligentes através de uma interface inteligente de utilizador que fornece acesso a insights e dados em tempo real. Permite que as empresas transformem e inovam através da entrega de uma lógica de negócio comprovada que lhes permite redesenhar os seus processos de negócio mais rapidamente. Dá às empresas a flexibilidade para crescerem ao seu ritmo e fazerem negócios em qualquer lugar, a qualquer momento, permitindo-lhes escalar as suas operações globalmente para satisfazer as necessidades empresariais.

Este guia de utilizador pode ajudá-lo a aprender a fazer com que a Dynamics 365 para Operações funcione para o seu negócio.

Consulte a [Dynamics 365 para operações ajude a Wiki](https://docs.microsoft.com/dynamics365/operations/) para obter mais informações.
