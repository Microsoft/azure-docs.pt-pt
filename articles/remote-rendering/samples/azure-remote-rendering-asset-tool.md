# <a name="azure-remote-rendering-asset-tool-arrt"></a>Ferramenta de ativo de renderização remota Azure (ARRT)

Azure Remote Rendering Asset Tool (ARRT) é uma aplicação de ambiente de trabalho de código aberto desenvolvida em C++/Qt.

![ARRT](./media/azure-remote-rendering-asset-tool.png "Screenshot arrt")

## <a name="features"></a>Funcionalidades

A aplicação pode ser usada para:

* Faça upload de um modelo 3D
* Controlar a conversão do modelo
* Criar e gerir uma sessão de renderização remota
* Carregue um modelo 3D
* Pré-visualizar o modelo 3D
* Modificar os seus materiais

Também pode usá-lo como uma amostra de código aberto para aprender a implementar uma extremidade frontal para o Azure Remote Rendering C++ SDK, utilizando a Biblioteca do Cliente de Armazenamento Azure para gerir a conversão do modelo 3D.

## <a name="source-repository"></a>Repositório de origem

Pode encontrar o código-fonte e a documentação no [repositório GitHub ARRT](https://github.com/Azure/azure-remote-rendering-asset-tool).
