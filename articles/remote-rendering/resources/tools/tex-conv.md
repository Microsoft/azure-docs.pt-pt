# <a name="texconv---texture-conversion-tool"></a>TexConv - Ferramenta de conversão de textura

A ferramenta TexConv é uma ferramenta externa de linha de comando para realizar conversões de textura em massa.
A ferramenta pode ser encontrada neste repositório de [gitHub: repositório GitHub para a ferramenta TexConv](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Para a renderização remota, as texturas não têm de ser convertidas, porque o passo de conversão pode lidar com formatos comuns de ficheiros de imagem e converte texturas para o formato de tempo de execução apropriado. No entanto, esta ferramenta pode ser útil para converter texturas localmente com configurações personalizadas.
