# <a name="azure-remote-rendering-tutorials"></a>Tutoriais de renderização remota Azure

Bem-vindos aos tutoriais de renderização remota do Azure. Além de aprender conceitos-chave de Renderização Remota Azure, estas lições interativas irão acompanhá-lo através da visualização, manipulação e personalização de modelos remotamente renderizados e destacar considerações para a construção de uma aplicação segura e pronta para o comércio.

### <a name="azure-remote-rendering-tutorials"></a>Tutoriais de renderização remota Azure

Recomendamos que complete os tutoriais abaixo, à medida que cada tutorial se baseia no anterior.

#### <a name="viewing-a-remotely-rendered-model"></a>[Visualização de um modelo prestado remotamente](view-remote-models/view-remote-models.md)

Neste tutorial, irá criar uma aplicação básica capaz de visualizar modelos remotamente renderizados utilizando renderização remota do Azure.

#### <a name="interfaces-and-custom-models"></a>[Interfaces e modelos personalizados](custom-models/custom-models.md)

Em seguida, você vai aprender a carregar seus próprios modelos em Azure Remote Rendering e vê-los na sua aplicação. [Ver um modelo renderizado remotamente](view-remote-models/view-remote-models.md) é um pré-requisito para este tutorial.

#### <a name="manipulating-models"></a>[Modelos manipuladores](manipulate-models/manipulate-models.md)

Neste tutorial, irá manipular e interagir com os seus modelos remotamente renderizados. Recomendamos que preencha a visualização de um modelo e [interfaces e modelos personalizados](custom-models/custom-models.md) [remotamente renderizados](view-remote-models/view-remote-models.md) antes deste tutorial.

#### <a name="refining-materials-lighting-and-effects"></a>[Refinação de materiais, iluminação e efeitos](materials-lighting-effects/materials-lighting-effects.md)

Em seguida, vamos cobrir conceitos importantes como modificar materiais, alterar a iluminação e aplicar efeitos nos seus modelos remotamente renderizados. [Visualizar um modelo remotamente renderizado,](view-remote-models/view-remote-models.md) [Interfaces e modelos personalizados](custom-models/custom-models.md), e [modelos manipuladores](manipulate-models/manipulate-models.md) são pré-requisitos recomendados para este tutorial.

#### <a name="securing-azure-remote-rendering-and-model-storage"></a>[Segurança da renderização remota do Azure e armazenamento de modelos](security/security.md)

Aqui, você aprenderá sobre a segurança da sua conta de renderização remota Azure e armazenamento Blob. Isto cobre os primeiros passos para a libertação de uma aplicação comercial que utiliza a renderização remota Azure. Este tutorial assume que completou as lições acima.


#### <a name="creating-a-commercial-ready-azure-remote-rendering-application"></a>[Criação de uma aplicação de renderização remota Azure](commercial-ready/commercial-ready.md)

Por último, discutiremos algumas considerações e dicas importantes para preparar uma aplicação comercial utilizando a Renderização Remota Azure. É útil se tiver completado todos os tutoriais de renderização remota do Azure antes de verificar este recurso.

**Começa com** [a visualização de um modelo renderizado remotamente.](view-remote-models/view-remote-models.md)
