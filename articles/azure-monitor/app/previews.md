# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Pré-visualizar alterações futuras para o Azure Application Insights 

O Application Insights lança frequentemente novos recursos. Se quiser ver pré-visualizações desses aprimoramentos, pode inscrever-se no painel de visualização de informações do aplicativo.  A equipe de desenvolvimento disponibiliza prévias de novos recursos de forma limitada antes do lançamento-los para todos os utilizadores. 

A imagem seguinte ilustra como definir suas preferências de pré-visualização.

![Painel de visualização de informações de aplicações](media/previews/preview.png)

## <a name="set-preferences"></a>Preferências de conjunto

No painel pré-visualização, pode selecionar de entre as opções seguintes quando vir as pré-visualizações.

- Sempre: Ver a que pré-visualização experiências assim que estiverem disponíveis.
- Automático: Verá que a Microsoft recomenda para a sua conta de experiências de pré-visualização. 
- Nunca: Apenas verá as experiências de visualização que selecionar.
