# <a name="manage-smart-groups"></a>Gerir grupos inteligentes

[Os grupos](./alerts-smartgroups-overview.md?toc=%2fazure%2fazure-monitor%2ftoc.json) inteligentes usam algoritmos de aprendizagem automática para agrupar alertas com base na coocorrência ou semelhança, para que o utilizador possa agora gerir grupos inteligentes em vez de ter que gerir cada alerta individualmente. Este artigo irá acompanhá-lo como aceder e usar grupos inteligentes no Azure Monitor.

1. Para ver os Grupos Inteligentes criados para os seus casos de alerta, pode:

     1. Clique em **Grupos Inteligentes** a partir da página **'Resumo' de Alertas.**    
    ![A screenshot mostra a página 'Resumo do Alerta' com grupos Inteligentes em destaque.](./media/alerts-managing-smart-groups/sg-alerts-summary.jpg)
    
     1. Clique em Alertas por Grupos Inteligentes a partir da página Todos os Alertas.   
     ![A screenshot mostra a página All Alerts com Alerta por Smart Group em destaque.](./media/alerts-managing-smart-groups/sg-all-alerts.jpg)

2. Isto leva-o à vista da lista de todos os Grupos Inteligentes criados sobre os seus casos de alerta. Em vez de vasculhar vários alertas, pode agora lidar com os grupos inteligentes.   
![A screenshot mostra a página All Alerts.](./media/alerts-managing-smart-groups/sg-list.jpg)

3. Clicar em qualquer Grupo Inteligente abre a página de detalhes, onde pode ver a razão de agrupamento, juntamente com os alertas dos membros. Esta agregação permite-lhe lidar com um grupo inteligente singular, em vez de vasculhar através de vários alertas.   
![A screenshot mostra a página Detalhes.](./media/alerts-managing-smart-groups/sg-details.jpg)
