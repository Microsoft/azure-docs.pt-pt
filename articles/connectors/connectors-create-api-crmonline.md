# <a name="create-and-manage-records-in-dynamics-365-by-using-azure-logic-apps"></a>Criar e gerir registos em Dynamics 365 utilizando apps Azure Logic

A Dynamics 365 utiliza o [Serviço Comum de Dados.](/powerapps/maker/common-data-service/data-platform-intro) Para ligações à Dinâmica 365, utilize o [conector Common Data Service](/connectors/commondataservice/).

> [!IMPORTANT]
> O [conector Dynamics 365](/connectors/dynamicscrmonline/) está precotado, mas continua a funcionar até ser removido. Não utilize o conector Dynamics 365 para novas aplicações lógicas. Ainda não foi anunciada nenhuma linha temporal para a remoção do conector Dynamics 365. Não precisa de converter as aplicações lógicas existentes no conector do Common Data Service ou noutro novo conector planeado, mas precisa de converter as aplicações lógicas quando o conector for removido. Para mais informações, consulte [o Conector Dynamics 365 está precotado.](/power-platform/important-changes-coming)
>
> O [conector do Common Data Service](/connectors/commondataservice/) proporciona as mesmas capacidades que o conector do Dynamics 365 preterido, mas inclui melhorias que aumentam a fiabilidade. Para obter informações sobre a utilização do conector do Common Data Service em aplicações lógicas, veja [Criar e gerir registos do Common Data Service com o Azure Logic Apps](../connectors/connect-common-data-service.md).

Para obter mais informações sobre o Serviço Comum de Dados, consulte estes tópicos:

* [Saiba: Começar com o Serviço Comum de Dados](/learn/modules/get-started-with-powerapps-common-data-service/)
* [Saiba: Conecte e analise os seus dados Dinâmicos 365 utilizando a Plataforma de Energia e o Serviço Comum de Dados](/learn/wwl/connect-analyze-dynamics-365-data/)
