# <a name="create-and-manage-records-in-dynamics-365-by-using-azure-logic-apps"></a>Crie e gerencie registos na Dynamics 365 utilizando aplicações da Azure Logic

A Dynamics 365 utiliza o [Serviço Comum](https://docs.microsoft.com/powerapps/maker/common-data-service/data-platform-intro)de Dados. Para ligações à Dynamics 365, utilize o [conector do Serviço De Dados Comuns](https://docs.microsoft.com/connectors/commondataservice/).

> [!IMPORTANT]
> O [conector Dynamics 365](https://docs.microsoft.com/connectors/dynamicscrmonline/) é depreciado, mas continua a funcionar até ser removido. Não utilize o conector Dynamics 365 para novas aplicações lógicas. Ainda não foi anunciada nenhuma linha temporal para remover o conector Dynamics 365. Não é obrigado a converter aplicações lógicas existentes para o conector do Common Data Service ou outro novo conector planeado, mas precisa de converter as suas aplicações lógicas quando o conector é removido. Para mais informações, consulte o [Conector Dynamics 365 .](https://docs.microsoft.com/power-platform/important-changes-coming)
>
> O [conector do Serviço Comum](https://docs.microsoft.com/connectors/commondataservice/) de Dados fornece as mesmas capacidades que o conector Deprecated Dynamics 365, mas inclui melhorias que aumentam a fiabilidade. Para obter informações sobre a utilização do conector do Serviço Comum de Dados em aplicações lógicas, consulte Criar e gerir registos do [Common Data Service utilizando aplicações lógicas do Azure](../connectors/connect-common-data-service.md).

Para obter mais informações sobre o Serviço Comum de Dados, consulte estes tópicos:

* [Saiba: Começar com o Serviço Comum de Dados](https://docs.microsoft.com/learn/modules/get-started-with-powerapps-common-data-service/)
* [Aprenda: Conecte e analise os seus dados Da Dinâmica 365 utilizando a Plataforma de Energia e o Serviço de Dados Comuns](https://docs.microsoft.com/learn/wwl/connect-analyze-dynamics-365-data/)
