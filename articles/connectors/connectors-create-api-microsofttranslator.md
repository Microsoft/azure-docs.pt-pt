# <a name="get-started-with-the-microsoft-translator-connector"></a>Começar com o conector Microsoft Tradutor

Ligue-se ao Microsoft Tradutor para traduzir texto, detetar um idioma, e muito mais. Com o Microsoft Tradutor, pode: 

* Construa o seu fluxo de negócios com base nos dados que obtém do Microsoft Tradutor. 
* Use ações para traduzir texto, detetar uma linguagem, e muito mais. Estas ações obtêm uma resposta e, em seguida, disponibilizam a produção para outras ações. Por exemplo, quando um novo ficheiro é criado no Dropbox, pode traduzir o texto no ficheiro para outro idioma usando o Microsoft Tradutor.

Você pode começar criando uma aplicação lógica agora, ver [Criar uma aplicação lógica](../logic-apps/quickstart-create-first-logic-app-workflow.md).
