# <a name="get-started-with-the-microsoft-translator-connector"></a>Começa com o conector microsoft Tradutor

Ligue-se ao Microsoft Tradutor para traduzir texto, detetar um idioma e muito mais. Com o Microsoft Tradutor, pode: 

* Construa o seu fluxo de negócios com base nos dados que obtém do Microsoft Tradutor. 
* Use ações para traduzir texto, detetar uma língua, e muito mais. Estas ações obtêm uma resposta e, em seguida, disponibilizam a produção para outras ações. Por exemplo, quando um novo ficheiro é criado no Dropbox, pode traduzir o texto no ficheiro para outro idioma utilizando o Microsoft Tradutor.

Pode começar por criar agora uma aplicação lógica, ver [Criar uma aplicação lógica.](../logic-apps/quickstart-create-first-logic-app-workflow.md)
