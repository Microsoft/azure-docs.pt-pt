# <a name="manage-your-facebook-timeline-and-page-by-using-azure-logic-apps"></a>Gerencie a sua linha de tempo e página do Facebook utilizando aplicações lógicas azure

Ligue-se ao Facebook e poste para uma linha do tempo, obtenha um feed de página, e muito mais. Com o Facebook, pode:

* Construa o seu fluxo de negócios com base nos dados que obtém do Facebook. 
* Use um gatilho quando um novo posto for recebido.
* Use ações que postam na sua linha do tempo, obtenha um feed de página, e muito mais. Estas ações obtêm uma resposta e, em seguida, disponibilizam a produção para outras ações. Por exemplo, quando há uma nova publicação na sua linha temporal, pode pegar nessa publicação e empurrá-la para o seu feed do Twitter. 

Você pode começar criando uma aplicação lógica agora, ver [Criar uma aplicação lógica](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-facebook"></a>Criar uma ligação ao Facebook

Quando adiciona este conector às suas aplicações lógicas, tem de autorizar aplicações lógicas para se conectar em seu Facebook.

1. Inscreva-se na sua conta de Facebook.

2. Selecione **Autorizar**, e permitir que as suas aplicações lógicas conectem e utilizem o seu Facebook.
