# <a name="get-started-with-the-facebook-connector"></a>Comece com o conector do Facebook
Ligue ao Facebook e publique numa linha cronológica, obtenha um feed de página e muito mais. Com o Facebook, pode:

* Crie o seu fluxo de negócios com base nos dados que obtém do Facebook. 
* Utilize um acionador, quando uma nova mensagem é recebida.
* Ações de utilização que publicar sua linha cronológica, obtenha um feed de página e muito mais. Estas ações obtém uma resposta e, em seguida, disponibilizá-a saída para outras ações. Por exemplo, quando existe uma nova publicação na linha cronológica do, pode tirar essa mensagem e enviá-la para o feed do Twitter. 

Pode começar a utilizar ao criar uma aplicação lógica agora, veja [criar uma aplicação lógica](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-facebook"></a>Criar uma ligação com o Facebook
Ao adicionar este conector às suas aplicações lógicas, tem de autorizar aplicações lógicas para se ligar ao seu Facebook.

1. Inicie sessão na sua conta do Facebook
2. Selecione **autorizar**e permitir que as aplicações lógicas para se ligar e utilizar o Facebook.
