# <a name="monitor-and-manage-wunderlist-in-azure-logic-apps"></a>Monitorize e gere o Wunderlist em Azure Logic Apps

A partir de 6 de maio de 2020, o [serviço Wunderlist fechou](https://www.wunderlist.com/) e pediu aos clientes para [migrarem os seus dados e começarem a usar o Microsoft To Do em vez disso.](https://www.wunderlist.com/blog/goodbye-from-wunderlist/) Pode utilizar o [conector Microsoft To Do](/connectors/todo/) para substituir o [conector Wunderlist](https://preview.flow.microsoft.com/connectors/shared_wunderlist/wunderlist/)prectilizado .
