# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Monitorize e gerencie o Wunderlist utilizando aplicações da Lógica Azure

A partir de 6 de maio de 2020, o [serviço Wunderlist fechou](https://www.wunderlist.com/) e pediu aos clientes que migrassem [os seus dados e começassem](https://www.wunderlist.com/blog/goodbye-from-wunderlist/)a usar o Microsoft Para Fazer em vez disso . Pode utilizar o [conector Microsoft To Do](https://docs.microsoft.com/connectors/todo/) para substituir o [conector Wunderlist](https://docs.microsoft.com/connectors/wunderlist/)depreciado .
