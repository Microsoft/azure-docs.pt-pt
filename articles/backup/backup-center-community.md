# <a name="access-community-resources-using-backup-center"></a>Aceda aos recursos comunitários usando o Backup Center

Você pode usar o Backup Center para aceder a vários recursos comunitários úteis para um administrador ou operador de reserva.

## <a name="using-community-hub"></a>Usando o Centro Comunitário

Para aceder ao Hub Comunitário, navegue no Backup Center no portal Azure e selecione o item do menu **comunitário.**

![Centro Comunitário](./media/backup-center-community/backup-center-community-hub.png)

Alguns dos recursos disponíveis através do Hub Comunitário são:

- **Microsoft Q&A**: Pode utilizar este fórum para fazer perguntas e descobrir perguntas sobre várias funcionalidades do produto e obter orientação da comunidade.

- **Pedidos de funcionalidades**: Pode navegar para o UserVoice e solicitar pedidos de funcionalidades de ficheiro.

- **Amostras para implementações automatizadas**: Utilizando o Hub Comunitário, pode descobrir modelos de gestão de recursos Azure (ARM) e Políticas de Azure que pode utilizar fora da caixa. Também pode encontrar scripts PowerShell, comandos CLI e backup de bases de dados da Microsoft.
