# <a name="update-the-azure-backup-central-reporting-content-pack"></a>Atualizar o Azure Backup central relatórios pacote de conteúdos 

O [pacote de conteúdos do Azure Backup](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) pode ser utilizado para ver relatórios sobre a cópia de segurança central. O pacote de conteúdos é atualizado regularmente para adicionar mais recursos e corrigir bugs. Este artigo mostra-lhe como atualizar o pacote de conteúdos. Ele também informa como adiar a atualização e ver as atualizações feitas ao longo do tempo.

## <a name="get-updates-to-the-content-pack"></a>Obter atualizações para o pacote de conteúdos

### <a name="get-the-updated-content-pack"></a>Obter o pacote de conteúdos atualizado
Se não tiver feito todas as alterações à sua cópia do pacote de conteúdos, ele atualiza automaticamente. Quando o pacote de conteúdos é alterada, receberá uma notificação no Power BI e uma notificação por e-mail. Pode optar por obter o pacote de conteúdos atualizado a sua necessidade. 

### <a name="postpone-the-update"></a>Adiar a atualização
A melhor prática é importar o pacote de conteúdos para uma [área de trabalho personalizada](https://youtu.be/26zyOtyHPJM?t=1m57s). Agora pode editar os relatórios.
Como mencionado anteriormente, se o pacote de conteúdos for alterado, verá uma notificação no Power BI. Pode optar por obter o pacote de conteúdos mais tarde. 

## <a name="coming-soon"></a>Brevemente
   
O pacote de conteúdos do Azure Backup é atualizado para suportar cargas de trabalho mais. Cargas de trabalho incluem o SQL Database do Azure para cópia de segurança de VM de IaaS e System Center Data Protection Manager. Esse suporte adiciona o suporte atual para o Azure backup e de cópias de segurança de VM do Azure. Esse suporte significa que pode ver e analisar todos os seus dados de cópia de segurança num único local central. [Relatórios também a podem ser personalizados](https://youtu.be/26zyOtyHPJM) de acordo com as necessidades da sua organização.

Os relatórios pré-configurados que vêm com o pacote de conteúdos do Azure Backup estão mudando. As mudanças façam os relatórios mais significativo em cargas de trabalho. Está disponível uma espiada no próximo conjunto de relatórios.
