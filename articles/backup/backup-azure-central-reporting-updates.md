# <a name="update-the-azure-backup-central-reporting-content-pack"></a>Atualizar o pacote de conteúdo de relatórios centrais do backup do Azure 

O [pacote de conteúdo do backup do Azure](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) pode ser usado para exibir relatórios sobre o backup central. O pacote de conteúdo é atualizado regularmente para adicionar mais recursos e corrigir bugs. Este artigo mostra como atualizar o pacote de conteúdo. Ele também informa como adiar a atualização e ver as atualizações feitas ao longo do tempo.

## <a name="get-updates-to-the-content-pack"></a>Obter atualizações para o pacote de conteúdo

### <a name="get-the-updated-content-pack"></a>Obter o pacote de conteúdo atualizado
Se você não tiver feito alterações na sua cópia do pacote de conteúdo, ele será atualizado automaticamente. Quando o pacote de conteúdo for alterado, você receberá uma notificação em Power BI e uma notificação por email. Você pode optar por obter o pacote de conteúdo atualizado à sua conveniência. 

### <a name="postpone-the-update"></a>Adiar a atualização
A prática recomendada é importar o pacote de conteúdo para um [espaço de trabalho personalizado](https://youtu.be/26zyOtyHPJM?t=1m57s). Agora você pode editar os relatórios.
Conforme mencionado anteriormente, se o pacote de conteúdo for alterado, você verá uma notificação em Power BI. Você pode optar por obter o pacote de conteúdo mais tarde. 

## <a name="coming-soon"></a>Brevemente
   
O pacote de conteúdo do backup do Azure é atualizado para dar suporte a mais cargas de trabalho. As cargas de trabalho incluem o banco de dados SQL do Azure para backup de VM IaaS e o System Center Data Protection Manager. Esse suporte adiciona ao suporte atual para backup do Azure e backups de VM do Azure. Esse suporte significa que você pode exibir e analisar todos os seus dados de backup em um local central. Os [relatórios também podem ser personalizados](https://youtu.be/26zyOtyHPJM) para atender às necessidades da sua organização.

Os relatórios pré-configurados fornecidos com o pacote de conteúdo do backup do Azure estão sendo alterados. As alterações tornam os relatórios mais significativos entre as cargas de trabalho. Uma espiada no próximo conjunto de relatórios está disponível aqui.
