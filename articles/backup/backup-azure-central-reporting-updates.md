# <a name="updating-azure-backup-central-reporting-content-pack"></a>A atualizar o Azure Backup Central relatórios pacote de conteúdos 

[Pacote de conteúdos de cópia de segurança do Azure](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) pode ser utilizado para ver relatórios sobre a cópia de segurança centralmente. O pacote de conteúdos é atualizado regularmente para adicionar mais recursos e para o bug correções. Este artigo orienta-o pelos passos para atualizar o pacote de conteúdos, para atrasar a atualização e as atualizações que são feitas ao longo do tempo.

## <a name="how-to-get-updates-to-the-content-pack"></a>Como obter as atualizações para o pacote de conteúdos

### <a name="to-get-the-updated-content-pack"></a>Para obter o pacote de conteúdos atualizado
Sua cópia do pacote de conteúdos será atualizada automaticamente se não tiver feito todas as alterações ao mesmo. No caso do pacote de conteúdos foi alterado, obterá uma notificação no Power BI e uma notificação por e-mail sobre o mesmo. Pode optar por obter o pacote de conteúdos atualizado de acordo com a sua comodidade. 

### <a name="to-delay-the-update"></a>Atrasar a atualização
A melhor prática é importar o pacote de conteúdos para uma [área de trabalho personalizada](https://youtu.be/26zyOtyHPJM?t=1m57s). Agora tem a capacidade de editar os relatórios.
Conforme mencionado acima, se o pacote de conteúdos for alterado, verá uma notificação no Power BI. Pode optar por obter o pacote de conteúdos mais tarde. 

## <a name="coming-soon"></a>Brevemente
   
Pacote de conteúdos de cópia de segurança do Azure está a ser atualizada para suportar mais cargas de trabalho, como SQL na cópia de segurança de VM de IaaS e SC DPM, além do suporte atual para o MAB e cópia de segurança do Azure VM. Isso significa que em breve poderá ver e analisar dados de todos os seus dados de cópia de segurança num único local central. O [relatórios também podem ser personalizados](https://youtu.be/26zyOtyHPJM) de acordo com as necessidades da sua organização.

Para fazer com que os relatórios mais significativo em cargas de trabalho, está a ser alterado o conjunto de relatórios pré-configurados com o pacote de conteúdos de cópia de segurança do Azure. Uma olhadela para o próximo conjunto de relatórios está disponível aqui:
