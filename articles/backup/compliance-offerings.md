# <a name="azure-backup-compliance-offerings"></a>Ofertas de conformidade com backup Azure

Para ajudar as organizações a cumprirem os requisitos nacionais, regionais e específicos da indústria que regem a recolha e utilização de dados individuais, o Microsoft Azure & Azure Backup oferecem um conjunto abrangente de certificações e atestados.

Pode encontrar abaixo as ofertas de conformidade para a Azure Backup para garantir que o seu serviço está regulado ao utilizar o serviço de Backup Azure.

## <a name="global"></a>Global

* [CSA-STAR-Atestado](/microsoft-365/compliance/offering-csa-star-attestation)
* [Certificação CSA-Estrela](/microsoft-365/compliance/offering-csa-star-certification)
* [CSA-STAR-Auto-avaliação](/microsoft-365/compliance/offering-csa-star-self-assessment)
* [ISO 20000-1:2011](/microsoft-365/compliance/offering-iso-20000-1-2011)
* [ISO 22301](/compliance/regulatory/offering-iso-22301)
* [ISO 27001](/microsoft-365/compliance/offering-iso-27017)
* [ISO 27017](/microsoft-365/compliance/offering-iso-27017)
* [ISO 27018](/microsoft-365/compliance/offering-iso-27018)
* [ISO 9001](/microsoft-365/compliance/offering-iso-9001)
* [SOC 1,2,3](/microsoft-365/compliance/offering-soc)

## <a name="us-government"></a>Governo dos Estados Unidos

* [DoD DISA L2, L4, L5](/microsoft-365/compliance/offering-dod-disa-l2-l4-l5)
