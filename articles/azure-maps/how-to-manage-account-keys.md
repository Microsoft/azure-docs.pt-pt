# <a name="manage-your-azure-maps-account"></a>Gerencie a sua conta Azure Maps

Pode gerir a sua conta Azure Maps através do portal Azure. Depois de ter uma conta, pode implementar as APIs no seu website ou aplicação móvel.

Se não tiver uma subscrição do Azure, crie uma [conta gratuita](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) antes de começar.

## <a name="create-a-new-account"></a>Criar uma nova conta

1. Inicie sessão no [Portal do Azure](https://portal.azure.com).

2. Selecione **Criar um recurso** no canto superior esquerdo do portal Azure.

3. Procure e selecione **Mapas**. Em seguida, selecione **Criar**.

4. Insira a informação para a sua nova conta.

[![Insira a informação da conta Azure Maps no portal Azure](./media/how-to-manage-account-keys/new-account-portal.png)](./media/how-to-manage-account-keys/new-account-portal.png#lightbox)

## <a name="delete-an-account"></a>Eliminar uma conta

Pode apagar uma conta do portal Azure. Navegue para a página de visão geral da conta e selecione **Eliminar**.

[![Apagar a sua conta Azure Maps no portal Azure](./media/how-to-manage-account-keys/account-delete-portal.png)](./media/how-to-manage-account-keys/account-delete-portal.png#lightbox)

Em seguida, vê uma página de confirmação. Pode confirmar a eliminação da sua conta digitando o seu nome.
