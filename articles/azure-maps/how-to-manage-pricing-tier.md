# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Gerir o nível de preços da sua conta Azure Maps

Pode gerir o nível de preços da sua conta Azure Maps através do portal Azure. Também pode ver ou alterar o nível de preços da sua conta depois de criar uma [conta](https://azure.microsoft.com/free/?WT.mc_id=A261C142F).

Obtenha mais informações sobre [a escolha do nível de preços certo no Azure Maps.](./choose-pricing-tier.md)

## <a name="view-your-pricing-tier"></a>Veja o seu nível de preços

Para ver o seu nível de preços escolhido, navegue para a opção **Priceing Tier** no menu de definições.

[![Ver nível ](./media/how-to-manage-pricing-tier/view-pricing-tier.png) de preços escolhidos](./media/how-to-manage-pricing-tier/view-pricing-tier.png#lightbox)

## <a name="change-a-pricing-tier"></a>Alterar um nível de preços

Depois de criar a sua conta Azure Maps, pode atualizar ou desvalorizar o nível de preços da sua conta Azure Maps. Para atualizar ou downgrade, navegue para a opção **Priceing Tier** no menu de definições. Selecione o nível de preços que não está realçado. Selecione o botão **Guardar** para guardar a opção de nível de preços escolhido.

[![Alterar um nível ](./media/how-to-manage-pricing-tier/change-pricing-tier.png) de preços](./media/how-to-manage-pricing-tier/change-pricing-tier.png#lightbox)

> [!NOTE]
> Não tem de gerar novas chaves de subscrição ou ID do cliente (para autenticação Azure AD) se atualizar ou desvalorizar o nível de preços da sua conta Azure Maps.
