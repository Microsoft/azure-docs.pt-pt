# <a name="provide-data-feedback-to-azure-maps"></a>Fornecer feedback de dados ao Azure Maps

O Azure Maps está disponível desde maio de 2018. O Azure Maps tem vindo a fornecer dados de mapas frescos, APIs REST fáceis de usar e SDKs poderosos para apoiar os nossos clientes empresariais com diferentes tipos de casos de uso de negócios. O mundo real está a mudar a cada segundo, e é crucial para nós fornecer uma representação digital factual aos nossos clientes. Os nossos clientes que planeiam abrir ou fechar instalações precisam dos nossos mapas para atualizar rapidamente. Assim, podem planear eficientemente a entrega, manutenção ou atendimento ao cliente nas instalações certas. Criámos o site de feedback de dados do Azure Maps para capacitar os nossos clientes a fornecerfeedback direto de dados. O feedback dos dados dos clientes vai diretamente para os nossos fornecedores de dados e os seus editores de mapas. Eles podem avaliar e incorporar rapidamente feedback nos nossos produtos de mapeamento.  

O site de feedback de [dados do Azure Maps](https://feedback.azuremaps.com) fornece uma forma fácil para os nossos clientes fornecerem feedback de dados de mapas, especialmente em pontos de interesse de negócio e endereços residenciais. Este artigo guia-o sobre como fornecer diferentes tipos de feedback através do site de feedback do Azure Maps.

## <a name="add-a-business-place-or-a-residential-address"></a>Adicione um local de negócios ou um endereço residencial 

Pode querer fornecer feedback sobre um ponto de interesse em falta ou um endereço residencial. Há duas maneiras de fazê-lo. Abra o site de feedback de dados do Mapa Azure, procure as coordenadas do local em falta e clique em "Adicionar um lugar"

  ![procurar localização em falta](./media/how-to-use-feedback-tool/search-poi.png)

Ou pode interagir com o mapa. Clique no local para deixar cair um pino na coordenada e clique em "Adicionar um lugar".

  ![adicionar pin](./media/how-to-use-feedback-tool/add-poi.png)

Ao clicar, será direcionado para um formulário para fornecer os detalhes correspondentes para o local.

  ![adicionar um lugar](./media/how-to-use-feedback-tool/add-a-place.png)

## <a name="fix-a-business-place-or-a-residential-address"></a>Fixar um local de negócios ou um endereço residencial 

O site de feedback também permite que você procure e localize um local de negócios ou um endereço. Pode fornecer feedback para corrigir o endereço ou a localização do pino, se não estiverem corretos. Para fornecer feedback para corrigir o endereço, utilize a barra de pesquisa para procurar um local de negócios ou morada residencial. Clique na localização do seu interesse a partir da lista de resultados. Clique em "Fix this place".

  ![local de pesquisa para corrigir](./media/how-to-use-feedback-tool/fix-place.png)

Para fornecer feedback para corrigir o endereço, preencha o formulário "Fixar um lugar" e, em seguida, clique no botão "submeter".

  ![forma de correção](./media/how-to-use-feedback-tool/fix-form.png)

Se a localização do pino do local estiver errada, verifique a caixa de verificação no formulário "Fixe um local" que diga "A localização do pino está incorreta". Mova o pino para a localização correta e, em seguida, clique no botão "submeter".

  ![mover localização pin](./media/how-to-use-feedback-tool/move-pin.png)

## <a name="add-a-comment"></a>Adicione um comentário 

Além de permitir que procure uma localização, a ferramenta de feedback também permite adicionar um comentário de texto de formulário gratuito para detalhes relacionados com a localização. Para adicionar um comentário, procure a localização ou clique na localização. Clique em "Adicionar um comentário", escreva um comentário e clique em "Submeter".

  ![adicionar comentário](./media/how-to-use-feedback-tool/add-comment.png)

## <a name="track-status"></a>Acompanhar o estado 

Também pode acompanhar o estado do seu pedido verificando a caixa "Quero rastrear o estado" e fornecendo o seu e-mail enquanto faz um pedido. Receberá um link de rastreio no e-mail que fornece um estado atualizado do seu pedido. 

  ![estado de feedback](./media/how-to-use-feedback-tool/feedback-status.png)
