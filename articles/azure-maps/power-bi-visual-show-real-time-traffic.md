# <a name="show-real-time-traffic"></a>Mostrar tráfego em tempo real

A camada de tráfego sobrepõe os dados de tráfego em tempo real no topo do mapa. Para ativar esta função, mova o deslizador da **camada de tráfego** no painel de **formato** para a posição **On.** Isto irá sobrepor os dados do fluxo de tráfego como estradas codificadas por cores.

> [!div class="mx-imgBorder"]
> ![Um mapa que mostra dados de tráfego em tempo real](media/power-bi-visual/traffic-layer.png)

As seguintes definições estão disponíveis na secção **camada de tráfego.**

| Definições         | Descrição    |
|-----------------|----------------|
| Mostrar incidentes  | Especifica se os incidentes de trânsito, tais como o encerramento de estradas e a construção, devem ser exibidos no mapa. |
| Controlo de tráfego | Adiciona um botão ao mapa que permite aos leitores reportarem ligar ou desligar a camada de tráfego.  |
