# <a name="azure-maps-community---open-source-projects"></a>Comunidade Azure Maps - Projetos de código aberto

Estas iniciativas de código aberto e orientadas para a comunidade são criadas e mantidas pela equipa do Azure Maps. Não fazem parte das ofertas padrão de produtos ou serviços.

O seguinte lista alguns dos mais populares projetos e amostras de código aberto Azure Maps.

| Nome do Projeto | Descrição |

## <a name="supportability-of-open-source-projects"></a>Apoio a projetos de código aberto

As seguintes declarações aplicam-se em todos os mapas Azure criados e mantidos projetos e amostras de código aberto:

- Os projetos e amostras de código aberto do Azure Maps são criados pela Microsoft e pela comunidade.
- Os projetos e amostras de código aberto do Azure Maps são mantidos pela Microsoft e pela comunidade.
- Azure Maps projetos e amostras de código aberto usam técnicas suportadas e recomendadas.
- Os projetos e amostras de código aberto da Azure Maps são uma iniciativa comunitária – pessoas que trabalham na iniciativa em benefício dos outros, e têm o seu trabalho normal de dia também.
- Os projetos e amostras de código aberto da Azure Maps NÃO são um produto, pelo que não é suportado pelo Premier Support ou por outros canais oficiais de apoio.
- Os projetos e amostras de código aberto do Azure Maps são suportados de forma semelhante a outros projetos de código aberto feitos pela Microsoft com o apoio da comunidade.
