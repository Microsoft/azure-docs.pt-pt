# <a name="azure-maps-community---open-source-projects"></a>Comunidade Azure Maps - Projetos de código aberto

Estas iniciativas de código aberto e orientadas para a comunidade são criadas e mantidas pela equipa do Azure Maps. Não fazem parte das ofertas padrão de produtos ou serviços.

O seguinte lista alguns dos mais populares projetos e amostras de código aberto do Azure Maps.

| Nome do Projeto | Descrição |
|-|-|
| [Amostras de código de mapas azure](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Uma coleção de amostras de código para a utilização do Azure Maps em aplicações baseadas na Web. |
| [Amostras de código de nuvem de mapas azure](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Uma coleção de amostras de código para a utilização de Mapas Azure através da Nuvem governamental azure. |
| [Azure Maps & amostras de diretório ativo Azure](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Uma coleção de amostras que mostram como usar o Diretório Ativo Azure com o Azure Maps. | 
| [Implementar a análise espacial IoT com o Azure Maps](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | Rastrear e capturar eventos relevantes que ocorrem no espaço e no tempo é um cenário ioT comum. |
| [Amostras de caderno sacana do Azure Maps](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Uma coleção de amostras de pitão utilizando os serviços De REPOUSO Azure Maps. |
| [Quadro Bot - Habilidade de Ponto de Interesse](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | A Habilidade de Ponto de Interesse fornece capacidades relacionadas com OPO a um Assistente Virtual usando o Azure Maps com o Serviço Bot Azure e a Bot Framework. |

Uma lista mais longa que inclui projetos criados pela comunidade de código aberto está disponível [aqui](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)

## <a name="supportability-of-open-source-projects"></a>Apoio à capacidade de projetos de código aberto

As seguintes declarações aplicam-se em todos os mapas azure criados e mantidos projetos e amostras de código aberto:

- Os projetos e amostras de código aberto do Azure Maps são criados pela Microsoft e pela comunidade.
- Os projetos e amostras de código aberto do Azure Maps são mantidos pela Microsoft e pela comunidade.
- Os projetos e amostras de código aberto do Azure Maps utilizam técnicas apoiadas e recomendadas.
- Os projetos e amostras de código aberto da Azure Maps são uma iniciativa comunitária – pessoas que trabalham na iniciativa em benefício dos outros, e que também têm o seu trabalho normal.
- Os projetos e amostras de código aberto do Azure Maps não são um produto, pelo que não é suportado pelo Premier Support ou por outros canais oficiais de apoio.
- Os projetos e amostras de código aberto do Azure Maps são suportados de forma semelhante a outros projetos de código aberto realizados pela Microsoft com o apoio da comunidade pela comunidade.
