## <a name="usage-reporting-infrastructure-for-cloud-service-providers"></a>Utilização de relatórios de infraestrutura para fornecedores de serviços Cloud

Pilha do Azure inclui a infraestrutura necessária para controlar utilização à que ocorre e reencaminha-lo para o Azure. No Azure, Azure comércio processa os dados de utilização e encargos de utilização para as subscrições do Azure adequadas, da mesma forma como a utilização que decorre na nuvem do Azure global.

Deve estar ciente de que determinados conceitos são consistentes entre o global Azure e pilha do Azure. Pilha do Azure tem subscrições locais que satisfazer uma função semelhante a uma subscrição do Azure. Subscrições locais localmente só são válidas. Subscrições locais estão mapeadas para as subscrições do Azure quando a utilização é reencaminhada para o Azure.

Pilha do Azure tem medidores utilização local. Utilização local está mapeada para medidores utilizados em comércio do Azure. No entanto, os IDs de medição são diferentes. Existem mais medidores disponíveis localmente do que Microsoft utiliza para faturação.

Existem algumas diferenças entre a forma como os serviços têm um preço na pilha do Azure e no Azure. Por exemplo, na pilha do Azure, os encargos para as VMs só baseia-se no vcore/horas, com a mesma velocidade para todas as séries VM, ao contrário do Azure. O motivo é que, no global Azure, os preços diferentes refletem um hardware diferente. Na pilha do Azure, o cliente fornece o hardware, pelo que não existe nenhum motivo para cobram taxas diferentes para diferentes classes VM.

Pode descobrir sobre medidores a pilha do Azure utilizados em comércio e os seus preços no Centro de parceiros, da mesma forma, tal como faria para serviços do Azure:

1. No Centro de parceiros, vá para o **menu do Dashboard** > **preços e ofertas**.
2. Em **baseada na utilização dos serviços**, selecione **atual**.
3. Abra o **do Azure, na lista de preços de Global CSP** folha de cálculo.
4. Filtrar **região = pilha do Azure**.

## <a name="terms-used-for-billing-and-usage"></a>Termos utilizados para utilização e faturação

Conceitos e termos seguintes são utilizados para utilização e faturação na pilha do Azure:

| Termo | Definição |
| --- | --- |
| Parceiro CSP direto | Um parceiro direta do fornecedor de solução em nuvem (CSP) recebe uma fatura diretamente diretamente a partir da Microsoft para utilização do Azure e pilha do Azure e os clientes de faturas. |
| Indireta CSP | Revendedores indiretas funcionam com um fornecedor de indireto (também conhecido como um distribuidor). Os revendedores recruit clientes finais; o fornecedor indireto contém a relação de faturação com a Microsoft, gere o cliente de faturação e fornece serviços adicionais, como o suporte técnico. |
| Cliente do fim | Os clientes de fim se a empresas e agências governamentais que possui as aplicações e outras cargas de trabalho que são executados na pilha do Azure. |
