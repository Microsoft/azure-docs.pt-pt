# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Microsoft Azure novo pedido de pedido de suporte enviar diretrizes de utilidade

Quando [cria um pedido de suporte,](https://portal.azure.com/#create/Microsoft.Support)a opção de upload de ficheiros pode ajudá-lo a carregar facilmente ficheiros de diagnóstico ou quaisquer outros ficheiros que considere relevantes para o incidente.  

> [!NOTE]
> Para proteger a sua privacidade, não inclua nenhuma informação pessoal no seu upload.
>
>

Siga estas diretrizes quando utilizar a opção de upload de ficheiros:

* Todos os ficheiros devem ter uma extensão de nome de ficheiro, como *.docx* ou *.xlsx*. Consulte a tabela para obter as extensões de nome de ficheiros permitidas.
* O nome do ficheiro não deve ter mais de 110 caracteres.
* Não pode enviar mais do que um ficheiro.
* Os ficheiros não podem ser maiores que 4 MB.
