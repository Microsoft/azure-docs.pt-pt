# <a name="supported-devices"></a>Dispositivos suportados

O [portal Azure](https://portal.azure.com) é uma consola baseada na Web e funciona no navegador de todos os desktops modernos e dispositivos tablet. Para utilizar o portal, tem de ter o JavaScript ativado no seu browser.

Se precisar de gerir os recursos do Azure a partir de um dispositivo móvel, experimente a [aplicação móvel Azure.](https://azure.microsoft.com/features/azure-portal/mobile-app/) Está disponível para iOS e Android.

## <a name="recommended-browsers"></a>Navegadores recomendados

Recomendamos que utilize o browser mais atualizado compatível com o seu sistema operativo. Os seguintes navegadores são suportados:

* Microsoft Edge (versão mais recente)
* Internet Explorer 11 (fim de suporte 17 de agosto de 2021 - consulte o nosso [blog post](https://techcommunity.microsoft.com/t5/microsoft-365-blog/microsoft-365-apps-say-farewell-to-internet-explorer-11-and/ba-p/1591666) para mais informações)
* Safari (versão mais recente, apenas Mac)
* Chrome (versão mais recente)
* Firefox (versão mais recente)
