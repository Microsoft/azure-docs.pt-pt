# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Abra um bilhete de apoio para Azure HPC Cache

Utilize o portal Azure para abrir um bilhete de apoio. Navegue para a sua instância de cache e, em seguida, clique no novo link **de pedido de suporte** que aparece na parte inferior da barra lateral.

Para abrir um bilhete quando não tiver uma cache ativa, utilize a página de suporte principal da Ajuda + a partir do portal Azure. Abra o menu do portal a partir do controlo na parte superior esquerda do ecrã, em seguida, desloque-se para a parte inferior e clique em **Suporte Help +**.

Escolha **novo pedido de suporte** e selecione **Técnico** para ajuda específica da Cache Azure HPC.

Selecione a sua subscrição da lista.

Para encontrar o serviço Cache Azure HPC, clique no botão **Todos os serviços** e procure por HPC.

![Screenshot do pedido de suporte - Separador básico, parcialmente preenchido como descrito](media/hpc-cache-support-request.png)

Preencha o resto dos campos com as suas informações e preferências e, em seguida, envie o bilhete quando estiver pronto.

Depois de submeter o pedido, receberá um e-mail de confirmação com um número de bilhete. Um membro do pessoal de apoio irá contactá-lo sobre o pedido.
