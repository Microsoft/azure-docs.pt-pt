# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Utilizar o serviço do Gestor de Dispositivos do StorSimple para resolver problemas na Matriz Virtual do StorSimple
## <a name="overview"></a>Descrição geral

O StorSimple Device Manager fornece um diagnóstico e resolve a definição de **problemas** dentro da lâmina de resumo do serviço, o que destaca alguns dos problemas que ocorrem com um conjunto virtual e como resolvê-los. Este tutorial introduz a capacidade de resolução de problemas auto-servida fornecida dentro do serviço StorSimple Device Manager.

![Diagnosticar e resolver problemas](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnosticar e resolver problemas

Pode ver alguns dos problemas comuns com o StorSimple Virtual Array e rever as soluções para esses problemas a partir da sua lâmina de resumo do serviço StorSimple Device Manager.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Para diagnosticar um problema com a sua matriz virtual

1. Clique em **Diagnosticar e resolver problemas** de regulação no painel esquerdo da sua lâmina de resumo do serviço StorSimple Device Manager para ver uma lista de problemas comuns.

2. **Expanda** o sintoma do problema que encontra para rever **as medidas recomendadas** para ajudá-lo a resolver o problema. Se desejar, também pode rever os links de documentação detalhados previstos para posterior referência.
   
    ![Problemas de resolução de um problema](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Se não conseguir encontrar uma referência ao seu problema ou resolvê-lo, contacte o Microsoft Support para obter mais assistência.
