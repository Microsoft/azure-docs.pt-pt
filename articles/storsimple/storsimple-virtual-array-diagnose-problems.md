# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Utilizar o serviço StorSimple Device Manager para resolver problemas relacionados com a matriz Virtual StorSimple
## <a name="overview"></a>Descrição geral

O StorSimple Device Manager fornece uma **diagnosticar e resolver problemas** definição dentro do painel de resumo do serviço, que destaca alguns dos problemas que normalmente que podem ocorrer com a sua matriz virtual e como resolvê-los. Este tutorial apresenta a funcionalidade resolução de problemas de Self-Service que fornecido no serviço StorSimple Device Manager.

![Diagnosticar e resolver problemas](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnosticar e resolver problemas

Pode ver alguns dos problemas comuns com o StorSimple Virtual Array e reveja as soluções para esses problemas certos de painel de resumo do serviço StorSimple Device Manager.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Para diagnosticar um problema com a sua matriz virtual

1. Clique em **diagnosticar e resolver problemas** definição no painel esquerdo do painel de resumo do serviço StorSimple Device Manager para ver uma lista dos problemas comuns.

2. **Expanda** sobre o sintoma do problema estão com problemas para rever **etapas recomendadas** para ajudá-lo a resolver o problema. Se desejar, poderá também ver as ligações de documentação detalhada fornecidas para referência ainda mais.
   
    ![Solucionar um problema](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Se não conseguir localizar uma referência para o seu problema ou resolvê-lo, entrar em contacto ao Support da Microsoft para obter mais assistência.
