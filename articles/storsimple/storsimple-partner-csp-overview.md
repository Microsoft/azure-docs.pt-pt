# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>O que é o StorSimple para o programa de fornecedores de soluções Cloud?


## <a name="overview"></a>Descrição geral

O Microsoft Azure StorSimple é uma abordagem única para o armazenamento de enterprise com capacidades de armazenamento verdadeiramente híbrido na cloud. Ele capacita os clientes para tirar partido do armazenamento na cloud económico para dados inativos, mantendo os seus dados críticos no local para os mais altos níveis de desempenho. 

Matriz Virtual StorSimple para o fornecedor de soluções de Cloud (CSP), os parceiros aproveitar essa oportunidade. Os parceiros o proprietário o ciclo de vida do cliente de ponto a ponto com direto de aprovisionamento, faturação e suporte dos serviços de cloud da Microsoft. Em resumo, os parceiros agora podem transacionar o StorSimple, juntamente com os serviços aos clientes.

Para obter mais informações sobre o StorSimple para CSP, visite o [descrição geral do Azure CSP](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview) página.

Para obter mais informações sobre faturação, preços, incentivos e obter suporte no CSP, aceda a [StorSimple no CSP: FAQ SOBRE](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Implementar e gerir o StorSimple para CSP

StorSimple para CSP está disponível como um serviço baseado em uso em todos os mercados em que o StorSimple está disponível hoje mesmo. StorSimple para CSP utiliza o portal do Azure e o serviço StorSimple Device Manager. Um parceiro CSP, pode criar um StorSimple Device Manager para gerir as matrizes virtuais do StorSimple, partilhas, volumes e cópias de segurança. Pode administrar todas as matrizes virtuais registadas ao seu serviço StorSimple Device Manager através do portal do Azure.

Para obter mais informações, aceda a [implementar e gerir a sua matriz Virtual StorSimple para CSP](storsimple-partner-csp-deploy.md).
