# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>O que é o Programa StorSimple para fornecedores de soluções cloud?


## <a name="overview"></a>Descrição geral

O Microsoft Azure StorSimple é uma abordagem única ao armazenamento da empresa com verdadeiras capacidades híbridas de armazenamento em nuvem. Capacita os clientes a aproveitarem o armazenamento económico em nuvem para os dados inativos, mantendo os seus dados críticos de missão no local para os mais altos níveis de desempenho. 

O StorSimple Virtual Array for Cloud Solutions Provider (CSP) permite aos parceiros capitalizar esta oportunidade. Os parceiros são donos do ciclo de vida do cliente de ponta a ponta com fornecimento direto, faturação e suporte aos serviços na nuvem da Microsoft. Em suma, os parceiros podem agora transacionar o StorSimple juntamente com os seus serviços aos clientes.

Para mais informações sobre o StorSimple para CSP, visite a página geral do [Azure CSP.](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview)

Para obter mais informações sobre faturação, preços, incentivos e obtenção de apoio em CSP, vá ao [StorSimple em CSP: FAQ](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Implementar e gerir o StorSimple para CSP

O StorSimple for CSP está disponível como um serviço baseado na utilização em todos os mercados onde o StorSimple está disponível hoje em dia. O StorSimple para CSP utiliza o portal Azure e o serviço StorSimple Device Manager. Um parceiro CSP pode criar um StorSimple Device Manager para gerir Matrizes Virtuais StorSimple, ações, volumes e backups. Pode administrar todas as matrizes virtuais registadas no seu serviço StorSimple Device Manager através do portal Azure.

Para mais informações, vá a [Deploy e gere o seu StorSimple Virtual Array para CSP](storsimple-partner-csp-deploy.md).
