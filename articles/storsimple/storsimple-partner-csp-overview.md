# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>O que é StorSimple para o Programa de Fornecedores de Soluções em Nuvem?


## <a name="overview"></a>Descrição geral

O Microsoft Azure StorSimple é uma abordagem única ao armazenamento empresarial com verdadeiras capacidades de armazenamento em nuvem híbrida. Capacita os clientes a aproveitarem o armazenamento económico em nuvem para os dados inativos, mantendo os seus dados críticos da missão no local para os mais altos níveis de desempenho. 

O StorSimple Virtual Array for Cloud Solutions Provider (CSP) permite aos parceiros capitalizar esta oportunidade. Os parceiros possuem o ciclo de vida do cliente de ponta a ponta com fornecimento direto, faturação e suporte aos serviços na nuvem da Microsoft. Em suma, os parceiros podem agora transacionar o StorSimple juntamente com os seus serviços aos clientes.

Para mais informações sobre storSimple para CSP, visite a página geral do [Azure CSP.](https://docs.microsoft.com/azure/cloud-solution-provider/overview/azure-csp-overview)

Para obter mais informações sobre faturação, preços, incentivos e obter apoio na CSP, vá ao [StorSimple em CSP: FAQ](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Implementar e gerir o StorSimple para a CSP

O StorSimple para CSP está disponível como um serviço baseado na utilização em todos os mercados onde o StorSimple está disponível hoje. StorSimple para CSP utiliza o portal Azure e o serviço StorSimple Device Manager. Um parceiro CSP pode criar um Gestor de Dispositivos StorSimple para gerir as Matrizes Virtuais StorSimple, partilhas, volumes e backups. Pode administrar todas as matrizes virtuais registadas no seu serviço StorSimple Device Manager através do portal Azure.

Para obter mais informações, aceda ao [Deploy e gere o seu StorSimple Virtual Array para CSP.](storsimple-partner-csp-deploy.md)
