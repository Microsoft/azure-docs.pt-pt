# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion-preview"></a>Alterar a vista de ecrã inteiro para uma sessão de vm: Bastion do Azure (pré-visualização)

Este artigo ajuda-o a alterar a vista de máquina virtual para tela inteira e volta no seu browser. Antes de trabalhar com uma VM, certifique-se de que seguiu os passos para [criar um anfitrião de bastião](bastion-create-host-portal.md). Em seguida, ligar à VM que pretende trabalhar com através de um [RDP](bastion-connect-vm-rdp.md) ou [SSH](bastion-connect-vm-ssh.md).

> [!IMPORTANT]
> Esta pré-visualização pública é disponibilizada sem um contrato de nível de serviço e não deve ser utilizada para cargas de trabalho de produção. Algumas funcionalidades podem não ser suportadas, podem ter capacidades restringidas ou podem não estar disponíveis em todas as localizações do Azure. Veja os [Termos Suplementares de Utilização para Pré-visualizações do Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/) para obter mais informações.
>

## <a name="launch-the-clipboard-tool"></a>Iniciar a ferramenta de área de transferência

Durante a sessão remota, inicie a paleta de ferramenta de acesso de área de transferência de Bastion selecionando duas setas que localizados no centro da esquerda da sessão.

![ferramentas](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>Selecione o ecrã inteiro

Selecione o **ecrã inteiro** botão para mudar a sessão para uma experiência de ecrã inteiro. Depois de mudar, a sessão irá ser inicializado para tela inteira.

![ecrã inteiro](./media/bastion-vm-manage/full-screen.png)
