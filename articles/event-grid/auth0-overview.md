# <a name="auth0-partner-topics"></a>Tópicos parceiros Auth0
![Logotipo Auth0](./media/auth0-overview/auth0-logo.png)

A Auth0, plataforma de identidade para construtores de aplicações, fornece aos desenvolvedores e empresas os blocos de construção de que precisam para garantir as suas aplicações.

O tópico parceiro Auth0 permite-lhe usar eventos que são emitidos pelo sistema da Auth0 para realizar uma série de tarefas. Envolva-se com os utilizadores de forma significativa após a autenticação ou automatizar as tarefas de segurança e infraestrutura.

A integração permite-lhe transmitir os seus eventos de log Auth0 com alta fiabilidade para o Azure. Lá, você pode consumir os eventos com os seus recursos Azure favoritos. Esta integração permite-lhe reagir a eventos, obter insights, monitorizar questões de segurança e interagir com outros poderosos oleodutos de dados.

Para organizações que usam Auth0 e Azure, esta integração permite-lhe integrar os dados de forma perfeita em toda a sua stack. 
 
## <a name="available-event-types"></a>Tipos de eventos disponíveis
A lista completa dos tipos de eventos Auth0 disponíveis e suas descrições está disponível [neste site.](https://auth0.com/docs/logs/references/log-event-type-codes)

## <a name="use-cases"></a>Casos de utilização

### <a name="engage-with-your-users"></a>Envolva-se com os seus utilizadores
Oferecer uma forte experiência de utilizador é fundamental para reduzir o impacto e manter os seus utilizadores. Oferece experiências de aplicação mais personalizadas utilizando eventos Auth0 com Funções Azure e Apps Azure Logic. 

### <a name="understand-user-behavior"></a>Compreender o comportamento do utilizador
Compreenda quando os utilizadores acedem ao seu produto, onde estão inscritos e quais os dispositivos que utilizam. Desenvolver uma compreensão das áreas de produtos que mais importam mantendo o registo destes sinais. Estes sinais ajudam-no a determinar:
- Que navegadores e dispositivos para suportar. 
- Quais os idiomas para localizar a sua aplicação. 
- Quando os teus tempos de tráfego são máximos. 

### <a name="manage-user-data"></a>Gerir dados de utilizadores
Manter e auditar as suas ações de utilizador é crucial para manter a segurança e seguir os regulamentos da indústria. A capacidade de editar, remover ou exportar dados dos utilizadores é cada vez mais importante para seguir leis de privacidade, como o Regulamento Geral de Proteção de Dados (RGPD) da União Europeia.

### <a name="secure-your-application"></a>Proteja a sua aplicação
Combinar procedimentos de monitorização de segurança e resposta a incidentes é importante quando protege um sistema distribuído. Por esta razão, é importante manter todos os dados no mesmo lugar e monitorizar toda a pilha.
