# <a name="release-notes-azure-event-grid-on-iot-edge"></a>Notas de lançamento: Azure Event Grid on IoT Edge

## <a name="100-preview1"></a>1.0.0-pré-visualização1

Lançamento inicial da Azure Event Grid no IoT Edge. Características incluídas:

* Criação de tópicos
* Criação de assinatura de eventos
* Filtros Avançados
* Criação de batches de saída
* Políticas de retíria
* Publicação de módulos
* Publique na WebHook como um destino
* Publique no IoT Edge Hub como destino
* Publique o serviço de cloud da Azure Event Grid como destino
* Estado persistente para metadados
* Integração do módulo de armazenamento blob

Etiquetas: `1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0-pré-visualização2

Pré-visualização 2 da Grelha de Eventos Azure em IoT Edge adicionado:

* Eventos configuráveis persistindo para o disco
* Métricas de tópicos
* Métricas de subscrição de eventos
* Publicar para Os Centros de Eventos como destino
* Publique para Service Bus Queues como destino
* Publique para Service Bus Topics como destino
* Publicar para storage queues como destino

Etiquetas: `1.0.0-preview2` , `1.0``latest`
