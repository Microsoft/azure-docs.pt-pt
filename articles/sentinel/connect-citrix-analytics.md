# <a name="connect-citrix-analytics-security"></a>Ligar a Citrix Analytics (Segurança) 

A integração da Citrix Analytics (Security) com o Azure Sentinel ajuda-o a exportar dados analisados para eventos de risco da Citrix Analytics (Security) para o Azure Sentinel. Pode criar dashboards personalizados, analisar dados de outras fontes, juntamente com os da Citrix Analytics (Security) e criar fluxos de trabalho personalizados usando Apps Lógicas para monitorizar e mitigar eventos de segurança. 



## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho Azure Sentinel.

- Direito à Citrix Security Analytics em Citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Ligue a Azure Sentinel à Citrix Analytics (Segurança)

1. No portal Azure, navegue nos conectores **Azure Sentinel**  >  **Data** e, em seguida, selecione o conector **Citrix Analytics (Segurança).**

2. Selecione **Abrir a página do conector**.

3. Siga as instruções na página **Citrix Analytics (Segurança).**
