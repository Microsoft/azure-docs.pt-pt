# <a name="connect-trend-micro-deep-security"></a>Conecte a tendência Micro Deep Security

O conector Trend Micro Deep Security permite-lhe ligar facilmente os seus registos de Segurança Profunda com o Azure Sentinel, para visualizar dashboards, criar alertas personalizados e melhorar a investigação. Isto dá-lhe mais informações sobre as redes e sistemas da sua organização e melhora as suas capacidades de operação de segurança.


## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho Azure Sentinel.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Ligue o Azure Sentinel à Trend Micro Deep Security

1. No portal Azure, navegue nos conectores **Azure Sentinel**  >  **Data** e, em seguida, selecione o conector **Trend Micro Deep Security.**

2. Selecione **Abrir a página do conector**.

3. Siga as instruções e os próximos passos.
