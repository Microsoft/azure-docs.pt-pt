# <a name="connect-trend-micro-deep-security"></a>Connect Trend Micro Deep Security

O conector Trend Micro Deep Security permite-lhe ligar facilmente os seus registos de Segurança Profunda com o Azure Sentinel, ver dashboards, criar alertas personalizados e melhorar a investigação. Isto dá-lhe mais informações sobre as redes e sistemas da sua organização, e melhora as suas capacidades de operação de segurança.


## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho Azure Sentinel.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Ligue o Sentinela Azure à Tendência Micro Segurança Profunda

1. No portal Azure, navegue para**os conectores** De Dados **Azure Sentinel** > e, em seguida, selecione o conector **Trend Micro Deep Security.**

2. Selecione página do **conector Aberto**.

3. Siga as instruções e os próximos passos.
