# <a name="about-your-application"></a>Acerca da sua aplicação

Este artigo mostra-lhe como obter informações sobre a sua aplicação IoT Central. Pode precisar:

- Esta informação se contactar o suporte.
- A subscrição Azure a sua aplicação utiliza para localizar informações de faturação no portal Azure.
- A identificação da aplicação é quando está a trabalhar com a API REST.
- A versão da aplicação para completar tarefas como adicionar um conector.

## <a name="get-information-about-your-application"></a>Obter informações sobre a sua aplicação

Para obter informações sobre a sua aplicação IoT Central:

1. Selecione o link **Ajuda** no menu superior.

1. Selecione **Sobre a sua aplicação.**

1. A **página sobre a sua aplicação** mostra informações sobre a sua aplicação:  

    :::image type="content" source="media/howto-get-app-info/about-your-app.png" alt-text="Sobre a sua imagem de aplicativo":::

    Utilize o botão **de informação Copy** para copiar as informações para a área de transferência.
