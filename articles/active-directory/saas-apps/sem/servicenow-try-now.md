# <a name="servicenow-and-azure-active-directory"></a>ServiceNow e Azure Active Directory

Acelere a adoção do ServiceNow em sua organização e aumente a adoção do usuário com nossa integração pré-criada. Simplifique seu gerenciamento de serviços de TI conectando o ServiceNow ao Azure Active Directory (Azure AD).

![Mostra o logotipo do aplicativo ServiceNow](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Experimente o ServiceNow com o Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>O que você pode fazer com o ServiceNow e o Azure AD?

- **Logon único fácil** -elimine vários nomes de usuário e senhas e simplifique o acesso permitindo que os usuários entrem no ServiceNow usando suas contas do Azure AD.
- **Automatizar o provisionamento de usuários** – Automatize a criação, a manutenção e a remoção de contas de usuário no ServiceNow quando os usuários ingressam, movem ou deixam sua organização.
- **Proteja, administre e monitore** o acesso seguro ao ServiceNow aplicando políticas de acesso condicional, administre o acesso por meio de revisões de acesso e monitore o acesso com logs de entrada.

## <a name="get-started"></a>Introdução

Configure o [provisionamento de usuário](https://aka.ms/ServiceNowProvisioning) e o [logon único](https://aka.ms/ServiceNowSSO) para o ServiceNow com nossos guias de configuração fáceis.
