# <a name="smartsheet-and-azure-active-directory"></a>SmartSheet e Azure Active Directory

Acelere a adoção de SmartSheet em sua organização e aumente a adoção do usuário com nossa integração predefinida. Permita que os usuários colaborem rapidamente conectando SmartSheet a Azure Active Directory (Azure AD).

![Mostra o logotipo do aplicativo SmartSheet](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Experimente o SmartSheet com o Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>O que você pode fazer com o SmartSheet e o Azure AD?

- **Logon único fácil** -elimine vários nomes de usuário e senhas e simplifique o acesso permitindo que os usuários entrem no SmartSheet usando suas contas do Azure AD.
- **Automatizar o provisionamento de usuários** – Automatize a criação, a manutenção e a remoção de contas de usuário no SmartSheet quando os usuários ingressam, movem ou deixam sua organização.
- **Proteja, controle e monitore** o acesso seguro ao SmartSheet aplicando políticas de acesso condicional, administre o acesso por meio de revisões de acesso e monitore o acesso com logs de entrada.

## <a name="get-started"></a>Introdução

Configure o [provisionamento de usuário](https://aka.ms/SmartsheetProvisioningDoc) e o [logon único](https://aka.ms/SmartsheetSSODoc) para o SmartSheet com nossos guias de configuração fáceis.
