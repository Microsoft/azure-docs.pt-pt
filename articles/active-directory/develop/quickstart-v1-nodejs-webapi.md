# <a name="quickstart-secure-a-web-api-with-azure-active-directory"></a>Início Rápido: proteger uma API Web com o Azure Active Directory

[!INCLUDE [active-directory-develop-applies-v1](../../../includes/active-directory-develop-applies-v1.md)]

A [plataforma de identidade da Microsoft](v2-overview.md) é uma evolução da plataforma de desenvolvedor do Azure Active Directory (Azure AD). Permite aos programadores criar aplicações que iniciam sessão em todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, como o Microsoft Graph, ou APIs criadas pelos programadores.

## <a name="next-steps"></a>Passos seguintes

Consulte o [exemplo da Web Azure Active Directory OIDC node. js](https://github.com/azureadquickstarts/appmodelv2-webapp-openidconnect-nodejs) , que demonstra como configurar a autenticação do OpenID Connect com o Azure ad v 2.0 em um aplicativo Web criado usando o Node. js com o Express. Ele foi projetado para ser executado em qualquer plataforma.
