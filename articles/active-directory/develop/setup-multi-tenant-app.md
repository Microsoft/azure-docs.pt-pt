Aqui está uma lista de tópicos recomendados para saber mais sobre candidaturas multi-inquilinos:

- Obter uma compreensão geral do [que significa ser uma aplicação multi-inquilino](https://docs.microsoft.com/azure/active-directory/develop/active-directory-dev-glossary#multi-tenant-application)
- Obtenha uma compreensão geral de [como configurar uma aplicação para ser multi-inquilino](https://docs.microsoft.com/azure/active-directory/develop/howto-convert-app-to-be-multi-tenant)

- Para mais profundidade, saiba [como é que uma aplicação multi-arrendatária é configurada e codificada de ponta a ponta](https://docs.microsoft.com/azure/active-directory/develop/active-directory-devhowto-multi-tenant-overview), incluindo como registar-se, utilizar o ponto final "comum", implementar o consentimento "utilizador" e "administrador", como implementar cenários multi-nível mais avançados
