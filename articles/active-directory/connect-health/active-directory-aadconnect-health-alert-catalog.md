# <a name="azure-active-directory-connect-health-alert-catalog"></a>Azure Active Directory Connect catálogo de alertas de estado de funcionamento 

Alertas de envio de serviço do Azure AD Connect Health indicam que a sua infraestrutura de identidade não está em bom estada. Este artigo inclui títulos de alertas, descrições e passos de remediação para cada alerta. <br />
Erro, aviso e Prewarning são três fases de alertas que são gerados a partir do serviço de integridade ligar. Recomendamos vivamente que tome medidas imediatas accionadas alertas. <br />
Alertas do Azure AD Connect Health obterem resolvidas numa condição de sucesso. Agentes de estado de funcionamento ligar do Azure AD detetar e comunicar as condições de êxito ao serviço periodicamente. Para alguns alertas, a supressão é baseados no tempo. Por outras palavras, se a condição de erro mesmo não é observada dentro de 72 horas da geração de alertas, o alerta é resolvido automaticamente.
