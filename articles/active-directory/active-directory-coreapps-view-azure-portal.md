# <a name="view-all-the-enterprise-apps-that-i-can-manage-in-azure-active-directory"></a>Ver todas as aplicações empresariais que pode gerir no Azure Active Directory
Pode gerir as aplicações da empresa no Azure Active Directory (Azure AD). Pode ver as aplicações que pode gerir, atribuir utilizadores ou grupos para uma aplicação, a manter as propriedades da aplicação, como o nome de aplicação/logótipo e mesmo desativar uma aplicação para que nenhum utilizador pode iniciar sessão no mesmo.

## <a name="how-do-i-view-all-my-apps"></a>Como posso ver as minhas aplicações?
1. Iniciar sessão para o [portal do Azure](https://portal.azure.com) com uma conta que seja um administrador global do diretório.
2. Selecione **todos os serviços**, introduza **do Azure Active Directory** na caixa de texto e, em seguida, selecione **Enter**.
3. No **Azure Active Directory -** ***directoryname*** painel (ou seja, o Azure AD para o diretório que está a gerir), selecione **aplicações empresariais**.

    ![Aplicações da empresa ao abrir](./media/active-directory-coreapps-view-azure-portal/open-enterprise-apps.png)
4. No **aplicações empresariais** painel, selecione **todas as aplicações**. Este painel, que pode selecionar aplicações para gerir, alterar as colunas apresentadas ou filtrar a lista para localizar essa aplicação que pretende (por exemplo, para ver apenas desativada aplicações).
