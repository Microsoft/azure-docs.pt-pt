# <a name="close-your-work-or-school-account-in-an-unmanaged-directory"></a>Fechar a sua conta escolar ou profissional num diretório não gerido

Se for um usuário numa organização não gerida do Azure Active Directory (Azure AD) e já não tem de utilizar aplicações dessa organização ou manter qualquer associação a é, pode fechar a sua conta em qualquer altura. Um diretório não gerido não tem um Administrador Global. Os usuários num diretório não gerido podem fechar a suas contas por conta própria, sem ter de contactar um administrador.

Os usuários num diretório não gerido são geralmente criados durante a Self-Service Inscreva-se. Um exemplo poderá ser um operador de informações numa organização que se inscreve para um serviço gratuito. Para obter mais informações sobre a inscrição Self-Service, consulte [o que é o self-service Inscreva-se para o Azure Active Directory?](directory-self-service-signup.md).

[!INCLUDE [GDPR-related guidance](../../../includes/gdpr-intro-sentence.md)]

## <a name="before-you-begin"></a>Antes de começar

Antes de fechar a sua conta, deve confirmar os seguintes itens:

* Certifique-se de que seja um utilizador do Azure não gerido diretório AD. Não é possível fechar a sua conta se pertence a um diretório gerido. Se pertence a um diretório gerido e quiser fechar a sua conta, tem de contactar o administrador. Para obter informações sobre como determinar se pertence a um diretório não gerido, consulte [eliminar o utilizador do inquilino não gerido](https://docs.microsoft.com/flow/gdpr-dsr-delete#delete-the-user-from-unmanaged-tenant).

* Guarde os dados que pretende manter. Para obter informações sobre como submeter um pedido de exportação, consulte [acessando e exportar registos gerados pelo sistema para os inquilinos não gerenciado](https://docs.microsoft.com/power-platform/admin/powerapps-gdpr-dsr-guide-systemlogs#accessing-and-exporting-system-generated-logs-for-unmanaged-tenants).

> [!WARNING]
> Fechar a sua conta é irreversível. Quando fecha a sua conta, todos os dados pessoais serão removidos. Já não terá acesso à sua conta e dados associados à sua conta.

## <a name="close-your-account"></a>Fechar a sua conta

Para fechar uma conta escolar ou profissional de não-gerenciado, siga estes passos:

1. Inicie sessão no [fechar a sua conta](https://go.microsoft.com/fwlink/?linkid=873123), utilizando a conta que pretende fechar.

1. No **os meus pedidos de dados**, selecione **Fechar conta**.

    ![Os meus pedidos de dados - Fechar conta](./media/users-close-account/close-account.png)

1. Reveja a mensagem de confirmação e, em seguida, selecione **Sim**.

    ![Os meus pedidos de dados - confirmar fechar](./media/users-close-account/confirm-close.png)
