# <a name="close-your-work-or-school-account-in-an-unmanaged-directory"></a>Feche o seu trabalho ou a sua conta escolar num diretório não gerido

Se for utilizador de uma organização não gerida do Azure Ative Directory (Azure AD) e já não precisar de utilizar aplicações dessa organização ou manter qualquer associação com a mesmo, pode fechar a sua conta a qualquer momento. Um diretório não gerido não tem um administrador global. Os utilizadores de um diretório não gerido podem fechar as suas contas por conta própria, sem terem de contactar um administrador.

Os utilizadores de um diretório não gerido são muitas vezes criados durante o autosserviço. Um exemplo pode ser um trabalhador da informação numa organização que se inscreve para um serviço gratuito. Para obter mais informações sobre a inscrição em self-service, consulte o que é a [inscrição de self-service para o Azure Ative Directory?](directory-self-service-signup.md)

[!INCLUDE [GDPR-related guidance](../../../includes/gdpr-intro-sentence.md)]

## <a name="before-you-begin"></a>Antes de começar

Antes de poder fechar a sua conta, deverá confirmar os seguintes itens:

* Certifique-se de que é utilizador de um diretório Azure AD não gerido. Não pode fechar a sua conta se pertencer a um diretório gerido. Se pertence a um diretório gerido e quer fechar a sua conta, deve contactar o seu administrador. Para obter informações sobre como determinar se pertence a um diretório não gerido, consulte [Eliminar o utilizador do Inquilino Não Gerido](https://docs.microsoft.com/flow/gdpr-dsr-delete#delete-the-user-from-unmanaged-tenant).

* Guarde todos os dados que quiser guardar. Para obter informações sobre como apresentar um pedido de exportação, consulte [registos de acesso e exportação gerados pelo sistema para inquilinos não geridos](https://docs.microsoft.com/power-platform/admin/powerapps-gdpr-dsr-guide-systemlogs#accessing-and-exporting-system-generated-logs-for-unmanaged-tenants).

> [!WARNING]
> Fechar a sua conta é irreversível. Quando fechar a sua conta, todos os dados pessoais serão removidos. Deixará de ter acesso à sua conta e dados associados à sua conta.

## <a name="close-your-account"></a>Fechar a conta

Para encerrar um trabalho não gerido ou uma conta escolar, siga estes passos:

1. Faça o sessão para fechar a [sua conta,](https://go.microsoft.com/fwlink/?linkid=873123)utilizando a conta que pretende fechar.

1. Nos **meus pedidos de dados,** selecione Conta **Fechar**.

    ![Meus pedidos de dados - Close account](./media/users-close-account/close-account.png)

1. Reveja a mensagem de confirmação e, em seguida, selecione **Sim**.

    ![Meus pedidos de dados - Confirme perto](./media/users-close-account/confirm-close.png)
