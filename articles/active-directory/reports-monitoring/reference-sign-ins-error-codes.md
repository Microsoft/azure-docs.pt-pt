# <a name="sign-in-activity-report-error-codes"></a>Códigos de erro do relatório de atividade de inscrição 

Com as informações fornecidas pelo [relatório de inscrições](concept-sign-ins.md)do utilizador, encontra respostas a questões como:

- Quem se inscreveu no meu pedido?
- Quais as candidaturas que foram assinadas?
- Que inscrições falharam e porquê?

Quando uma sindagem falha, vê-se um código de erro correspondente à falha. [Encontre a descrição de um código de erro](https://login.microsoftonline.com/error).
