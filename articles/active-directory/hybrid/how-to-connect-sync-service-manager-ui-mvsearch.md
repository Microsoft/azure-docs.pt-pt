# <a name="sync-service-manager-metaverse-search"></a>Pesquisa metaversa do gestor de serviço sincronizado

![Gestor de Serviços de Sincronização](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

O separador de pesquisa metaverso é útil para resolver problemas relacionados com dados. Na metade superior, pode criar uma consulta baseada numa combinação de atributos. Quando estiver satisfeito com a sua consulta, clique em **Procurar**. O resultado é visível na grelha inferior. Pode selecionar quais as colunas que devem ser visíveis com **definições**da coluna .

Nos resultados da pesquisa, selecione um objeto e **propriedades** para ver as propriedades do [objeto metaverso](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
