# <a name="sync-service-manager-metaverse-search"></a>Pesquisa de Metaverso do Gestor do serviço de sincronização

![Gestor do serviço de sincronização](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

O separador de pesquisa de metaverso é útil para resolução de problemas relacionados com dados. Na metade superior, pode criar uma consulta com base numa combinação de atributos. Quando estiver satisfeito com a sua consulta, clique em **pesquisa**. O resultado é visível na grelha na parte inferior. Pode selecionar as colunas que devem estar visíveis com **definições de coluna**.

Nos resultados da pesquisa, selecione um objeto e **propriedades** para ver a [propriedades do objeto de metaverso](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
