# <a name="sync-service-manager-metaverse-search"></a>Pesquisa de Metaverse do Gestor de Serviços de Sincronização

![Gestor de Serviço sincronizado](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

O separador de pesquisa de metaverso é útil para resolver problemas relacionados com problemas relacionados com os dados. Na metade superior, pode criar uma consulta baseada numa combinação de atributos. Quando estiver satisfeito com a sua consulta, clique em **Procurar**. O resultado é visível na grelha inferior. Pode selecionar quais as colunas que devem ser visíveis com **as definições da coluna**.

Nos resultados da pesquisa, selecione um objeto e **propriedades** para ver as propriedades do [objeto metaverso](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
