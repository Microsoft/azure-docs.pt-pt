# <a name="azure-active-directory-connect-health-alert-catalog"></a>Azure Active Directory Connect Health catálogo de alertas 

Azure AD Connect Health alertas de envio do serviço indicam que sua infraestrutura de identidade não está íntegra. Este artigo inclui títulos de alertas, descrições e etapas de correção para cada alerta. <br />
Erro, aviso e preaviso são três estágios de alertas gerados do serviço de integridade de conexão. É altamente recomendável que você faça ações imediatas em alertas disparados. <br />
Azure AD Connect Health alertas são resolvidos em uma condição de êxito. Os agentes de Azure AD Connect Health detectam e relatam as condições de sucesso para o serviço periodicamente. Para alguns alertas, a supressão é baseada em tempo. Em outras palavras, se a mesma condição de erro não for observada em 72 horas da geração de alertas, o alerta será resolvido automaticamente.

| Nome do Alerta | Descrição | Remediação |
| --- | --- | ----- |

Os últimos dados processados pelo Serviço de Integridade são mais antigos que 2 horas. | Verifique se os agentes de integridade têm conectividade de saída com os pontos de extremidade de serviço necessários.
