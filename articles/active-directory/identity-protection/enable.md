# <a name="enabling-azure-active-directory-identity-protection"></a>Habilitando Azure Active Directory Identity Protection

Azure Active Directory Identity Protection é um recurso do Azure Active Directory (AD do Azure). Com Azure AD Identity Protection, você pode:

- Obtenha uma exibição consolidada de usuários sinalizados e detecções de risco detectadas usando algoritmos de aprendizado de máquina
- Definir políticas de acesso condicional com base em risco para proteger automaticamente seus usuários
- Melhore a postura de segurança agindo com vulnerabilidades 

Para obter mais informações, consulte [Azure Active Directory Identity Protection](../active-directory-identityprotection.md).

Este artigo mostra como habilitar Azure Active Directory Identity Protection.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Etapas para habilitar o Azure Active Directory Identity Protection

1. Faça logon no seu [portal do Azure](https://ms.portal.azure.com/) como administrador global. 

2. Na portal do Azure, clique em **Marketplace**.
   
    ![Criar](./media/enable/01.png "Criar")

3. Na lista de aplicativos, clique em **identidade**.
   
    ![Criar](./media/enable/05.png "Criar")

4. Clique em **Azure ad Identity Protection**.
   
    ![Criar](./media/enable/06.png "Criar")

5. Na folha **Azure ad Identity Protection** , clique em **criar**.
   
    ![Criar](./media/enable/04.png "Criar")
