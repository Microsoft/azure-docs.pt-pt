# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Get It Now - adicione uma aplicação do Azure Marketplace

Está quase aí! 

Se está a tentar usar o Azure AD como fornecedor de identidade para uma aplicação, então está no lugar certo. Só precisa adicioná-lo ao seu inquilino da AD Azure. Para aprender a fazê-lo, siga a série quickstart aqui: [Veja aplicativos no seu inquilino AD Azure.](view-applications-portal.md)

Se estiver a tentar instalar uma aplicação no seu computador ou dispositivo móvel local, então está no lugar errado. O Azure Marketplace é projetado para organizações que usam apps com seus inquilinos Azure. Para computadores e dispositivos pessoais, consulte a [loja microsoft Apps](https://www.microsoft.com/store/apps).
