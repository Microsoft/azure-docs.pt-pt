# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Como configurar uma aplicação proxy de aplicação para usar o PingAccess

A nossa colaboração com o PingAccess permite agora alargar os benefícios do Proxy de Aplicação a aplicações utilizando a autenticação baseada em cabeçalhos. Se as suas aplicações não utilizarem cabeçalhos, consulte a nossa [documentação De Assinatura Única](application-proxy-configure-single-sign-on-with-kcd.md) para mais detalhes sobre outras opções.

## <a name="overview-of-steps-and-recommended-documents"></a>Visão geral dos passos e documentos recomendados

Para configurar uma aplicação com o PingAccess, existem quatro etapas:

1.  Configure Conectores proxy de aplicação

2.  Criar uma aplicação de procuração de aplicação ad ida e resumida

3.  Baixar & Configure PingAccess

4.  Configure aplicações no PingAccess

Para mais detalhes sobre cada um destes passos, consulte a nossa [documentação Single Sign-On com Cabeçalhos](application-proxy-configure-single-sign-on-with-ping-access.md).
