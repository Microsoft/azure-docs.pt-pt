# <a name="azure-ad-federation-compatibility-list"></a>Lista de compatibilidades de federação do Azure AD
Azure Active Directory fornece o início de sessão único em e modulação de segurança de acesso de aplicação para Office 365 e outros serviços Online da Microsoft para implementações de apenas na nuvem e de híbrida sem necessidade de qualquer solução de terceiros. Office 365, como a maioria dos Serviços Online da Microsoft, está integrado com o Azure Active Directory para serviços de diretório, autenticação e autorização. Azure Active Directory também fornece início de sessão único a milhares de aplicações SaaS e no local as aplicações web. Consulte o Azure Active Directory [Galeria de aplicações](https://azuremarketplace.microsoft.com/marketplace/apps) suportam aplicações SaaS. 

## <a name="idp-validation"></a>Validação do IDP
Se a sua organização utiliza uma solução de Federação de terceiros, pode configurar o início de sessão único para os utilizadores do Active Directory no local com os Serviços Online da Microsoft, como o Office 365, desde a solução de Federação de terceiros é compatível com o Azure Do Active Directory.  Para perguntas sobre a compatibilidade, contacte o fornecedor de identidade.  Se gostaria de ver uma lista de fornecedores de identidade que anteriormente foram testadas para compatibilidade com o Azure AD, pela Microsoft, clique em [aqui](https://www.microsoft.com/download/details.aspx?id=56843). 

>[!NOTE]
>A Microsoft fornece já não validação de testes para fornecedores de identidade independente para compatibilidade com o Azure Active Directory. Se gostaria de testar o produto de interoperabilidade, consulte estes [diretrizes](https://www.microsoft.com/download/details.aspx?id=56843).
