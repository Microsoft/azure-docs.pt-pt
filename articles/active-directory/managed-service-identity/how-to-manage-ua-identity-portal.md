# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Criar, lista ou eliminar um utilizador atribuído Managed Service Identity (MSI) com o portal do Azure

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Identidade de serviço gerida fornece serviços do Azure com uma identidade gerida no Azure Active Directory. Pode utilizar esta identidade para autenticar para serviços que suportam a autenticação do Azure AD, sem a necessidade de credenciais no seu código. 

Neste momento, não é possível criar, listar ou eliminar um utilizador atribuído a identidade gerida no portal do Azure.  Verifique novamente a existência de atualizações.
