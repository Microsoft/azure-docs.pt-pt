# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Criar, lista ou eliminar um utilizador atribuído geridos serviço de identidade (MSI) no portal do Azure

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Identidade de serviço geridas fornece serviços do Azure com uma identidade gerido no Azure Active Directory. Pode utilizar esta identidade para se autenticarem em serviços que suportam a autenticação do Azure AD, sem necessitar de credenciais no seu código. 

Neste momento, não é possível criar, lista ou eliminar um utilizador atribuído gerida identidade no portal do Azure.  Verifique novamente para as atualizações.
