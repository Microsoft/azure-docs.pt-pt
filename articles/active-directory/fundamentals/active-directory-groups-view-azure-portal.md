# <a name="view-or-search-for-your-user-groups-in-azure-active-directory"></a>Veja ou procure os seus grupos de utilizadores no Azure Active Directory
Este artigo explica como ver grupos no Azure Active Directory (Azure AD). Uma das funcionalidades da gestão de utilizadores do Azure AD é poder utilizar grupos para realizar tarefas de gestão, como atribuir licenças ou permissões a um número de utilizadores ao mesmo tempo.

## <a name="how-do-i-see-all-the-groups"></a>Como posso ver todos os grupos?
1. Iniciar sessão no [Portal do Azure](https://portal.azure.com) com uma conta que seja um administrador global do diretório.
2. Selecione **Todos os serviços**, introduza **Utilizadores e grupos** na caixa de texto e, em seguida, selecione **Enter**.

   ![Abrir utilizadores e grupos](./media/active-directory-groups-view-azure-portal/search-user-management.png)
3. No painel **Utilizadores e grupos**, selecione **Todos os grupos**.

   ![Abrir o painel de grupos](./media/active-directory-groups-view-azure-portal/view-groups-blade.png)
4. No painel **Utilizadores e grupos – Todos os grupos**, pode adicionar ou remover colunas apresentadas, filtrar a lista para procurar um grupo ou fazer alterações aos grupos que tem permissões suficientes para alterar.
