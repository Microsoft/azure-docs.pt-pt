# <a name="securing-external-collaboration-in-azure-active-directory-and-microsoft-365"></a>Assegurar a colaboração externa no Azure Ative Directory e microsoft 365

A colaboração segura com parceiros externos garante que os parceiros externos certos tenham acesso adequado aos recursos internos durante o tempo certo. Através de uma abordagem de governação holística, você pode reduzir os riscos de segurança, cumprir objetivos de conformidade, e garantir que você sabe quem tem acesso.

A colaboração desgovernada conduz à falta de clareza sobre a propriedade do acesso e à possibilidade de serem expostos recursos sensíveis. Avançar para uma colaboração segura e governada pode garantir que existem linhas claras de propriedade e responsabilidade pelo acesso dos utilizadores externos. O que está incluído:

* Gerir as organizações externas, e os utilizadores dentro delas, que têm acesso a recursos.

* Garantindo que o acesso é adequado, revisto e limitado a tempo, se for caso disso.

* Capacitar os empresários para gerir a colaboração dentro dos trilhos de guarda criados em TI.

Se tiver de cumprir os quadros de conformidade, a colaboração governada permite-lhe atestar a adequação do acesso.

A Microsoft oferece um conjunto abrangente de ferramentas para um acesso externo seguro.  A Azure Ative Directory (Azure AD) B2B Collaboration está no centro de qualquer plano de colaboração externa. O Azure AD B2B pode integrar-se com outras ferramentas em AZure AD, e ferramentas nos serviços da Microsoft 365, para ajudar a garantir e gerir o seu acesso externo.

Este conjunto de documentos destina-se a permitir-lhe passar de colaboração externa ad hoc ou livremente governada para um estado mais seguro.
