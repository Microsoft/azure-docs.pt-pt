# <a name="find-help-and-open-a-support-ticket-for-azure-active-directory"></a>Encontre ajuda e abra um bilhete de apoio para o Diretório Ativo Azure
A Microsoft fornece suporte técnico, pré-venda, faturação e subscrição global para o Azure Ative Directory (Azure AD). O suporte está disponível tanto online como por telefone para subscrições pagas e experimentais do Microsoft Azure. O suporte por telefone e o suporte de faturação online estão disponíveis em idiomas adicionais. 

## <a name="find-help-without-opening-a-support-ticket"></a>Obter ajuda sem abrir um pedido de suporte

Antes de criar um pedido de suporte, veja os seguintes recursos para obter respostas e informações. 

* Para conteúdos como informações sobre como fazer informações ou amostras de código para profissionais e desenvolvedores de TI, consulte a [documentação técnica em docs.microsoft.com](https://docs.microsoft.com/azure/active-directory/).

* A [Microsoft Technical Community](https://techcommunity.microsoft.com/) é o local para os nossos parceiros e clientes profissionais de TI colaborarem, partilharem e aprenderem. O [Microsoft Technical Community Info Center](https://techcommunity.microsoft.com/t5/Community-Info-Center/ct-p/Community-Info-Center) é utilizado para anúncios, posts de blog, interações de "ask-me-anything- anything" (AMA) com especialistas, entre outros. Pode também [juntar-se à comunidade para apresentar as suas ideias.](https://techcommunity.microsoft.com/t5/Communities/ct-p/communities)


## <a name="open-a-support-ticket"></a>Abrir um pedido de suporte

Se não conseguir encontrar respostas utilizando recursos de autoajuda, pode abrir um bilhete de apoio online. Você deve abrir cada bilhete de apoio para apenas um problema, para que possamos ligá-lo aos engenheiros de suporte que são especialistas em assuntos para o seu problema. Além disso, as equipas de engenharia do Azure Ative Directory priorizam o seu trabalho com base em incidentes que são gerados, pelo que muitas vezes está a contribuir para melhorias de serviço.

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-azure-portal"></a>Como abrir um bilhete de apoio para a Azure AD no portal Azure

> [!NOTE]
> Para problemas de faturação ou subscrição, deve utilizar o centro de administração microsoft [365](https://admin.microsoft.com).
> 

1. Inscreva-se [no portal Azure](https://portal.azure.com) e abra o **Diretório Ativo Azure.**
   
2. Desloque-se até **à resolução de problemas + Suporte** e selecione Novo pedido de **suporte**.
   
3. Na lâmina **Basics,** para **tipo de emissão,** selecione **Técnico**.

4. Selecione a sua **Subscrição**.

5. Para **serviço,** selecione **Azure Ative Directory**.

6. Crie um **resumo** para o pedido. O resumo deve ser inferior a 140 caracteres.
  
7. Selecione um **tipo de problema**e, em seguida, selecione uma categoria para esse tipo. Neste momento, também lhe é oferecida informação de autoajuda para a sua categoria de problemas.
  
7. Adicione o resto das suas informações sobre o problema e clique em **Next**. 

8. Neste ponto, são-lhe oferecidas soluções de autoajuda e documentação na lâmina **Soluções.** Se nenhuma das soluções lá resolver o seu problema, clique em **Next**. 

9. Na lâmina **Detalhes,** preencha os detalhes necessários e selecione uma [Severidade](https://azure.microsoft.com/support/plans/response/). 
 
![image](https://user-images.githubusercontent.com/13383753/76565580-1c284900-6468-11ea-8c0f-85af98097b6f.png)
 
10. Forneça as suas informações de contacto e selecione **Next**. 


8. Forneça as suas informações de contacto e selecione **Criar**.
  ![Imagem de autoajuda da categoria problemática](./media/active-directory-troubleshooting-support-howto/open-support-ticket.png)

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-microsoft-365-admin-center"></a>Como abrir um bilhete de suporte para o Azure AD no centro de administração da Microsoft 365

> [!NOTE]
> O suporte para Azure AD no centro de administração da [Microsoft 365](https://admin.microsoft.com) é oferecido apenas para administradores. 

1. Inscreva-se no centro de administração da [Microsoft 365](https://admin.microsoft.com) com uma conta que tem uma licença de Mobilidade Empresarial + Segurança (EMS).

2. No azulejo **de suporte,** selecione Novo pedido de **serviço:**

3. Na página **'Visão Geral** de Suporte', selecione **Gestão de Identidade** ou **Gestão de Utilizador e domínio:**

4. Para **funcionalidades**, selecione a função Azure AD para a qual deseja suporte.

5. Para **o Sintoma**, selecione um sintoma apropriado, sintetmente o seu problema e forneça detalhes relevantes e, em seguida, selecione **Next**.

6. Selecione um dos recursos de autoajuda oferecidos, ou selecione **Sim, continue** ou **não, cancele**o pedido .

7. Se continuar, é-lhe pedido mais detalhes. Pode anexar quaisquer ficheiros que representem o problema e, em seguida, selecionar **Next**.

8. Forneça as suas informações de contacto e selecione **Enviar pedido**.
