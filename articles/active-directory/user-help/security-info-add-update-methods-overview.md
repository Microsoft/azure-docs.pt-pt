# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>Visão geral sobre como alterar suas informações e métodos de informações de segurança (versão prévia)

Você pode adicionar, editar, excluir e alterar o método de informações de segurança padrão de dentro das páginas de **informações de segurança** . Para obter mais informações sobre cada método e como alterar suas informações, consulte o artigo associado baseado em método.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>Verificação de dois fatores versus autenticação de redefinição de senha

Os métodos de informações de segurança são usados para verificação de dois fatores e para redefinição de senha. No entanto, nem todos os métodos podem ser usados para ambos.

| Método | Utilizado para |
| ------ | -------- |
| Aplicativo autenticador | Verificação de dois fatores e autenticação de redefinição de senha. |
| Mensagens de texto | Verificação de dois fatores e autenticação de redefinição de senha. |
| Chamadas telefônicas | Verificação de dois fatores e autenticação de redefinição de senha. |
| Conta de email | Somente autenticação de redefinição de senha. Você precisará escolher outro método para verificação de dois fatores. |
| Perguntas de segurança | Somente autenticação de redefinição de senha. Você precisará escolher outro método para verificação de dois fatores. |
