# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>Descrição geral sobre como alterar seus métodos de informações (pré-visualização) de segurança e informações
Pode adicionar, editar, eliminar e alterar o seu método de informações de segurança do padrão de dentro do **informações de segurança** páginas. Para obter mais informações sobre cada método e como alterar as informações, consulte o artigo associado, com base no método.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>Verificação de dois fatores em comparação com a palavra-passe de reposição de autenticação
Métodos de informações de segurança são utilizados para ambos os efeitos de verificação de dois fatores e para a reposição de palavra-passe. No entanto, nem todos os métodos podem ser utilizados para ambos.

|Método|Utilizado para|
|------|--------|
|Aplicação de autenticação|Verificação de dois fatores e a palavra-passe de reposição de autenticação.|
|Mensagens de texto|Verificação de dois fatores e a palavra-passe de reposição de autenticação.|
|Chamadas telefónicas|Verificação de dois fatores e a palavra-passe de reposição de autenticação.|
|Conta de e-mail|Apenas a autenticação de reposição de palavra-passe. Terá de escolher outro método para a verificação de dois fatores.|
|Perguntas de segurança|Apenas a autenticação de reposição de palavra-passe. Terá de escolher outro método para a verificação de dois fatores.|
