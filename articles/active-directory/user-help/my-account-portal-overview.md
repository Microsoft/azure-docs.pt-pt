# <a name="what-is-the-my-account-portal"></a>O que é o portal A Minha Conta?

O portal **My Account** ajuda-o a gerir o seu trabalho ou conta escolar, configurando e gerindo as suas informações de segurança, gerindo as suas organizações e dispositivos conectados, visualizando como a sua organização utiliza os seus dados.

Pode chegar ao portal **My Account** (https://myprofile.microsoft.com)), a partir da versão atual de qualquer um dos seguintes navegadores:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

![Meu Portal de contas, página de visão geral](media/my-account-portal/my-account-portal-overview.png)

>[!Important]
>Este artigo destina-se a usuários que estão tentando acessar o portal minha conta para atualizar suas informações de segurança, informações de dispositivo, senha, organizações conectadas, privacidade ou informações de entrada anteriores. Se é um administrador que procura informações sobre como ativar a autenticação e outras funcionalidades do Azure Ative Directory (Azure AD) para os seus colaboradores e outras utilizações, consulte a [documentação da AD Azure para administradores](https://docs.microsoft.com/azure/active-directory/).
