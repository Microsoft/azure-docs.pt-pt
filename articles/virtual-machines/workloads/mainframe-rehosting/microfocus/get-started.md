# <a name="micro-focus-devtest-environments-on-azure"></a>Micro Focus dev/test ambientes em Azure

Começar em Azure com um ambiente de desenvolvimento e teste é uma ótima maneira de impulsionar os seus esforços de modernização. As aplicações mainframe que dependem de sistemas antigos como o IBM CICS (Customer Information Control System) e o IMS (Information Management System) podem ser recandidaturados no Azure utilizando soluções parceiras.

A Micro Focus estabeleceu uma parceria com a Microsoft para oferecer ferramentas de migração e rehosagem de computadores principais convincentes. Pode executar aplicações mainframe em todo o desenvolvimento, testes e cenários de produção no Azure e na plataforma Microsoft Windows.

A Micro Focus oferece soluções de rehosagem e de teste de desenvolvimento que alargam ao Azure as aplicações comerciais existentes de COBOL e PL/I.
