# <a name="mainframe-workloads-supported-on-azure"></a>Cargas de trabalho do mainframe suportadas no Azure

As soluções listadas aqui podem ajudá-lo a migrar a sua carga de trabalho do computador principal para o Microsoft Azure. Algumas cargas de trabalho podem ser migradas com relativa facilidade. Outras cargas de trabalho que dependem do software do sistema legado podem ser rehosted. 

Para obter mais ajuda com a emulação e serviços do mainframe, consulte o [centro de migração do Azure Mainframe.](https://azure.microsoft.com/migration/mainframe/)

## <a name="migrate-mainframe-closer-to-azure"></a>Migrar o mainframe mais perto de Azure

- [RPG Visual ASNA para](https://asna.com/us/products/visual-rpg) compilador .NET para o quadro .NET via visual Studio plug-in.
- Ambiente de desenvolvimento [cobol amt asysco AMT](https://www.asysco.com/cobol/) (Unisys, mainframes IBM e outros dialetos COBOL, como Micro Focus COBOL).
- [Asysco AMT GO](https://www.asysco.com/amt-go/) arquitetura de implantação baseada em nuvem para cargas de trabalho de alta qualidade.
- [Transformação assco AMT](https://www.asysco.com/amt-transform/) para conversão de dados, código, script, segurança, interfaces e outros artefactos do computador principal.
- [Ferramentas fujitsu NetCOBOL](https://www.fujitsu.com/global/products/software/developer-tool/netcobol/) de desenvolvimento e integração.
- [Micro Focus Visual COBOL](https://www.microfocus.com/products/visual-cobol/) ferramentas de desenvolvimento e integração.
- [Micro Focus PL/I](https://www.microfocus.com/campaign/download/pli-modernization/) compilador legado para a plataforma .NET, suportando a sintaxe PL/I do computador principal, tipos de dados e comportamento.
- [Micro Focus Enterprise Server](https://www.microfocus.com/products/enterprise-suite/enterprise-server/) plataforma de integração do computador principal.
- [Sistemas Modernos CTU (COBOL-To-Universal)](https://modernsystems.com/automatic-cobol-to-java-conversion/) ferramentas de desenvolvimento e integração.
- Ferramentas de desenvolvimento e integração [da NTT Data Enterprise COBOL.](https://us.nttdata.com/en/digital/application-development-and-modernization)
- [NTT Open PL/I](https://us.nttdata.com/en/digital/application-development-and-modernization) compilador legado pl/I para a plataforma .NET, suportando a sintaxe PL/I do computador principal, tipos de dados e comportamento.
- Ferramentas de desenvolvimento e integração [do compilador COBOL de raincode.](https://www.raincode.com/products/cobol/)
- [O compilador PL/I](https://www.raincode.com/products/pli/) do Raincode para a plataforma .NET suporta a sintaxe PL/I do computador principal, tipos de dados e comportamento.
- [Compilador RAINCODE ASM370](https://www.raincode.com/technical-landscape/asm370/) para a sintaxe do mainframe Assembler 370 e HLASM.

## <a name="deploy-an-emulation-environment-for-online-and-batch-processing"></a>Implementar um ambiente de emulação para processamento online e de lote

- [Asysco AMT GO](https://www.asysco.com/amt-go/) arquitetura de implantação suportando CICS, IMS, TIP, HVTIP e outros ambientes comuns do mainframe.
- [Micro Focus Enterprise Server](https://www.microfocus.com/products/enterprise-suite/enterprise-server/) plataforma de integração do computador principal.
- [NTT DATA Mainframe Rehosting Development Environment ambiente](https://us.nttdata.com/en/-/media/assets/white-paper/apps-mainframe-re-hosting-development-environment-whitepaper.pdf) de processamento de transações nativas.
- [NTT DATA Processuando Ambiente](https://us.nttdata.com/en/-/media/assets/white-paper/apps-mainframe-re-hosting-development-environment-whitepaper.pdf) (BPE) incluindo capacidades de transação JCL.
- [Emulador CICS de código de chuva](https://www.raincode.com/technical-landscape/cics/) para plataformas .NET e Azure.
- Intérprete JCL compatível com ficha [JCL.](https://www.raincode.com/products/jcl/)

## <a name="code-conversion"></a>Conversão de código

- Tecnologia de conversão do sistema [Asysco](https://www.asysco.com/azure-cloud/) que abrange código fonte, dados, lote, agendamento, monitores TP, interfaces, segurança, gestão e muito mais.
- [Serviços Asysco AMT serviços](https://www.asysco.com/migration-services/) de ponta a ponta para projetos de migração, incluindo inventário e análise, formação de design, ensaios de vestuário, go-live e apoio pós-migração.
- [Ferramentas Blu Age](https://www.bluage.com/) para digitalizar aplicações e bases de dados de negócios antigas.
- [Serviços de computação heirloom](https://www.heirloomcomputing.com/tag/convert-cobol-to-java/) para converter o computador principal COBOL, CICS e VSAM para Java.
- [LzLabs Software Defined Mainframe](https://www.lzlabs.com/) geriu o contentor de software para migrar aplicações mainframe para computadores Linux ou ambientes de nuvem privados, públicos e híbridos.
