# <a name="red-hat-enterprise-linux-rhel-extended-lifecycle-support"></a>Red Hat Enterprise Linux (RHEL) Extended Lifecycle Support
Este artigo fornece informações sobre suporte de ciclo de vida prolongado para as imagens da Red Hat Enterprise:
* Red Hat Enterprise Linux 6  

## <a name="red-hat-enterprise-linux-6-lifecycle"></a>Red Hat Enterprise Linux 6 Lifecycle
A partir de 30 de novembro de 2020, a Red Hat Enterprise Linux 6 chegará ao fim da fase de manutenção. A fase de manutenção é seguida pela Fase de Vida Prolongada. À medida que a Red Hat Enterprise Linux 6 sai das Fases Full/Manutenção, é fortemente recomendado o upgrade para Red Hat Enterprise Linux 7 ou 8. Se os clientes térmiam em Red Hat Enterprise Linux 6, é aconselhável adicionar o Suporte ao Ciclo de Vida Prolongado (ELS) da Red Hat Enterprise.

## <a name="steps-to-add-extended-lifecycle-support-on-marketplace-pay-as-you-go-vms"></a>Passos para adicionar suporte de ciclo de vida alargado em VMs pay-as-you-go do mercado
1. Preencha o [formulário ELS disponível aqui](https://aka.ms/els-form) com os seus dados de contacto e informações de subscrição de VMs para os quais pretende adicionar suporte ELS. O complemento de detalhes sobre os preços também está disponível no formulário.
1. A equipa da Azure Red Hat Enterprise Linux irá contactar-te com a lista de VMs para suporte ELS adicionar dentro de 1-2 dias úteis. Por favor, reveja a lista e responda concordando com o adição de preços.
1. A equipa da Azure Red Hat Enterprise Linux partilhará os passos para adicionar o pacote de clientes ELS aos VMs. Siga as etapas, que serão fornecidas no e-mail, para continuar a receber manutenção de software (correções de bugs e segurança) e suporte para Red Hat Enterprise Linux 6.

> [!Note]
> Não partilhe os passos para a utilização do RHEL ELS com alguém fora da sua organização. Contacte AzureRedHatELS@microsoft.com para obter apoio ou para quaisquer perguntas adicionais.

## <a name="frequently-asked-questions"></a>Perguntas Mais Frequentes

#### <a name="im-running-red-hat-enterprise-linux-6-and-cant-migrate-to-a-later-version-at-this-time-what-options-do-i-have"></a>Estou a comandar o Red Hat Enterprise Linux 6 e não posso migrar para uma versão posterior neste momento. Que opções tenho?
* Continue a executar a Red Hat Enterprise Linux 6 e compre os repositórios de suporte ao ciclo de vida prolongado (ELS) Add-On para continuar a receber manutenção limitada de software e suporte técnico (ver processo para atualizar e definir detalhes de preços abaixo).
* Migrar para red hat enterprise Linux 7 ou 8 o mais rápido possível.

#### <a name="what-is-the-additional-charge-for-using-red-hat-enterprise-linux-extended-life-cycle-support-els-add-on"></a>Qual é o custo adicional para a utilização do Red Hat Enterprise Linux Extended Life Cycle Support (ELS) Add-On?
Os custos relacionados com o suporte de ciclo de vida prolongado podem ser encontrados com o [formulário ELS](https://aka.ms/els-form)
