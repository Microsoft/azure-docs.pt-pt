# <a name="red-hat-enterprise-linux-rhel-extended-lifecycle-support"></a>Red Hat Enterprise Linux (RHEL) Extended Lifecycle Support
Este artigo fornece informações sobre suporte de ciclo de vida prolongado para as imagens da Red Hat Enterprise:
* Red Hat Enterprise Linux 6  

## <a name="red-hat-enterprise-linux-6-lifecycle"></a>Red Hat Enterprise Linux 6 Lifecycle
A partir de 30 de novembro de 2020, a Red Hat Enterprise Linux 6 chegará ao fim da fase de manutenção. A fase de manutenção é seguida pela Fase de Vida Prolongada. À medida que a Red Hat Enterprise Linux 6 sai das Fases Full/Manutenção, é fortemente recomendado o upgrade para Red Hat Enterprise Linux 7 ou 8. Se os clientes térmiam em Red Hat Enterprise Linux 6, é aconselhável adicionar o Suporte ao Ciclo de Vida Prolongado (ELS) da Red Hat Enterprise.

## <a name="frequently-asked-questions"></a>Perguntas Mais Frequentes

* Migrar para red hat enterprise Linux 7 ou 8 o mais rápido possível.
