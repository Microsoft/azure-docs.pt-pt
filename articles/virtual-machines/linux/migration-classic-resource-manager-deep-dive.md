Vamos uma descrição profunda sobre a migração a partir do modelo de implementação clássica do Azure para o modelo de implementação Azure Resource Manager. Vamos ver recursos a um nível de recursos e funcionalidades para ajudar a compreender a forma como a plataforma do Azure efetua a migração de recursos entre os modelos de implementação de dois.
