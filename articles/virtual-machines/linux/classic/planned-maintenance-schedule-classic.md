# <a name="how-to-schedule-planned-maintenance-on-azure-vms"></a>Como agendar a manutenção planeada em VMs do Azure
> [!IMPORTANT]
> O Azure tem dois modelos de implementação diferentes para criar e trabalhar com recursos: [Resource Manager e clássica](../../../resource-manager-deployment-model.md). Este artigo explica como utilizar o modelo de implementação clássica. A Microsoft recomenda que as implementações mais novas utilizem o modelo Resource Manager. Para obter informações sobre a manutenção planeada no modelo do Resource Manager, consulte [aqui](../planned-maintenance.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
