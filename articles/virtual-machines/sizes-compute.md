# <a name="compute-optimized-virtual-machine-sizes"></a>Compute otimizado tamanhos de máquina virtual

Os tamanhos de VM otimizados com uma elevada relação CPU-memória. Estes tamanhos são bons para servidores web de tráfego médio, eletrodomésticos de rede, processos de lote e servidores de aplicações. Este artigo fornece informações sobre o número de vCPUs, discos de dados e NICs. Também inclui informações sobre a entrada de armazenamento e a largura de banda da rede para cada tamanho neste agrupamento.

A [série Fsv2 baseia-se](fsv2-series.md) no processador Intel® Xeon® Platinum 8168. Possui uma velocidade de relógio Turbo de 3,4 GHz e uma frequência turbo de núcleo único máxima de 3,7 GHz. As instruções ® INTEL AVX-512 são novas em processadores Intel Scalable. Estas instruções fornecem até um aumento de desempenho de 2X para o processamento de volumes de trabalho de processamento de vetores em operações de ponto flutuante de precisão única e dupla. Por outras palavras, são muito rápidos para qualquer carga de trabalho computacional.

A um preço de lista mais baixo por hora, a série Fsv2 é o melhor valor no desempenho de preços no portefólio Azure com base na Unidade de Computação Azure (ACU) por vCPU.
