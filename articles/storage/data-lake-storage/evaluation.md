# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Orientação para os clientes de geração 1 de armazenamento do Azure Data Lake

O Azure Data Lake armazenamento Gen2 ofertas nativa Blob armazenamento funcionalidade de pré-visualização, incluindo a criação de camadas de dados, o gerenciamento de ciclo de vida e a segurança de rede aprimorada, além do sistema de ficheiros Hadoop compatível disponível na geração 1 de armazenamento do Data Lake.

Se for um cliente existente do Data Lake Storage Gen1, a Microsoft incentiva a avaliar todos os recursos novos e aprimorados de geração 2 de armazenamento do Data Lake e fornecer comentários por [participar na pré-visualização do](https://aka.ms/adlsgen2signup). Além disso, Gen1 de armazenamento do Data Lake irá continuar a ser suportado. Data Lake Storage Gen1 e Gen2 podem coexistir, pelo que seus recursos existentes e aplicações que tiram partido da geração 1 de armazenamento do Data Lake continuam a funcionar sem as atualizações. Pode desenvolver um plano de migração para a geração 2 de armazenamento do Data Lake a sua necessidade.

Se estiver atualmente a avaliar Gen1 de armazenamento do Data Lake, a Microsoft incentiva a participar na pré-visualização de geração 2 e avaliar os recursos novos e aprimorados, oferecidos pela geração 2 de armazenamento do Data Lake. Da mesma forma, se for um cliente de armazenamento de Blob existente, recomendamos que avalie se a um sistema de ficheiros Hadoop compatível é apropriado para suas necessidades de análise.
