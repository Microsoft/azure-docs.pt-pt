# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Orientações para clientes de Gen1 de armazenamento do Azure Data Lake

Azure Data Lake pré-visualização do armazenamento Gen2 ofertas nativa Blob funcionalidade de armazenamento, incluindo a criação de camadas de dados, gestão de ciclo de vida e segurança de rede avançada, para além do Hadoop compatível com o sistema de ficheiros disponível no Gen1 de armazenamento do Data Lake.

Se tiver um cliente existente do Data Lake armazenamento Gen1, Microsoft encoraja avaliar todas as capacidades novas e avançadas Gen2 de armazenamento do Data Lake e fornecer comentários por [participar na pré-visualização](https://aka.ms/adlsgen2signup). Além disso, Gen1 de armazenamento do Data Lake irá continuar a ser suportado. Gen1 de armazenamento do Data Lake e Gen2 podem coexistir, pelo que os seus recursos existentes e aplicações que tiram partido Gen1 de armazenamento do Data Lake continuam a funcionar sem as atualizações. Pode desenvolver um plano de migração para o Data Lake armazenamento Gen2 convier.

Se estiver a avaliar atualmente Gen1 de armazenamento do Data Lake, a Microsoft encoraja participar na pré-visualização Gen2 e avaliar as funcionalidades novas e avançadas oferecidas pelo Gen2 de armazenamento do Data Lake. Da mesma forma, se tiver um cliente de armazenamento de Blob existente, Microsoft encoraja para avaliar se o sistema de ficheiros Hadoop compatível é adequado para as suas necessidades de análise.
