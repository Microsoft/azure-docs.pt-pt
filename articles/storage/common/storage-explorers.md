# <a name="microsoft-client-tools-for-working-with-azure-storage"></a>Ferramentas para clientes da Microsoft para trabalhar com o Azure Storage

A Microsoft fornece várias ferramentas gráficas de interface de utilizador (GUI) para trabalhar com os dados na sua conta de Armazenamento Azure. Todas as ferramentas descritas na tabela a seguir são gratuitas.

| Ferramenta cliente de armazenamento Azure                 | Plataformas suportadas | Blob de Bloco | Blob de página | Blob do Apêndio | Tables | Filas | Ficheiros |
|-------------------------------------------|---------------------|------------|-----------|-------------|--------|--------|-------|
| [Portal do Azure](https://portal.azure.com)                    | Web                 | Yes          | Yes         | Yes           | Yes      | Yes      | Yes     |
| [Explorador do Storage do Azure](https://azure.microsoft.com/features/storage-explorer/)          | Windows, OSX        | Yes          | Yes         | Yes           | Yes      | Yes      | Yes     |
| [Microsoft Visual Studio Cloud Explorer](/visualstudio/azure/vs-azure-tools-resources-managing-with-cloud-explorer) | Windows             | Yes          | Yes         | Yes           | Yes      | Yes      | No      |

Há também uma série de ferramentas de terceiros disponíveis para trabalhar com dados de armazenamento Azure.
