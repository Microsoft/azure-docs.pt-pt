# <a name="microsoft-client-tools-for-working-with-azure-storage"></a>Ferramentas para clientes da Microsoft para trabalhar com o Azure Storage

A Microsoft fornece várias ferramentas gráficas de interface de utilizador (GUI) para trabalhar com os dados na sua conta de Armazenamento Azure. Todas as ferramentas descritas na tabela a seguir são gratuitas.

| Ferramenta cliente de armazenamento Azure                 | Plataformas suportadas | Blob de Bloco | Blob de página | Blob do Apêndio | Tabelas | Filas | Ficheiros |
|-------------------------------------------|---------------------|------------|-----------|-------------|--------|--------|-------|
| [Portal do Azure](https://portal.azure.com)                    | Web                 | Sim          | Sim         | Sim           | Sim      | Sim      | Sim     |
| [Explorador de Armazenamento do Azure](https://azure.microsoft.com/features/storage-explorer/)          | Windows, OSX        | Sim          | Sim         | Sim           | Sim      | Sim      | Sim     |
| [Microsoft Visual Studio Cloud Explorer](/visualstudio/azure/vs-azure-tools-resources-managing-with-cloud-explorer) | Windows             | Sim          | Sim         | Sim           | Sim      | Sim      | Não      |

Há também uma série de ferramentas de terceiros disponíveis para trabalhar com dados de armazenamento Azure.
