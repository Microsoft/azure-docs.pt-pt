# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Parceiros de armazenamento Azure para armazenamento primário e secundário

Este artigo destaca as empresas parceiras da Microsoft que fornecem uma solução de armazenamento anexa (NAS) ou de rede de área de armazenamento (SAN). A solução pode ser no local, em Azure, ou uma solução híbrida que usa o Azure Storage como um nível rentável. Estas soluções podem permitir que os clientes utilizem a mesma solução em qualquer um dos seus ambientes.
