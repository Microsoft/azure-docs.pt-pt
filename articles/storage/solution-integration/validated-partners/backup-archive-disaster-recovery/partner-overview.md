# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Azure Storage archive, backup e parceiros de recuperação de desastres

Este artigo destaca os parceiros da Microsoft que estão integrados com o Azure Storage para arquivo, backup e para a continuidade do negócio e recuperação de desastres (BCDR). Estas soluções parceiras aproveitam a escala e os benefícios de custos da Azure Storage. Pode utilizar as soluções para ajudar a resolver desafios de backup, criar um site de recuperação de desastres ou arquivar conteúdo não utilizado para retenção a longo prazo. Com todos os padrões de conformidade que o Azure Storage cumpre, e com funcionalidades Azure como [armazenamento imutável](../../../blobs/storage-blob-immutable-storage.md) e [gestão do ciclo](../../../blobs/storage-lifecycle-management-concepts.md)de vida, estas soluções podem facilmente substituir backups baseados em fita, e oferecer um site de recuperação económica a pedido.
