# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Azure Storage Archive, Backup e Parceiros de Recuperação de Desastres

Este artigo destaca os parceiros da Microsoft que estão integrados com o Azure Storage for Archive e bcdr. Aproveitam a escala e os benefícios de [custos](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) do Azure Storage para resolver eficazmente os desafios de backup, criar um site de recuperação de desastres ou arquivar conteúdo não aproveitado para retenção a longo prazo. Com todos os padrões de conformidade que o Azure Storage cumpre, e com [armazenamento imutável](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) e gestão do ciclo de vida, estas soluções podem facilmente substituir backups baseados em fita, e oferecer um site de recuperação económica a pedido.
