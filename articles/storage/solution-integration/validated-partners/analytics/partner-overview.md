# <a name="azure-storage-analytics-and-big-data-partners"></a>Azure Storage analytics e grandes parceiros de dados

Este artigo destaca as empresas parceiras da Microsoft que estão integradas com o Azure Data Lake Storage. Estas soluções parceiras cobrem cargas de trabalho como cargas de trabalho modernas de armazém de dados, análises avançadas e análises em tempo real. Estes parceiros aproveitam o [espaço hierárquico](../../../blobs/data-lake-storage-namespace.md) no Azure Storage para otimizar a sua solução e executá-la de forma eficiente em Azure.
