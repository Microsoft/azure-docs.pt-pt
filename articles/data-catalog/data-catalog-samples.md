# <a name="azure-data-catalog-developer-samples"></a>Amostras de desenvolvedores do Catálogo de Dados Azure

Começar a desenvolver aplicações do Catálogo de Dados Azure utilizando a Data Catalog REST API. O Data Catalog REST API é uma API baseada em REST que fornece acesso programático aos recursos do Data Catalog para registar, anotar e pesquisar ativos de dados programáticos.

## <a name="samples-available-on-githubcom"></a>Amostras disponíveis em GitHub.com

* [Introdução ao Catálogo de Dados do Azure](https://github.com/Azure-Samples/data-catalog-dotnet-get-started/)
  
   A amostra de início de partida mostra como autenticar com a AD Azure para registar, pesquisar e eliminar um ativo de dados utilizando a API REST do Catálogo de Dados.
   
* [Começar com o Catálogo de Dados Azure usando o Principal de Serviço](https://github.com/Azure-Samples/data-catalog-dotnet-service-principal-get-started/)

   Esta amostra mostra-lhe como registar, pesquisar e eliminar um ativo de dados utilizando a API REST do Catálogo de Dados. Esta amostra utiliza a autenticação do Diretor de Serviço.

* [Ferramenta de importação/exportação para catálogo de dados Azure](https://github.com/Azure-Samples/data-catalog-dotnet-import-export/)

   Esta amostra que mostra como usar a Data Catalog REST API para recolher ativos do Catálogo de Dados Azure e serializá-los num ficheiro. Também demonstra como pegar num conjunto de ativos serializados como JSON e empurrá-los para o catálogo. Suporta a exportação de um subconjunto do catálogo utilizando uma consulta de pesquisa.

* [Registo a granel e anotação no Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-dotnet-excel-register-data-assets/)
  
   Esta amostra que mostra como registar os ativos de dados a granel de um livro do Excel utilizando data catalog REST API e Open XML.
  
* [Termos glossários de importação a granel para o Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-bulk-import-glossary/)

   Esta amostra mostra-lhe como importar termos glossários de ficheiros CSV para glossário ADC.

* [Relações de importação a granel no Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-bulk-import-relationship/)

   Esta amostra mostra-lhe como importar programáticamente informações de relacionamento de um ficheiro CSV para um catálogo de dados.

* [Publicar relações no Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-dotnet-publish-relationships/)

   Esta amostra mostra como pode publicar programáticamente informações de relacionamento num catálogo de dados.
