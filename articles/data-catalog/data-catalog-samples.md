# <a name="azure-data-catalog-developer-samples"></a>Amostras de desenvolvedores do Catálogo de Dados Azure

[!INCLUDE [Azure Purview redirect](../../includes/data-catalog-use-purview.md)]

Começar a desenvolver aplicações do Catálogo de Dados Azure utilizando o Catálogo de Dados REST API. O Data Catalog REST API é uma API baseada em REST que fornece acesso programático aos recursos do Catálogo de Dados para registar, anotar e pesquisar os ativos de dados programáticamente.

## <a name="samples-available-on-githubcom"></a>Amostras disponíveis em GitHub.com

* [Introdução ao Catálogo de Dados do Azure](https://github.com/Azure-Samples/data-catalog-dotnet-get-started/)
  
   A amostra de início mostra-lhe como autenticar com AZure AD para registar, pesquisar e eliminar um ativo de dados utilizando o Catálogo de Dados REST API.
   
* [Começar com o Azure Data Catalog usando o Chefe de Serviço](https://github.com/Azure-Samples/data-catalog-dotnet-service-principal-get-started/)

   Esta amostra mostra-lhe como registar, pesquisar e apagar um ativo de dados utilizando a API do Catálogo de Dados REST. Esta amostra utiliza a autenticação principal do serviço.

* [Ferramenta de importação/exportação para catálogo de dados Azure](https://github.com/Azure-Samples/data-catalog-dotnet-import-export/)

   Esta amostra que mostra como usar o Catálogo de Dados REST API para recolher ativos do Catálogo de Dados Azure e serializá-los em um ficheiro. Também demonstra como pegar um conjunto de ativos serializados como JSON e empurrá-los para o catálogo. Suporta a exportação de um subconjunto do catálogo utilizando uma consulta de pesquisa.

* [Registo a granel e anotação no Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-dotnet-excel-register-data-assets/)
  
   Esta amostra que mostra como registar em massa os ativos de dados de um livro do Excel utilizando o Data Catalog REST API e o Open XML.
  
* [Termos glossários de importação a granel para catálogo de dados Azure](https://github.com/Azure-Samples/data-catalog-bulk-import-glossary/)

   Esta amostra mostra-lhe como importar termos glossários de ficheiros CSV para glossário ADC.

* [Relações de importação a granel no Catálogo de Dados Azure](https://github.com/Azure-Samples/data-catalog-bulk-import-relationship/)

   Esta amostra mostra-lhe como importar informações de relacionamento programáticamente de um ficheiro CSV para um catálogo de dados.

* [Publicar relações no Catálogo de Dados da Azure](https://github.com/Azure-Samples/data-catalog-dotnet-publish-relationships/)

   Esta amostra mostra-lhe como pode publicar programáticamente informações de relacionamento a um catálogo de dados.
