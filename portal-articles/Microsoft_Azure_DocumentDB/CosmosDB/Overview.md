# <a name="azure-cosmos-db-documentation"></a>Documentação da BD do Cosmos para o Azure

A BD do Cosmos para o Azure é um serviço de base de dados globalmente distribuído concebido para lhe permitir dimensionar o débito e o armazenamento, de forma independente e elástica, num qualquer número de regiões geográficas com um SLA abrangente. Pode desenvolver bases de dados de gráficos, de chave/valor ou de documentos com a BD do Cosmos através de um conjunto de APIs populares e modelos de programação. Saiba como utilizar a BD do Cosmos com os nossos guias de introdução, tutoriais e amostras.
