[O modelo do Gestor de Recursos](/azure/azure-resource-manager/templates/overview) é um ficheiro JavaScript Object Notation (JSON) que define a infraestrutura e a configuração para o seu projeto. O modelo usa sintaxe declarativa, que permite indicar o que pretende implementar sem ter de escrever a sequência de comandos de programação para criá-la. Se quiser saber mais sobre o desenvolvimento de modelos de Gestor de Recursos, consulte a [documentação do Gestor](/azure/azure-resource-manager/) de Recursos e a referência do [modelo](/azure/templates).
