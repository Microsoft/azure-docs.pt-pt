Uma máquina virtual (VM) é atribuída automaticamente um endereço IP privado de um intervalo que especificou, com base no que é implementada na sub-rede. O endereço é mantido por uma VM até que a VM é eliminada. Azure atribui de forma dinâmica o endereço do próximo IP privado disponível da sub-rede que é criar uma VM no. Atribua um endereço IP estático se pretender que um endereço IP específico da sub-rede atribuído à VM.
