In Unitity, abra o projeto na `Unity` pasta. A unidade pode levar-te a uma versão diferente da Unidade entre o projeto e a que instalaste na tua máquina. Este aviso está bom, desde que a sua versão do Editor de Unidade seja mais recente do que aquela com que o projeto foi criado. Nesse caso, basta clicar **em Continuar.** Se a sua versão Unity Editor for mais antiga do que a que o projeto precisa, clique em **Quit e**atualize o seu Editor de Unidade.
