In Unitity, abra o projeto na pasta *Unidade.* A unidade pode levar-te a uma diferença entre a versão do projeto e a versão que está instalada na tua máquina. Este aviso está bom, desde que a sua versão do Editor de Unidade seja mais recente do que aquela com que o projeto foi criado. Se a sua versão for mais recente, **selecione Continue**. Se a sua versão for mais antiga do que a que o projeto precisa, **selecione Quit** e atualize o seu Editor de Unidade.
