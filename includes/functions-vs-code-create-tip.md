> [!TIP]
> Por predefinição, os recursos Azure exigidos pela sua aplicação de função são criados com base no nome da aplicação de função que fornece. Por padrão, também são criados no mesmo novo grupo de recursos com a aplicação de função. Se quiser personalizar os nomes destes recursos ou reutilizar os recursos existentes, tem de [publicar o projeto com opções avançadas de criação.](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options)
