1. Na **configurações** secção da página da rede virtual, selecione **servidores DNS** para abrir o **servidores DNS** página.

2. Sobre o **servidores DNS** página, preencha os valores seguintes:

  - **Servidores DNS**: selecione **personalizado**.

  - **Adicionar servidor DNS**: introduza o endereço IP do servidor DNS que pretende utilizar para a resolução de nome.

2. Quando tiver terminado de adicionar os servidores DNS, selecione **guardar**.

    ![Especificar um servidor DNS](./media/vpn-gateway-add-dns-rm-portal-include/specify-dns-server.png "Especificar um Servidor DNS")
