## <a name="create-an-azure-resource"></a>Criar um recurso Azure

Comece a utilizar a API de Verificação ortográfica Bing, criando um dos seguintes recursos Azure:

[Bing Spell Check recurso](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSpellCheck-v7)
   * Disponível através do portal Azure até eliminar o recurso.
   * Use o nível de preços gratuitos para experimentar o serviço e atualize mais tarde para um nível pago para produção.
   * A API de Verificação ortográfica de Bing também é oferecida em alguns níveis do [recurso Bing Search v7](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7).
    
[Recurso multi-serviço](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * Disponível através do portal Azure até eliminar o recurso.  
   * Use a mesma chave e ponto final para as suas aplicações, através de vários Serviços Cognitivos.
