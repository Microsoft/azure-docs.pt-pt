Tem de ter uma [conta de API dos serviços cognitivos](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) com acesso à API de verificação de ortografia do Bing. Se não tiver uma subscrição do Azure, pode [criar uma conta](https://azure.microsoft.com/try/cognitive-services/?api=text-analytics) gratuita. Antes de continuar, terá da chave de acesso [fornecida após a ativação de sua avaliação gratuita](https://azure.microsoft.com/try/cognitive-services/my-apis/), ou uma chave de subscrição paga do dashboard do Azure.
