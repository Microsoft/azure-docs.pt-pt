Caso seja necessário, instale o módulo Azure PowerShell com as instruções no [guia do Azure PowerShell](/powershell/azure/azurerm/) e, em seguida, execute `Connect-AzureRmAccount` para criar uma ligação ao Azure. Além disso, tem de ter uma chave pública SSH com o nome `id_rsa.pub` no diretório .ssh do seu perfil de utilizador.
