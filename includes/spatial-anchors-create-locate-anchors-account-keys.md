## <a name="set-up-authentication"></a>Configurar autenticação

Para aceder ao serviço, terá de fornecer uma chave de conta, o token de acesso ou o token de autenticação do Azure Active Directory.

### <a name="account-keys"></a>Chaves de conta

Chaves de conta são uma credencial que permite que a sua aplicação para autenticar com o serviço de âncoras espaciais do Azure. A finalidade das chaves de conta é que o ajudarão a começar a trabalhar rapidamente. Especialmente durante a fase de desenvolvimento de integração de seu aplicativo com âncoras espaciais do Azure. Como tal, pode utilizar chaves de conta ao incorporá-los em seus aplicativos de cliente durante o desenvolvimento. À medida que progride além do desenvolvimento, é altamente recomendado para mover para um mecanismo de autenticação que é ao nível da produção, suportados por Tokens de acesso ou a autenticação de utilizador do Azure Active Directory. Para obter uma chave de conta para o desenvolvimento, visite a sua conta de âncoras espaciais do Azure e navegue até ao separador de "Chaves".
