A configuração de um nome de domínio personalizado em um aplicativo Web integrado ao Gerenciador de tráfego só está disponível para o tipo de preço **Standard** .  

Para obter mais informações sobre os tipos de preço do serviço de aplicativo, incluindo como alterar o tipo de preço do aplicativo, consulte [escalar verticalmente um aplicativo no Azure](../articles/app-service/manage-scale-up.md).
