A definição de um nome de domínio personalizado numa aplicação web que está integrada com o Gestor de tráfego só está disponível para o **padrão** escalão de preço.  

Para obter mais informações sobre os escalões de preço do serviço de aplicações, incluindo como alterar a sua aplicação do escalão de preço, consulte [aumentar verticalmente uma aplicação no Azure](../articles/app-service/web-sites-scale.md).
