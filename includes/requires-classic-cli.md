> [!IMPORTANT]
> Este artigo contém conteúdo que requer a CLI clássica do Azure. A versão atual da CLI do Azure não tem suporte para as funcionalidades descritos neste artigo, pelo que a CLI clássica é necessária. 
> 
> A CLI clássica pode ser instalada lado a lado com a CLI do Azure modernos, mas recomendamos que utilize a CLI do Azure para todos os novos scripts e implementações em que está disponível o suporte de funcionalidades. Para instalar a CLI clássica, veja [instalar o Azure CLI clássica](/cli/azure/install-classic-cli) e para instalar a CLI atual, veja [instalar a CLI do Azure](/cli/azure/install-azure-cli).
