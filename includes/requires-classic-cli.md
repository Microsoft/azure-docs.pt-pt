> [!IMPORTANT]
> Este artigo contém conteúdo que requer a CLI clássica do Azure. A versão atual da CLI do Azure não tem suporte para as funcionalidades descritos neste artigo, pelo que a CLI clássica é necessária. 
> 
> A CLI clássica pode ser instalada lado a lado com a CLI do Azure moderna, mas recomendamos que a CLI do Azure para todos os novos scripts e implementações em que está disponível o suporte de funcionalidades. Para instalar a CLI clássica, veja [instalar a CLI do Azure clássica](https://docs.microsoft.com/cli/azure/install-classic-cli?view=azure-cli-latest). 
> Para instalar a CLI atual, veja [instalar a CLI do Azure](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest).
