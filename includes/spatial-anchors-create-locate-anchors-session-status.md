Como descrito anteriormente, você precisa de dados ambientais suficientes capturados antes de tentar criar uma nova âncora espacial em nuvem. Isso significa que `ReadyForCreateProgress` tem de ser acima de 1, embora recomendemos que espere até estar acima de `RecommendedForCreateProgress` 1 para o fazer.
