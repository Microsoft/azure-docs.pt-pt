Como descrito anteriormente, você precisa de dados ambientais suficientes capturados antes de tentar criar uma nova âncora espacial em nuvem. Isto significa que `ReadyForCreateProgress` tem de estar acima de 1, embora recomendemos que espere até `RecommendedForCreateProgress` esteja acima de 1 para o fazer.
