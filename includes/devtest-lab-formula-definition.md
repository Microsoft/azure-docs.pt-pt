Uma fórmula em Azure DevTest Labs é uma lista de valores de propriedade padrão usados para criar uma máquina virtual (VM). Ao criar um VM a partir de uma fórmula, os valores predefinidos podem ser usados como -e, ou modificados. Tal como [as imagens personalizadas](../articles/lab-services/devtest-lab-create-template.md) e [as imagens](../articles/lab-services/devtest-lab-configure-marketplace-images.md)do Marketplace, as fórmulas fornecem um mecanismo para o fornecimento rápido de VM. Semelhante a [imagens personalizadas,](../articles/lab-services/devtest-lab-create-template.md)as fórmulas permitem criar uma imagem base a partir de um ficheiro VHD. A imagem base pode então ser usada para fornecer um novo VM. Para ajudar a decidir qual é o certo para o seu ambiente particular, consulte o artigo, [comparando imagens e fórmulas personalizadas em DevTest Labs](../articles/lab-services/devtest-lab-comparing-vm-base-image-types.md).
