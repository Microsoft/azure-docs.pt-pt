Uma fórmula em Azure DevTest Labs é uma lista de valores de propriedade padrão usados para criar uma máquina virtual (VM). Ao criar um VM a partir de uma fórmula, os valores predefinidos podem ser usados como-é, ou modificados. Tal como [imagens personalizadas](../articles/devtest-labs/devtest-lab-create-template.md) e [imagens do Marketplace,](../articles/devtest-labs/devtest-lab-configure-marketplace-images.md)as fórmulas fornecem um mecanismo para o fornecimento rápido de VM. Semelhantes a [imagens personalizadas,](../articles/devtest-labs/devtest-lab-create-template.md)as fórmulas permitem criar uma imagem base a partir de um ficheiro VHD. A imagem base pode então ser usada para provisão de um novo VM. Para ajudar a decidir qual é o certo para o seu ambiente específico, consulte o artigo, [comparando imagens e fórmulas personalizadas em DevTest Labs](../articles/devtest-labs/devtest-lab-comparing-vm-base-image-types.md).
