Se não vai continuar a utilizar esta aplicação, pode eliminar o recurso azure static Web App através dos seguintes passos:

1. Abra o [portal do Azure](https://portal.azure.com)
1. Na barra de pesquisa superior, procure a sua aplicação pelo nome que forneceu anteriormente
1. Clique na aplicação
1. Clique no botão **Eliminar**
1. Clique **Sim** para confirmar a ação de exclusão
