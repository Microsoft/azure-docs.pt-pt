[O Azure App Service](/azure/app-service/overview) permite-lhe construir e hospedar código. Este código pode ser sob a forma de aplicações web, backends móveis e APIs RESTful. Utilizando o Serviço de Aplicações, pode desenvolver o seu código utilizando a linguagem de programação à sua escolha sem gerir a infraestrutura. O Serviço de Aplicações suporta tanto o Windows como o Linux. As implementações automatizadas de qualquer repo Git são suportadas, incluindo GitHub e Azure DevOps.
