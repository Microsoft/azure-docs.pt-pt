[Serviço de aplicações do Azure](/azure/app-service/overview) permite-lhe criar e alojar o código. Esse código pode ser na forma de aplicações web, back-ends móveis e RESTful APIs. Utilizar o serviço de aplicações, pode desenvolver código usando a linguagem de programação da sua preferência sem ter de gerir infraestrutura. Serviço de aplicações suporta Windows e Linux. São suportadas implementações automáticas a partir de qualquer repositório de Git, incluindo o GitHub e do Azure DevOps.
