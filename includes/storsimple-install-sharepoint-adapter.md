#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Para instalar o Adaptador StorSimple para SharePoint
1. Copie o instalador para o servidor frontal (WFE) que também está configurado para executar a aplicação web da Administração Central do SharePoint. 
2. Utilize uma conta com privilégios de administrador para iniciar scontabilidade no servidor WFE.
3. Clique duas vezes no instalador. Começa o adaptador StorSimple para o Assistente de Configuração SharePoint. Clique **em seguida** para iniciar a instalação.
   
    ![Página de início de configuração do adaptador StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. No adaptador StorSimple para a página de configuração do SharePoint, selecione um local de instalação, digite o endereço IP para a interface de rede DATA 0 no seu dispositivo StorSimple e, em seguida, clique em **Seguinte**. 
   
    ![Página de configuração do adaptador StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Na página de confirmação de configuração, clique **em Instalar**.
   
    ![Página de confirmação de configuração do adaptador StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Clique **em Terminar** para fechar o Assistente de Configuração.
   
    ![Página terminada de configuração do adaptador StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Abra a página da Administração Central do SharePoint. Deverá ver um grupo de configuração StorSimple que contém o Adaptador StorSimple para ligações SharePoint.
8. Vá para o próximo passo: [Configurar RBS](#configure-rbs).
