**_Antes de começar,_** utilize o seletor acima para escolher o tipo de política que está a configurar. O Azure AD B2C oferece dois métodos para definir como os utilizadores interagem com as suas aplicações: embora [fluxos](../articles/active-directory-b2c/user-flow-overview.md)de utilizador predefinidos, ou através de [políticas personalizadas](../articles/active-directory-b2c/custom-policy-overview.md)totalmente configuráveis . Os passos exigidos neste artigo são diferentes para cada método.
