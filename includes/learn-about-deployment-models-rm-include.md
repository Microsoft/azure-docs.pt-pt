> [!NOTE]
> O Azure tem dois modelos de implementação diferentes que pode utilizar para criar e trabalhar com recursos: [Azure Resource Manager e clássico](../articles/azure-resource-manager/management/deployment-models.md). Este artigo abrange a utilização do modelo de implementação do Gestor de Recursos. Recomendamos o modelo de implementação do Gestor de Recursos para novas implementações em vez do modelo clássico de implementação.
