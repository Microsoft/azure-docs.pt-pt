> O Azure tem dois modelos de implementação diferentes, que pode usar para criar e trabalhar com recursos: [O Azure Resource Manager e clássica](../articles/azure-resource-manager/resource-manager-deployment-model.md). Este artigo aborda o uso do modelo de implementação do Resource Manager. Recomendamos que o modelo de implementação do Resource Manager para novas implementações em vez do modelo de implementação clássica.
