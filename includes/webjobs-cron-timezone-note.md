> [!NOTE]  
> O fuso horário padrão usado para executar expressões CRON é UTC (tempo Universal Coordenado). Para que sua expressão CRON seja executada com base em outro fuso horário, crie uma configuração de aplicativo para seu aplicativo de funções chamado WEBSITE_TIME_ZONE. Para saber mais, consulte [NCRONTAB Time Zones](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
