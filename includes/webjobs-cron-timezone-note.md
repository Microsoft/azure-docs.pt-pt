> [!NOTE]  
> O fuso horário padrão utilizado para executar expressões CRON é Tempo Universal Coordenado (UTC). Para que a sua expressão CRON funcione com base em outro fuso horário, crie uma definição de aplicação para a sua aplicação de função denominada WEBSITE_TIME_ZONE. Para saber mais, consulte os [fusos horários NCRONTAB.](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones)
