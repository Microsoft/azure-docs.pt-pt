FedEx, UPS ou DHL pode ser utilizado para enviar o pacote para o datacenter do Azure. 

- Forneça uma operadora FedEx, UPS ou DHL válida conta o número de que a Microsoft irá utilizar para enviar as unidades de volta. 
    
    - Um número de conta FedEx, UPS ou DHL é necessário para unidades de envio anterior a partir de localizações dos EUA e na Europa. 
    - Um número de conta DHL é preferido para unidades de envio anterior a partir de localizações Ásia e da Austrália. 
    - Se não tiver um número de conta, crie um [FedEx](http://www.fedex.com/us/oadr/) ou [DHL](http://www.dhl.com/) conta operadora.
- Quando os pacotes de envio, tem de seguir a [termos de serviço do Microsoft Azure](https://azure.microsoft.com/support/legal/services-terms/).
