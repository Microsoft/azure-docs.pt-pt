FedEx, UPS ou DHL podem ser usados para enviar o pacote para o centro de dados Azure. Se quiser utilizar uma transportadora diferente da FedEx/DHL, contacte a equipa de Operações da Caixa de Dados da Azure `adbops@microsoft.com`

* Forneça um número de conta de transportador FedEx, UPS ou DHL válido que a Microsoft utilizará para enviar as unidades de volta.
  * Um número de conta FedEx, UPS ou DHL é necessário para enviar unidades de volta das localizações dos EUA e da Europa.
    * Um número de conta DHL é preferido para envio de unidades de volta da Ásia e Austrália locais.
    * Se não tiver um número de conta, crie uma conta de transportadora [FedEx](https://www.fedex.com/us/oadr/) ou [DHL.](http://www.dhl.com/)
* Ao enviar os seus pacotes, deve seguir os Termos de [Serviço Microsoft Azure](https://azure.microsoft.com/support/legal/services-terms/).
* Embale corretamente os seus discos para evitar potenciais danos e atrasos no processamento.
