FedEx, UPS ou DHL podem ser usados para enviar o pacote para o centro de dados Azure. Se pretender utilizar uma transportadora que não a FedEx/DHL, contacte a equipa de Operações de Caixa de Dados da Azure em`adbops@microsoft.com`

* Forneça um número de conta de porta-aviões FedEx, UPS ou DHL válido que a Microsoft utilizará para enviar as unidades de volta.
  * Um número de conta FedEx, UPS ou DHL é necessário para envios de volta dos locais dos EUA e da Europa.
    * Um número de conta DHL é preferido para envios de volta dos locais da Ásia e Austrália.
    * Se não tiver um número de conta, crie uma conta de porta-aviões [FedEx](https://www.fedex.com/us/oadr/) ou [DHL.](http://www.dhl.com/)
* Ao enviar os seus pacotes, deve seguir os Termos de [Serviço Microsoft Azure](https://azure.microsoft.com/support/legal/services-terms/).
* Embabem corretamente os seus discos para evitar potenciais danos e atrasos no processamento.
