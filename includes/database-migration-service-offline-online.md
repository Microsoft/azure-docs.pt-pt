> [!TIP]
> Quando migrar bases de dados para o Azure com o Azure Database Migration Service, pode fazer uma migração *offline* ou *online*. Com uma migração offline, o período de indisponibilidade da aplicação começa quando inicia a migração. Com uma migração online, o período de indisponibilidade tem um tempo limitado para transitar no final da migração. Sugerimos que teste uma migração offline para determinar se o período de inatividade é aceitável; se não for, faça uma migração online.
