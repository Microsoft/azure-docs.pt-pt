A tabela seguinte descreve os limites máximos para os clusters Azure Data Explorer.

| Recurso | Limite |
| --- | --- |
| Clusters por região por subscrição | 20 |
| Instâncias por cluster | 1000 | 
| Número de bases de dados num cluster | 10,000 |
| Número de configurações de base de dados anexas num cluster | 70 |

A tabela seguinte descreve os limites das operações de gestão realizadas nos clusters Azure Data Explorer.

| Âmbito | Operação | Limite |
| --- | --- | --- |
| Cluster | ler (por exemplo, obter um cluster) | 500 por 5 minutos |
| Cluster | escrever (por exemplo, criar uma base de dados) | 1000 por hora |
