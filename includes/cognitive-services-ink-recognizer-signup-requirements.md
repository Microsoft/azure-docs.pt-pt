Você deve ter uma [conta de API de serviços cognitivas](../articles/cognitive-services/cognitive-services-apis-create-account.md) com acesso à API do reconhecedor de tinta. Se não tiver uma subscrição do Azure, pode [criar uma conta](https://azure.microsoft.com/try/cognitive-services/) gratuita. Você pode obter sua chave de assinatura do [portal do Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#get-the-keys-for-your-resource) depois de criar sua conta ou [site do Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) depois de ativar uma avaliação gratuita.
