Você deve ter uma conta API de [Serviços Cognitivos](../articles/cognitive-services/cognitive-services-apis-create-account.md) com acesso à API do Reconhecimento de Tinta. Se não tiver uma subscrição do Azure, pode [criar uma conta](https://azure.microsoft.com/try/cognitive-services/) gratuita. Pode obter a sua chave de subscrição no [portal Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#get-the-keys-for-your-resource) depois de criar a sua conta, ou [o site Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) depois de ativar um teste gratuito.
