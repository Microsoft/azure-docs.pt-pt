As identidades geridas para os recursos do Azure são uma funcionalidade do Azure Active Directory. Cada um dos [serviços do Azure que suportam as identidades geridas para os recursos do Azure](../articles/active-directory/managed-identities-azure-resources/services-support-managed-identities.md) estão sujeitos à sua própria linha de tempo. Certifique-se de que revê o estado de [disponibilidade](../articles/active-directory/managed-identities-azure-resources/services-support-managed-identities.md) das identidades geridas para o seu recurso e [problemas conhecidos](../articles/active-directory/managed-identities-azure-resources/known-issues.md) antes de começar.
