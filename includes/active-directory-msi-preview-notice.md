As identidades geridas para os recursos do Azure são uma funcionalidade do Azure Active Directory. Cada um dos [serviços do Azure que suportam as identidades geridas para os recursos do Azure](/azure/active-directory/managed-identities-azure-resources/services-support-msi) estão sujeitos à sua própria linha de tempo. Certifique-se de que revê o estado de [disponibilidade](/azure/active-directory/managed-identities-azure-resources/services-support-msi) das identidades geridas para o seu recurso e [problemas conhecidos](/azure/active-directory/managed-identities-azure-resources/known-issues) antes de começar.
