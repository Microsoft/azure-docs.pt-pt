O Azure Spatial Anchors SDK para a versão 2.9 ou posterior deve ser configurado na [Unidade 2020.3 (LTS)](https://unity3d.com/unity/whats-new/2020.3.0) com o [Quadro Plug-in Unity XR](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) utilizando os seguintes pacotes:

- Fundação AR: [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@4.0/manual/index.html)
- Plugin Windows XR (para suporte a dispositivos HoloLens) : [4.4.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr@4.4/manual/index.html)
- ARCore XR Plugin (para suporte a dispositivos Android): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.0/manual/index.html)
- ARKit XR Plugin (para suporte ao dispositivo iOS): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arkit@4.0/manual/index.html)

As versões Azure Spatial Anchors SDK 2.8 ou anteriormente não suportam o [Quadro Plug-in Unitário XR](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) e devem ser configuradas na [Unidade 2019.4 (LTS)](https://unity.com/releases/2019-lts) utilizando os seguintes pacotes:

- Fundação AR: [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html)
- Windows Mixed Reality (para suporte ao dispositivo HoloLens): [4.2.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html)
- ARCore XR Plugin (para suporte a dispositivos Android): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html)
- ARKit XR Plugin (para suporte a dispositivos iOS): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html)
