O Azure Spatial Anchors SDK deve ser configurado na <a href="https://unity.com/releases/2019-lts" target="_blank">Unidade 2019.4 (LTS)</a> utilizando os seguintes pacotes:
- Fundação AR: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- ARCore XR Plugin (para suporte a dispositivos Android): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR Plugin (para suporte a dispositivos iOS): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> O ASA SDK não suporta atualmente o <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">quadro plug-in Unity XR</a>
