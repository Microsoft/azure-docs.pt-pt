* Uma conta ativa do Azure. (Se não tiver uma conta, pode criar uma [conta gratuita](https://azure.microsoft.com/pricing/free-trial/) em apenas alguns minutos.)

* [Recomenda-se a versão python 3.7 ou posterior.](https://www.python.org/downloads/) Certifique-se de que utiliza a instalação de 32 ou 64 bits, conforme exigido pela sua configuração. Quando lhe for pedido durante a instalação, confirme que adiciona Python à variável de ambiente específica da sua plataforma. Para outras versões de Python suportadas, consulte [as funcionalidades do dispositivo Azure IoT](https://github.com/Azure/azure-iot-sdk-python/tree/master/azure-iot-device#azure-iot-device-features) na documentação SDK. Se optar por utilizar o Python 2.7, poderá ter de [instalar ou atualizar *pip,* o sistema de gestão de pacotes Python](https://pip.pypa.io/en/stable/installing/).
