#### <a name="to-exit-maintenance-mode"></a>Para sair do modo manutenção
1. No tipo de solicitação de comando:
   
     `Exit-HcsMaintenanceMode`
2. Aparecerá uma mensagem de aviso e uma mensagem de confirmação. Tipo **Y** para sair do modo manutenção.
   
    Ambos os controladores vão reiniciar. Quando o reinício estiver concluído, aparecerá outra mensagem indicando que o dispositivo se encontra no modo Normal.
