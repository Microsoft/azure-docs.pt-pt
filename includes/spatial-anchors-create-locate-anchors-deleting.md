## <a name="delete-anchors"></a>Eliminar as âncoras

Para eliminar uma âncora de geográficos na cloud, use o `DeleteAnchor()` método. A eliminar as âncoras quando já não é utilizada é uma boa prática para incluir no início, no processo de desenvolvimento e práticas. Se o fizer, mantém os recursos limpos. Além disso, mesmo que pode usar âncoras espaciais do Azure sem encargos, os preços podem ser alterados no futuro. Pode encontrar detalhes sobre os preços durante a pré-visualização em nosso [página de preços](https://azure.microsoft.com/pricing/details/spatial-anchors/).
