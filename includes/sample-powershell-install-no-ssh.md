Este exemplo requer o Azure PowerShell. Executar `Get-Module -ListAvailable Az` para ver se está instalado. Se precisar de instalar, consulte [módulo de instalar o Azure PowerShell](/powershell/azure/install-az-ps). 

Execute `Connect-AzAccount` para criar uma ligação com o Azure.
