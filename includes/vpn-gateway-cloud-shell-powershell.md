Este artigo usa cmdlets PowerShell. Para executar os cmdlets, você pode usar Azure Cloud Shell, um ambiente interativo de concha hospedado em Azure e usado através do navegador. A Azure Cloud Shell vem com os cmdlets Azure PowerShell pré-instalados.

Para executar qualquer código contido neste artigo sobre a Azure Cloud Shell, abra uma sessão Cloud Shell, use o botão **Copy** num bloco de código para copiar o código e cole-o na sessão Cloud Shell com __Ctrl+Shift+V__ no Windows e Linux, ou __Cmd+Shift+V__ no macOS. O texto colado não é executado automaticamente, por isso prima **Introduza** o código de execução.

Pode lançar a Azure Cloud Shell utilizando qualquer um dos seguintes métodos:

|  |   |
|-----------------------------------------------|---|
| Selecione **Experimentar** no canto superior direito de um bloco de código. Isto __não__ copia automaticamente o texto para a Cloud Shell. | ![Exemplo de Try It for Azure Cloud Shell](./media/cloud-shell-try-it/hdi-azure-cli-try-it.png) |
| Abra [shell.azure.com](https://shell.azure.com) no seu navegador. | [![lançar botão Azure Cloud Shell](./media/cloud-shell-try-it/hdi-launch-cloud-shell.png)](https://shell.azure.com) |
| Selecione o botão **Cloud Shell** no menu, no canto superior direito do [portal do Azure](https://portal.azure.com). | ![Botão Cloud Shell no portal do Azure](./media/cloud-shell-try-it/hdi-cloud-shell-menu.png) |
