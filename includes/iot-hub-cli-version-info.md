> [!NOTE]
> Este artigo utiliza a versão mais recente da extensão Azure IoT, chamada `azure-iot`. A versão antiga chama-se `azure-iot-cli-ext`. Só deve ter uma versão instalada de cada vez. Pode utilizar o comando `az extension list` para validar as extensões atualmente instaladas.
>
> Utilize `az extension remove azure-iot-cli-ext` para remover a versão antiga da extensão.
> Utilize `az extension add -name azure-iot` para adicionar a nova versão da extensão. 
>
> Para ver que extensões instalou, use `az extension show `.
>
