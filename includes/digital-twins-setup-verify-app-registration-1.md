Para verificar se as permissões foram configuradas corretamente, siga [este link](https://portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/RegisteredApps) para navegar para a página geral de registo da aplicação AD Azure no portal Azure. Esta página mostra todos os registos de aplicações que foram criados na sua subscrição.

Você deve ver o registo de aplicações que acabou de criar na lista geral. Selecione-o para abrir os seus detalhes.
