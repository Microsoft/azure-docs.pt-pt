## <a name="supported-ssh-key-formats"></a>Formatos de chaves SSH suportados

Azure atualmente suporta SSH protocolo 2 RSA (SSH-2) públicas-privadas pares de chaves com um comprimento mínimo de 2048 bits. Não são suportados outros formatos chaves como ED25519 e ECDSA.
