Este exemplo requer o Azure PowerShell. Executar `Get-Module -ListAvailable Az` para localizar a versão. Se precisar de instalar ou atualizar, veja [Install Azure PowerShell module](/powershell/azure/install-az-ps)(Instalar o módulo do Azure PowerShell). 

Execute `Connect-AzAccount` para criar uma ligação com o Azure.
