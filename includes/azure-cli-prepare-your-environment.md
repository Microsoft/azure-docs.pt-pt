## <a name="prepare-your-environment"></a>Preparar o ambiente

1. Para executar os comandos Azure CLI, faça uma [instalação local](/cli/azure/install-azure-cli) do Azure CLI, ou inicie [a Azure Cloud Shell](/azure/cloud-shell/quickstart) utilizando o ambiente de bash.

1. Se estiver a utilizar uma instalação local, inicie sessão com o Azure CLI utilizando o comando [de login az.](/cli/azure/reference-index#az-login)  Siga os passos apresentados no seu terminal para concluir o processo de autenticação.  Consulte [iniciar súm em Azure CLI](/cli/azure/authenticate-azure-cli) para obter um sinal adicional nas opções.
