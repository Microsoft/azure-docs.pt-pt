> [!IMPORTANT]
> Existem dois módulos PowerShell usados para interagir com o Tecido de Serviço. [A Azure PowerShell](/powershell/azure/install-az-ps) é usada para gerir os recursos Azure, como um cluster de tecido de serviço hospedado em Azure. [A Azure Service Fabric SDK](../articles/service-fabric/service-fabric-get-started.md) é usada para ligar diretamente ao cluster de Tecido de Serviço (independentemente do local onde está hospedado) e gerir o cluster, aplicações e serviços.
