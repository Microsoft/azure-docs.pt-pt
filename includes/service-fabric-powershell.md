> [!IMPORTANT]
> Existem dois módulos PowerShell usados para interagir com o Tecido de Serviço. [A Azure PowerShell](/powershell/azure/install-az-ps) é usada para gerir recursos Azure, como um cluster de Tecido de Serviço azure. [O Azure Service Fabric SDK](../articles/service-fabric/service-fabric-get-started.md) é utilizado para ligar diretamente ao cluster Service Fabric (independentemente do local onde está hospedado) e gerir o cluster, aplicações e serviços.
