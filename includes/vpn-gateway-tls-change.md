A partir de 1 de Julho de 2018, está a ser removido suporte para TLS 1.0 e 1.1 do Gateway de VPN do Azure. Gateway de VPN irá suportar apenas TLS 1.2. Apenas as ligações de ponto a site são afetadas; ligações site a site não serão afetadas. Se estiver a utilizar o TLS para VPNs ponto a site nos clientes do Windows 10, não precisa de efetuar qualquer ação. Se estiver a utilizar o TLS para ligações ponto a site em clientes Windows 7 e Windows 8, consulte o [FAQs do VPN Gateway](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) para obter instruções de atualização.
