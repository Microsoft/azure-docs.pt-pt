A partir de 1 de julho de 2018, o suporte será removido para o TLS 1.0 e 1.1 do Gateway de VPN do Azure. O Gateway de VPN irá suportar apenas o TLS 1.2. Apenas as ligações ponto a site são afetadas; ligações site a site não serão afetadas. Se estiver a utilizar TLS para as VPNs ponto a site em clientes do Windows 10, não precisa de tomar qualquer ação. Se estiver a utilizar o TLS para ligações ponto a site em clientes Windows 7 e Windows 8, consulte a [FAQ do Gateway de VPN](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) para obter instruções de atualização.
