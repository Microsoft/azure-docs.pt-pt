A partir de 1 de julho de 2018, o suporte será removido para o TLS 1.0 e 1.1 do Gateway de VPN do Azure. O Gateway de VPN irá suportar apenas o TLS 1.2. Apenas as ligações ponto-a-local são impactadas; As ligações site-to-site não serão afetadas. Se estiver a utilizar o TLS para VPNs ponto-a-site nos clientes do Windows 10, não precisa de tomar qualquer medida. Se estiver a utilizar o TLS para ligações ponto-a-site nos clientes do Windows 7 e Windows 8, consulte o [VPN Gateway FAQ](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) para obter instruções de atualização.
