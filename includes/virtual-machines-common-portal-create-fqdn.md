1. Selecione o seu VM no portal. No **nome DNS,** clique em **Configurar**.
   
   ![Clique em Configurar em nome DNS](./media/virtual-machines-common-portal-create-fqdn/configure.png)

2. Introduza o nome DNS desejado e, em seguida, **selecione Guardar**.
   
   ![Introduza um rótulo de nome DNS para o seu recurso IP público](./media/virtual-machines-common-portal-create-fqdn/configure-pane.png)


3. Para voltar à lâmina de visão geral VM, feche a lâmina *do endereço IP público.* Verifique se o *nome DNS* está agora indicado.
