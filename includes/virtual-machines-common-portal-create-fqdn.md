1. Selecione o seu VM no portal. Sob **o nome DNS,** clique **em Configurar**.
   
   ![Clique em Configurar sob o nome DNS](./media/virtual-machines-common-portal-create-fqdn/configure.png)

2. Introduza o nome DNS desejado e, em seguida, selecione **Guardar**.
   
   ![Introduza uma etiqueta de nome DNS para o seu recurso IP público](./media/virtual-machines-common-portal-create-fqdn/configure-pane.png)


3. Para voltar à lâmina de visão geral vM, feche a lâmina de *endereço IP pública.* Verifique se o *nome DNS* está agora mostrado.
