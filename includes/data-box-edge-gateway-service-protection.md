- Para aceder ao serviço Azure Stack Edge, a sua organização precisa de ter uma subscrição de Acordo de Empresa (EA) ou Cloud Solution Provider (CSP). Para mais informações, consulte [O Inscreva-se para uma subscrição do Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Como este serviço de gestão está hospedado em Azure, está protegido pelas funcionalidades de segurança do Azure. Para mais informações sobre as funcionalidades de segurança fornecidas pelo Azure, vá ao [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Para operações de gestão de SDK, pode obter a chave de encriptação para o seu recurso nas **propriedades do Dispositivo**. Só é possível visualizar a chave de encriptação se tiver permissões para a API do Graph Graph.
