- Para aceder ao serviço de borda de caixa de dados, a sua organização tem de ter uma subscrição Enterprise Agreement (EA) ou do fornecedor de soluções Cloud (CSP). Para obter mais informações, consulte [Inscreva-se uma subscrição do Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Uma vez que este serviço de gestão está alojado no Azure, é protegida pelos recursos de segurança do Azure. Para obter mais informações sobre as funcionalidades de segurança fornecida pelo Azure, vá para o [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Para operações de gestão do SDK, pode obter a chave de encriptação para o seu recurso no **propriedades do dispositivo**. Pode ver a chave de encriptação apenas se tiver permissões para a Graph API do recurso.
