### <a name="open-the-mfa-page"></a><a name="mfa"></a>Abra a página do MFA

1. Inicie sessão no Portal do Azure.
2. Navegue para **Azure Ative Directory -> Todos os utilizadores**.
3. Selecione **Autenticação multi-factor** para abrir a página de autenticação de vários fatores.

   ![Iniciar sessão](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Selecione utilizadores

1. Na página **de autenticação de vários fatores,** selecione o(s) utilizador(s) para quem pretende ativar o MFA.
2. Selecione **Ativar**.

   ![Selecione](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
