> [!IMPORTANT]
> * O Azure Security Center não pode atualmente realizar [uma pesquisa de vulnerabilidade de imagem](../articles/security-center/defender-for-container-registries-introduction.md?bc=%2fazure%2fcontainer-registry%2fbreadcrumb%2ftoc.json&toc=%2fazure%2fcontainer-registry%2ftoc.json) num registo que restringe o acesso a pontos finais privados, sub-redes selecionadas ou endereços IP. 
> * Casos de serviços Azure, incluindo serviços de devops Azure, Aplicações Web e Instâncias de Contentores Azure também não conseguem aceder a um registo restrito de contentores em rede. 
> * Algumas outras instâncias de serviço Azure podem aceder de forma segura a um registo restrito de contentores em rede. Para obter mais informações, consulte [Permitir que os serviços fidedignos acedam de forma segura a um registo restrito de contentores em rede.](../articles/container-registry/allow-access-trusted-services.md)
