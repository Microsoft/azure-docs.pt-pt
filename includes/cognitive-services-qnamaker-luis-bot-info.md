## <a name="more-information-about-bots"></a>Obter mais informações sobre bots

Para obter mais informações sobre como utilizar este serviço com bots, começar com os seguintes recursos:

|Resource|Objetivo|
|--|--|
|[O Azure Bot service](https://dev.botframework.com/)|O Azure Bot service fornece um serviço web alojado na cloud completa com um ponto de final do bot. As utilizações de serviços [Bot framework](https://github.com/Microsoft/botframework), que está disponível em várias linguagens.|
|[Arquitetura de bot](https://github.com/Microsoft/botframework)|O Microsoft Bot Framework é uma plataforma abrangente para a criação de experiências de ia de conversação e de nível empresarial.|
|[Emulador do bot Framework](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|O emulador do Bot Framework é um aplicativo de desktop de várias plataformas que permite aos desenvolvedores testar e depurar bots, criados com o SDK do Bot Framework de bot. Pode utilizar o emulador do Bot Framework para testar bots em execução localmente no seu computador ou para ligar aos bots executar remotamente.|
|[Ferramentas de bot](https://github.com/Microsoft/botbuilder-tools)|As ferramentas de arquitetura de Bot são uma coleção de ferramentas de linha de comandos para várias plataformas criados para abranger o fluxo de trabalho de desenvolvimento de bot-a-ponto. |
|[Exemplos de construtor de bot](https://github.com/Microsoft/BotBuilder-Samples)|Exemplos de bot desenvolvidos por completo foram concebidos para ilustrar os cenários, que precisará implementar a criação de bots excelente.|
