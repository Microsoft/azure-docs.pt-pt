## <a name="more-information-about-bots"></a>Mais informações sobre bots

Para mais informações sobre a utilização deste serviço com bots, comece com os seguintes recursos:

|Recurso|Objetivo|
|--|--|
|[Serviço Azure Bot](https://dev.botframework.com/)|O serviço Azure Bot fornece um serviço web completo hospedado em nuvem com um ponto final de bot. Os serviços utilizam [o quadro Bot](https://github.com/Microsoft/botframework), que está disponível em várias línguas.|
|[Bot Framework](https://github.com/Microsoft/botframework)|O Microsoft Bot Framework é uma plataforma abrangente para a construção de experiências de IA de conversação de nível empresarial.|
|[Emulador-quadro bot](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|O Bot Framework Emulator é uma aplicação de ambiente de trabalho transversal que permite aos desenvolvedores de bots testar e depurar bots construídos usando o Bot Framework SDK. Pode utilizar o Emulador Bot Framework para testar bots que correm localmente na sua máquina ou para se ligar a bots que estão a funcionar remotamente.|
|[Ferramentas bot](https://github.com/Microsoft/botbuilder-tools)|As ferramentas Bot Framework são uma coleção de ferramentas de linha de comando de plataforma cruzada projetadas para cobrir o fluxo de trabalho de desenvolvimento de bots de ponta a ponta. |
|[Amostras de construtor de bots](https://github.com/Microsoft/BotBuilder-Samples)|As amostras de bots desenvolvidas para ilustrar cenários que terá de implementar para construir grandes bots.|
