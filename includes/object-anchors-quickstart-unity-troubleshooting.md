## <a name="troubleshooting"></a>Resolução de problemas

### <a name="tips"></a>Sugestões

Se o seu objeto não estiver a ser detetado, pode experimentar os seguintes passos:

1. Verifique duas vezes se está a usar o modelo correto para o seu objeto.
2. Visualize a área de pesquisa e certifique-se de que cobre o objeto alvo.
3. Tente reduzir o `MinSurfaceCoverage` .
4. No Portal do Dispositivo Windows, clique em **Visualizações -> vistas 3D** e verifique se a verificação está completa.
