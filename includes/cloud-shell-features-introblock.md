O Azure Cloud Shell é uma experiência de shell baseada no browser para gerir e desenvolver recursos do Azure.

A Cloud Shell oferece uma experiência de concha pré-configurada e acessível ao navegador para gerir os recursos do Azure sem a sobrecarga de instalar, ver versão e manter uma máquina por si mesmo.

A Cloud Shell fornece máquinas por pedido e, como resultado, o estado da máquina não persistirá em todas as sessões. Uma vez que a Cloud Shell é construída para sessões interativas, as conchas terminam automaticamente após 20 minutos de inatividade da concha.
