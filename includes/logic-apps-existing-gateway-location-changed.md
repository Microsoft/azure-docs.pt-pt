**As perguntas e**: por que fez a localização para o meu alterações de gateway existente? <br/>
**A**: recursos de Gateway criados antes de 3 de Maio de 2017 foram movidos para as localizações originais do inquilino do Azure AD para a conta escolar ou profissional que criou desses gateways. No entanto, essas alterações não devem afetar atualmente em execução logic apps, o que devem continuar a trabalhar como de costume. Com a disponibilidade geral do gateway em Maio, localizações de recurso do gateway podem ser diferente das localizações de aplicação lógica.
