**Q**: Por que foi alterado a localização para o meu gateway existente? <br/>
**A**: Recursos de gateway criados antes de 3 de Maio de 2017 foram movidos para as localizações originais do inquilino do Azure AD para o trabalho conta escolar ou profissional que criou desses gateways. No entanto, essas alterações não devem afetar atualmente em execução logic apps, o que devem continuar a trabalhar como de costume. Com a disponibilidade geral do gateway em Maio, localizações de recurso do gateway podem ser diferente das localizações de aplicação lógica.
