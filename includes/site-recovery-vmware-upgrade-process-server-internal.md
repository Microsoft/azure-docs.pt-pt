1. Inscreva-se no servidor de processos como administrador.
2. Descarregue a versão mais recente do [Instalador de Configuração Unificado](https://aka.ms/unifiedinstaller).
3. Faça duplo clique no instalador para iniciar o processo de atualização.
4. O instalador deteta os componentes de Recuperação do Site que estão instalados e atualiza-os para a versão mais recente.
