## <a name="clean-up-resources"></a>Limpar recursos

Outros guias de introdução desta coleção têm por base este guia de introdução. Se planeia continuar com os quickstarts subsequentes ou com os tutoriais, não limpe os recursos criados neste arranque rápido. Se não pretender continuar, utilize o seguinte comando para eliminar todos os recursos criados neste arranque rápido:

```azurecli-interactive
az group delete --name myResourceGroup
```
Selecione `y` quando solicitado.
