## <a name="clean-up-resources"></a>Limpar recursos

Outros guias de introdução desta coleção têm por base este guia de introdução. Se você planeja continuar com os guias de início rápido subsequentes ou com os tutoriais, não limpe os recursos criados neste guia de início rápido. Se você não planeja continuar, use o seguinte comando para excluir todos os recursos criados neste guia de início rápido:

```azurecli-interactive
az group delete --name myResourceGroup
```
Selecione `y` quando lhe for pedido.
