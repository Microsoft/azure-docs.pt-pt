Ligue-se ao Seu VM windows utilizando o Protocolo de Ambiente de Trabalho Remoto (RDP) através do IP que passou durante a criação de VM.

1. No seu cliente, lance RDP. Ir para **iniciar** e escrever **mstsc**.
1. Introduza o endereço IP do VM e as credenciais de acesso que utilizou no ficheiro de parâmetros do modelo VM.

    ![Ligue via RDP ao Windows VM](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)
1. Pode ser necessário aprovar a ligação a uma máquina não fidedquiríss em si. 
1. Está agora ligado ao seu VM a funcionar no aparelho.
