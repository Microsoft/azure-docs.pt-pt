Ligue-se ao Seu Windows VM utilizando o Protocolo de Ambiente de Trabalho Remoto (RDP) através do IP que passou durante a criação de VM.

1. No seu cliente, lance RDP. Ir para **iniciar** e escrever **mstsc**.
1. Introduza o endereço IP do VM e as credenciais de acesso que utilizou no ficheiro de parâmetros do modelo VM.

    ![Screenshot que mostra como ligar via RDP ao seu Windows VM.](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)
1. Talvez precise de aprovar a ligação a uma máquina não fidedquiríssia. 
1. Está agora assinado no seu VM que funciona no aparelho.
