Geração de armazenamento 1 do Azure Data Lake é um repositório de hiperescala de toda a empresa para cargas de trabalho de análise de macrodados. Pode utilizar o Data Lake Storage Gen1 capturar dados de qualquer tamanho, tipo e velocidade de ingestão num único local para análises exploratórias e operacionais. Não existe nenhum limite à quantidade de dados, pode armazenar numa conta de geração 1 de armazenamento do Data Lake.

| **Recurso** | **Limite predefinido** | **Comentários** |
| --- | --- | --- |
| Número máximo de contas do Data Lake Storage Gen1, por subscrição, por região |10 | Para pedir um aumento para este limite, contacte o suporte. |
| Número máximo de acesso ACLs, por ficheiro ou pasta |32 | Este é um limite rígido. Utilize grupos para gerir o acesso com menos de entradas. |
| Número máximo de ACLs padrão, por ficheiro ou pasta |32 | Este é um limite rígido. Utilize grupos para gerir o acesso com menos de entradas. |
