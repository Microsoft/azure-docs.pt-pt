## <a name="prerequisites"></a>Pré-requisitos

Antes de poder prosseguir com as tarefas neste artigo, tem de ter:

- [As ferramentas do cliente instaladas](../articles/azure-arc/data/install-client-tools.md)

   As ferramentas incluem `azdata` , Azure Data Studio, e Azure CLI.

   Para o Azure Data Studio, instale extensões para **Azure Data CLI,** **Azure Arc**e **PostgreSQL** (dependendo dos serviços de dados)

- [Um controlador de dados Azure Arc instalado](../articles/azure-arc/data/create-data-controller.md)
