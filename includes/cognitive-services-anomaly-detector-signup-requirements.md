Tem de ter uma [conta de API dos serviços cognitivos](../articles/cognitive-services/cognitive-services-apis-create-account.md) com acesso à API de detetor de anomalias. Se não tiver uma subscrição do Azure, pode [criar uma conta](https://azure.microsoft.com/try/cognitive-services/) gratuita. Pode obter a chave de subscrição a partir da [portal do Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource) depois de criar a sua conta.
