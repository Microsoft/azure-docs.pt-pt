Verifique o tipo de contrato para determinar se tem acesso a uma conta de faturação de um Contrato de Cliente Microsoft.
 
1. Acesse a [portal do Azure](https://portal.azure.com) para verificar o acesso à conta de cobrança. Pesquise e selecione **Gerenciamento de custos + cobrança**.

   ![Procure gerenciamento de custos e cobrança, portal do Microsoft Azure](./media/billing-check-mca/billing-search-cost-management-billing.png)
 
2. Se você tiver acesso a apenas um escopo de cobrança, selecione **Propriedades** no menu. Tem acesso a uma conta de faturação de um Contrato de Cliente Microsoft se o tipo de conta de faturação for **Contrato de Cliente Microsoft**.
 
    ![Contrato do cliente da Microsoft, tipo de conta de cobrança, Propriedades portal do Microsoft Azure](./media/billing-check-mca/billing-mca-property.png)
 
3. Se tiver acesso a vários âmbitos de faturação, verifique o tipo na coluna da conta de faturação. Tem acesso a uma conta de faturação de um Contrato de Cliente Microsoft se o tipo de conta de faturação de qualquer um dos âmbitos for **Contrato de Cliente Microsoft**.
 
    [![contrato do cliente da Microsoft, tipo de conta de cobrança, lista de contas de cobrança, portal do Microsoft Azure](./media/billing-check-mca/billing-mca-in-the-list.png)](./media/billing-check-mca/billing-mca-in-the-list-zoomed-in.png#lightbox)
