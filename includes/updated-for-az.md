> [!NOTE]
>
> Este artigo foi actualizado recentemente para utilizar comandos do módulo Azure PowerShell Az novo. Este módulo é o novo módulo do PowerShell padrão do .NET para o Azure, compatível com versões do PowerShell 5.x e 6. AzureRM irá continuar a receber atualizações críticas bugfix, mas as novas funcionalidades estarão no módulo Az apenas.
>
> * Para saber mais sobre o módulo de Az e obter suporte contínuo para `AzureRM`, consulte [apresentando o novo módulo Azure PowerShell Az](/powershell/azure/new-azureps-module-az).
> * O módulo de Az já está disponível no Cloud Shell, com a compatibilidade do AzureRM ativada automaticamente.
>   Para obter instruções sobre como instalar o módulo de Az, consulte [instalar o Azure PowerShell](/powershell/azure/install-az-ps).
> * Para ativar os comandos de compatibilidade do AzureRM, execute `Enable-AzureRMAlias` após a instalação.
>   Para obter mais informações sobre a compatibilidade, consulte [migrar do Azure PowerShell AzureRM para Az](/powershell/azure/migrate-from-azurerm-to-az).
