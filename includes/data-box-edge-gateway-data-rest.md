Para os dados em repouso:

- O acesso aos dados armazenados em ações é restrito.

    - Os clientes SMB que acedam a dados de partilha precisam de credenciais de utilizador associadas à partilha. Estas credenciais são definidas quando a partilha é criada.
    - Os endereços IP dos clientes NFS que acedem a uma ação precisam de ser adicionados quando a ação é criada.
