> [!IMPORTANT]
> Ao criar identidades atribuídas ao utilizador, apenas são suportados caracteres alfanuméricos\_(0-9, a-z, A-Z), o sublinhado ( ) e o hífen (-) são suportados. Além disso, o nome deve ter pelo menos 3 caracteres e até 128 caracteres de comprimento para que a atribuição à VM/VMSS funcione corretamente. Volte mais tarde para obter atualizações. Para obter mais informações, veja [FAQs e problemas conhecidos](/azure/active-directory/managed-service-identity/known-issues).
