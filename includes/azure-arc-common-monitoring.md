- **[Monitor com dashboards Azure Data Studio](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> O Azure Data Studio é uma ferramenta de base de dados transversal para profissionais de dados que utilizam a família Microsoft de plataformas de dados em nuvem e no Windows, MacOS e Linux. O Azure Data Studio oferece uma experiência de editor moderna com o IntelliSense, snippets de código, integração de controlo de fontes e um terminal integrado. É projetado com o utilizador da plataforma de dados em mente, com gráficos incorporados de conjuntos de resultados de consulta e dashboards personalizáveis.

**[Monitor com Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana é uma aplicação web de origem aberta multi-plataforma e visualização interativa. Fornece gráficos, gráficos e alertas para a web quando ligado a fontes de dados suportadas. É expansível através de um sistema plug-in. Os utilizadores finais podem criar dashboards de monitorização complexos utilizando consultas interativas.

**[Registar análise com Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana é um painel de visualização de dados de código aberto para o elasticsearch. Fornece capacidades de visualização em cima do conteúdo indexado num cluster de elasticsearca. Os utilizadores podem criar parcelas de barras, linhas e dispersões, ou gráficos de tortas e mapas em cima de grandes volumes de dados.
