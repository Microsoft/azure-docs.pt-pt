Ao trabalhar com o modelo de implantação clássico, não pode usar a Azure Cloud Shell. Em vez disso, tem de instalar a versão mais recente dos cmdlets PowerShell (Direção de Serviços Azure) (SM) localmente no seu computador. Estes cmdlets são diferentes dos cmdlets AzureRM ou Az. Para instalar os cmdlets SM, consulte instalar os [cmdlets](/powershell/azure/servicemanagement/install-azure-ps)de Gestão de Serviço . Para obter mais informações sobre o Azure PowerShell em geral, consulte a [documentação Azure PowerShell](/powershell/azure/).
