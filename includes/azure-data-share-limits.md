A Azure Data Share permite que as organizações partilhem dados de forma simples e segura com os seus clientes e parceiros.

| **Recurso** | **Limite** |
| --- | --- |
| Número máximo de recursos de partilha de dados por subscrição do Azure |50 |
| Número máximo de ações enviadas por recurso De partilha de dados |100 |
| Número máximo de ações recebidas por recurso De partilha de dados |100 |
| Número máximo de convites por ação enviada |100 |
| Número máximo de subscrições de ações por ação enviada |100 |
| Número máximo de conjuntos de dados por ação |100 |
| Número máximo de horários instantâneos por ação |1 |
