O Azure Data Lake Analytics facilita a complexa tarefa de gerir infraestruturas distribuídas e códigos complexos. Fornece recursos dinamicamente, e pode usá-lo para fazer análises em exabytes de dados. Quando o trabalho termina, acaba com os recursos automaticamente. Paga-se apenas pelo poder de processamento que foi usado. À medida que aumenta ou diminui o tamanho dos dados armazenados ou a quantidade de cálculo utilizado, não tem de reescrever código. Para aumentar os limites predefinidos para a sua subscrição, contacte o suporte.

| **Recurso** | **Limite** | **Comentários** |
| --- | --- | --- |
| Número máximo de empregos simultâneos |20 | |
| Número máximo de unidades de análise (AUs) por conta |250 | Use qualquer combinação de até um máximo de 250 AUs em 20 postos de trabalho. Para aumentar este limite, contacte o Microsoft Support. |
| Tamanho máximo do script para submissão de emprego | 3 MB | |
| Número máximo de contas data lake analytics por região por subscrição | 5 | Para aumentar este limite, contacte o Microsoft Support. |
