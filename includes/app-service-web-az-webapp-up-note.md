> [!NOTE]
> O comando `az webapp up` executa as seguintes ações:
>
>- Criar um padrão [grupo de recursos](https://docs.microsoft.com/cli/azure/group?view=azure-cli-latest#az-group-create).
>
>- Criar um padrão [plano do serviço de aplicações](https://docs.microsoft.com/cli/azure/appservice/plan?view=azure-cli-latest#az-appservice-plan-create).
>
>- [Criar uma aplicação](https://docs.microsoft.com/cli/azure/webapp?view=azure-cli-latest#az-webapp-create) com o nome especificado.
>
>- [Implementar Zip](https://docs.microsoft.com/azure/app-service/deploy-zip) ficheiros do diretório de trabalho atual para a aplicação.
>
