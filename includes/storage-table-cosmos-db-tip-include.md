> [!TIP]
> O conteúdo deste artigo aplica-se ao armazenamento de Tabelas do Azure original. No entanto, existe agora uma oferta alternativa para armazenamento de mesas: a AZure Cosmos DB Table API. Esta API oferece um maior desempenho e disponibilidade, distribuição global e índices secundários automáticos. Também está disponível num modo [sem servidor](../articles/cosmos-db/serverless.md) baseado em consumo. Existem [algumas diferenças](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) entre a Tabela API no Azure Cosmos DB e o armazenamento de mesa Azure. Para mais informações, consulte [a Azure Cosmos DB Table API](../articles/cosmos-db/table-introduction.md).
