* [Gerir os recursos azure utilizando o Azure PowerShell](../articles/azure-resource-manager/management/manage-resources-powershell.md)
* [Gerir os recursos azure utilizando o Azure CLI](../articles/azure-resource-manager/management/manage-resources-cli.md).
