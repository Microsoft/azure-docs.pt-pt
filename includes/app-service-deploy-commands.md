* [Gerenciar recursos do Azure usando o Azure PowerShell](../articles/azure-resource-manager/management/manage-resources-powershell.md)
* [Gerencie recursos do Azure usando CLI do Azure](../articles/azure-resource-manager/management/manage-resources-cli.md).
