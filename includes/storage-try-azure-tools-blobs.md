> [!TIP]
> 
> Gerencie os recursos de armazenamento da Azure Blob com o Azure Storage Explorer. 
> [O Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/) é uma aplicação gratuita e autónoma da Microsoft que lhe permite gerir os recursos de [armazenamento do Azure Blob.](../articles/vs-azure-tools-storage-explorer-blobs.md) Utilizando o Azure Storage Explorer, pode criar, ler, atualizar e eliminar visualmente recipientes e bolhas blobs, bem como gerir o acesso aos seus recipientes e bolhas.
