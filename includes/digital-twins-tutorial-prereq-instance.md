### <a name="prepare-an-azure-digital-twins-instance"></a>Prepare uma instância Azure Digital Twins

Para completar este tutorial, você precisará de uma **instância Azure Digital Twins** para programar contra. 

Se já tiver uma instância Azure Digital Twins configurada a partir de trabalhos anteriores, pode usar esse caso e saltar para a secção seguinte.

Caso contrário, siga estes passos para configurar o seu caso:
1. **Crie um caso** utilizando as instruções em [*Como-a-fazer: Criar uma instância Azure Digital Twins*](../articles/digital-twins/how-to-set-up-instance.md). 
2. **Crie um registo de aplicações Azure Ative Directory para o seu exemplo** com a Criar uma secção de registo de *aplicações* de [*Como-a-: Autenticar uma aplicação do cliente.*](../articles/digital-twins/how-to-authenticate-client.md#create-an-app-registration)
