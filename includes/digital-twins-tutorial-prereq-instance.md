### <a name="prepare-an-azure-digital-twins-instance"></a>Prepare uma instância Azure Digital Twins

Para completar este tutorial, você precisará de uma **instância Azure Digital Twins** para programar contra. Se já tem uma instância Azure Digital Twins configurada a partir de trabalhos anteriores, pode usar esse caso.

* Caso contrário, **configurar uma instância e autenticação** utilizando as instruções em [*Como-a-fazer: Configurar uma instância e autenticação*](../articles/digital-twins/how-to-set-up-instance-portal.md). As instruções também contêm passos para verificar se completou cada passo com sucesso e estão prontos para passar a usar a sua nova instância.

Neste tutorial, você precisará dos seguintes valores a partir de quando configurar o seu exemplo. Se precisar de voltar a recolher estes valores, utilize os links abaixo para as secções correspondentes no artigo de configuração para os encontrar no [portal Azure](https://portal.azure.com).
* **_Nome_** de instância Azure Digital Twins[(encontre no portal)](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values)
* Nome **_de anfitrião_** de exemplo de Azure Digital Twins[(encontre no portal)](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values)
* Aplicação de registo de aplicações AD AZure **_(cliente) ID_** [(find in portal)](../articles/digital-twins/how-to-set-up-instance-portal.md#collect-important-values)
* Diretório de registo de aplicativoS Azure **_AD (inquilino) ID_** [(find in portal)](../articles/digital-twins/how-to-set-up-instance-portal.md#collect-important-values)
