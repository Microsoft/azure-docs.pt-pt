### <a name="prepare-an-azure-digital-twins-instance"></a>Prepare uma instância Azure Digital Twins

Para completar este tutorial, você precisará de uma **instância Azure Digital Twins** para programar contra. Se já tiver uma instância Azure Digital Twins configurada a partir de trabalhos anteriores, pode usar esse caso e saltar para a secção seguinte.

Caso contrário, **configurar uma instância e autenticação** utilizando as instruções em [*Como-a-fazer: Configurar uma instância e autenticação*](../articles/digital-twins/how-to-set-up-instance-scripted.md). As instruções também contêm passos para verificar se completou cada passo com sucesso e estão prontos para passar a usar a sua nova instância.
