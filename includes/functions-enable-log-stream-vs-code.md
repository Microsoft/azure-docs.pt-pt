Para ativar os registos de transmissão em fluxo para a sua aplicação de função no Azure:

1. chave de prima F1 para abrir a paleta de comandos, em seguida, procure e execute o comando **as funções do Azure: Iniciar transmissão em fluxo registos**.

1. Selecione a sua aplicação de função no Azure, em seguida, selecione **Sim** para ativar o registo de aplicação para a sua aplicação de função.

1. Acionar as suas funções no Azure e tenha em atenção que os dados de registo são apresentados na janela de saída no Visual Studio Code.

1. Quando terminar, lembre-se executar o comando **as funções do Azure: Parar a transmissão em fluxo registos** para desativar o registo para a aplicação de funções.
