## <a name="overview-of-azure-resource-manager-templates"></a>Descrição Geral dos Modelos do Azure Resource Manager
Os modelos Azure Resource Manager permitem-lhe especificar de forma declarativa a infraestrutura IaaS do Azure em linguagem JSON, definindo as dependências entre os recursos.

Veja [Criação de modelos de extensão](../articles/virtual-machines/windows/template-description.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json) para saber mais sobre a criação de modelos para utilizar extensões.

Neste artigo, aprenderemos sobre resolução de problemas de algumas das falhas de extensão de VM comuns.
