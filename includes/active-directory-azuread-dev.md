> [!IMPORTANT]
> A [plataforma de identidade da Microsoft (v2.0)](/azure/active-directory/develop/v2-overview) é uma evolução da plataforma de desenvolvimento azure Ative Directory (Azure AD) (v1.0). Permite aos programadores criar aplicações que iniciam sessão em todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, como o Microsoft Graph, ou APIs criadas pelos programadores.
> Este conteúdo destina-se ao ponto final do Azure AD v1.0. Recomendamos que utilize o ponto final v2.0 para novos projetos. Para mais informações, leia [por que atualizar para a plataforma de identidade da Microsoft (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) bem como limitações da [plataforma de identidade microsoft.](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)
