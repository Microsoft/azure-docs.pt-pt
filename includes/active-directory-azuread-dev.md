> [!IMPORTANT]
> [A plataforma de identidade Microsoft (v2.0)](/azure/active-directory/develop/v2-overview) é uma [evolução da plataforma de desenvolvimento Azure Ative Directory (Azure AD) (v1.0)](../articles/active-directory/azuread-dev/about-microsoft-identity-platform.md). Permite aos programadores criar aplicações que iniciam sessão em todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, como o Microsoft Graph, ou APIs criadas pelos programadores.
> Este conteúdo é para os mais velhos, Azure AD v1.0 ponto final. Recomendamos que utilize o ponto final v2.0 para novos projetos. Para mais informações, leia [Por que atualizar para a plataforma de identidade da Microsoft (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) bem como [as limitações da plataforma de identidade da Microsoft.](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)
