Para aceder ao armazenamento do Azure, terá uma subscrição do Azure. Se ainda não tiver uma subscrição, em seguida, crie uma [conta gratuita](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) antes de começar.

Todos os acessos ao armazenamento do Azure ocorre através de uma conta de armazenamento. Neste início rápido, crie uma conta de armazenamento com o [portal do Azure](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM), o Azure PowerShell ou da CLI do Azure. Para obter ajuda na criação da conta, veja [Criar uma conta de armazenamento](../articles/storage/common/storage-quickstart-create-account.md).
