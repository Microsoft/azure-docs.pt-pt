Para aceder ao armazenamento do Azure, terá uma subscrição do Azure. Se ainda não tiver uma subscrição, em seguida, crie uma [conta gratuita](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) antes de começar.

Todos os acessos ao armazenamento do Azure ocorre através de uma conta de armazenamento. Se ainda não tiver uma conta de armazenamento, pode criar facilmente um que utilize o [portal do Azure](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Também pode criar uma conta de armazenamento com o Azure PowerShell ou a CLI do Azure. Para obter ajuda na criação da conta, veja [Criar uma conta de armazenamento](../articles/storage/common/storage-quickstart-create-account.md).
