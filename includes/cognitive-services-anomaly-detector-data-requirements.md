> [!NOTE]
> Para [melhores resultados](../articles/cognitive-services/anomaly-detector/concepts/anomaly-detection-best-practices.md) ao utilizar a API de detetor de anomalias, os seus dados de séries de tempo de formato JSON devem incluir:
> * pontos de dados separados pelo mesmo intervalo, com mais do que 10% do número esperado de pontos em falta.
> * pontos de dados, pelo menos, 12 se seus dados não tem um padrão de limpeza sazonal.
> * pelo menos 4 ocorrências de padrão se seus dados tiver um padrão de limpeza sazonal.
