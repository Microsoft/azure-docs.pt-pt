Configure o software parceiro para ligar ao cliente. Para se ligar ao software do parceiro, normalmente precisaria das seguintes informações (podem variar) recolhidas a partir da página **Connect e copy** do UI web local no passo anterior:

- Nome da conta de armazenamento
- Chave de acesso
- Ponto final de serviço blob
 
Para verificar se a ligação está estabelecida com sucesso, utilize o Storage Explorer para se ligar a uma conta de armazenamento externa. Se não tiver o Storage Explorer, baixe o Storage Explorer.

Se esta for a primeira vez que estiver a utilizar o Storage Explorer, tem de executar os seguintes passos.

1. A partir da barra de comando superior, vá a **Editar > Target Azure Stack**.

    ![Painel de instrumentos da Caixa de Dados](media/data-box-configure-partner-software/data-box-connect-via-rest-6.png)

2. Reinicie o Explorador de Armazenamento para que as alterações entrem em vigor.

    ![Painel de instrumentos da Caixa de Dados](media/data-box-configure-partner-software/data-box-connect-via-rest-7.png)
