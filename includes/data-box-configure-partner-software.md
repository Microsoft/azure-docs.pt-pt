Configure o software de parceiro para ligar ao cliente. Para ligar para o software de parceiro, normalmente, terá de ter as seguintes informações (podem variar) obtidas com o **Connect e a cópia** página da web local da interface do Usuário no passo anterior:

- Nome da conta de armazenamento
- Chave de acesso
- Ponto final de serviço blob
 
Para verificar que a ligação é estabelecida com êxito, utilize o Explorador de armazenamento para anexar a uma conta de armazenamento externo. Se não tiver o Explorador de armazenamento, transfira o Explorador de armazenamento.

Se esta for a primeira vez que está a utilizar o Explorador de armazenamento, terá de efetuar os passos seguintes.

1. A partir da barra de comando superior, aceda a **Editar > Azure Stack de destino**.

    ![Dashboard de caixa de dados](media/data-box-configure-partner-software/data-box-connect-via-rest-6.png)

2. Reinicie o Explorador de armazenamento para que as alterações entrem em vigor.

    ![Dashboard de caixa de dados](media/data-box-configure-partner-software/data-box-connect-via-rest-7.png)
