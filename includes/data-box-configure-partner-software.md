Configure o software parceiro para se conectar ao cliente. Para se ligar ao software do parceiro, normalmente necessitaria das seguintes informações (pode variar) recolhidas a partir da página **de Connect e copy** da UI web local no passo anterior:

- Nome da conta de armazenamento
- Chave de acesso
- Ponto final de serviço blob
 
Para verificar se a ligação está estabelecida com sucesso, utilize o Storage Explorer para anexar a uma conta de armazenamento externo. Se não tiver Storage Explorer, descarregue o Storage Explorer.

Se esta for a primeira vez que utiliza o Storage Explorer, tem de executar os seguintes passos.

1. A partir da barra de comando superior, vá ao **Edit > Target Azure Stack**.

    ![Painel de caixa de dados](media/data-box-configure-partner-software/data-box-connect-via-rest-6.png)

2. Reinicie o Explorador de Armazenamento para que as alterações entrem em vigor.

    ![Painel de instrumentos da Caixa de Dados 2](media/data-box-configure-partner-software/data-box-connect-via-rest-7.png)
