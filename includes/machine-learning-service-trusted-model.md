> [!IMPORTANT]
> Deve utilizar apenas modelos que cria ou obtém a partir de uma fonte fidedigna. Deve tratar os modelos serializados como código, porque as vulnerabilidades de segurança foram descobertas em vários formatos populares. Além disso, os modelos podem ser intencionalmente treinados com intenção maliciosa de fornecer uma saída tendenciosa ou imprecisa.
