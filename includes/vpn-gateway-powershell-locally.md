Também pode instalar e executar os cmdlets Azure PowerShell localmente no seu computador. Os cmdlets PowerShell são atualizados frequentemente. Se não tiver instalado a versão mais recente, os valores especificados nas instruções podem falhar. Para encontrar as versões do Azure PowerShell `Get-Module -ListAvailable Az` instaladas no seu computador, utilize o cmdlet. Para instalar ou atualizar, consulte [Instalar o módulo PowerShell Azure](/powershell/azure/install-az-ps).
