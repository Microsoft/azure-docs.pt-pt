Um controlo de segurança é uma qualidade ou característica de um serviço Azure que contribui para a capacidade do serviço de prevenir, detetar e responder a vulnerabilidades de segurança.

Para cada controlo, utilizamos "Sim" ou "Não" para indicar se está atualmente em vigor para o serviço, "N/A" para um controlo que não é aplicável ao serviço. Também podemos fornecer uma nota ou links para mais informações sobre um atributo.
