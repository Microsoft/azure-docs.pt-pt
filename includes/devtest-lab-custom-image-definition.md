No Azure DevTest Labs, as imagens personalizadas permitem criar VMs rapidamente sem esperar que todo o software necessário seja instalado na máquina-alvo. As imagens personalizadas permitem-lhe pré-instalar todo o software de que necessita num ficheiro VHD e, em seguida, utilizar o ficheiro VHD para criar um VM. Como o software já está instalado, o tempo de criação de VM é muito mais rápido. Além disso, imagens personalizadas podem ser usadas para clonar VMs criando uma imagem personalizada a partir de um VM e, em seguida, criando VMs com base nessa imagem personalizada.
