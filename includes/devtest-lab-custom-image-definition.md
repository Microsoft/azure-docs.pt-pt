No Azure DevTest Labs, as imagens personalizadas permitem-lhe criar VMs rapidamente sem esperar que todo o software necessário seja instalado na máquina-alvo. As imagens personalizadas permitem-lhe pré-instalar todo o software que necessita num ficheiro VHD e, em seguida, usar o ficheiro VHD para criar um VM. Como o software já está instalado, o tempo de criação vM é muito mais rápido. Além disso, imagens personalizadas podem ser usadas para clonar VMs criando uma imagem personalizada a partir de um VM, e, em seguida, criando VMs com base nessa imagem personalizada.
