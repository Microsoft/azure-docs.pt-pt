Para evitar que ficheiros desnecessários sejam incluídos no instantâneo, faça um ficheiro de ignorar `.gitignore` `.amlignore` (ou) no diretório. Adicione os ficheiros e diretórios para excluir este ficheiro. Para obter mais informações sobre a sintaxe a utilizar dentro deste ficheiro, consulte [sintaxe e padrões](https://git-scm.com/docs/gitignore) para `.gitignore` . O `.amlignore` ficheiro usa a mesma sintaxe. _Se ambos os ficheiros existirem, o `.amlignore` ficheiro tem precedência._
