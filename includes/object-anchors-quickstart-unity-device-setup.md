## <a name="set-up-your-device"></a>Configurar o dispositivo

Para implementar aplicativos para os seus HoloLens, terá de emparelhar os HoloLens com o seu computador.

1. Nos seus HoloLens, navegue para **Settings -> Update & Security -> For Developers**.
2. Clique **em Emparelhar** e deixe o ecrã aberto até introduzir o PIN no Estúdio Visual durante a sua primeira implantação.
