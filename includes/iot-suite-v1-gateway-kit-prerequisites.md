## <a name="prerequisites"></a>Pré-requisitos

Para concluir este tutorial, precisa de uma subscrição ativa do Azure.

> [!NOTE]
> Se não tiver uma conta, pode criar uma de avaliação gratuita em apenas alguns minutos. Para obter mais detalhes, consulte [Azure Free Trial (Avaliação Gratuita do Azure)][lnk-free-trial].

### <a name="required-software"></a>Software necessário

Terá de cliente SSH no seu computador de ambiente de trabalho que lhe permite aceder remotamente a linha de comandos a NUC Intel.

- Windows não inclui um cliente SSH. Recomendamos que utilize [PuTTY](http://www.putty.org/).
- A maioria das distribuições de Linux e Mac OS incluem o utilitário da linha de comandos do SSH.

### <a name="required-hardware"></a>Hardware necessário

Um computador de secretária que lhe permite ligar remotamente à linha de comandos no NUC Intel.

[Kit de Gateway de IoT comercial][lnk-starter-kits]. Este tutorial utiliza os seguintes itens do kit:

- Intel® NUC Kit DE3815TYKE com cartões de expansão de memória e Bluetooth 4g
- Adaptador de energia
