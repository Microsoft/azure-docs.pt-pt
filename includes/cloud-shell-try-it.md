## <a name="use-azure-cloud-shell"></a>Utilizar o Azure Cloud Shell

Azure aloja Azure Cloud Shell, um ambiente de shell interativo que pode utilizar através do browser. Cloud Shell permite-lhe utilizar qualquer um `bash` ou `PowerShell` para trabalhar com os serviços do Azure. Pode utilizar os comandos de pré-instaladas do Cloud Shell para executar o código neste artigo, sem precisar instalar nada no seu ambiente local.

Para iniciar o Azure Cloud Shell:

| Opção | Exemplo/de ligação |
|-----------------------------------------------|---|
| Selecione **Experimentar** no canto superior direito de um bloco de código. Selecionando **experimentar** não copia automaticamente o código para o Cloud Shell. | ![Exemplo de Experimente-o para o Azure Cloud Shell](./media/cloud-shell-try-it/cli-try-it.png) |
| Aceda a [ https://shell.azure.com ](https://shell.azure.com) ou selecione o **Launch Cloud Shell** botão para abrir o Cloud Shell no seu browser. | <a href="https://shell.azure.com" title="Inicie o Azure Cloud Shell"><img name="launch-cloud-shell" src="https://docs.microsoft.com/azure/includes/media/cloud-shell-try-it/launchcloudshell.png" /></a> |
| Selecione o **Cloud Shell** botão na barra de menu do canto superior direito no [portal do Azure](https://portal.azure.com). | ![Botão Cloud Shell no portal do Azure](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Para executar o código neste artigo no Azure Cloud Shell:

1. Inicie Cloud Shell.
1. Selecione o **cópia** botão num bloco de código para copiar o código. 
1. Cole o código para a sessão do Cloud Shell com **Ctrl**+**Shift**+**V** em Windows e Linux, ou **Cmd** + **Shift**+**V** em macOS. 
1. Prima **Enter** para executar o código.
