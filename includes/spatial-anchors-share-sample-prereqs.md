## <a name="prerequisites"></a>Pré-requisitos

Para concluir este tutorial, confirme que tem:

* Leia a visão geral das [âncoras espaciais de Azure](../articles/spatial-anchors/overview.md).
* Completou um dos [Quickstarts de 5 minutos.](../articles/spatial-anchors/index.yml)
* Conhecimento básico sobre C# e Unidade.
* Conhecimento básico no <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore</a> se quiser utilizar o Android ou <a href="https://developer.apple.com/arkit/" target="_blank">o ARKit</a> se quiser utilizar o iOS.
* Um computador Windows no qual <a href="https://www.visualstudio.com/downloads/" target="_blank">o Visual Studio 2017</a> ou mais tarde é instalado com a ASP.NET e a carga de trabalho **de desenvolvimento web.**
* O [.NET Core 3.1 SDK](https://dotnet.microsoft.com/download).
* Um ou mais dispositivos (iOS ou Android) para implementar e executar uma aplicação.
  * Se estiver a utilizar o Android, tem de ter:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> ou mais tarde, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.4 (LTS)</a>e <a href="https://git-scm.com/download/win" target="_blank">Git for Windows</a> instalados no seu computador Windows.
    * Um dispositivo Android <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">habilitado para o programador</a> e <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">capaz de ARCore.</a>
  * Se estiver a usar o iOS, tem de ter:
    * Um computador macOS no qual o <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> ou mais tarde, <a href="https://cocoapods.org" target="_blank">Os Cacau</a>e <a href="https://unity3d.com/get-unity/download" target="_blank">a Unidade 2019.4 (LTS)</a> estão instalados.
    * Um dispositivo iOS <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">compatível com ARKit ativado pelo</a> desenvolvedor.
    * Git instalado via Homebrew. Introduza o seguinte comando numa única linha no Terminal: `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` Em seguida, corra `brew install git` .
