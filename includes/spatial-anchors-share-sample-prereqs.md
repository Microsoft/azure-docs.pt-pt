## <a name="prerequisites"></a>Pré-requisitos

Para concluir este tutorial, confirme que tem:

* Um computador Windows, no qual <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017</a> ou posterior é instalado com o **desenvolvimento na web e ASP.NET** carga de trabalho.
* O [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* Um ou mais dispositivos (iOS ou Android) em que pretende implementar e executar uma aplicação.
  * Se estiver a utilizar o Android, tem de ter:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> ou posterior, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> ou posterior, e <a href="https://git-scm.com/download/win" target="_blank">Git para Windows</a> instalado no seu computador Windows.
    * R <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">habilitados para desenvolvedor</a> e <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">com capacidade de ARCore</a> dispositivo Android.
  * Se estiver a utilizar iOS, tem de ter:
    * Um computador macOS, no qual <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> ou posterior <a href="https://cocoapods.org" target="_blank">CocoaPods</a>, e <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> ou posterior está instalado.
    * Um desenvolvedor-ativado <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">compatível com ARKit</a> dispositivo iOS.
    * Git instalado através do Homebrew. Introduza o seguinte comando numa única linha no Terminal: `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` Em seguida, execute `brew install git`.
