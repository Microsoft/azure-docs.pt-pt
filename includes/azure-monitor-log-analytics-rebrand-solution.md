>[!NOTE]
>Versões anteriores deste artigo referem-se ao Log Analytics como o seu próprio serviço. A sua funcionalidade não mudou, mas foi renomeada para a funcionalidade de registo do Azure Monitor. Com este renome, este artigo descreve os seus dados sendo armazenados num espaço de trabalho log analytics no Azure Monitor. Para obter mais informações, consulte [as alterações de marca Azure Monitor](../articles/azure-monitor/terminology.md).
