>[!NOTE]
>Versões anteriores deste artigo, o Log Analytics é referido como seu próprio serviço. Sua funcionalidade não foi alterada, mas ele é foi renomeado para a funcionalidade de registo do Azure Monitor. Com esta mudança de nome, este artigo descreve os dados sejam armazenados numa área de trabalho do Log Analytics no Azure Monitor. Para obter mais informações, consulte [do Azure Monitor, imagem corporativa alterações](../articles/azure-monitor/azure-monitor-rebrand.md).
