Verifique se consegue ver dados sobre a máquina virtual a partir do painel do portal Azure.

1. No portal do Azure, selecione **Dashboard**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="Navegação do portal Azure para painel":::

1. Na página do painel de instrumentos, selecione **Simple VM Dashboard**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="Navegue para o simples painel VM":::

1. Reveja o painel de instrumentos que o modelo de Gestor de Recursos Azure (modelo ARM) criou. Pode ver que alguns dos conteúdos são estáticos, mas também existem gráficos que mostram o desempenho do VM que criou no início.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="Rever simples painel VM":::
