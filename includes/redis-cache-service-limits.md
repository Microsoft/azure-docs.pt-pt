Azure Cache para os limites e tamanhos redis são diferentes para cada nível de preços. Para ver os níveis de preços e os seus tamanhos associados, consulte [a Cache Azure para os preços de Redis.](https://azure.microsoft.com/pricing/details/cache/)

Para obter mais informações sobre o Cache Azure para os limites de configuração do Redis, consulte a [configuração do servidor Redis padrão](../articles/azure-cache-for-redis/cache-configure.md#default-redis-server-configuration).

Como a configuração e gestão do Azure Cache para instâncias Redis é feita pela Microsoft, nem todos os comandos Redis são suportados em Azure Cache para Redis. Para obter mais informações, consulte [os comandos Redis não suportados em Cache Azure para Redis](../articles/azure-cache-for-redis/cache-configure.md#redis-commands-not-supported-in-azure-cache-for-redis).
