## <a name="create-a-cloud-spatial-anchor"></a>Criar uma âncora de geográficos de cloud

Para criar uma âncora de cloud, crie primeiro uma âncora no sistema de AR de sua plataforma e, em seguida, crie um equivalente na cloud. Utilizar o `CreateAnchorAsync()` método.
