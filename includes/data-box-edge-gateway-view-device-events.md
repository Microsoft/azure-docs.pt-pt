Tome os seguintes passos no portal Azure para ver um evento do dispositivo. 

1. No portal Azure, vá ao seu recurso Azure Stack Edge / Data Box Gateway e, em seguida, vá a **Eventos de Monitorização > Dispositivo**.
2. Selecione um evento e veja os detalhes do alerta. Tome as medidas adequadas para resolver a condição de alerta.

    ![Selecione detalhes do evento e vista](media/data-box-edge-gateway-view-device-events/view-device-events.png)
