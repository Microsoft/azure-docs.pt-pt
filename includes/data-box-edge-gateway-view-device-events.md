Dê os seguintes passos no portal Azure para ver um evento de dispositivo.

1. No portal Azure, vá ao seu recurso Data Box Edge/ Data Box Gateway e, em seguida, vá para **monitorizar > eventos do Dispositivo**.
2. Selecione um evento e veja os detalhes do alerta. Tome as medidas adequadas para resolver a condição de alerta.

    ![Selecione detalhes do evento e ver](media/data-box-edge-gateway-view-device-events/view-device-events.png)
