## <a name="scenario"></a>Cenário
Para melhor ilustrar como configurar um endereço IP estático para uma VM, este documento utilizará o cenário abaixo.

![Cenário de VNet](./media/virtual-networks-static-ip-scenario-include/static-ip-scenario.png)

Neste cenário criará uma VM com o nome **DNS01** no **front-end** sub-rede e defina-o para utilizar um endereço IP estático de **192.168.1.101**.
