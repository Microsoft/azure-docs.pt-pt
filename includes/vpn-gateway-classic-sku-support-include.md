>  * Para obter informações sobre como trabalhar com os SKUs (Basic, Standard e HighPerformance) de gateway legados, veja [trabalhar com SKUs (SKUs legados) de gateway VPN](../articles/vpn-gateway/vpn-gateway-about-skus-legacy.md).
>  * Para SKUs de gateway ExpressRoute, veja [Gateways de rede Virtual para o ExpressRoute](../articles/expressroute/expressroute-about-virtual-network-gateways.md).
