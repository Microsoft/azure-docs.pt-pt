>  * Para obter informações sobre como trabalhar com o gateway legado SKUs (básico, Standard e HighPerformance), consulte [trabalhar com o gateway de VPN SKUs (SKUs legados)](../articles/vpn-gateway/vpn-gateway-about-skus-legacy.md).
>  * Para SKUs de gateway ExpressRoute, consulte [Gateways da Virtual Network para o ExpressRoute](../articles/expressroute/expressroute-about-virtual-network-gateways.md).
