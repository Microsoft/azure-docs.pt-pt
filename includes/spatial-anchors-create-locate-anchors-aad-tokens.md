### <a name="azure-active-directory-authentication"></a>Autenticação do Azure Active Directory

A Azure Spatial Anchors também permite que as aplicações se autentiquem com tokens De AD (Ative Directory) do utilizador. Por exemplo, pode utilizar fichas Azure AD para se integrar com âncoras espaciais Azure. Se uma Empresa mantiver os utilizadores em Azure AD, pode fornecer um token Azure AD no Azure Spatial Anchors SDK. Fazê-lo permite-lhe autenticar diretamente o serviço De Âncoras Espaciais Azure para uma conta que faz parte do mesmo inquilino azure AD.
