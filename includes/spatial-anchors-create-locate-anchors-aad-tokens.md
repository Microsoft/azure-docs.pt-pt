### <a name="azure-active-directory-authentication"></a>Autenticação do Azure Active Directory

Âncoras espaciais do Azure também permite que os aplicativos para se autenticar com tokens do utilizador do Azure AD (Active Directory). Por exemplo, pode utilizar tokens de acesso do Azure AD para integrar com as âncoras espaciais do Azure. Se uma empresa mantém os utilizadores no Azure AD, pode fornecer um token de utilizador do Azure AD no Azure geográficos âncoras SDK. Se o fizer, permite-lhe autenticar diretamente para o serviço de âncoras espaciais do Azure para uma conta que faz parte do mesmo inquilino do Azure AD.
