### <a name="azure-active-directory-authentication"></a>Autenticação do Azure Active Directory

A Azure Spatial Anchors também permite que as aplicações autentem com tokens AZURE AD (Ative Directory) do utilizador. Por exemplo, pode utilizar fichas AD AZure para se integrar com âncoras espaciais Azure. Se uma Enterprise mantiver os utilizadores em Azure AD, pode fornecer um token AD do utilizador Azure no Azure Spatial Anchors SDK. Ao fazê-lo, permite-lhe autenticar diretamente o serviço Azure Spatial Anchors para uma conta que faz parte do mesmo inquilino da AD Azure.
