## <a name="create-a-repository"></a>Criar um repositório

Este artigo usa um repositório de modelos GitHub para facilitar o seu início. O modelo possui uma aplicação inicial usada para implementar usando aplicações web estáticas Azure.

# <a name="no-framework"></a>[Sem Enquadramento](#tab/vanilla-javascript)

1. Certifique-se de que está inscrito no GitHub e navegue para o seguinte local para criar um novo repositório:
    1. https://github.com/staticwebdev/vanilla-basic/generate - se a ligação não funcionar, por favor [inscreva-se no GitHub](https://github.com/login?return_to=%2Fstaticwebdev%2Fvanilla-basic%2Fgenerate) e tente novamente
1. Nomeie o seu repositório **my-first-static-web-app**

> [!NOTE]
> A azure Static Web Apps requer pelo menos um ficheiro HTML para criar uma aplicação web. O repositório que cria neste passo inclui um único ficheiro _index.html._
