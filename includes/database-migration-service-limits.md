O Azure Database Migration Service é um serviço totalmente gerido projetado para permitir migrações sem emenda de múltiplas fontes de base de dados para plataformas de dados Azure com tempo de inatividade mínimo.

| **Recurso** | **Limite** | **Comentários** |
| --- | --- | --- |
| Número máximo de serviços por subscrição, por região |2 | Para solicitar um aumento para este limite, contacte o suporte. |
