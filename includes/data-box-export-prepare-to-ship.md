O passo final é preparar o dispositivo para envio. Neste passo, todas as partilhas do dispositivo são colocadas offline. Não é possível aceder às partilhas depois de iniciar este processo. Também pode optar por limpar o dispositivo pelo qual os dados locais do dispositivo são apagados permanentemente. Uma vez concluído o passo, o seu ecrã de tinta-e-tinta mostrará a etiqueta de envio de devolução.

> [!IMPORTANT]
> - Pode optar por apagar os dados do dispositivo permanentemente neste passo e limpar o dispositivo. Os dados da sua conta de Armazenamento Azure ficarão e acumularão encargos. Recomendamos que exclua estes dados apenas depois de ter verificado que a cópia de dados para um servidor de dados no local está completa.

1. Vá a **Preparar-se para enviar** e selecione **Iniciar a preparação**. 
   
    ![Preparar para enviar 1](media/data-box-export-prepare-to-ship/prepare-to-ship1.png)

 
2. A preparação para envio e iniciada e as partilhas do dispositivo ficam offline. Por predefinição, a limpeza do dispositivo é realizada e a Data Box apaga os dados nos seus discos. 


    ![Preparar para enviar 2](media/data-box-export-prepare-to-ship/prepare-to-ship2.png)

    Pode optar por não fazer o procedimento de limpeza desmarcando a caixa de combinação. Neste caso, os dados do dispositivo são apagados mais tarde no datacenter.

    ![Preparar para enviar 3](media/data-box-export-prepare-to-ship/prepare-to-ship3.png)


3. Assim que a **preparação para o navio** começar, verá uma notificação de progresso com uma barra de progresso.

    ![Preparar para enviar 4](media/data-box-export-prepare-to-ship/prepare-to-ship4.png)

4. Após a preparação para o envio e a limpeza do dispositivo (se selecionada) estiver concluída, o visor de tinta E no seu dispositivo também mostrará a devolução da etiqueta de envio. 

    Você também vê um número de referência de conclusão que é usado para coisas diferentes dependendo do país em que você está localizado. Também pode ver as instruções para a remessa de devolução. Dependendo do tipo de envio que escolheu, a Microsoft geriu ou geriu por si própria, as instruções variarão. 
        
    ![Preparar para enviar 5](media/data-box-export-prepare-to-ship/prepare-to-ship5.png)


5. Encerre o dispositivo. Localize o botão para ligar/desligar no painel frontal operativo do dispositivo. Desligue o dispositivo.

    ![Botão para ligar/desligar do Data Box](media/data-box-export-prepare-to-ship/data-box-powered-door-open.png)

6. Remova os cabos. O passo seguinte é enviar o dispositivo para a Microsoft.
