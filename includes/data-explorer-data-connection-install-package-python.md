## <a name="install-python-package"></a>Instalar pacote do Python

Para instalar o pacote do Python para o Data Explorer do Azure (Kusto), abra um prompt de comando que tenha o Python em seu caminho. Execute o seguinte comando:

```python
pip install azure-common
pip install azure-mgmt-kusto
```
