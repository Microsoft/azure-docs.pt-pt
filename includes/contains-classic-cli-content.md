> [!IMPORTANT]
> Este artigo contém conteúdo que requer a CLI clássica do Azure. Esta é uma versão mais antiga da CLI do Azure que deve ser utilizada apenas com o modelo de implementação clássica do Azure.
>
> A CLI clássica pode ser instalada lado a lado com a CLI do Azure moderna, mas recomendamos que utilize a CLI do Azure para todos os novos scripts e implementações.
> Para instalar a CLI clássica, veja [Instalar a CLI clássica do Azure](/cli/azure/install-classic-cli) e para instalar a CLI atual, veja [Instalar a CLI do Azure](/cli/azure/install-azure-cli).
