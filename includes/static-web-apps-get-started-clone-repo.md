## <a name="clone-the-repository"></a>Clonar o repositório

Com o repositório criado na sua conta GitHub, clone o projeto para a sua máquina local usando o seguinte comando.

```bash
git clone https://github.com/<YOUR_GITHUB_ACCOUNT_NAME>/my-first-static-web-app.git
```

Certifique-se de que substitui `<YOUR_GITHUB_ACCOUNT_NAME>` pelo seu nome de utilizador GitHub.
