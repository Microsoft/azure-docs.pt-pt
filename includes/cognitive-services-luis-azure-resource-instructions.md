Se estiver familiarizado com os serviços cognitivos ou familiarizado com o Azure, crie uma [chave temporária](https://azure.microsoft.com/try/cognitive-services/). Se tiver uma conta do Azure, crie um novo recurso no portal. Consultas de ponto final de compreensão de linguagem suportam dois tipos de recursos do Azure: **Compreensão de idiomas** ou o **serviço cognitivo** recursos. É necessário o nome do recurso quando [atribuir](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md) a chave para a aplicação.
