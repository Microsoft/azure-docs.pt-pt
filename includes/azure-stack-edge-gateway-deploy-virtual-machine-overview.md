Pode criar e gerir máquinas virtuais (VMs) num dispositivo Azure Stack Edge utilizando APIs. Estas APIs são APIs standard Azure Resource Manager chamadas usando o ponto final local Azure Stack Edge. As APIs do Gestor de Recursos Azure fornecem uma camada de gestão consistente que, neste caso, permite criar, atualizar e eliminar VMs numa subscrição local que existe no dispositivo. Pode ligar-se ao Gestor de Recursos Azure em execução no Azure Stack Edge através de cmdlets Azure PowerShell.
