[O controlo de acesso baseado em funções Azure (Azure RBAC)](../articles/role-based-access-control/overview.md) é o sistema de autorização que utiliza para gerir o acesso aos recursos Azure. Para determinar a que recursos os utilizadores, grupos, diretores de serviços ou identidades geridas têm acesso, lista as suas atribuições de funções.
