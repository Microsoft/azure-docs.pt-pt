O [controlo de acesso baseado em funções azure (RBAC)](../articles/role-based-access-control/overview.md) é o sistema de autorização que utiliza para gerir o acesso aos recursos do Azure. Para determinar a que recursos os utilizadores, grupos, diretores de serviços ou identidades geridas têm acesso, você lista as suas atribuições de funções.
