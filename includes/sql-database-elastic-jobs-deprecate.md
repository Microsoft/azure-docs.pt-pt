> [!IMPORTANT]
> Este artigo é para a versão hospedado pelo cliente do *tarefas de bases de dados elásticas*. Tarefas elásticas da base de dados estão a ser preterido e substituído pela nova alojado no Azure **tarefas de bases de dados elásticas**. Para novos trabalhos, utilize [a versão mais recente **tarefas de bases de dados elásticas**](../articles/sql-database/elastic-jobs-overview.md). Se utilizar atualmente o as tarefas de hospedado pelo cliente mais antigas, veja [migrar para as novas tarefas de bases de dados elásticas](../articles/sql-database/elastic-jobs-migrate.md) as direções e scripts de migração para atualizar rapidamente para a versão mais recente.
