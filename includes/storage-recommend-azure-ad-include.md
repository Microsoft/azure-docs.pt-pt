> [!TIP]
> O armazenamento do Azure dá suporte a solicitações de autorização para armazenamento de BLOBs e filas usando Azure Active Directory (Azure AD). Autorizar usuários ou aplicativos usando um token OAuth 2,0 retornado pelo AD do Azure fornece segurança superior e facilidade de uso sobre a autorização de chave compartilhada. Com o Azure AD, não é necessário armazenar a chave de acesso da conta com seu código e arriscar as vulnerabilidades de segurança potenciais.
>
> Além disso, o armazenamento do Azure dá suporte à assinatura de acesso compartilhado (SAS) de delegação de usuário para o armazenamento de BLOBs. A SAS de delegação de usuário é assinada com as credenciais do Azure AD. Quando seu design de aplicativo requer assinaturas de acesso compartilhado para acesso ao armazenamento de BLOB, use as credenciais do Azure AD para criar uma SAS de delegação de usuário para segurança superior.
>
> A Microsoft recomenda usar o Azure AD com seus aplicativos de armazenamento do Azure quando possível. Para obter mais informações, consulte autorizar o [acesso a BLOBs e filas do Azure usando o Azure Active Directory](../articles/storage/common/storage-auth-aad.md).
