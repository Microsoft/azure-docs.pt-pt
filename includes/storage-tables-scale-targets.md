|----------|---------------|
| Tamanho máximo de uma tabela | 500 TiB |
| Tamanho máximo de uma entidade de tabela | 1 MiB |
| Número máximo de propriedades numa entidade de tabela | 255 (incluindo 3 propriedades do sistema: PartitionKey, RowKey e Timestamp) |
| Número máximo de políticas de acesso armazenado por tabela | 5 |
| Taxa máxima de pedidos por conta de armazenamento | 20.000 transações por segundo (a assumir um tamanho de entidade de 1 KiB) |
| Taxa de transferência de destino para uma partição de uma única tabela (entidades de 1 KiB) | Até 2000 entidades por segundo |
