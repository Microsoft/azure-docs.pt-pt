> [!IMPORTANT]
> Para evitar encargos quando não houver emprego, **coloque os nós mínimos para 0**. Esta definição permite que o Azure Machine Learning desaloque os nós quando não estão a ser utilizados. Qualquer valor maior do que 0 manterá esse número de nós em funcionamento, mesmo que não estejam a ser utilizados.
