* Consulte a documentação de referência do serviço Blob na Biblioteca do [Cliente de Armazenamento Azure para obter referências .NET](/dotnet/api/overview/azure/storage) para obter detalhes completos sobre as APIs disponíveis.

  * Para trabalhar com as filas de armazenamento Azure, [inicie com o armazenamento da Fila Azure usando .NET](../articles/storage/queues/storage-dotnet-how-to-use-queues.md).
  * Para armazenar dados relacionais, consulte [Connect to SQL Database utilizando .NET (C#)](../articles/azure-sql/database/connect-query-dotnet-visual-studio.md).
