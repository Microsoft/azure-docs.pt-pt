> [!WARNING]
>  **O Serviço de Contentores Azure (ACS) está a ser depreciado. Não estão a ser adicionadas novas funcionalidades ou funcionalidades ao ACS. Todas as APIs, experiência do portal, comandos CLI e documentação são marcados como preced.**
>
> Em 2017, introduzimos o Azure Kubernetes Service (AKS) para simplificar a gestão, implantação e operações da Kubernetes. Se utilizar o orquestrador Kubernetes, por favor, emigre para a AKS até 31 de janeiro de 2020. Para começar, consulte [migrar para o Serviço Azure Kubernetes.](../articles/aks/aks-migration.md)
>
> Para mais informações, consulte o [anúncio de depreciação do Serviço de Contentores Azure no Azure.com](https://azure.microsoft.com/updates/azure-container-service-will-retire-on-january-31-2020/).
