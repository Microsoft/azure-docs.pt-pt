> [!IMPORTANT]
>
> A utilização desta função Azure da PowerShell requer o `AzureRM` módulo instalado. Este é um módulo mais antigo apenas disponível para o Windows PowerShell 5.1 que já não recebe novas funcionalidades.
> Os `Az` `AzureRM` módulos e módulos __não__ são compatíveis quando instalados para as mesmas versões do PowerShell.
> Se precisar de ambas as versões:
>
> 1. [Desinstale o módulo Az](/powershell/azure/uninstall-az-ps) a partir de uma sessão PowerShell 5.1.
> 2. [Instale o módulo AzureRM](/powershell/azure/azurerm/install-azurerm-ps) a partir de uma sessão PowerShell 5.1.
> 3. [Faça o download e instale o PowerShell Core 6.x ou mais tarde](/powershell/scripting/install/installing-powershell-core-on-windows).
> 4. [Instale o módulo Az](/powershell/azure/install-az-ps) numa sessão PowerShell Core.
