Quando utilizar um Gestor de tráfego do Microsoft Azure para o tráfego de balanceamento de carga para seu site do Azure, nesse Web site pode ser acedido através da  **\*. trafficmanager.net** nome de domínio atribuído pelo Azure. Também pode associar um nome de domínio personalizado, como www\.contoso.com, com o seu Web site a fim de fornecer um nome de domínio mais reconhecível para os seus utilizadores.
