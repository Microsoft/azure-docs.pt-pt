## <a name="version-the-device-template"></a>Versão o modelo do dispositivo

A análise de vídeo - aplicação de deteção de objetos e movimentos inclui um modelo de dispositivo **LVA Edge Gateway.** Para utilizar o ficheiro manifesto de implementação atualizado com este modelo de dispositivo, tem de criar e publicar uma nova versão do modelo.

Para criar uma nova versão do modelo:

1. Na sua aplicação IoT Central, navegue para **os modelos do dispositivo** e selecione o modelo do dispositivo **LVA Edge Gateway.**

1. Selecione **versão** para criar um novo modelo chamado **LVA Edge Gateway v2** e, em seguida, selecione **Create**.

O novo modelo de dispositivo **LVA Edge Gateway v2** está agora na sua lista de modelos de dispositivos.
