## <a name="use-the-azure-portal-to-move-a-vm-to-a-different-subscription"></a>Use o portal Azure para mover um VM para uma subscrição diferente
Pode mover um VM e os seus recursos associados para uma subscrição diferente utilizando o portal Azure.

1. Vá ao [portal Azure](https://portal.azure.com) para gerir o grupo de recursos que contém o VM para se mover. Procure e selecione **grupos de recursos.**
2. Escolha o grupo de recursos que contém o VM que gostaria de mover.
3. No topo da página para o grupo de recursos, selecione **Move** e, em seguida, selecione **Mover para outra subscrição**. A página **de recursos move** abre.
4. Selecione cada um dos recursos para se mover. Na maioria dos casos, deve mover todos os recursos relacionados que estão listados.
5. Selecione a **Subscrição** para onde deseja que o VM seja movido.
6. Selecione um grupo de **Recursos** existente, ou insira um nome para ter um novo grupo de recursos criado.
7. Quando terminar, selecione que compreende que serão criados novos IDs de recursos e que os novos IDs terão de ser utilizados com o VM depois de ser movido e, em seguida, selecione **OK**.
