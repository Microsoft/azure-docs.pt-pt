## <a name="use-the-azure-portal-to-move-a-vm-to-a-different-subscription"></a>Use o portal Azure para mover um VM para uma subscrição diferente
Pode mover um VM e os seus recursos associados para uma subscrição diferente utilizando o portal Azure.

1. Dirija-se ao [portal Azure](https://portal.azure.com) para gerir o grupo de recursos que contém o VM para se mover. Procure e selecione **grupos de Recursos**.
2. Escolha o grupo de recursos que contém o VM que gostaria de mover.
3. Na parte superior da página para o grupo de recursos, selecione **Move** e, em seguida, selecione **Mover para outra subscrição**. A página de **recursos move** abre.
4. Selecione cada um dos recursos para se mover. Na maioria dos casos, deve mover todos os recursos relacionados que estão listados.
5. Selecione a **Subscrição** onde pretende que o VM seja movido.
6. Selecione um **grupo de Recursos**existentes, ou insira um nome para ter um novo grupo de recursos criado.
7. Quando terminar, selecione que entenda que serão criados novos IDs de recurso e que os novos IDs terão de ser usados com o VM depois de movidos, e depois selecione **OK**.
